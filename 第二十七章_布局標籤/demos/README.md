# 第二十七章｜布局標籤示例

這個資料夾收錄第二十七章筆記對應的可執行示例頁。  
這章的重點是理解 HTML5 布局標籤如何描述頁面區塊角色，而不是取代 CSS 版面配置。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第二十七章 Markdown 筆記。
- 想直接對照完整頁面骨架與標籤角色時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [layout-tags-demo.html](./layout-tags-demo.html)：整合展示 `header`、`nav`、`main`、`article`、`section`、`aside`、`footer`、`hgroup`、`div` 的角色，並對照 `article` / `section`、語意標籤 / CSS 排版、常見錯誤與判斷流程。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
