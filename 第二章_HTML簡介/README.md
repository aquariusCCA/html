# 第二章｜HTML 簡介

> 章節定位：從網頁、網站與 HTML 這些入門名詞開始，建立後續學習 HTML 的共同語言  
> 建議閱讀方式：先把本章的基本名詞釐清，再往後看標籤、結構與實作內容

## 本章在學什麼

這一章先處理學 `HTML` 前最容易混淆的基礎名詞。  
如果沒有先分清網頁、網站、瀏覽器與 `HTML` 的角色，後面學標籤與頁面結構時很容易失去主線。

- 父元素、子元素、祖先元素、後代元素、兄弟元素的差別
- `.html`、`.htm` 與瀏覽器之間的關係

- 不同版本的 `DOCTYPE` 大致長什麼樣
