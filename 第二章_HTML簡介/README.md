# 第二章｜HTML 簡介

> 章節定位：從網頁、網站與 HTML 這些入門名詞開始，建立後續學習 HTML 的共同語言  
> 建議閱讀方式：先把本章的基本名詞釐清，再往後看標籤、結構與實作內容

## 本章在學什麼

這一章先處理學 `HTML` 前最容易混淆的基礎名詞。  
如果沒有先分清網頁、網站、瀏覽器與 `HTML` 的角色，後面學標籤與頁面結構時很容易失去主線。

## 本章與整體地圖的關係

- 前置知識：第一章已先建立 `Web`、瀏覽器與前後端的基本概念。
- 這章的核心任務：把 `HTML` 的基本定位、頁面骨架與標籤關係先建立起來。
- 讀完後通常接：更細的標籤、屬性、文件結構與實作內容。

## 易混淆主題

- 網頁 vs 網站：單一頁面和整體網站是不同層次的概念。
- `HTML` 骨架 vs 頁面內容標籤：前者是頁面的基本外框，後者是在外框中描述具體內容。
- 父子元素 vs 祖先後代元素：前者是直接一層，後者可以跨多層。
- `HTML5` vs 舊版 `HTML`：現在學習主線通常以 `HTML5` 為主，但舊版本仍需要能辨認。
