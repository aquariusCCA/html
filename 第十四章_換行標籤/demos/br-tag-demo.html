`<br>` 的工作很單純：在同一段內容裡強制換到下一行。它適合地址、詩句、歌詞、署名這類本來就需要逐行呈現的內容；如果你想表達的是新段落或版面間距，就不該優先用它。

`<br>` 是換行，不是分段 它只讓內容往下一行，不會產生新的段落語意。
適合逐行內容 地址、詩句、歌詞、聯絡資訊這種一行一行有意義的內容很適合。
`<p>` 負責段落 當內容已經是兩個完整意思單位時，應優先拆成兩個段落。
不要拿來撐間距 視覺距離應交給 CSS，例如 `margin`、`padding`、`line-height`。

2. `<br>` 和 `<p>` 的差異

        

          

            段落分開
            
兩個完整意思單位，應用 `<p>`

            
<p>第一段介紹課程目標與學習方式。</p>
<p>第二段補充修課前的建議與準備事項。</p>

            

              
第一段介紹課程目標與學習方式。

              
第二段補充修課前的建議與準備事項。

            

            

              這裡是兩個獨立段落，所以應該讓 HTML 直接表達出「分段」。
            

          


          

            同段換行
            
同一段資訊內的分行，適合 `<br>`

            
<p>
  客服專線：02-1234-5678<br>
  服務時間：週一到週五 09:00 - 18:00<br>
  電子郵件：service@example.com
</p>

            

              客服專線：02-1234-5678

              服務時間：週一到週五 09:00 - 18:00

              電子郵件：service@example.com
            

            

              這還是一組聯絡資訊，只是每一行都需要清楚排列。
            

          

        

      


      

        
3. 為什麼按 Enter 不等於換行

        

          

            

              
              
              
            

          


          

            
空白折疊

            
原始碼換行不會自動變成畫面換行

            

              HTML 會把多餘空白折疊，所以你在程式裡按 Enter，通常不會直接得到多行顯示效果。
            

            

            

              

            

          

        

      


      

        
4. 不要拿 `<br>` 來硬做間距

        

          

            

              
              
            

          


          

            
不建議

            
間距是樣式問題，不是內容換行問題

            

              如果只是想讓兩塊區域離遠一點，應該交給 CSS，而不是插入一串沒有語意的 `<br>`。
            

            

            

            

              同樣的畫面距離，交給 CSS 控制會更穩定，也比較不會讓 HTML 難讀。
            

          

        

      


      

        
5. 實務提醒

        

          

            優先判斷是不是新段落
            若上下兩段已經是不同意思，先考慮拆成兩個 `<p>`。
          

          

            逐行資訊才用 `<br>`
            一行一行本身有意義的內容，才是 `<br>` 最自然的落點。
          

          

            版面 spacing 用 CSS
            `margin`、`padding`、`line-height` 才是控制視覺距離的正規工具。
