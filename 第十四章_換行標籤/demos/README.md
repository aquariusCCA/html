# 第十四章｜換行標籤示例

這個資料夾收錄第十四章筆記對應的可執行示例頁。  
這章的重點是理解 `<br>` 用來做單純換行，不是拿來切段落，也不適合拿來硬撐版面間距。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十四章 Markdown 筆記。
- 想直接看 `<br>` 的使用情境時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [br-tag-demo.html](./br-tag-demo.html)：整合展示 `<br>` 的基本用途、和 `<p>` 的差異、空白折疊行為，以及不建議的濫用方式。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
