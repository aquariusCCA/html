# 第十七章｜文本格式化標籤示例

這個資料夾收錄第十七章筆記對應的可執行示例頁。  
這章的重點是理解文字看起來相似，不代表背後語意相同；有些標籤偏外觀，有些標籤會同時表達內容意義。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十七章 Markdown 筆記。
- 想直接看成對標籤差異時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [text-formatting-tags-demo.html](./text-formatting-tags-demo.html)：整合展示 `b/strong`、`i/em`、`u/ins`、`s/del` 的差異，以及何時該交給 CSS 處理純外觀需求。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
