# 第十七章｜文本格式化標籤

> 章節定位：建立 HTML 文字外觀效果與語意強調差異的基本判讀框架  
> 建議閱讀方式：先分清哪些標籤偏外觀、哪些標籤偏語意，再回頭看成對標籤的對照

## 本章在學什麼

這一章整理 HTML 常見的文本格式化標籤。  
重點不是只記住文字會變粗、變斜、加底線或刪除線，而是理解這些效果背後是否同時帶有語意。

## 本章與整體地圖的關係

- 前置知識：前面已整理段落、換行、水平線與預格式化文本等基礎文字顯示方式。
- 這章的核心任務：分清純視覺格式化和語意強調之間的差別。
- 讀完後通常接：之後在處理重點提示、內容修改痕跡或語氣強調時，可以回來快速查這一章。

## 易混淆主題

- 外觀效果 vs 語意強調：兩者常長得像，但用途不同。
- HTML 標籤 vs CSS 樣式：前者可帶語意，後者更偏視覺控制。
