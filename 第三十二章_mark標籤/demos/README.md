# 第三十二章｜`mark` 標籤示例

這個資料夾收錄第三十二章筆記對應的可執行示例頁。  
這章的重點是理解 `<mark>` 如何標示目前上下文中特別相關的文字，而不是只把它當成黃色背景效果。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第三十二章 Markdown 筆記。
- 想直接操作搜尋命中與語意對照時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [mark-tags-demo.html](./mark-tags-demo.html)：整合展示搜尋命中高亮、引用片段、目前討論詞、CSS 客製化，以及 `<mark>` / `<strong>` / `<em>` / `<span>` 的語意差異與常見誤用。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
