# 第三十二章｜`mark` 標籤

> 章節定位：理解 `<mark>` 如何標示在當前上下文中特別相關的文字  
> 建議閱讀方式：先掌握語意，再回看它和 `<strong>`、純 CSS 高亮的差別

## 本章在學什麼

這一章整理 HTML5 的 `<mark>` 標籤。  
重點是建立一個穩定觀念：它不是單純的黃底效果，而是標示某段文字在目前脈絡下特別值得被注意。

## 本章與整體地圖的關係

- 前置知識：前面已學過多種文字與語意標籤，這一章補的是「高亮標示但不等於重要性」這個語意。
- 這章的核心任務：理解 `<mark>` 的語意，並和 `<strong>`、`<span>` 做出區分。
- 讀完後通常接：之後在做搜尋命中標示、文件高亮或語意化標記時，都可以回查這一章。
