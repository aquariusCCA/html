`spellcheck` Demo

這個示例把筆記中的三種常見情境排在同一頁：文字型 input 啟用拼寫檢查、 textarea 關閉拼寫檢查、以及 contenteditable 區塊啟用拼寫檢查。

文字輸入框

這個欄位設定 spellcheck="true"，允許瀏覽器檢查輸入文字。

spellcheck="true"

留言區

這個 textarea 設定 spellcheck="false"，不要做拼寫檢查。

spellcheck="false"

可編輯內容

這個區塊可以直接編輯，並且設定 spellcheck="true"。

You can edit this txt and let the browser try spell checking.
contenteditable + spellcheck="true"
注意：是否顯示紅底線、如何給出建議、檢查語言支援到什麼程度，都由瀏覽器決定。 這個示例主要用來對照 true / false 與不同元素類型的差別。
