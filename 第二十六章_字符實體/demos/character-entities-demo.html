第二十六章：字符實體 Demo

這個示例把筆記中的 HTML character references 做成可對照畫面：原始碼怎麼寫、瀏覽器怎麼顯示、什麼時候該轉義，什麼時候該交給 CSS。

一句話核心

字符實體是把容易和 HTML 語法衝突、難直接輸入，或具有特殊顯示行為的字符，用安全明確的方式顯示在頁面上。

名稱寫法

&copy; 顯示成 ©，可讀性最好，初學先記這種。

十進位寫法

&#169; 也顯示成 ©，使用 Unicode 十進位編號。

十六進位寫法

&#xA9; 同樣顯示成 ©，使用 Unicode 十六進位編號。

1. 高頻字符實體速查

想顯示 HTML 原始碼 用途
< &lt; 顯示小於符號，常用於展示 HTML 標籤。
> &gt; 顯示大於符號，常和 &lt; 成對使用。
& &amp; 顯示 & 本身，避免被當成字符實體開頭。
" &quot; 顯示雙引號，常用於屬性值內容。
' &#39; 顯示單引號，常用於屬性值內容。
不換行空格 &nbsp; 產生 non-breaking space，不只是普通空格。
© &copy; 顯示版權符號。

2. 顯示 HTML 標籤本身

直接寫標籤會被解析

<p><button>送出</button></p>

轉義後才會顯示原始碼

<p>&lt;button&gt;送出&lt;/button&gt;</p>

<button>送出</button>

3. 普通空格會被合併

HTML 一般顯示規則下，連續普通空格、換行、Tab 通常會合併成一個空格。

<p>hello     world</p>

hello world

4. &nbsp; 是不換行空格

&nbsp; 可以保留明顯空白，也會避免瀏覽器在這個空格位置自動換行。

<p>hello&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;world</p>

hello     world

5. 保留空白格式：pre 與 CSS white-space

pre 保留原始格式

<pre>
hello     world
line 2
    line 3
</pre>
hello     world
line 2
    line 3

white-space: pre-wrap

.text {
  white-space: pre-wrap;
}

hello world line 2 line 3

6. &nbsp; 不該拿來大量排版

不建議

姓名：&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;小明
姓名：     小明

建議交給 CSS

.user-name {
  display: flex;
  gap: 16px;
}
姓名： 小明

7. URL 參數中的 &amp;

在 HTML 屬性中，URL 查詢參數的 & 建議寫成 &amp;。瀏覽器解析後，連結意義仍然是 &。

<a href="/search?keyword=html&amp;page=1">搜尋</a>
搜尋 keyword=html&page=1

8. HTML 屬性中的引號

雙引號內容用 &quot;

<input value="他說：&quot;Hello&quot;">

單引號內容用 &#39;

<input value='It&#39;s OK'>

9. textContent 與 innerHTML 對照

輸入一段文字後分別套用到下方兩個區塊。textContent 會當文字顯示；innerHTML 會把字串當 HTML 解析。

textTarget.textContent = userInput;
htmlTarget.innerHTML = userInput;

textContent

innerHTML

10. 什麼時候該用字符實體

  • 想顯示 HTML 標籤本身，例如 <div>。
  • 文字中有 < 或 &，需要避免語法混淆。
  • 屬性值內含引號或特殊字符。
  • 需要顯示 &copy;、&reg;、&trade; 等符號。

11. 什麼時候改用 CSS

  • 只是要控制元素間距。
  • 只是要讓文字縮排。
  • 只是要保留多行文字格式。
  • 只是要避免整段文字換行，可以考慮 white-space。
