# 第二十六章｜字符實體示例

這個資料夾收錄第二十六章筆記對應的可執行示例頁。  
這章的重點是理解字符實體如何安全顯示特殊字符、HTML 原始碼、空白與常見符號。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第二十六章 Markdown 筆記。
- 想直接對照原始碼與顯示結果時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [character-entities-demo.html](./character-entities-demo.html)：整合展示 `&lt;`、`&gt;`、`&amp;`、`&quot;`、`&#39;`、`&nbsp;`、`&copy;`，並對照普通空格合併、`pre`、CSS `white-space`、URL 中的 `&amp;`、屬性引號，以及 `textContent` / `innerHTML` 的差異。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
