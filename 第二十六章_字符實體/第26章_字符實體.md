# 字符實體（HTML 實體）

> 所屬章節：第二十六章｜字符實體  
> 關鍵字：字符實體、HTML 實體、entity、`&nbsp;`、`&lt;`、`&gt;`、`&amp;`、`&copy;`、特殊字符、空格  
> 建議回查情境：想知道為什麼有些符號不能直接寫進 HTML、想顯示多個空格、想輸出 `<` 或 `>` 本身、想快速查常見實體符號怎麼寫

## 本節導讀

這篇整理 HTML 中的字符實體，也常叫做 HTML 實體。  
它的核心用途，是讓原本會被瀏覽器當成語法的一些符號，能以「普通文字」的方式正確顯示出來。

原始內容方向正確，但像課堂速記，讀者容易只記住幾個例子，卻沒建立穩定判斷：到底什麼時候需要用字符實體、哪些情況最常碰到、以及為什麼多個空格不會照原樣顯示。  
這裡改成先講問題，再講常見實體與例子，讓這篇更適合回查。

## 你會在這篇學到什麼

- 什麼是字符實體
- 為什麼 `<`、`>`、`&` 這類符號常需要特殊寫法
- 為什麼連續空格不會照原樣顯示
- `&nbsp;`、`&lt;`、`&gt;`、`&amp;`、`&copy;` 分別在做什麼
- 什麼時候該查表，而不是硬背一長串實體名稱

## 30 秒複習入口

- 字符實體是讓特殊符號能在 HTML 中被正確顯示的寫法。
- 常見形式是 `&名稱;`，例如 `&lt;`、`&gt;`、`&nbsp;`、`&copy;`。
- `<` 和 `>` 若直接寫進 HTML，可能會被當成標籤語法的一部分。
- 連續多個普通空格通常會被瀏覽器合併顯示。
- 想保留多個空格時，最常用的是 `&nbsp;`。

## 速查區

### 核心概念

- 字符實體是 HTML 用來表示特殊字符的一種寫法。
- 它常出現在「符號本身容易和 HTML 語法衝突」或「直接輸入後顯示效果不符合預期」的情況。
- 重點不是背所有實體，而是知道常見問題該怎麼處理。

### 關鍵規則 / 判準

- 若想顯示 `<`、`>`、`&` 這些容易和 HTML 語法混在一起的符號，常會使用實體。
- 若想保留多個空格，常會用 `&nbsp;`。
- 常見寫法可以先記成：`&名稱;`。
- 真的遇到少見符號時，查表通常比硬背更有效率。

### 常見使用場景

- 顯示 HTML 標籤本身，例如 `<div>`
- 顯示多個空格
- 顯示版權符號
- 顯示文字中的特殊符號，而不想被瀏覽器當成語法

### 常見混淆點

- 字符實體不是只有拿來顯示冷門符號，最常見的其實是處理 `<`、`>`、`&` 與空格。
- 多個空格在原始碼裡看得到，不代表頁面上就會照樣顯示。
- `&nbsp;` 是不換行空格，重點是它能避免被當成一般空白直接合併掉。

### 一句話抓核心

- 字符實體是把「不方便直接寫」或「容易被誤解成語法」的字符，安全地顯示在 HTML 頁面上的方法。

## 正文筆記

### 1. 什麼是字符實體？

- 在 HTML 中，有些符號直接寫進頁面原始碼時，瀏覽器可能不會把它當成普通文字。
- 這時就可以用字符實體，讓這些符號被正確顯示。
- 例如 `<` 常被拿來表示標籤開始，如果你只是想把 `<` 這個符號印出來，就不能直接把它當一般文字理解。

### 2. 常見形式怎麼看？

- 常見的字符實體可以先記成：`&名稱;`
- 例如：
  - `&lt;` 代表 `<`
  - `&gt;` 代表 `>`
  - `&amp;` 代表 `&`
  - `&nbsp;` 代表空格
  - `&copy;` 代表版權符號

### 3. 為什麼空格也會用到字符實體？

- 在 HTML 中，連續多個普通空格通常不會照原樣顯示。
- 瀏覽器常會把它們合併處理，所以原始碼裡就算打了很多空格，畫面上可能還是只看到一個。
- 如果想明確保留多個空格，最常見的做法就是使用 `&nbsp;`。

### 4. 範例怎麼理解？

```html
<!-- 單詞之間有 5 個空格，頁面上通常只會看到 1 個空格 -->
hello     world <br />

<!-- 用字符實體保留多個空格 -->
hello&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;world <br />

<!-- 顯示版權符號 -->
&copy;
```

這段例子主要在說兩件事：

- 第一，原始碼中的多個普通空格，瀏覽器通常不會原封不動顯示。
- 第二，若想保留空白效果或輸出特殊符號，就可以改用字符實體。

### 5. 哪些實體最值得先記？

原始課程的取向是先記常用的，這個方向是對的。  
真正高頻回查的，通常是這幾個：

- `&nbsp;`：保留空格
- `&lt;`：顯示 `<`
- `&gt;`：顯示 `>`
- `&amp;`：顯示 `&`
- `&copy;`：顯示版權符號

其他比較少見的符號，不必一開始就全部硬背；需要時查表即可。

## 常見問法

### 為什麼 `<` 不能直接當普通文字來理解？

- 因為在 HTML 中，`<` 常和標籤語法有關。
- 若直接寫進原始碼，瀏覽器可能不會把它單純當成一個普通字符。

### 為什麼我明明打了很多空格，頁面上卻只看到一個？

- 因為連續多個普通空格通常會被瀏覽器合併顯示。
- 若要保留多個空格，常會用 `&nbsp;`。

### 字符實體是不是都要硬背？

- 不需要。
- 先記高頻幾個就夠了，其餘需要時查表會更實際。

### `&copy;` 是做什麼的？

- 用來顯示版權符號。
- 它是常見的特殊字符實體之一。
