# 第二十六章｜字符實體

> 章節定位：理解如何在 HTML 中安全地顯示特殊字符  
> 建議閱讀方式：先理解字符實體要解決的問題，再回看高頻實體與空格顯示規則

## 本章在學什麼

這一章整理 HTML 的字符實體。  
重點不是把一大串實體名稱全部背起來，而是先理解：什麼時候字符不能直接寫、為什麼多個空格不會照原樣顯示，以及哪些實體最常用。

## 本章與整體地圖的關係

- 前置知識：前面已學過各種標籤，這一章補的是「當你想把某些符號當文字顯示時，該怎麼寫」。
- 這章的核心任務：建立字符實體的用途觀念，並掌握高頻實體與空格處理。
- 讀完後通常接：之後在寫教學頁、展示 HTML 原始碼或處理特殊符號時，都可以回來查這一章。

## 適合快速回查的主題

- 什麼是字符實體
- 為什麼 `<`、`>`、`&` 不能總是直接寫
- 為什麼多個空格不會照原樣顯示
- `&nbsp;`、`&lt;`、`&gt;`、`&amp;`、`&copy;` 怎麼分

## 易混淆主題

- 普通空格 vs `&nbsp;`：前者可能被合併顯示，後者常用來保留空白效果。
- 符號本身 vs HTML 語法：像 `<`、`>`、`&` 這類字符很容易和語法混在一起。
- 高頻實體 vs 全部硬背：前者值得先掌握，後者通常不必要。
