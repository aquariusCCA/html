## 正文筆記

- 我們目前自己寫的網站，通常都放在自己的電腦上，因此只有自己能訪問。
- 如果希望其他人也能訪問這個網站，就需要把專案放到服務器上，這樣更多使用者才能透過網路存取網站內容。

![服務器示意圖](./images/1775985638808_YKITp9xzK6.png)

- 服務器是提供計算服務的設備，本質上也是一台電腦。
- 在網路環境下，依照提供的服務類型不同，服務器可以分為文件服務器、資料庫服務器、應用程式服務器、Web 服務器等。
- 服務器也常被稱為主機。

### 甚麼是 Web 服務器？

- Web 服務器通常指網站服務器，也就是駐留在網際網路上的一類程式或系統。
- 它可以向瀏覽器等 Web 客戶端提供文件與資源，讓使用者能夠瀏覽網站內容，或下載網站上提供的資料。

### Web 服務器的兩種常見形式

以下提到的服務器，主要都是指 Web 服務器。依照它在網路中的位置與使用方式，常可先分成兩類：

#### 本地服務器

- 可以把自己的電腦設定成一台本地服務器。
- 這樣同一個局域網中的其他人，就有機會訪問你放在這台電腦上的專案。

#### 遠程服務器

- 遠程服務器通常是由其他公司提供的一台主機。
- 只要把網站專案上傳到這台主機，其他人就能透過網路和域名來訪問你的網站。
- 例如像 `www.baidu.com` 這樣的網址，背後就對應到一台對外提供服務的遠程主機。
