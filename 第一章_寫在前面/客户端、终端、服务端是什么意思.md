# 客户端、终端、服务端是什么意思

> 所屬章節：第一章｜寫在前面  
> 本文定位：補充 [甚麼是 Web 的前端和後端](./甚麼是%20Web%20的前端和後端.md) 中「客戶端 / 服務端」相關名詞  
> 關鍵字：客戶端、用戶端、服務端、伺服器、終端、命令列、CMD、Shell  
> 建議回查情境：看到客戶端、終端、服務端三個詞混在一起時、想補充理解「服務端 / 客戶端」與實際設備的對應、想知道終端在日常開發裡通常指什麼

## 本節導讀

這一節是 [甚麼是 Web 的前端和後端](./甚麼是%20Web%20的前端和後端.md) 的補充閱讀，主要把 `客戶端`、`終端`、`服務端` 這三個容易混用的詞拆開來看。  
第一次閱讀時，可以先看「服務端和客戶端」的部分，再看「終端」的補充說明。

## 你會在這篇學到什麼

- 服務端與客戶端最直觀的意思

## 正文筆記

### 先說明這篇在補什麼

- 同一個概念在不同語境中，意思可能不完全一樣。
- `終端`、`客戶端`、`服務端` 這三個詞，常常會在不同場合被混用，所以初學時特別容易搞混。
- 這篇主要是用比較直觀的方式，補充理解它們在日常語境中的常見意思。

### 1. 什么是服务端和客户端

- `服務端` 主要是從功能角度來命名的，重點在於它是否提供服務。
- 可以先把它想成餐廳裡的服務員：他和你一樣都是人，但因為他在提供服務，所以會被叫做服務員。
- 在電腦世界裡，服務端通常也是一台機器，只是這台機器被拿來承擔某種對外提供功能的角色，因此常被稱為 `伺服器`。

- 以網站為例：
- 網站由許多網頁檔案組成。
- 這些檔案放在一台對外提供網站內容的電腦上。
- 這台電腦可以被叫做網站伺服器，也就是服務端。
- 而使用者自己的電腦如果拿來開啟這個網站，這台電腦此時就可以叫做客戶端，也可以叫用戶端。

- 以手機 App 為例：
- App 背後通常也會有一套放在伺服器上的檔案和服務。
- 提供這些內容與功能的那一端，可以叫做服務端。
- 使用者手機上安裝並使用這個 App，手機此時就可以叫做客戶端。

### 2. 什么是终端

- `終端` 這個詞比 `客戶端`、`服務端` 更容易因情境不同而改變意思。
- 這份原始筆記裡，終端主要是用「操作介面」的角度來理解。
- 可以先把它想成一個讓人和系統互動、輸入指令、控制操作的界面。

- 以日常開發環境來說：
- 在 `Windows` 上，像 `cmd` 這類命令列視窗，常會被叫做終端。
- 在 `macOS` 或 `Linux` 上，命令列介面也常會被叫做終端。

- 簡單理解時，可以先把終端看成「讓你向系統下指令的入口」。
