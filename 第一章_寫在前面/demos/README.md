# 第一章｜寫在前面示例

這個資料夾收錄第一章筆記對應的互動示例頁。  
第一章的內容偏概念建立，所以示例不會只展示單一標籤，而是把瀏覽器、Web 請求、前後端分工與 Web 標準放進同一個可操作情境裡。

## 這個資料夾怎麼用

- 想先理解名詞時：先讀第一章筆記。
- 想把零散概念串成一個流程時：再打開這裡的示例頁。
- 想拿來教學或自我複習時：可以一邊切換互動區塊，一邊對照章節內容。

## 目前示例

- [web-basics-demo.html](./web-basics-demo.html)：用單頁互動方式串起瀏覽器、伺服器、HTTP 回應、前端 / 後端分工，以及 HTML / CSS / JavaScript 三層分工。
