## 正文筆記

- Web 是 `World Wide Web`（全球資訊網）的簡稱。它建立在網際網路之上，透過超文本連結把世界各地的資訊和資源串接起來，並使用標準化的網路協議與格式，例如 `HTTP` 和 `HTML`，讓這些內容可以被存取與呈現。
- Web 可以看作一個巨大的資訊系統，其中包含許多不同類型的資源，例如網頁、圖片、音訊、影片等。使用者通常透過網際網路與瀏覽器來訪問、瀏覽和使用這些資源。

### 甚麼是 JavaWeb？

- 所有使用 Java 語言編寫，並且可以透過瀏覽器訪問的程式，通常都可以歸類為 JavaWeb。
- Web 是一個可以存取與共享資源的平台，因此也能承載 Java 程式。這些程式可以執行在 Web 應用伺服器上，再由瀏覽器提供給使用者操作與查看。
