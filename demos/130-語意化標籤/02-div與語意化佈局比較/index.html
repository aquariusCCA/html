比較設定

同畫面，不同結構

畫面相同，不代表 HTML 語意相同

觀察兩邊的標籤名稱；語意化版本不需要靠 class 才能先讀出內容角色。

全用 div

頭部標籤
導航欄標籤

最新消息

這是一段與最新消息相關的內容。

<div class="header">頭部標籤</div>
<div class="nav">導航欄標籤</div>
<div class="section">
  <h2>最新消息</h2>
  <p>這是一段與最新消息相關的內容。</p>
</div>

結構摘要：標籤本身都是 <div>，角色主要藏在 class 名稱裡。

語意化標籤

頭部標籤
導航欄標籤

最新消息

這是一段與最新消息相關的內容。

<header>頭部標籤</header>
<nav>導航欄標籤</nav>
<section>
  <h2>最新消息</h2>
  <p>這是一段與最新消息相關的內容。</p>
</section>

結構摘要：標籤直接表達頁首、導覽與主題區塊。

錯誤修正

不是所有容器都要換成 section

誤用

<section class="button-group">
  <button>儲存</button>
  <button>取消</button>
</section>

這只是按鈕群組，沒有形成一個有標題的主題區段。

修正

<div class="button-group">
  <button>儲存</button>
  <button>取消</button>
</div>

當目的只是分組與排版，<div> 是合理的通用容器。
