Demo 05

列表小型整合：課程學習頁

這個 demo 把無序列表、有序列表、描述列表與巢狀列表放在同一個頁面中， 觀察不同內容如何選擇不同列表語意。

章節目錄

  1. HTML 基礎
    1. 文件結構
    2. 常用全局屬性
  2. 內容標籤
    1. 標題與段落
    2. 圖片與連結
    3. 列表標籤

本節學習目標

  • 分辨 <ul>、<ol>、<dl> 的語意。
  • 寫出合法的 <li> 列表項目。
  • 將子列表放在正確的父層 <li> 裡。

建議練習步驟

  1. 先閱讀列表標籤筆記。
  2. 開啟本章 demos，觀察不同列表的用途。
  3. 修改列表內容，確認畫面與語意是否仍合理。
  4. 自己建立一個導覽列或 FAQ 區塊。

本節術語

無序列表
用於沒有先後順序的一組項目。
有序列表
用於順序、步驟或排名有意義的一組項目。
描述列表
用於名詞與說明、欄位與內容的配對。

實務注意事項

  • 列表的外觀交給 CSS，不要為了樣式破壞 HTML 結構。
  • 如果內容是步驟，優先使用 <ol>。
  • 如果內容是導覽連結群組，可以用 <nav> 搭配 <ul>。
