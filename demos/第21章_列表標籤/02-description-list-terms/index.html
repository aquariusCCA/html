描述列表：dt 與 dd

這個 demo 專門觀察 <dl>、<dt> 與 <dd> 如何表達「項目名稱」與「項目說明」。

HTML 常見名詞

HTML
用來描述網頁內容與結構的標記語言。
它負責說明頁面有哪些內容，以及這些內容具有什麼語意。
CSS
用來控制網頁外觀、排版、顏色與間距的樣式語言。
DOM
瀏覽器根據 HTML 建立出的文件物件模型。
JavaScript 通常透過 DOM API 操作頁面節點。

什麼時候不適合用 dl？

如果內容只是「一組並列項目」，例如課程公告、商品特色、選單連結， 通常使用 <ul> 比較清楚。
