Demo 03

常見錯誤：列表巢狀位置

這個 demo 比較錯誤與正確的巢狀列表。重點是觀察： 子列表應該放在對應的父層 <li> 裡。

錯誤與正確寫法比較

錯誤寫法

子列表被放成父層 <ul> 的直接子元素。

  • 上海
    • 外灘
    • 迪士尼樂園
  • 西安
<ul>
  <li>上海</li>
  <ul>
    <li>外灘</li>
    <li>迪士尼樂園</li>
  </ul>
  <li>西安</li>
</ul>

正確寫法

子列表放在「上海」這個 <li> 裡。

  • 上海
    • 外灘
    • 迪士尼樂園
  • 西安
<ul>
  <li>
    上海
    <ul>
      <li>外灘</li>
      <li>迪士尼樂園</li>
    </ul>
  </li>
  <li>西安</li>
</ul>

判斷規則

  1. 先找出子列表屬於哪一個父項目。
  2. 把子列表放進該父項目的 <li> 裡。
  3. 確認父層 <ul> 或 <ol> 的直接子元素都是 <li>。
