# 基本超鏈接：href、target、rel

## 1. Demo 目標

這個 demo 用來示範：

- `<a>` 如何透過 `href` 指定連結目標。
- 外部連結、內部頁面連結與頁內區塊連結的基本差異。
- `target="_self"` 與 `target="_blank"` 的差異。
- 外部新分頁連結為什麼建議搭配 `rel="noopener noreferrer"`。

## 4. 觀察重點

1. 外部連結會前往 `https://example.com`，並嘗試開啟新分頁。
2. 內部連結會開啟同資料夾中的 `target-page.html`。
3. 頁內區塊連結會跳到目前頁面中的指定 `id`。
4. 鍵盤按 Tab 時，可以看到連結的 focus 樣式。

## 5. 程式碼重點

### HTML

- 使用 `<a href="...">連結文字</a>` 建立連結。
- 使用 `target="_blank"` 示範新分頁。
- 使用 `rel="noopener noreferrer"` 補足外部新分頁連結的實務安全寫法。

### CSS

- CSS 只用於排版、卡片樣式與 focus 狀態。

### JavaScript

- 此 demo 不需要 JavaScript。

## 6. 修改練習

請嘗試修改以下內容，觀察結果差異：

1. 移除外部連結的 `target="_blank"`，觀察開啟方式。
2. 將內部連結改成錯誤檔名，觀察瀏覽器反應。
3. 修改連結文字，讓它更清楚描述目的。
4. 新增一個指向 `#target-title` 的頁內連結。

## 7. 常見錯誤

| 錯誤 | 可能造成的問題 | 修正方式 |
|---|---|---|
| 連結文字只寫「點我」 | 使用者不知道連結目的 | 改成具體文字，例如「查看本地目標頁面」 |
| `href` 路徑寫錯 | 目標頁面無法開啟 | 確認檔案位置與相對路徑 |
| 外部新分頁連結缺少 `rel` | 可能有安全與隱私風險 | 加上 `rel="noopener noreferrer"` |
