Demo 02

錨點連結：href="#id"

這個 demo 示範如何使用 href="#區塊id" 跳到目前頁面中的指定位置。

本頁目錄

  1. 超鏈接用途
  2. target 開啟方式
  3. 檔案連結
  4. 應用程式連結

超鏈接用途

<a> 可以讓使用者從目前頁面前往其他頁面、其他網站、檔案， 或頁面中的特定位置。頁面內部跳轉就是錨點連結的常見用途。

回到頁首

target 開啟方式

target="_self" 代表在目前頁面開啟，target="_blank" 代表嘗試在新分頁或新視窗開啟。

回到頁首

檔案連結

href 可以指向圖片、PDF、文字檔、壓縮檔等資源。瀏覽器會依檔案類型與伺服器設定決定開啟或下載。

回到頁首

應用程式連結

tel: 與 mailto: 這類 URL scheme 可以嘗試喚起電話或郵件應用，但實際結果會依裝置與瀏覽器而不同。

回到頁首

對應 HTML

<a href="#file-link">檔案連結</a>

<section id="file-link">
  <h2>檔案連結</h2>
</section>
