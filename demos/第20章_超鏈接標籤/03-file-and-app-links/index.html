Demo 03

檔案連結、download、tel、mailto

這個 demo 示範 href 如何指向本地檔案， 以及如何使用 tel:、mailto: 這類 URL scheme。

檔案連結

直接開啟文字檔 使用 download 下載文字檔

同一個檔案可以直接開啟，也可以加上 download 提示瀏覽器下載。實際行為仍可能受到瀏覽器與伺服器設定影響。

喚起應用程式

電話聯繫：10010 寄信給 demo@example.com

點擊電話或郵件連結時，瀏覽器可能詢問是否開啟對應應用程式。 如果裝置沒有可用應用，可能不會有明顯效果。

對應 HTML

<a href="./assets/files/sample-download.txt">直接開啟文字檔</a>

<a href="./assets/files/sample-download.txt" download="hyperlink-demo-note.txt">
  使用 download 下載文字檔
</a>

<a href="tel:10010">電話聯繫：10010</a>

<a href="mailto:demo@example.com?subject=HTML%20a%20demo">
  寄信給 demo@example.com
</a>
