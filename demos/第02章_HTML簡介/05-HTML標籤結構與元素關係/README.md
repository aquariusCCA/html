# HTML 標籤結構與元素關係 HTML 教學範例

來源筆記：`notes/第02章_HTML簡介/05-HTML標籤結構與元素關係.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-標籤基本組成.html` | 標籤與元素的基本組成 | 起始標籤、內容與結束標籤如何形成一個完整元素 | 修改段落內容，並在表格中補上一個新的雙標籤範例 |
| 2 | `02-雙標籤與不寫結束標籤元素.html` | 雙標籤與不寫結束標籤的元素 | 哪些元素會包住內容，哪些元素本身不包住文字內容 | 新增一個筆記中出現過的雙標籤範例，並觀察它是否需要結束標籤 |
| 3 | `03-元素巢狀與關係.html` | 父子、祖先後代與兄弟元素關係 | 外層元素包住內層元素時，層級與關係如何形成 | 把其中一個段落移出 `section`，觀察它和其他元素的關係如何改變 |
| 4 | `04-常見錯誤對照.html` | 常見結構錯誤與修正 | 標籤交錯閉合、忘記結束標籤、把 `img` 寫成雙標籤的差異 | 將錯誤區塊改寫成只保留正確寫法，並確認結構仍然清楚 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
