# HTML 文件骨架與基本結構 HTML 教學範例

來源筆記：`notes/第02章_HTML簡介/04-HTML文件骨架與基本結構.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-基本HTML5文件骨架.html` | 基本 HTML5 文件骨架 | 檔案從 `<!doctype html>` 開始，並包含 `html`、`head`、`meta`、`title` 與 `body` | 修改 `title`、`h1` 與段落文字，觀察分頁標題與頁面內容的差異 |
| 2 | `02-結構標籤角色.html` | 骨架成員的作用 | `doctype`、`html`、`head` 與 `body` 各自負責不同工作 | 在表格中補上一列 `title`，用自己的話說明它的作用 |
| 3 | `03-head與body分工.html` | `head` 與 `body` 的分工 | 文件設定放在 `head`，使用者看得到的主體內容放在 `body` | 修改分頁標題與頁面段落，確認哪一個出現在分頁，哪一個出現在頁面 |
| 4 | `04-常見錯誤對照.html` | 常見錯誤與修正 | `title` 不是頁面主標題；可見內容應放在 `body`；`meta charset` 應放在 `head` | 把錯誤區塊改寫成只保留正確做法，並加入一段新的 body 內容 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
