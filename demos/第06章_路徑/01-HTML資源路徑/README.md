# HTML資源路徑 HTML 教學範例

來源筆記：`notes/第06章_路徑/01-HTML資源路徑.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-路徑從HTML位置出發.html` | 路徑的用途與出發點 | 相對路徑是從目前 HTML 檔案所在位置開始找資源 | 把 `logo.png` 改成 `product.jpg`，觀察路徑最後一段代表檔案名稱 |
| 2 | `02-絕對路徑與本機路徑.html` | 絕對路徑與本機絕對路徑 | 完整網址和本機磁碟路徑的差異，以及本機路徑為什麼容易失效 | 把本機路徑改成 `images/photo.jpg`，練習改用相對路徑 |
| 3 | `03-目前目錄與下一層資料夾.html` | `./`、省略 `./` 與下一層資料夾 | `images/logo.png` 和 `./images/logo.png` 都是從目前目錄開始找 | 移除第二個範例中的 `./`，確認兩種寫法代表同一件事 |
| 4 | `04-回到上一層目錄.html` | `../` 回到上一層目錄 | 當 HTML 在 `pages/` 裡，要先回上一層再進入 `images/` | 把正確寫法中的 `logo.png` 改成 `banner.png`，保留 `../images/` |
| 5 | `05-常見錯誤與搬動後修正.html` | 路徑常見錯誤 | 比較本機路徑、`/` 開頭誤用，以及搬動 HTML 後忘記更新路徑 | 逐一檢查三組正確寫法，說明每個路徑是從哪一個位置出發 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
