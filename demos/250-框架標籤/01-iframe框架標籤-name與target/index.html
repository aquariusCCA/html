iframe 的 name 與 target 配合

觀察 <iframe name> 如何和 <a target>、<form target> 配合， 讓連結與表單的結果載入到指定框架；並比較名稱對不上時的後果。

① 連結 target 導向（正確：名稱一致）

三個連結都設定 target="container"，右側 iframe 設定 name="container"。 點不同連結，內容會載入到同一個框架。

② 表單 target 導向

表單設定 target="container"，送出後結果會載入到上方同一個名為 container 的 iframe，而不會跳離本頁。輸入關鍵字試試看。

提示：表單以預設的 GET 方式送出，結果頁會把 keyword 顯示出來。請看上方框架。

③ 常見錯誤：target 與 name 對不上

下面這個連結的 target="content"，但本頁沒有 name="content" 的框架。 瀏覽器找不到對應框架，會改用新分頁／新視窗開啟，而不是載入到下方框架。

<a target="content"> ✗ 對不上

⚠️ 點我（會另開視窗）

這個框架叫 result，和上面連結的 content 不一致，所以不會被它更新。

修正方式

<!-- 讓 target 與 name 使用相同名稱 -->
<a href="news.html" target="result">開啟頁面</a>
<iframe name="result" title="頁面內容"></iframe>

只要兩邊名稱一致，連結內容就會正確載入到指定框架。
