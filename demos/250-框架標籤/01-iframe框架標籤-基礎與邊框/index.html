② 對照：frameborder（舊） vs CSS border（新）

兩種寫法都能去掉邊框，但 frameborder 是已過時的舊式屬性，新內容應改用 CSS。

舊式寫法（不建議）

<iframe src="home.html"
        frameborder="0"></iframe>

能運作，但屬於舊式屬性，且樣式與結構沒有分離。

現代寫法（建議）

<iframe src="home.html" title="..."
        style="border: 0;"></iframe>

用 CSS 控制邊框，符合樣式與結構分離，也提供了 title。

③ 進階屬性速覽（唯讀說明）

初學階段先掌握上面的基礎即可。下列屬性在嵌入第三方服務或考量安全/效能時才會用到，這裡僅說明、不做互動。

屬性作用
sandbox限制 iframe 內頁面的能力，例如是否允許執行腳本、提交表單或開新視窗。
allow指定 iframe 可使用哪些瀏覽器功能，例如全螢幕、相機、麥克風、剪貼簿等。
allowfullscreen允許 iframe 內容進入全螢幕，常見於影音播放器或互動展示。
referrerpolicy控制載入 iframe 時，是否以及如何送出來源頁面的 referrer 資訊。
loading設為 lazy 時，iframe 接近可視範圍才載入，減少初始負擔。

關於 PDF：iframe 也能嵌入瀏覽器可顯示的文件（如 PDF），寫法與嵌入網頁相同，只是 src 指向文件路徑。本 demo 未附 PDF 檔，故以本機 HTML 頁面示意。
