觀察 head 裡的 meta 資訊

<meta> 不會直接變成頁面上的文字，但瀏覽器、搜尋引擎、社群平台或 JavaScript 可以讀取這些文件層級資訊。

範例中的 head 設定

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="這個範例示範 charset、viewport、description、robots 與 Open Graph metadata。">
<meta name="robots" content="index, follow">
<meta property="og:title" content="基本 head metadata Demo">
