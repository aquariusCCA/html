這個 demo 把常見錯誤整理成可觀察的對照。頁面中也故意放入一個位於 `body` 的 <meta>，讓你觀察它在 DOM 中的位置。

錯誤寫法與修正方式

錯誤：把 meta 放在 body

<body>
  <meta name="description" content="頁面描述">
</body>

修正：metadata 通常應放在 <head>。

錯誤：description 缺少 content

<meta name="description">

修正：補上具體描述，例如 `content="本頁介紹 HTML meta 標籤"`。

錯誤：替 meta 寫結束標籤

<meta charset="UTF-8"></meta>

修正：<meta> 是 void element，寫成 <meta charset="UTF-8">。

注意：濫用 keywords 或 refresh

<meta name="keywords" content="HTML, SEO, 排名">
<meta http-equiv="refresh" content="3;url=https://example.com/">

修正：不要把 `keywords` 視為排名保證，`refresh` 跳轉也應謹慎使用。
