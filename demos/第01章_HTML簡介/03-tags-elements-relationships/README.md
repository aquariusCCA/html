# HTML 標籤、元素與階層關係

## 1. Demo 目標

這個 demo 用來示範：

- 標籤與元素的差異。
- 雙標籤與單標籤的差異。
- HTML 元素之間的父子、兄弟、祖先與後代關係。

## 3. 如何執行

方式一：直接打開檔案

1. 找到 `index.html`
2. 使用瀏覽器開啟
3. 點擊上方關係按鈕，觀察不同元素被標示

方式二：使用 VS Code Live Server

1. 在 demo 資料夾上按右鍵
2. 選擇 `Open with Live Server`
3. 在瀏覽器中查看結果

## 4. 觀察重點

1. 黃色外框代表目前主要觀察的元素。
2. 綠色外框代表和主要元素相關的子元素或後代元素。
3. `img` 是單標籤元素，但 `p`、`h2`、`section` 仍需要結束標籤。

## 5. 修改練習

1. 在 `article` 中新增一個 `footer`，觀察它和其他元素的關係。
2. 新增一個 `a` 連結，並把它加入某個模式的 `related` 選取器中。
3. 嘗試移除某個結束標籤，觀察瀏覽器是否仍能解析，以及結果是否符合預期。

## 6. 常見錯誤

| 錯誤 | 問題 | 修正方式 |
|---|---|---|
| 把 `<p>` 稱為整個元素 | `<p>` 只是開始標籤 | 將開始標籤、內容、結束標籤合起來稱為元素 |
| 認為所有標籤都可以單獨使用 | 只有特定 void elements 才不需要結束標籤 | 查清楚元素是否屬於單標籤 |
| 巢狀結構不清楚 | CSS 與 JavaScript 選取目標容易錯誤 | 使用縮排整理父子與兄弟關係 |
