# HTML 簡介對應 Demo 規劃

## 一、Notes 摘要

本章介紹 HTML 與網頁、網站、瀏覽器、DOM 之間的關係，並建立 HTML5 文件骨架、標籤與元素、單標籤與雙標籤、元素階層、語意化結構等基礎觀念。

## 二、Demo 清單

| 編號 | Demo 名稱 | 類型 | 對應知識點 | 建議路徑 |
|---:|---|---|---|---|
| 01 | `01-basic-html-document` | 單一概念 | HTML5 文件骨架、`head` 與 `body` | `demos/第01章_HTML簡介/01-basic-html-document/` |
| 02 | `02-html-browser-dom-flow` | 對照型 | HTML 原始碼、瀏覽器呈現、DOM 形成流程 | `demos/第01章_HTML簡介/02-html-browser-dom-flow/` |
| 03 | `03-tags-elements-relationships` | 單一概念 | 標籤、元素、父子、兄弟、祖先與後代關係 | `demos/第01章_HTML簡介/03-tags-elements-relationships/` |
| 04 | `04-semantic-article-structure` | 常見錯誤 | 語意化元素、標題層級、過度使用 `div` 的問題 | `demos/第01章_HTML簡介/04-semantic-article-structure/` |

## 四、使用方式

每個 demo 都可以直接用瀏覽器開啟 `index.html`。若使用 VS Code，也可以在 demo 資料夾上使用 Live Server 觀察結果。
