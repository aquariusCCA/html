Demo 02

觀察 `width`、`height`、`margin`、`padding` 的差異

這個 demo 對照一般行內元素、塊級元素、`inline-block` 與表單控制項。 請觀察相同尺寸設定在不同元素上的效果。

一般行內元素：尺寸不會像區塊一樣生效

前方文字 span 設定 width 與 height 後方文字

觀察重點：`span` 仍然跟文字排在同一行，`width` 與 `height` 不會把它變成固定大小的區塊。

塊級元素：可以直接控制寬高

div 設定 width 與 height

觀察重點：`div` 會形成固定寬高的獨立區塊。

`inline-block`：保留同行排列，也可以控制尺寸

前方文字 span + display: inline-block 後方文字

觀察重點：`inline-block` 仍可跟文字在同一行，但能套用寬高。

表單控制項：常有自己的瀏覽器行為

觀察重點：`input` 常出現在行內排版中，但瀏覽器會讓它可以設定尺寸。
