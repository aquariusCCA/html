# 塊級與行內元素排列

## 1. Demo 目標

這個 demo 用來示範：

- 塊級元素預設會獨佔一行。
- 行內元素預設會跟相鄰文字排在同一行。
- 區塊容器中可以包含行內內容。
- 選元素時應先看語意，再交給 CSS 調整視覺效果。

方式一：直接打開檔案

2. 使用瀏覽器開啟
3. 觀察每個元素外框的排列方式

## 4. 觀察重點

1. `h3`、`p`、`div` 預設會由上到下排列。
2. `span`、`strong`、`a` 會跟文字排在同一行。
3. 外層區塊可以包住標題、段落與行內標記。
4. CSS 外框只是輔助觀察，沒有改變 HTML 元素的語意。

## 5. 程式碼重點

### HTML

- 第一段用 `h3`、`p`、`div` 展示塊級排列。
- 第二段用 `span`、`strong`、`a` 展示行內排列。
- 第三段用 `article`、`h3`、`p`、`span`、`strong` 組成混合結構。

### CSS

- `.sample-frame` 提供觀察容器。
- 外框顏色用來區分塊級與行內元素。

### JavaScript

- 此 demo 不需要 JavaScript。

## 6. 修改練習

請嘗試修改以下內容，觀察結果差異：

1. 把第二段的 `span` 改成 `div`。
2. 在第一段新增一個 `section`，觀察它是否另起一行。
3. 在 CSS 加上 `.inline-frame span { display: block; }`，觀察視覺排列是否改變。

## 7. 常見錯誤

| 錯誤 | 可能造成的問題 | 修正方式 |
|---|---|---|
| 用元素預設外觀判斷語意 | 可能把標題、段落、連結用錯 | 根據內容意義選元素 |
| 所有內容都用 `div` 包起來 | 結構缺少語意，較難閱讀與維護 | 優先使用適合的語意元素 |
| 把 `span` 當成區塊排版工具 | 不容易控制間距與尺寸 | 需要區塊結構時使用合適區塊元素或 CSS `display` |
