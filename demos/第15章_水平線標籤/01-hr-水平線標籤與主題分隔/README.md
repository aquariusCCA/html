# 01-hr-水平線標籤與主題分隔 HTML 教學範例

來源筆記：`notes/第15章_水平線標籤/01-hr-水平線標籤與主題分隔.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-hr表示主題分隔.html` | `<hr>` 表示主題分隔 | `<hr>` 放在兩個不同主題中間，表示內容進入下一個主題 | 把兩個主題改成「產品特色」與「購買注意事項」，保留 `<hr>` 在中間 |
| 2 | `02-hr是單標籤.html` | `<hr>` 是單標籤 | `<hr>` 沒有內容，不需要 `</hr>`，也不包住文字 | 暫時刪除 `<hr>` 再放回兩個不同主題中間，觀察分段是否清楚 |
| 3 | `03-瀏覽器預設外觀與CSS調整.html` | 瀏覽器預設樣式與 CSS 外觀調整 | CSS 可以改變 `<hr>` 的線條、寬度與間距，但不改變主題分隔語意 | 調整 `width` 與 `margin`，觀察外觀改變 |
| 4 | `04-不要為了裝飾使用hr.html` | 分辨語意分隔與純裝飾線 | 如果線條只是標題裝飾，應該用 CSS，不應使用 `<hr>` | 刪除裝飾用 `<hr>`，改用 `border-bottom` 控制標題底線 |
| 5 | `05-不要用大量hr取代結構.html` | `<hr>` 不應取代標題與段落結構 | 內容小節應優先用標題與段落建立結構，不要只靠大量 `<hr>` 切開 | 替第二段與第三段加上明確標題，再移除不必要的 `<hr>` |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
