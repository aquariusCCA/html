Demo 02

比較標題語意與 CSS 外觀

同樣想讓文字變大或變小時，應該保留正確的標題層級， 再用 CSS 控制外觀，而不是為了視覺效果改用錯誤的標題標籤。

做法 A：先選正確語意，再調整外觀

晨光閱讀燈

適合書桌與床邊閱讀的可調式桌燈。

產品特色

三段亮度、低藍光模式與可折疊支架。

規格

USB-C 充電，重量 420g，續航約 8 小時。

觀察重點：兩個區塊標題都是 h2，只是透過 CSS 呈現不同大小。

做法 B：只為了外觀挑標籤

晨光閱讀燈

適合書桌與床邊閱讀的可調式桌燈。

產品特色

三段亮度、低藍光模式與可折疊支架。

規格

USB-C 充電，重量 420g，續航約 8 小時。

觀察重點：畫面可能看起來像標題，但結構上出現多個最高層級標題，也有段落假裝成標題。

關鍵差異

<h2 class="large-section-title">產品特色</h2>
<h2 class="small-section-title">規格</h2>

上面的寫法讓 HTML 保留章節層級，CSS 負責字體大小、粗細與間距。
