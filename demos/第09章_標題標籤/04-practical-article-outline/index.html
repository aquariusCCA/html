Demo 04

用標題建立可掃描的文章結構

這個 demo 把文章頁分成清楚的標題層級，並用 JavaScript 從 DOM 中產生右側 outline， 觀察標題如何幫助使用者快速理解內容。

如何開始學習 HTML

HTML 是用來描述網頁內容與結構的標記語言。學習 HTML 時， 先理解內容層級，再學習常見標籤，會比只記畫面效果更穩定。

先理解 HTML 的用途

HTML 的工作是描述頁面有哪些內容，以及這些內容彼此之間的關係。 標題標籤就是用來指出主題、段落主題與子段落主題。

內容與結構

同一段文字可以被做成普通段落，也可以被做成標題。 選擇哪個標籤，應該取決於它是不是一個新的內容區塊名稱。

再學習常見標籤

初學者可以從標題、段落、連結、圖片與表單元素開始。 每個標籤都先看語意，再看預設外觀。

標題標籤

h1 表示最高層級標題，h2 表示主要段落， h3 可以表示更細的子段落。

段落與連結

段落用來承載一般文字，連結用來表示可前往其他位置的文字或內容。

安排學習順序

先用標題建立頁面骨架，再補上段落與其他內容。 這樣比較容易檢查整篇文章是否有清楚的主題與章節。

文章標題 outline

    觀察重點：outline 是從文章內的 h1、h2、h3 產生的。
