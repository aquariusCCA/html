# 實務應用：產品卡片中的圖片

## 1. Demo 目標

這個 demo 用來示範：

- 在產品卡片中正確使用 `<img>`。
- 使用 `alt` 描述產品圖片內容。
- 用 CSS 控制圖片寬度並維持比例。
- 重要文字不要只放在圖片裡，應使用 HTML 文字呈現。

## 2. 對應 notes

- 筆記路徑：`notes/第17章_圖片標籤/圖片標籤.md`
- 對應章節：`7. 實務應用場景`、`8. 常見錯誤與修正`、`9. 與其他概念的關係`
- 對應知識點：
  - 產品圖片
  - `src` 與本地圖片路徑
  - `alt` 替代文字
  - CSS 控制圖片尺寸
  - 圖片與 HTML 文字分工

## 3. 如何執行

方式一：直接打開檔案

1. 找到 `index.html`
2. 使用瀏覽器開啟
3. 觀察產品卡片中的圖片與文字分工

方式二：使用 VS Code Live Server

1. 在 demo 資料夾上按右鍵
2. 選擇 `Open with Live Server`
3. 在瀏覽器中查看結果

## 4. 觀察重點

1. 產品圖片使用 `src` 指向本地 SVG。
2. `alt` 描述圖片內容，而產品名稱與價格使用真正文字呈現。
3. `.product-image` 使用 `width: 100%; height: auto;` 維持比例。
4. 縮小瀏覽器視窗時，卡片版面會改為單欄。

## 5. 程式碼重點

### HTML

- 使用 `<article>` 表示一張完整產品卡片。
- 使用 `<img>` 放產品圖片。
- 使用文字節點呈現產品名稱、描述與價格。

### CSS

- `.product-image` 控制圖片填滿卡片寬度並維持比例。
- CSS 負責視覺排版，HTML 負責內容與語意。

### JavaScript

- 此 demo 不需要 JavaScript。

## 6. 修改練習

請嘗試修改以下內容，觀察結果差異：

1. 將 `alt` 改得更精準，例如加入產品顏色或角度。
2. 移除 `.product-image` 的 `height: auto`，觀察圖片比例是否受影響。
3. 將產品名稱改成另一個商品，但保持圖片 `alt` 與內容一致。
4. 嘗試把價格刪除，只留在圖片中，思考為什麼這不是好做法。

## 7. 常見錯誤

| 錯誤 | 可能造成的問題 | 修正方式 |
|---|---|---|
| 產品圖片沒有 `alt` | 輔助工具無法理解圖片內容 | 補上描述圖片用途的替代文字 |
| 商品名稱與價格只做成圖片 | 圖片失敗時重要資訊消失 | 重要資訊使用 HTML 文字 |
| 用固定高度壓縮產品圖 | 圖片比例可能變形 | 使用 `height: auto` 或明確裁切策略 |
| 圖片檔名與內容不一致 | 後續維護困難 | 使用有語意的檔名與清楚資料夾結構 |
