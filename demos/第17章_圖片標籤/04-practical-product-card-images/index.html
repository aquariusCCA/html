Demo 04

實務應用：產品卡片中的圖片

這個 demo 把 <img> 放進常見的產品卡片情境。 重點是圖片要有正確來源、清楚替代文字，產品名稱與價格也應使用真正的 HTML 文字呈現。

新品

Nimbus 無線耳機

適合通勤、線上會議與日常聆聽。產品名稱、價格與重點資訊使用文字呈現， 不依賴圖片本身承載所有資訊。

NT$ 2,490

這張產品圖片做了什麼

  • src 指向本地圖片檔案。
  • alt 說明圖片是黑色無線耳機產品照片。
  • 圖片使用 CSS 維持比例並適應卡片寬度。
  • 產品名稱與價格不是只放在圖片裡。

關鍵 HTML

<img
  class="product-image"
  src="../assets/images/product-headphones.svg"
  alt="黑色無線耳機產品照片"
>
