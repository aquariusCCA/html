# 基本圖片標籤：src、alt、title

## 1. Demo 目標

這個 demo 用來示範：

- `<img>` 如何透過 `src` 載入圖片。
- `alt` 如何描述圖片內容。
- `title` 可以提供額外提示，但不應取代 `alt`。
- 只設定 `width` 時，圖片高度會依比例縮放。

## 2. 對應 notes

- 筆記路徑：`notes/第17章_圖片標籤/圖片標籤.md`
- 對應章節：`3. 核心概念`、`4. 語法與基本用法`
- 對應知識點：
  - `<img>` 是用來顯示圖片的單標籤
  - `src` 是圖片來源
  - `alt` 是替代文字
  - `title` 是提示文字

## 3. 如何執行

方式一：直接打開檔案

1. 找到 `index.html`
2. 使用瀏覽器開啟
3. 觀察圖片是否正常顯示

方式二：使用 VS Code Live Server

1. 在 demo 資料夾上按右鍵
2. 選擇 `Open with Live Server`
3. 在瀏覽器中查看結果

## 4. 觀察重點

1. `src` 指向正確圖片時，瀏覽器會載入圖片。
2. 圖片有 `alt`，代表它有可讀的替代描述。
3. 滑鼠停在圖片上時，可能會看到 `title` 提示文字。
4. HTML 中沒有 `</img>` 結束標籤。

## 5. 程式碼重點

### HTML

- 使用完整的 `<img>` 屬性範例。
- 使用 `<figure>` 與 `<figcaption>` 提供圖片說明。

### CSS

- CSS 只用於排版與讓圖片自適應容器寬度。
- `height: auto` 用來維持圖片比例。

### JavaScript

- 此 demo 不需要 JavaScript。

## 6. 修改練習

請嘗試修改以下內容，觀察結果差異：

1. 修改 `alt` 文字，讓它更精準描述圖片。
2. 將 `width="420"` 改成 `width="260"`。
3. 移除 `title`，觀察圖片仍然可以正常顯示。
4. 將 `src` 改成錯誤路徑，觀察圖片載入失敗時的結果。

## 7. 常見錯誤

| 錯誤 | 可能造成的問題 | 修正方式 |
|---|---|---|
| `src` 寫錯 | 圖片無法顯示 | 確認圖片與 HTML 的相對位置 |
| 用 `title` 取代 `alt` | 圖片失敗或輔助工具讀取時資訊不足 | 圖片內容應放在 `alt` |
| 忘記寫 `alt` | 可訪問性不足 | 為有內容意義的圖片提供替代文字 |
