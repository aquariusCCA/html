# 比較 width、height 與圖片比例

## 1. Demo 目標

這個 demo 用來示範：

- 只設定 `width` 時，圖片高度通常會依比例縮放。
- 同時設定錯誤的 `width` 與 `height` 時，圖片可能變形。
- CSS 的 `max-width: 100%; height: auto;` 可以讓圖片更適合響應式版面。

## 4. 觀察重點

1. 只設定 `width` 的圖片仍維持原本比例。
2. 同時設定不合比例的 `width` 與 `height`，圖片會被壓縮變形。
3. 響應式圖片可以跟著容器縮放。
4. CSS 的 `height: auto` 是保持比例的關鍵。

## 5. 程式碼重點

### HTML

- 每個卡片使用同一張圖片，方便比較不同尺寸寫法。
- 錯誤比例範例刻意寫成 `width="240" height="240"`。

### CSS

- `.responsive-image` 示範 `max-width` 與 `height: auto`。
- CSS 只用於輔助觀察尺寸差異。

### JavaScript

- 此 demo 不需要 JavaScript。

## 6. 修改練習

請嘗試修改以下內容，觀察結果差異：

1. 將錯誤範例的 `height="240"` 改成 `height="135"`。
2. 將第一張圖片的 `width="320"` 改成 `width="500"`。
3. 移除 `.responsive-image` 的 `height: auto`。
4. 將瀏覽器視窗縮小，觀察響應式圖片如何變化。

## 7. 常見錯誤

| 錯誤 | 可能造成的問題 | 修正方式 |
|---|---|---|
| 同時亂設寬高 | 圖片比例被破壞 | 只設定其中一個尺寸，或確保比例正確 |
| CSS 設了固定高度 | 圖片被拉伸或裁切 | 搭配 `height: auto` 或使用合適的裁切策略 |
| 忽略小螢幕寬度 | 圖片超出容器 | 使用 `max-width: 100%` |
