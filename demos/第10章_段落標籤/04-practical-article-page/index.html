這個 demo 使用文章頁情境示範標題與段落的分工， 並透過控制按鈕調整段落間距，觀察 HTML 結構與 CSS 樣式的責任差異。

HTML 基礎

為什麼文章需要清楚的段落？

段落是文章閱讀節奏的基本單位。當一篇文章被拆成多個清楚段落時，讀者可以更快掌握每一段正在說明的重點。

在 HTML 中，段落應使用 <p> 表示。這不只是為了讓畫面看起來有空隙，而是為了讓文件結構清楚地表達「這是一段正文」。

如果需要調整段落之間的距離，應該修改 CSS，而不是在 HTML 中加入多個空白、空段落或連續換行標籤。

這樣的分工可以讓內容結構保持穩定，也讓樣式在不同頁面或不同裝置上更容易統一管理。
