img 屬性與替代文字判斷

完整寫法

建議

HTML

<img src="assets/portrait.jpg" alt="穿黑色西裝的男子在典禮背景前揮手" title="活動現場照片" width="320">

替代文字輸出

穿黑色西裝的男子在典禮背景前揮手

判斷

內容圖片應該用具體 alt 說明圖片傳達的資訊；title 只能補充提示，不能取代 alt。

Reference

屬性的分工

src 指向圖片檔案，alt 提供替代文字，title 是滑鼠停留時的補充提示，width 與 height 控制顯示尺寸。
