`lang`、`dir` 與 `translate` 的文字屬性

這個頁面用多語內容示範語言標示、文字方向與不建議翻譯的專有名詞。

`lang`：標示內容語言

繁體中文段落

這段內容使用繁體中文，外層頁面也宣告了 lang="zh-Hant"。

English paragraph

This paragraph is marked with lang="en".

`dir`：控制文字方向

預設方向

This sentence flows from left to right.

從右到左

HTML 可以用 dir="rtl" 指定從右到左的呈現方向。

自動判斷

HTML Global Attributes 是本章的英文術語。

`translate`：提示是否適合被翻譯

本章會提到 HTML Global Attributes 這個術語。這個片段使用 translate="no"，表示它不建議被自動翻譯。

品牌名、書名、程式語法或專有名詞，常會用這種方式保留原文。

注意事項

`lang`、`dir`、`translate` 是給瀏覽器、搜尋引擎、翻譯工具與輔助工具參考的資訊。 它們不是保護資料的安全機制，也不應拿來隱藏內容。
