Demo 05

`tabindex`、`accesskey` 與 `draggable`

這個 demo 示範鍵盤焦點控制與拖放互動。請用 Tab 鍵移動焦點，也可以拖曳標籤到目標區。

鍵盤焦點觀察

自然焦點 2：前往拖放區
`tabindex="0"`：讓非互動元素進入自然 Tab 順序

`tabindex="-1"`：不會被 Tab 自然聚焦，但可以透過 JavaScript 聚焦。

本 demo 不使用正整數 `tabindex`，因為它會打亂 DOM 自然順序，通常不建議濫用。

`accesskey` 觀察

下方按鈕使用 accesskey="h"。實際快捷鍵組合會因作業系統與瀏覽器不同。

你觸發了按鈕。若是透過快捷鍵觸發，代表瀏覽器支援目前的快捷鍵組合。
