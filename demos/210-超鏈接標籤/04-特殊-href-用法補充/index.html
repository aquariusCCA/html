特殊 href 用法補充

這些寫法都能出現在 href，但它們不一定適合當作正式互動功能。

點擊下方範例，觀察每種 href 的用途與限制。
#

回到頂部或空片段

<a href="#">回到頂部</a>
測試 href="#"
#top

明確錨點

<a href="#top">回到頂部</a>
<header id="top">...</header>
回到頂部
""

空 href

<a href="">刷新本頁面</a>
測試空 href
javascript:

不建議作為主要互動

<a href="javascript:alert('Hello')">點擊我</a>
測試 javascript:
tel:

電話連結

<a href="tel:10010">電話聯繫</a>
測試 tel:
mailto:

郵件連結

<a href="mailto:10010@qq.com">郵件聯繫</a>
測試 mailto:

觸發操作時，用 button 分工更清楚

不建議

<a href="javascript:submitForm()">送出</a>

較合適

<button type="submit">送出</button>

app scheme 應提供備用方案

<a href="line://msg/text/Hello%20World">使用 LINE 發送消息</a>
