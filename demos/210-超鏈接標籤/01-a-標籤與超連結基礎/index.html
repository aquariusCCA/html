a 標籤與超連結基礎

點擊下方範例觀察連結語意。

同一個標籤，重點是導航目標

<a> 的核心任務是把使用者帶到另一個位置。`href` 決定目標，`target` 決定開啟方式。

本頁目標區 內部連結會定位到這裡

有 href：可導航

<a href="#internal-section">前往本頁目標</a>
前往本頁目標

沒有 href：不是一般超連結

<a>前往首頁</a>
前往首頁

target 對照

<a href="https://example.com" target="_blank" rel="noopener">外部網站</a>
_self 範例 _blank 範例

前往位置用 a，觸發動作用 button

<a href="product.html">查看商品</a>
查看商品 
<button type="button">儲存</button>

判斷準則

  • 前往網站、頁面、檔案或錨點：使用 <a href="...">。
  • 送出、儲存、切換狀態、打開彈窗：通常使用 <button>。
  • 需要新頁籤時才使用 target="_blank"，不要無條件套用。
