# 顯示狀態與補充提示屬性 HTML 教學範例

來源筆記：`notes/第05章_全局屬性/04-顯示狀態與補充提示屬性.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-hidden基本隱藏狀態.html` | `hidden` 基本隱藏狀態 | 有 `hidden` 屬性的段落不會呈現在頁面上 | 移除第二個段落的 `hidden` 屬性，重新整理後觀察畫面 |
| 2 | `02-until-found可被尋找的隱藏內容.html` | `hidden="until-found"` | 內容平時隱藏，但可由頁面搜尋或片段連結揭示 | 修改隱藏段落中的關鍵字，再使用頁面搜尋測試 |
| 3 | `03-checkbox切換hidden欄位.html` | 用 JavaScript 設定 `hidden` | 勾選核取方塊後，額外欄位從隱藏變成顯示 | 把欄位標籤改成「公司抬頭」，再觀察勾選與取消勾選 |
| 4 | `04-title補充提示資訊.html` | `title` 提供補充提示資訊 | 滑鼠移到有 `title` 的元素上時，瀏覽器可能顯示提示文字 | 修改其中一個 `title` 內容，再用滑鼠懸停觀察提示 |
| 5 | `05-title與hidden常見誤用.html` | `title` 與 `hidden` 的實務注意事項 | 重要資訊不要只放在 `title`；內容仍應出現時不要用 `hidden` | 把警告文字改成另一種重要提醒，確認提醒直接顯示在畫面上 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
