# 鍵盤操作與焦點順序屬性 HTML 教學範例

來源筆記：`notes/第05章_全局屬性/05-鍵盤操作與焦點順序屬性.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-accesskey基本快捷鍵.html` | `accesskey` 基本快捷鍵 | 元素可以設定快捷鍵，但啟用方式依瀏覽器與作業系統而不同 | 把「儲存」按鈕的快捷鍵從 `s` 改成 `v`，並同步更新頁面說明 |
| 2 | `02-accesskey不宜大量使用.html` | `accesskey` 的使用風險 | 快捷鍵太多會難以記憶，也可能和其他快捷鍵衝突 | 移除錯誤範例中三個不必要的 `accesskey`，只保留一個高頻操作 |
| 3 | `03-依DOM順序排列焦點.html` | 不寫 `tabindex` 時依 DOM 順序聚焦 | 按 `Tab` 時，焦點依 HTML 寫法移動到姓名、電子郵件、送出 | 交換兩個輸入欄位的 HTML 位置，再觀察焦點順序 |
| 4 | `04-tabindex常見值.html` | `tabindex="0"` 與 `tabindex="-1"` | `0` 會加入自然 Tab 順序，`-1` 不會被 Tab 停到但可用程式聚焦 | 移除中間區塊的 `tabindex="0"`，觀察它是否仍會被 Tab 停到 |
| 5 | `05-正整數tabindex常見誤用.html` | 避免使用大於 `0` 的 `tabindex` | 正整數 `tabindex` 會讓焦點順序脫離 HTML 內容順序 | 移除正整數 `tabindex`，改用正確的 HTML 順序 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
