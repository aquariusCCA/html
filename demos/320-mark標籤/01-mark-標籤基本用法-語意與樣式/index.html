mark 標籤：語意與樣式

<mark> 用來標示「在目前上下文中需要注意的文字片段」。 多數瀏覽器預設給它黃色背景，但那只是預設樣式，不是它的全部意義。 下面切換樣式時請注意：不管底色怎麼變，被標記文字的語意都沒有改變。

原始 HTML

<p>學習 HTML 時，先理解 <mark>標籤語意</mark> 會比只記語法更重要。</p>

實際呈現

學習 HTML 時，先理解 標籤語意 會比只記語法更重要。

這堂課會先介紹 HTML5 新增標籤 的用途。

切換 mark 的樣式

目前樣式：瀏覽器預設黃底。語意不變，只是視覺呈現不同。

mark {
  /* 使用瀏覽器預設樣式 */
}
