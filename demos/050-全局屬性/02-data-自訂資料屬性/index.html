data-* 與 dataset

點選商品卡片，觀察畫面上的文字和元素攜帶的自訂資料不是同一件事。

商品清單

<button
  data-product-id="A-102"
  data-category="keyboard"
  data-stock="18">
  機械鍵盤
</button>

JavaScript 讀到的 dataset

屬性名稱會從 `data-product-id` 轉成 `dataset.productId`。

dataset.productId
A-102
dataset.category
keyboard
dataset.stock
18
`data-*` 適合存放少量、和元素直接相關的資料；不應把重要可見文字只藏在屬性裡。 
selected.dataset.productId
selected.dataset.category
selected.dataset.stock
