Demo 01

基本表格結構：table、tr、th、td

這個 demo 示範最基本的 HTML 表格：一張表格由列與儲存格組成， 表頭使用 <th>，一般資料使用 <td>。

學生資料表

每一列 <tr> 代表一筆學生資料，每一欄代表同一種資料欄位。

結構觀察

<table>

定義整張表格，是所有表格列與儲存格的容器。

<tr>

定義表格中的一列，列裡面才放表頭或資料儲存格。

<th>

表示表頭儲存格，適合放欄位名稱或列標題。

<td>

表示一般資料儲存格，放實際資料內容。
