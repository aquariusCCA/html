這個 demo 的分工

HTML

使用 <caption>、<thead>、<tbody>、<tfoot> 表達表格結構。

CSS

控制表格寬度、邊框、間距、狀態標籤與小螢幕捲動。

JavaScript

只處理篩選與排序，不改變表格本身應有的 HTML 語意。
