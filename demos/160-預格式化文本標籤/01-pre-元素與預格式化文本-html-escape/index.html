HTML Demo

在 pre 中展示 HTML 原始碼

錯誤

直接寫 HTML 標籤

`pre` 會保留空白與換行，但裡面的 HTML 標籤仍會被瀏覽器解析。

HTML

<pre>
<div class="card">內容</div>
</pre>

Result

內容
修正

先轉義特殊字元

要讓標籤以文字出現，就把 `<`、`>`、`&` 改成字元實體。

HTML

<pre>
&lt;div class="card"&gt;內容&lt;/div&gt;
</pre>

Result

<div class="card">內容</div>

常見轉義對照

原字元 轉義寫法 用途
< &lt; 避免被解析成標籤開頭
> &gt; 避免被解析成標籤結尾
& &amp; 避免被解析成字元實體開頭
