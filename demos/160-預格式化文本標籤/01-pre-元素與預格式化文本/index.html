pre 元素與預格式化文本

p

一般段落

段落文字會折疊連續空白與換行，原本的縮排不會完整保留。

HTML

<p>
body {
  background: #fff;
  font: 12px/24px Arial, sans-serif;
}
</p>

Result

body { background: #fff; font: 12px/24px Arial, sans-serif; }

pre

預格式化文本

`pre` 會保留內容中的空格、Tab 與換行，適合展示程式碼或排版好的文字。

HTML

<pre>
body {
  background: #fff;
  font: 12px/24px Arial, sans-serif;
}
</pre>

Result

body {
  background: #fff;
  font: 12px/24px Arial, sans-serif;
}

觀察重點

  • `p` 適合一般段落，不負責保留程式碼排版。
  • `pre` 表示內容已經預先排版，瀏覽器會照原本空白與換行呈現。
  • 當空白、縮排、換行本身就是資訊時，使用 `pre` 更符合語意。
