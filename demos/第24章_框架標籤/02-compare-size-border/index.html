Demo 02

iframe 尺寸與邊框控制對照

這個 demo 對照固定 width、height、 舊式 frameborder 與現代 CSS 控制方式。

寫法 A：固定寬高與 frameborder

這種寫法常見於舊教材。它看得懂即可，新程式碼建議把視覺樣式交給 CSS。

寫法 B：CSS 控制尺寸與邊框

這種寫法把 iframe 的視覺呈現交給 CSS，較容易配合不同螢幕寬度。

觀察重點

  • 縮小瀏覽器視窗，觀察固定寬度 iframe 是否需要水平捲動。
  • 比較 frameborder="0" 與 border: 0; 的角色差異。
  • 修改 CSS 中的 .responsive-frame 高度與邊框，觀察畫面變化。
