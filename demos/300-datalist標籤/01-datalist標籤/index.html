<datalist> 標籤：替輸入框提供建議選項

<datalist> 替指定的 <input> 提供「候選建議值」，但不強制， 使用者仍可輸入清單外的內容。下面三個區塊分別示範：基本綁定與自由輸入、與 <select> 的差異、 以及 list / id 對不上的常見錯誤。

① 建議輸入：list 與 id 的綁定

點進輸入框時，瀏覽器會跳出 <datalist> 的建議清單。 你可以選建議值，也可以直接輸入清單以外的瀏覽器名稱（例如 Edge、Brave）。

② 對照：<datalist> 與 <select>

兩邊都是「瀏覽器」選項。左邊 datalist 允許自由輸入，右邊 select 只能從固定選項挑一個。 各自下方會即時顯示「表單會送出的值」。

試試輸入 Edge 再送出 —— 清單外的值也能送出。

無法輸入 Edge —— 只能在三個固定選項裡選。

準則：需要「提示但允許自由輸入」用 <datalist>； 需要「限定固定選項」用 <select>。
