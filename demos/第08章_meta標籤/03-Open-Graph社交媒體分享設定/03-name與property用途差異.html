name 與 property 用途差異

搜尋摘要與社群分享的不同寫法

<meta name="description" content="這是給搜尋摘要參考的頁面描述。">
<meta property="og:description" content="這是給社群分享預覽參考的頁面描述。">

兩者都在描述頁面，但主要使用者不同：前者偏向搜尋引擎，後者偏向社群平台與分享工具。

Open Graph 的屬性寫法

錯誤寫法

<meta name="og:title" content="網頁標題">

建議寫法

<meta property="og:title" content="網頁標題">

建議修改任務

分別修改 name="description" 和 property="og:description" 的內容，練習讓搜尋摘要與社群摘要服務不同目的。
