description 與頁面內容一致

正確寫法

頁面內容：介紹 HTML meta 標籤的用途、description、keywords 與 robots。

<meta name="description" content="介紹 HTML meta 標籤的用途，包含 description、keywords 與 robots。">

這段描述和頁面內容一致，使用者能快速判斷是否要點進來。

錯誤寫法

頁面內容：只介紹 HTML meta 標籤。

<meta name="description" content="最完整的前端課程，保證搜尋第一名。">

錯誤寫法：摘要寫得很吸引人，但正文沒有提供對應內容。

修正方式：改成準確描述本頁內容的簡短文字。
