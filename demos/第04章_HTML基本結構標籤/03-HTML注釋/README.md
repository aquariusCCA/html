# 03-HTML注釋 HTML 教學範例

來源筆記：`notes/第04章_HTML基本結構標籤/03-HTML注釋.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-注釋基本語法.html` | HTML 注釋語法 | 觀察 `<!--` 與 `-->` 如何包住注釋內容 | 修改注釋文字，確認頁面畫面不會跟著改變 |
| 2 | `02-注釋不會顯示.html` | 注釋不會成為頁面內容 | 比較一般段落、注釋內容與錯誤寫法的顯示差異 | 把一段注釋移出注釋符號，觀察它是否顯示 |
| 3 | `03-標示結構與待辦.html` | 標示結構與留下 TODO | 觀察注釋如何標示主要內容開始、結束與後續調整提醒 | 替 `main` 或 `nav` 加上一段有意義的注釋 |
| 4 | `04-注釋結尾錯誤對照.html` | 注釋必須正確結束 | 比較少寫 `-->` 與正確結尾的差異 | 在錯誤範例中補上 `-->`，確認寫法變完整 |
| 5 | `05-注釋內容使用原則.html` | 有價值的注釋與敏感資訊 | 比較重複描述標籤的注釋與說明原因的注釋 | 將不太有價值的注釋改寫成說明原因的注釋 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
