DOCTYPE 放置位置

觀察範例

正確寫法

<!doctype html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
</head>
<body>
  頁面內容
</body>
</html>

<!doctype html> 放在 <html> 之前。

錯誤寫法

<html lang="zh-Hant">
  <!doctype html>
  <head>
    <meta charset="UTF-8">
  </head>
</html>

<!doctype html> 不應該放進 <html> 裡面。

修改練習

在正確寫法中新增 <body> 內容，確認 <!doctype html> 仍然留在第一行。
