# HTML標籤屬性 HTML 教學範例

來源筆記：`notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-屬性名與屬性值.html` | 一般屬性的基本組成 | 屬性由屬性名與屬性值組成，寫在開始標籤內 | 修改 `title` 的屬性值，觀察提示文字是否改變 |
| 2 | `02-屬性位置.html` | 屬性的正確位置 | 屬性要寫在開始標籤或單標籤內，不寫在結束標籤 | 把正確寫法中的 `title` 改成另一段提示文字 |
| 3 | `03-多個屬性與空格.html` | 多個屬性與空格分隔 | 同一個標籤可以有多個屬性，屬性之間用空格分隔 | 修改 `placeholder`，觀察輸入欄位中的提示文字 |
| 4 | `04-布林屬性disabled.html` | 布林屬性的基本寫法 | `disabled` 只要出現屬性名，就代表欄位停用 | 移除第二個輸入欄位的 `disabled`，觀察欄位狀態 |
| 5 | `05-常見錯誤對照.html` | 屬性常見錯誤 | 比較缺少空格、重複屬性、屬性寫到結束標籤的錯誤與正確寫法 | 檢查每個正確寫法的屬性位置與空格 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
