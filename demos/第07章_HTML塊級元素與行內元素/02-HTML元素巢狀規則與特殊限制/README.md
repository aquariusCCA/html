# HTML 元素巢狀規則與特殊限制 HTML 教學範例

來源筆記：`notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-父子關係與內容模型.html` | 父元素、子元素與內容模型 | 外層元素包住內層元素時的父子關係，以及巢狀判斷不只看塊級或行內 | 將其中一個子元素移出父元素，觀察外框與結構關係改變 |
| 2 | `02-void-elements不能包內容.html` | void elements 不能有子內容 | `input` 不能包住 `span` 或文字，需要提示文字時可使用 `label` | 把提示文字改成「電子郵件」，並同步修改 `input` 的 `name` |
| 3 | `03-標題元素不可互相巢狀.html` | `h1` 到 `h6` 不能相互嵌套 | 標題應該平行排列，用層級表達章節關係，不是互相包住 | 將正確寫法中的 `h2` 改成 `h3`，觀察標題層級仍然是平行結構 |
| 4 | `04-p段落不能包含區塊內容.html` | `p` 不能包含區塊內容 | 瀏覽器通常會在 `div` 出現前自動結束 `p`，所以原始碼和 DOM 可能不同 | 把錯誤範例中的 `div` 改成 `span`，比較是否仍需要拆成獨立區塊 |
| 5 | `05-a連結不可互相巢狀.html` | `a` 可以包允許的內容，但不能包另一個 `a` | 多個可點擊目標應拆成獨立連結；卡片式連結中不要再放其他連結 | 將卡片式連結改成兩個獨立連結，觀察結構是否更清楚 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
