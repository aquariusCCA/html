Demo 01

觀察 pre 如何保留文字格式

這個 demo 使用 <pre> 顯示需要保留空格、縮排與換行的文字內容。

命令列輸出

npm run build

> html-demo@1.0.0 build
> vite build

✓ 42 files transformed
dist/index.html        1.2 kB
dist/assets/app.css    3.8 kB
dist/assets/app.js     8.5 kB

觀察重點：空行、縮排與每一列的對齊都會保留。

項目        數量    狀態
HTML        12      完成
CSS         08      進行中
JavaScript  03      待整理

觀察重點：這類固定寬度資料依賴空格對齊，適合用 <pre> 保留格式。

本 demo 重點

保留空格

連續空格不會被折疊成單一空格。

保留換行

原始碼中的換行會在畫面上呈現。

常見預設字體

瀏覽器通常會用等寬字體顯示 <pre> 內容。
