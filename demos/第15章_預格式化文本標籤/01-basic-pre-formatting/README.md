# 基本 pre 格式保留

## 1. Demo 目標

這個 demo 用來示範：

- `<pre>` 會保留文字中的空格。
- `<pre>` 會保留原始碼中的換行。
- `<pre>` 適合顯示命令列輸出、純文字表格等固定格式內容。
- CSS 可以改善 `<pre>` 的外觀，但不改變它的 HTML 語意。

## 4. 觀察重點

1. `<pre>` 中的空行會保留。
2. 連續空格可以用來對齊純文字內容。
3. 內容通常會以等寬字體呈現。
4. CSS 只負責讓區塊更容易觀察。

## 5. 程式碼重點

### HTML

- 使用 `<pre>` 包住命令列輸出。
- 使用 `<pre>` 顯示固定寬度純文字表格。

### CSS

- `.pre-box` 設定等寬字體、邊框與水平捲動。
- 樣式只輔助閱讀，不取代 `<pre>` 的語意。

### JavaScript

- 此 demo 不需要 JavaScript。

## 6. 修改練習

請嘗試修改以下內容，觀察結果差異：

1. 在命令列輸出中新增一個空行。
2. 在純文字表格中調整空格數量，觀察對齊變化。
3. 將其中一個 `<pre>` 改成 `<p>`，觀察格式是否被保留。
4. 移除 `.pre-box` 的 `font-family`，觀察字體變化。

## 7. 常見錯誤

| 錯誤 | 可能造成的問題 | 修正方式 |
|---|---|---|
| 用 `<pre>` 顯示普通段落 | 內容可能出現不自然的空白與換行 | 普通文字使用 `<p>` |
| 忽略標籤內縮排 | 多餘縮排也會被顯示 | 將真正要呈現的文字對齊 |
| 只靠 CSS 模擬固定格式 | 容易失去 HTML 語意 | 需要保留原始格式時使用 `<pre>` |
