Demo 02

比較 p 與 pre 的空白處理

這個 demo 使用同一段 CSS 文字，比較一般段落與 <pre> 對空格、縮排與換行的處理差異。

同一段文字放在不同元素中

使用 p

body { background: #fff; font-size: 16px; line-height: 1.6; }

一般段落會依照 HTML 的空白處理規則顯示，縮排與換行不會完整保留。

使用 pre

body {
  background: #fff;
  font-size: 16px;
  line-height: 1.6;
}

<pre> 會保留原始碼中的換行與縮排。

選用判斷

需求 建議元素 原因
一般文章段落 <p> 段落是文字內容結構，不需要保留所有空白。
展示程式碼或固定格式文字 <pre> 縮排、空格與換行本身就是內容的一部分。
只想調整畫面間距 CSS 視覺距離應交給樣式控制，不應改變 HTML 語意。
