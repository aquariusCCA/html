Demo 03

常見錯誤：以為 pre 會自動顯示 HTML 原始碼

這個 demo 比較未轉義 HTML 與使用 HTML entity 後的結果。

錯誤寫法：直接放 HTML 標籤

下方區塊刻意保留錯誤寫法。目標是顯示原始碼，但瀏覽器仍會解析其中的標籤。 

Hello HTML


這是一段文字。

正確寫法：轉成 HTML entity

<h1>Hello HTML</h1>
<p>這是一段文字。</p>

觀察重點：要顯示 <h1> 這種標籤文字時，需要把 < 與 > 轉義。

本章常用轉義對照

想顯示的字元 應寫成 使用情境
< &lt; 顯示 HTML 開始標籤的左尖括號。
> &gt; 顯示 HTML 標籤的右尖括號。
