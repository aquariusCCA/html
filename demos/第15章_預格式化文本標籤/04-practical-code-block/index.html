Demo 04

用 pre 與 code 建立可閱讀的程式碼區塊

這個 demo 把 <pre> 放進教學文章情境，並搭配 <code> 與 CSS 改善閱讀體驗。

範例：CSS 基本設定

下方程式碼區塊使用 <pre> 保留換行與縮排， 使用 <code> 表示內容是一段程式碼。 

body {
  margin: 0;
  background: #fff;
  color: #222831;
  font-family: Arial, "Noto Sans TC", sans-serif;
  line-height: 1.6;
}

.very-long-selector-name-used-to-test-horizontal-scroll {
  display: grid;
  grid-template-columns: minmax(0, 1fr) minmax(240px, 360px);
}

觀察重點：長行不會把整個頁面撐壞，而是在程式碼區塊內水平捲動。

HTML 與 CSS 的分工

部分 負責內容 本 demo 中的例子
HTML 表達內容結構與語意。 <pre> 保留格式，<code> 表示程式碼。
CSS 控制視覺呈現。 overflow-x: auto;、背景色、內距與邊框。
