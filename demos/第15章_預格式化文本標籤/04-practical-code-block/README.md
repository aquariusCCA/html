# 實務程式碼區塊

## 1. Demo 目標

這個 demo 用來示範：

- 使用 `<pre>` 保留程式碼的換行與縮排。
- 使用 `<code>` 表示內容是一段程式碼。
- 使用 CSS 改善程式碼區塊的可讀性。
- 使用 `overflow-x: auto` 處理較長的程式碼行。

## 2. 對應 notes

- 筆記路徑：`notes/第15章_預格式化文本標籤/預格式化文本標籤.md`

  - 透過 CSS 改善 `<pre>` 閱讀體驗

## 4. 觀察重點

1. `<pre>` 中的 CSS 程式碼保留縮排與換行。
2. `<code>` 表示這段內容是程式碼。
3. 長行會在程式碼區塊內水平捲動。
4. 背景色、邊框與內距都由 CSS 控制。

## 5. 程式碼重點

### HTML

- 使用 `<pre class="code-block"><code>...</code></pre>` 建立程式碼區塊。
- 將程式碼區塊放在文章情境中觀察實務用途。

### CSS

- `.code-block` 設定 `overflow-x: auto`。
- `.code-block code` 移除 inline `code` 的背景與內距，避免雙層樣式干擾。

### JavaScript

- 此 demo 不需要 JavaScript。

## 6. 修改練習

請嘗試修改以下內容，觀察結果差異：

1. 刪除 `.code-block` 的 `overflow-x: auto`，觀察長行如何影響版面。
2. 新增一段更長的 CSS 選擇器，觀察水平捲動。
3. 移除 `<code>` 標籤，只保留 `<pre>`，思考語意差異。
4. 修改 `.code-block` 的背景色與內距，觀察閱讀體驗變化。

## 7. 常見錯誤

| 錯誤 | 可能造成的問題 | 修正方式 |
|---|---|---|
| 只用 `<pre>` 表示程式碼 | 少了程式碼語意 | 使用 `<pre><code>...</code></pre>` |
| 沒有處理長行 | 程式碼可能撐破版面 | 使用 `overflow-x: auto` |
| 把樣式寫進 HTML 結構中 | 維護性變差 | 將視覺呈現交給 CSS |
