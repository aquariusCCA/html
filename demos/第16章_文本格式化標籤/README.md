# HTML 文本格式化標籤對應 Demo 規劃

## 一、Notes 摘要

本章說明 HTML 文本格式化標籤的用途與語意差異，核心知識點包含 `<b>` 與 `<strong>`、`<u>` 與 `<ins>`、`<i>` 與 `<em>`、`<s>` 與 `<del>` 的比較，以及「視覺效果」不等於「HTML 語意」的觀念。

## 二、Demo 清單

| Demo | 類型 | 對應 notes 知識點 | 學習目標 | 建議路徑 |
|---|---|---|---|---|
| `01-basic-text-formatting-tags` | 單一概念 | 四組文本格式化標籤的基本效果與語意 | 觀察各標籤的預設呈現，理解外觀相近但語意不同 | `demos/第16章_文本格式化標籤/01-basic-text-formatting-tags/` |
| `02-compare-visual-and-semantic` | 對照型 | 視覺效果與 HTML 語意不是同一件事 | 比較外觀相近但語意不同的寫法 | `demos/第16章_文本格式化標籤/02-compare-visual-and-semantic/` |
| `03-common-mistake-style-as-meaning` | 常見錯誤 | 不要只為了外觀誤用語意標籤 | 觀察錯誤寫法與 CSS 修正方式 | `demos/第16章_文本格式化標籤/03-common-mistake-style-as-meaning/` |
| `04-practical-product-and-notice-text` | 實務應用 | 重要提醒、文件修訂、原價折扣等實務文字 | 在實務頁面片段中正確使用文本格式化標籤 | `demos/第16章_文本格式化標籤/04-practical-product-and-notice-text/` |
