# 基本文本格式化標籤

## 1. Demo 目標

這個 demo 用來示範：

- `<b>`、`<strong>` 的加粗效果與語意差異。
- `<u>`、`<ins>` 的底線效果與語意差異。
- `<i>`、`<em>` 的傾斜效果與語意差異。
- `<s>`、`<del>` 的刪除線效果與語意差異。

## 2. 對應 notes

## 3. 如何執行

方式一：直接打開檔案

1. 找到 `index.html`
2. 使用瀏覽器開啟
3. 觀察四組標籤的畫面呈現

方式二：使用 VS Code Live Server

1. 在 demo 資料夾上按右鍵
2. 選擇 `Open with Live Server`
3. 在瀏覽器中查看結果

## 4. 觀察重點

1. `<b>` 與 `<strong>` 預設都可能呈現粗體。
2. `<i>` 與 `<em>` 預設都可能呈現斜體。
3. 外觀相近不代表語意相同。
4. 選用標籤時應先判斷文字的內容意義。

## 5. 程式碼重點

### HTML

- 每張卡片展示一組相近標籤。
- 使用表格整理選用判斷。
- 使用 `<code>` 顯示標籤名稱。

### CSS

- CSS 只用於排版與輔助觀察。
- 沒有使用 CSS 來改變標籤本身的語意。

### JavaScript

- 此 demo 不需要 JavaScript。

## 6. 修改練習

請嘗試修改以下內容，觀察結果差異：

1. 將 `<strong>` 改成 `<b>`，觀察外觀是否明顯不同。
2. 將 `<em>` 改成 `<i>`，再重新思考語意是否正確。
3. 將 `<ins>` 改成 `<u>`，觀察畫面相近但語意變弱。
4. 替每張卡片新增一個自己的範例句。

## 7. 常見錯誤

| 錯誤 | 可能造成的問題 | 修正方式 |
|---|---|---|
| 只根據外觀選標籤 | HTML 語意可能不準確 | 先判斷內容意義，再選標籤 |
| 把 `<strong>` 當粗體工具 | 會把不重要的內容標成重要 | 純外觀使用 CSS |
| 把 `<em>` 當一般斜體工具 | 會把沒有強調語氣的文字標成強調 | 術語或外文詞可用 `<i>` |
