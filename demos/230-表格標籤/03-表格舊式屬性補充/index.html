B. 舊式屬性與 CSS 的分工

這些屬性能讀懂即可；正式專案的外觀控制應集中在 CSS。

目的舊式 HTML 屬性現代實務方向
表格對齊alignCSS 版面與 margin
邊框borderCSS border、border-collapse
儲存格內距cellpaddingCSS padding
儲存格間距cellspacingCSS border-spacing 或 border-collapse
表格寬度widthCSS width、max-width、響應式

HTML 負責資料結構與語意，CSS 負責視覺表現。border 只提供基礎邊框，無法精細控制框線顏色、雙線或合併——那要交給 CSS。
