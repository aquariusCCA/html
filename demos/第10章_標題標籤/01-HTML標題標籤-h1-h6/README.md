# 01-HTML標題標籤-h1-h6 HTML 教學範例

來源筆記：`notes/第10章_標題標籤/01-HTML標題標籤-h1-h6.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-h1到h6語意層級.html` | `<h1>` 到 `<h6>` 表示標題層級 | 數字越小，層級越高；數字越大，層級越深 | 把範例文字改成課程大綱，保留標題層級順序 |
| 2 | `02-文章標題結構.html` | 標題層級和文章結構的關係 | `<h1>` 是文章主題，`<h2>` 是主要段落，`<h3>` 是小節 | 改成旅遊指南，新增主要章節與小節 |
| 3 | `03-瀏覽器預設樣式觀察.html` | 瀏覽器會提供標題預設外觀 | 標題預設會加粗、獨佔一行，且常由大到小 | 只修改標題與正文文字，不改標籤，觀察預設外觀 |
| 4 | `04-外觀大小交給CSS.html` | HTML 負責語意，CSS 負責外觀 | 不要為了字小而把重要小節改成 `<h6>` | 修改 CSS 的 `font-size`，確認 HTML 層級仍然合理 |
| 5 | `05-不要任意跳過層級.html` | 標題層級不應任意跳級 | 從 `<h1>` 直接跳到 `<h4>` 會讓文件結構不清楚 | 新增主要章節，判斷應使用 `<h2>` 還是更深層標題 |
| 6 | `06-標題概括正文承載細節.html` | 標題用來概括內容，正文承載細節 | 一般說明文字不應整段包成標題 | 把長活動文字拆成短標題與正文 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
