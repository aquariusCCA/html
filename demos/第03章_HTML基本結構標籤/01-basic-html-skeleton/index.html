HTML 文件基本骨架

這個頁面示範一份 HTML 文件從第一行宣告到可見內容的基本組成。

本頁的基本骨架

<!doctype html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>HTML 文件基本骨架</title>
</head>
<body>
  <h1>HTML 文件基本骨架</h1>
  <p>這裡放使用者看得到的內容。</p>
</body>
</html>

各部分的角色

結構 用途 觀察方式
<!doctype html> 告訴瀏覽器用 HTML5 的方式解析文件。 檢視原始碼時應該出現在第一行。
<html lang="zh-Hant"> 整份文件的根元素，並宣告主要語言。 使用開發者工具查看最外層 html 元素。
<head> 放文件設定，例如編碼、標題與外部 CSS。 內容通常不會直接顯示在頁面主畫面。
<body> 放使用者主要看得到的頁面內容。 本頁標題、段落、表格都在 body 裡。

觀察重點

<!doctype html> 不是一般 HTML 元素，不會在畫面上顯示，也沒有結束標籤。
