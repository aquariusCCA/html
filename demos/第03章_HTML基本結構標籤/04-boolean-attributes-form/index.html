布林屬性只要出現就代表啟用

這個頁面用表單欄位觀察 disabled、checked、required 的行為。

可直接觀察的表單

disabled 出現後，欄位不能輸入或聚焦。

disabled="false" 仍然代表有 disabled 屬性，所以欄位依然停用。

checked 出現後，核取方塊預設被勾選。

required 出現後，送出前瀏覽器會檢查欄位是否填寫。

程式碼對照

<input type="text" name="studentId" disabled>
<input type="checkbox" name="newsletter" checked>
<input type="email" name="email" required>

布林屬性不是只有屬性值，像 disabled 本身就是屬性名。

取消布林狀態的方式

若要取消停用、預設勾選或必填，不是把值寫成 false，而是移除整個屬性。
