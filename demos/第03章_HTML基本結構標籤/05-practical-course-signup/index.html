Demo 05

HTML 入門課程報名

這個整合 demo 使用基本骨架、屬性、布林屬性與註解，建立一個小型報名頁。

填寫報名資料

本頁整合的結構

寫法 在本頁的用途
<!doctype html> 宣告使用 HTML5 解析方式。
lang="zh-Hant" 宣告頁面主要語言為繁體中文。
<meta charset="UTF-8"> 讓中文、標點與特殊符號能用 UTF-8 正確解析。
class="intro" 提供 CSS 選取與樣式補充資訊。
required 讓姓名、Email 與確認欄位成為必填。
<!-- 註解 --> 說明程式碼區塊用途，不會成為頁面可見內容。

註解觀察方式

畫面上看不到 HTML 註解。若要觀察註解，請使用瀏覽器的「檢視原始碼」或開發者工具。
