頁尾資訊

模擬 html5shiv 後的狀態

`html5shiv` 的核心目的，是讓 IE6、IE7、IE8 能建立並識別部分 HTML5 新元素，使它們比較能被樣式化。

目前右側區塊正在模擬舊版 IE 未識別 HTML5 元素的狀態。
