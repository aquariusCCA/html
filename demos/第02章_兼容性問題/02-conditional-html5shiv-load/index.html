Demo 02

html5shiv 條件載入對照

這個 demo 用可操作的方式比較「所有瀏覽器都載入」與「只讓低於 IE9 的 IE 載入」兩種寫法。

常見錯誤：所有瀏覽器都載入

<script src="./js/html5shiv.min.js"></script>

這種寫法會讓不需要它的現代瀏覽器也載入舊版相容腳本。

修正：低於 IE9 才載入

<!--[if lt IE 9]>
<script src="./js/html5shiv.min.js"></script>
<![endif]-->

條件註解讓符合條件的舊版 IE 才執行其中內容。現代瀏覽器會把它視為普通 HTML 註解。
