## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-html5shiv服務的語意元素問題.html` | html5shiv 服務的舊 IE 語意元素問題 | `header`、`main`、`article` 在現代瀏覽器可正常呈現；html5shiv 是舊 IE 相容方案 | 修改文章標題與內容，觀察語意元素仍保有清楚結構 |
| 2 | `02-低於IE9才載入.html` | 用 IE 條件註解限制載入 html5shiv | `lt IE 9` 表示只有低於 IE9 的 IE 才需要讀取相容性腳本 | 將程式碼文字中的 `lt IE 9` 改成 `lte IE 8`，並說出範圍差異 |
| 3 | `03-條件判斷符號對照.html` | `lt`、`lte`、`gt`、`gte`、`!` 的讀法 | 從條件符號判斷哪些 IE 版本會看到內容 | 新增一列 `gt IE 7` 的理解說明 |
| 4 | `04-限制與常見錯誤對照.html` | html5shiv 與條件註解的限制、常見錯誤 | html5shiv 不是完整 HTML5 支援工具，也不應對所有瀏覽器載入；IE10 不支援條件註解 | 把其中一組錯誤與修正改寫成自己的說法，保留「錯誤寫法」與「修正理解」標示 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
