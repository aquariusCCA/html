# Web 標準與三層分離 HTML 教學範例

來源筆記：`notes/第01章_寫在前面/06-Web-標準與三層分離.md`

## 建議學習順序

| 順序 | Demo | 對應觀念 | 觀察重點 | 建議修改任務 |
| --- | --- | --- | --- | --- |
| 1 | `01-結構表現行為分工.html` | HTML、CSS、JavaScript 的三層分工 | HTML 放內容，CSS 控制按鈕外觀，JavaScript 處理點擊後的文字變化 | 修改按鈕文字，觀察 HTML 內容變化；再修改 CSS 顏色，觀察外觀變化 |
| 2 | `02-語意與層級.html` | HTML 負責語意與層級 | `h1` 表示主要標題，`p` 表示段落；CSS 只改外觀，不取代語意 | 修改文章標題與段落內容，保持 `h1` 與 `p` 的語意角色 |
| 3 | `03-樣式分離對照.html` | 不把樣式直接塞進 HTML 結構 | 比較 `style` 屬性與 `class` 搭配 CSS 的差異 | 把錯誤寫法中的第二個樣式也改成使用 `class` |
| 4 | `04-先結構再樣式再行為.html` | 實務建立順序 | 先寫內容與 HTML 層級，再處理 CSS 外觀，最後加入必要 JavaScript | 先新增一段提醒文字，再決定是否需要樣式或點擊行為 |

## 學習提醒

這組 demo 的目標不是取代正式筆記，而是幫助你把筆記中的觀念轉成可觀察、可修改、可練習的 HTML 範例。
