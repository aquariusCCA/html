HTML5 輸入類型與表單屬性

用更精確的 type 描述資料格式，讓瀏覽器提供合適的輸入介面與基本驗證。

對應筆記：notes/240-表單標籤/05-HTML5輸入類型與表單屬性.md

A. 觀察：不同 type 帶來不同輸入介面

點點下面各個欄位，注意 date 會跳出日曆、number 有上下箭頭、color 有取色器、行動裝置上鍵盤也會不同。

<input type="email">
<input type="url">
<input type="date">
<input type="number">
<input type="tel">
<input type="search">
<input type="color">

日期、顏色等控件外觀依瀏覽器不同，版面要保留彈性。

B. 互動：required + email 的原生驗證

不填、或填一個格式不對的信箱（例如 abc），直接按提交，瀏覽器會自己擋下來並提示。

提交結果

  • 通過驗證後才會送出（這裡攔截顯示）。

提醒：原生驗證是「使用者體驗」，使用者仍可能繞過，重要資料一定要在後端再驗證一次。
