input 基礎與文字、密碼輸入

同一個 <input>，靠 type 變成不同控件；value 是「值」，placeholder 只是「提示」。

對應筆記：notes/240-表單標籤/02-input基礎與文字密碼輸入.md

A. type 決定輸入型態

三個都是 <input>，只因 type 不同就變成完全不同的控件。試著各輸入一次看看。

<input type="text">
<input type="password">
<input type="file">
<!-- 一次選多個檔案 -->
<input type="file" multiple>

密碼框只是「畫面遮蔽」，不代表資料已加密；真正安全要靠 HTTPS 與後端。

B. 錯誤修正：value 當提示 vs placeholder 當提示

在兩個框「不輸入任何字」直接看下方的「目前真正的值」，差別立刻出現。

C. 互動：密碼框只是遮蔽畫面

輸入一段文字後，點按鈕切換 type，會發現「值沒變，只是顯示方式變了」。
