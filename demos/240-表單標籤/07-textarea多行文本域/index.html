textarea 多行文本域

多行文字用 <textarea>；它沒有 type，初始內容也不是寫在 value。

對應筆記：notes/240-表單標籤/07-textarea多行文本域.md

A. 對照：單行 input vs 多行 textarea

在兩個欄位都試著輸入多行留言（按 Enter 換行），體會差異。

<input type="text" name="single">

<textarea name="multi" rows="5">
</textarea>

留言、評論、地址這類可能超過一行的內容，用 textarea。

B. 錯誤修正：初始內容寫在哪？

兩個 textarea 都「想要」預設文字「這是預設文字」，但只有正確寫法真的出現了文字。

C. textarea 沒有 type 屬性

需要單行用 <input type="text">；需要多行用 <textarea>。正式版面的尺寸通常交給 CSS，而非只靠 rows/cols。
