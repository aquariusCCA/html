Demo 04

實務應用：卡片容器與狀態文字

這個 demo 使用 <div> 包住卡片區塊， 使用 <span> 標記狀態、價格與可被 JavaScript 更新的數字。

課程卡片

狀態： 開放報名

HTML 盒子標籤

學習如何選擇 `div` 與 `span`，並理解無語意容器的使用限制。

價格： NT$ 990

目前人數： 0

這裡每個元素的角色

  • div.course-card：包住整張卡片。
  • span.status：標記段落中的狀態文字。
  • span.price：標記段落中的價格文字。
  • span#count：讓 JavaScript 精準更新數字。
  • button：表示真正可以操作的按鈕。

關鍵程式碼

<p>
  目前人數：
  <span id="count">0</span>
</p>

這裡只把需要更新的數字包進 span，JavaScript 就能精準改變局部內容。
