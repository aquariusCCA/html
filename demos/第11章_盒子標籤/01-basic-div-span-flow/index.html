觀察 div 與 span 的預設排列

這個 demo 讓你直接觀察 <div> 通常會獨佔一行， 而 <span> 通常可以和其他行內內容排在同一行。

`div`：區塊型容器

第一個 div：我通常會形成一個獨立區塊。
第二個 div：我會排在下一行。
第三個 div：即使文字很短，也通常會佔一整行。

觀察重點：多個 `div` 會由上到下排列。

`span`：行內型容器

HTML CSS JavaScript DOM

觀察重點：多個 `span` 會跟文字一樣盡量排在同一行。
