# 第十六章｜預格式化文本標籤示例

這個資料夾收錄第十六章筆記對應的可執行示例頁。  
這章的重點是理解 `<pre>` 用來保留文字原本的空格、縮排與換行，不是單純讓文字看起來像程式碼。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十六章 Markdown 筆記。
- 想直接看 `<pre>` 的使用情境時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [pre-tag-demo.html](./pre-tag-demo.html)：整合展示 `<pre>` 與一般段落的差異、常見使用場景、和 `<code>` 的搭配方式，以及不建議的濫用方式。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
