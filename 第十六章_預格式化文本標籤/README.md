# 第十六章｜預格式化文本標籤

> 章節定位：建立 HTML 保留文字原始排版格式的基本判讀框架  
> 建議閱讀方式：先分清 `<pre>` 是保留文本格式，不只是等寬字體，再回頭看它的常見使用場景

## 本章在學什麼

這一章整理 HTML 的預格式化文本標籤 `<pre>`。  
重點不是只記住它看起來像程式碼區塊，而是理解它在保留空格、換行與縮排上的作用。

## 本章與整體地圖的關係

- 前置知識：前面幾章已整理段落、換行與水平線等基礎文本結構。
- 這章的核心任務：分清一般文字排版和預格式化文字顯示的差別。
- 讀完後通常接：之後在展示程式碼、命令輸出或格式敏感文字時，可以回來快速查 `<pre>` 的使用時機。

## 易混淆主題

- 保留格式 vs 等寬字體：前者是核心用途，後者是常見顯示結果。
- `<pre>` vs `<p>`：前者保留原始排版，後者偏一般段落閱讀。
