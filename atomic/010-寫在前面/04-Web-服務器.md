# Web 服務器

> 來源：origin/010-寫在前面/03-Web 服務器.md / 全文

我們目前寫的項目都是放在自己的電腦上，只能自己訪問。如果想要很多人訪問我們的網站，可以把項目放到服務器上，這樣就可以讓很多人訪問我們的網站了。

![多位使用者連線至服務器示意圖](../../origin/010-寫在前面/assets/images/web-server-img-001-7d7f51.png)

服務器是提供計算服務的設備，它也是一台計算機。在網路環境下，根據服務器提供的服務類型不同，服務器又分為文件服務器、數據庫服務器、應用程序服務器、Web 服務器等。

服務器我們也稱為主機。

Web 服務器可以指提供 HTTP/HTTPS 服務的軟體，也可以指承載這類服務的主機；它會向瀏覽器等 Web 客戶端提供網頁文件、圖片、資料或下載資源。

## 本地服務器與遠程服務器

以下服務器主要指的是 Web 服務器。根據服務器在網路中所在的位置不同，可以分為以下兩種：

- 本地服務器：我們可以把自己的電腦設置為本地服務器，這樣同一個局域網的同學就可以訪問你的項目了。
- 遠程服務器：通常是別的公司為我們提供的一台電腦（主機），網站項目上傳後，還需要完成網路、DNS、權限或防火牆等設定，目標使用者才可以透過域名或 IP 訪問網站。比如：www.baidu.com。
