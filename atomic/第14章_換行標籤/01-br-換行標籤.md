# br 換行標籤

> 來源：origin/第14章_換行標籤/01-換行標籤.md / 全文

`<br>` 的標籤語意是強制換行。

在 HTML 中，文字會依書寫方向在可用行寬內排列，空間不足時瀏覽器會自動換行。當換行本身具有內容意義時，可以使用換行標籤 `<br>` 強制換行；不要用 `<br>` 取代段落或控制版面間距。

## 使用場景

- 當換行本身具有內容意義時，讓文字強制換行顯示。

## 名稱來源

- `br` 是英文單詞 `break` 的縮寫，意思是打斷、換行。

## 特點

- `<br>` 是 HTML void element，不需要結束標籤；`<br />` 在 HTML 中也常見，但斜線不是必要語法。
- `<br>` 標籤只是簡單地開始新的一行，跟段落不一樣；段落之間會插入一些垂直間距。

```html
第一行文本<br>第二行文本
```
