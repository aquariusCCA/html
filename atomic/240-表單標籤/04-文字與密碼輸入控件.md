# 文字與密碼輸入控件

## 文本框

```html
<input type="text">
```

<aside>

**常用屬性：**

- `name` 屬性：數據的名稱。
- `value` 屬性：輸入框的默認輸入值。
- `maxlength` 屬性：輸入框最大可輸入長度。

</aside>

## 密碼框

```html
<input type="password">
```

<aside>

**常用屬性如下：**

- `name` 屬性：數據的名稱。
- `value` 屬性：可設定默認輸入值，但密碼框通常不建議預填，避免安全與使用體驗問題。
- `maxlength` 屬性：輸入框最大可輸入長度。

</aside>
