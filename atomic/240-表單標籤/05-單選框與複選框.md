# 單選框與複選框

> 來源：origin/240-表單標籤/03-表單控件.md / `### 單選框`、`### 複選框`

## 單選框

```html
<input type="radio" name="sex" value="female"> 女
<input type="radio" name="sex" value="male"> 男
```

<aside>

**常用屬性如下：**

- `name` 屬性：數據的名稱。
  - 注意想要單選效果，多個 `radio` 的 `name` 屬性值需要保持一致。
- `value` 屬性：提交的數據值。
- `checked` 屬性：讓該單選按鈕默認選中。

</aside>

## 複選框

```html
<input type="checkbox" name="hobby" value="smoke"> 抽菸
<input type="checkbox" name="hobby" value="drink"> 喝酒
<input type="checkbox" name="hobby" value="perm"> 燙頭髮
```

<aside>

**常用屬性如下：**

- `name` 屬性：數據的名稱。
- `value` 屬性：提交的數據值。
- `checked` 屬性：讓該複選框默認選中。

</aside>
