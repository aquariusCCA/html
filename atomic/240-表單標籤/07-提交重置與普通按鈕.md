# 提交、重置與普通按鈕

## 提交按鈕

```html
<input type="submit" value="點我提交表單">
<button type="submit">點我提交表單</button>
```

注意點：

- `button` 標籤未指定 `type` 時，在表單中通常會作為提交按鈕。
- `input` 標籤編寫的按鈕，使用 `value` 屬性指定按鈕文字。

## 重置按鈕

```html
<input type="reset" value="點我重置">
<button type="reset">點我重置</button>
```

注意點：

- `input` 標籤編寫的按鈕，使用 `value` 屬性指定按鈕文字。

## 普通按鈕

普通按鈕的 `type` 值為 `button`。如果 `<button>` 不寫 `type`，在表單中可能會引起表單提交。

```html
<input type="button" value="普通按鈕">
<button type="button">普通按鈕</button>
```

## button 標籤的 type 類型

- `submit`：提交按鈕。
- `reset`：重置按鈕。
- `button`：普通按鈕。

```html
<button type="submit">提交按鈕</button>
<button type="reset">重置按鈕</button>
<button type="button">普通按鈕</button>
```
