# input 文字框與密碼框

> 來源：origin/測試/03-表單控件.md / ### 文本框、### 密碼框

## 文本框

```html
<input type="text">
```

常用屬性：

- `name` 屬性：數據的名稱。
- `value` 屬性：輸入框的默認輸入值。
- `maxlength` 屬性：輸入框最大可輸入長度。

## 密碼框

```html
<input type="password">
```

常用屬性：

- `name` 屬性：數據的名稱。
- `value` 屬性：輸入框的默認輸入值，一般不用，無意義。
- `maxlength` 屬性：輸入框最大可輸入長度。
