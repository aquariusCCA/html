# form 表單域與提交屬性

> 來源：origin/測試/02-表單域.md / # 表單域

> 表單域是一個包含表單元素的區域。

- 在 HTML 標籤中，`<form>` 標籤用於定義表單域，以實現用戶信息的收集和傳遞。
- `<form>` 會把它範圍內的表單元素信息提交給服務器。
- 表單域常用屬性如下：

![form 標籤 action、method、name 屬性說明表](../../origin/測試/assets/images/form-field-img-001-dedcdd.png)

```html
<form action="url地址" method="提交方式" name="表单域的名称" target="如何打開頁面">
  <!-- 省略... -->
</form>
```
