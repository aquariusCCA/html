# HTML 標題標籤 h1 到 h6

> 標籤語意：作為標題使用，並且依據重要性遞減。

場景：在新聞和文章的頁面中，都離不開標題，用來凸顯文章主題。

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
```

<aside>
👉

**特點：**

- 在瀏覽器預設樣式下，文字通常會加粗。
- 瀏覽器預設樣式通常會讓 h1 最大、h6 最小，從 h1 到 h6 字號逐漸變小。
- 標題元素預設通常會以區塊方式呈現，因此會獨佔一行；實際外觀可由 CSS 調整。
</aside>
