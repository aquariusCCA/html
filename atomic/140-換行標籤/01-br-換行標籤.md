# br 換行標籤

> 標籤語意: 強制換行。

在一般水平書寫情境下，文字會在所在容器的可用寬度內排列；到達行尾時，瀏覽器會依排版規則自動換行。如果希望某段文本在指定位置強制換行顯示，就可以使用換行標籤 `<br>`。

- 場景 : 讓文字強制換行顯示。
- 單詞 break 的縮寫，意為打斷、換行。
- 特點 :
    - `<br>` 是 HTML 的空元素（void element），只需要開始標籤，不需要結束標籤；`<br />` 也能被瀏覽器解析，但斜線在 HTML 中沒有額外作用。
    - `<br>` 標籤只會讓內容從新的一行繼續；`<p>` 表示段落，瀏覽器預設樣式通常會讓段落上下有 margin，因此看起來會有垂直間距。
    - `<br>` 適合用在地址、詩句等換行本身有意義的文本；如果只是要調整段落或區塊間距，應使用段落元素或 CSS 的 margin、padding，不建議連續使用 `<br>` 排版。

```html
第一行文本 <br>第二行文本
```
