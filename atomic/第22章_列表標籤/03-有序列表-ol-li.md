# 有序列表與 `ol`、`li` 標籤

> 來源：origin/第22章_列表標籤/01-列表標籤.md / # 3. 有序列表

> 有序列表即為有排列順序的列表，其各個列表會按照一定的順序排列定義。

```html
<h4>粉丝排行榜</h4>
<ol>
    <li>刘德华 10000</li>
    <li>刘若英 1000</li>
    <li>pink老师 1</li>
</ol>
```

- 在 HTML 標籤中，`<ol>` 標籤用於定義有序列表，列表預設通常以數字標記，也可透過屬性或 CSS 調整標記樣式，並且使用 `<li>` 標籤來定義列表項。
- `<ol></ol>` 中只能嵌套 `<li></li>`，直接在 `<ol></ol>` 標籤中輸入其他標籤或者文字的作法是不被允許的。
- `<li>` 與 `</li>` 之間相當於一個容器，可以容納文字與多數常見元素，但仍需符合 HTML 內容模型。
- 瀏覽器會替有序列表套用預設樣式，例如編號、縮排與間距；實際使用時，我們會使用 CSS 來設置。
