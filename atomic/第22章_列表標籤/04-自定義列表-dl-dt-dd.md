# 自定義列表與 `dl`、`dt`、`dd` 標籤

> 來源：origin/第22章_列表標籤/01-列表標籤.md / # 4. 自定義列表

> 使用場景：常用於對術語或名詞進行解釋和描述，定義列表的列表項前沒有任何項目符號。

- 在 HTML 標籤中，`<dl>` 標籤用於定義描述列表，該標籤會與 `<dt>` 和 `<dd>` 一起使用。
  - `<dl>`：定義列表。
  - `<dt>`：定義項目 / 名字。
  - `<dd>`：描述每一個項目 / 名字。

```html
<dl>
  <dt>关注我们</dt>
  <dd>新浪微博</dd>
  <dd>官方微信</dd>
  <dd>联系我们</dd>

  <dt>关注我们</dt>
  <dd>新浪微博</dd>
  <dd>官方微信</dd>
  <dd>联系我们</dd>
</dl>
```

- `<dl></dl>` 裡面只能包含 `<dt>` 和 `<dd>`。
- `<dt>` 和 `<dd>` 裡面可以放文字與多數常見元素，但仍需符合 HTML 內容模型。
- `<dt>` 和 `<dd>` 個數沒有限制，經常是一個 `<dt>` 對應多個 `<dd>`。
