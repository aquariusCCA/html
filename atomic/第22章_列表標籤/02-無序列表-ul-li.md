# 無序列表與 `ul`、`li` 標籤

> 來源：origin/第22章_列表標籤/01-列表標籤.md / # 2. 無序列表

> `<ul>` 標籤表示 HTML 頁面中項目的無序列表，一般會以項目符號呈現列表項，而列表項使用 `<li>` 標籤定義。

```html
<h4>您喜歡的食物?</h4>
<ul>
    <li>榴槤</li>
    <li>臭豆腐</li>
    <li>鲱鱼罐头</li>
    <li>
        <p>123</p>
    </li>
</ul>
```

- 無序列表的各個列表之間沒有順序級別之分，是並列的。
- `<ul></ul>` 中只能嵌套 `<li></li>`，直接在 `<ul></ul>` 標籤中輸入其他標籤或者文字的做法是不被允許的。
- `<li>` 與 `</li>` 之間相當於一個容器，可以容納文字與多數常見元素，但仍需符合 HTML 內容模型。
- 瀏覽器會替無序列表套用預設樣式，例如項目符號、縮排與間距；實際使用時，我們會使用 CSS 來設置。
