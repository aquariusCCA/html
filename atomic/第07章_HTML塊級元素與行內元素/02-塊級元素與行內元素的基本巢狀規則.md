## 塊級元素中常見可寫行內元素和塊級元素

像 `div` 這類通用容器可以包含多種行內元素和塊級元素；其他元素仍需依各自的內容模型判斷。

```html
<!-- 規則一: div 這類通用容器中可寫行內元素、塊級元素 -->
<div>
  <span>尚硅谷1</span>
  <input type="text">
  <div>尚硅谷2</div>
</div>
```

## 行內元素需依內容模型判斷可放內容

行內元素能否放入其他元素，需依各自的內容模型判斷。例如 `span` 可包含 phrasing content，但 `img`、`input`、`br` 這類 void elements 不能有子內容。

```html
<!-- 規則二: span 可包含其他 phrasing content -->
<span>
  <span>尚硅谷1</span>
  <input type="text">
  <span>尚硅谷2</span>
</span>
```
