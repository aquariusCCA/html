# p 標籤不能包含塊級元素

> 來源：origin/第07章_HTML塊級元素與行內元素/01-HTML塊級元素與行內元素.md / # **6. 特殊規則 → p標籤中不能寫塊元素**

`p` 標籤中不能寫塊級元素。

```html
<!-- 特殊規則 -> p 標籤中不能寫塊元素 -->
<p>
  <div>尚硅谷</div>
</p>
```
