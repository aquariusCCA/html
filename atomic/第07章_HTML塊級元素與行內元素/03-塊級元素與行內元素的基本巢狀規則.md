## 塊級元素中能寫行內元素、塊級元素

```html
<!-- 規則一: 塊級元素中能寫行內元素、塊級元素(幾乎什麼都能寫) -->
<div>
  <span>尚硅谷1</span>
  <input type="text">
  <div>尚硅谷2</div>
</div>
```

## 行內元素中能寫行內元素，但不能寫塊級元素

```html
<!-- 規則二: 行內元素中能寫行內元素，但不能寫塊級元素 -->
<span>
    <span>尚硅谷1</span>
    <input type="text">
    <span>尚硅谷2</span>
</span>
```
