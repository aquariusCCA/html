## 塊級元素獨佔一行

```html
<body>
  <!-- 塊級元素特點: 獨佔一行 -->
  <h1>尚硅谷</h1>
  <p>尚硅谷</p>
  <div>尚硅谷</div>
</body>
```

## 行內元素不獨佔一行

```html
<!-- 行內元素特點 : 不獨佔一行 -->
<input type="text">
<span>尚硅谷</span>
```
