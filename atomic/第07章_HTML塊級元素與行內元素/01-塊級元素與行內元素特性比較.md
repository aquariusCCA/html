## 塊級元素

塊級元素，顧名思義，該元素呈現塊狀，所以它有自己的寬度和高度，也就是可自訂 `width` 和 `height`。除此之外，塊級元素會獨自佔據一行高度（`float` 浮動除外），一般可以作為其他容器使用，可容納塊級元素和行內元素。

塊級元素有以下特點：

- 每個塊級元素都是獨自佔一行。
- 高度、行高、外邊距（`margin`）以及內邊距（`padding`）都可以控制。
- 元素的寬度如果不設定，通常會佔滿父元素中的可用寬度；這不等同於固定設定 `width: 100%`。
- 多個塊級元素標籤寫在一起，預設排列方式為從上至下。
- 區塊元素常見包括 `div`、`p`、`h1` ~ `h6`、`ul`、`ol`、`li`、`dl`、`dt`、`dd`、`form`、`table`、`hr`、`blockquote`、`address`、`menu`、`pre` 等等。

## 行內元素

一般非置換行內元素設定寬（`width`）和高（`height`）通常不會生效，但可以與其他行內元素位於同一行。行內元素能否包含其他元素，應依該元素的內容模型判斷，不能只用「行內 / 塊級」分類決定。行內元素的高度多由文字大小與 `line-height` 影響，寬度通常由內容長度控制；`img`、`input` 等置換元素或改變 `display` 時是例外。

行內元素有以下特點：

- 不會獨佔一行，相鄰的行內元素會排列在同一行裡，直到一行排不下才會自動換行，其寬度隨元素的內容而變化。
- 一般非置換行內元素的高寬設定通常不生效；左右方向的外邊距（`margin`）和內邊距（`padding`）會影響排列，上下方向的版面效果較有限。
- 設定行高（`line-height`）有效，會影響該元素參與的行盒高度；若要整段文字一致，通常設定在父元素讓子元素繼承。
- 一般非置換行內元素的寬度通常由內容決定；置換元素或改變 `display` 時可有不同表現。
- 行內元素能否包含塊級內容需依內容模型判斷；`a` 鏈接裡面不能再放鏈接。
- 行內元素常見包括 `span`、`em`、`i`、`b`、`strong`、`a`、`img`、`input`、`br`、`select`、`textarea`、`q`、`bdo`、`sub`、`sup` 等等。

## 塊級元素獨佔一行

```html
<body>
  <!-- 塊級元素特點: 獨佔一行 -->
  <h1>尚硅谷</h1>
  <p>尚硅谷</p>
  <div>尚硅谷</div>
</body>
```

## 行內元素不獨佔一行

```html
<!-- 行內元素特點: 不獨佔一行 -->
<input type="text">
<span>尚硅谷</span>
```
