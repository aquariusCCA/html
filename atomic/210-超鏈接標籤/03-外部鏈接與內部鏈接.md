# 外部鏈接與內部鏈接

## 外部鏈接

```html
<h4>外部鏈接</h4>
<a href="http://www.qq.com" target="_self"> 騰訊</a>
<a href="http://www.itcast.cn" target="_blank">傳智播客</a>
```

## 內部鏈接

> 網站內部頁面之間的相互鏈接，直接鏈接內部頁面名稱即可。

```html
<body>
  <h4>內部鏈接</h4>
  <a href="./assets/files/hyperlink-tags-file-001-b99c80.html" target="_blank">公司簡介</a>
</body>
```
