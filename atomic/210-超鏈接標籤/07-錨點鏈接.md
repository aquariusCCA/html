# 錨點鏈接

> 來源：origin/210-超鏈接標籤/01-超鏈接標籤.md / ## 3.9 錨點鏈接(實用)

作用: 點擊鏈接，可以快速定位到頁面中的某個位置。

使用步驟:

- 在鏈接文本的 `href` 屬性中，設置屬性值為 `#名字` 的形式。
- 找到目標位置標籤，裡面添加一個 `id 屬性=剛才的名字`。

```html
<!-- 跳轉到本頁的錨點 -->
<a href="#two">第二季</a>
<h3 id="two">第二季介绍</h3>

<!-- 跳轉到其他頁面的錨點 -->
<a href="./assets/files/hyperlink-tags-file-001-b99c80.html#test1">跳轉到測試頁面的 test1 錨點</a><br />
```
