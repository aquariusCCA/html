# a 標籤基本語法

> 來源：origin/第21章_超鏈接標籤/01-超鏈接標籤.md / # 1. 鏈接標籤的介紹

主要作用: 從當前頁面進行跳轉。

- 單詞 anchor 的縮寫，意為 **錨**。

```html
<a href="目标地址">文字内容</a>

<!-- 舉例: -->
<a href="https://www.baidu.com/">百度</a>
```

- 💡 雖然 a 是行內元素，但是 a 元素可以包裹多數非互動內容，例如文字、圖片或區塊元素；但不能巢狀另一個 a，也應避免包裹 button、input 等互動元素。

```html
<a href="https://example.com">
    <div>這是一個包含在 a 標籤中的區塊元素</div>
</a>
```
