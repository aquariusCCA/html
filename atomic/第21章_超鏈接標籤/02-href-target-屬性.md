# href 與 target 屬性

> 來源：origin/第21章_超鏈接標籤/01-超鏈接標籤.md / # 2. 屬性介紹

![超鏈接標籤 href 與 target 屬性說明表](../../origin/第21章_超鏈接標籤/assets/images/hyperlink-tags-img-001-910f05.png)

- **href**: 用於指定鏈接目標的 url 地址；若要建立可點擊並可導航的超鏈接，通常需要設定 href。沒有 href 的 a 元素不會形成一般超鏈接。
- **target**: 用於指定連接頁面的打開方式。
    - **_self**: 為默認值，在本頁簽中打開。
    - **_blank**: 為在新頁簽中打開。
