# meta 文件資訊與版權設定

> 來源：origin/第08章_meta標籤/01-meta標籤.md / Notes：配置網頁作者、配置網頁生成工具、配置定義網頁版權信息

這組 meta 標籤用來描述網頁作者、網頁生成工具與版權資訊。

## 配置網頁作者

```html
<!-- 配置網頁作者  -->
<meta name="author" content="kevin">
```

## 配置網頁生成工具

```html
<!-- 配置網頁生成工具   -->
<meta name="generator" content="Visual Studio Code">
```

## 配置定義網頁版權信息

```html
<!-- 配置定義網頁版權信息  -->
<meta name="copyright" content="2023-2027版權所有">
```
