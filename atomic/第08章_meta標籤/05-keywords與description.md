# keywords 與 description meta

> 來源：origin/第08章_meta標籤/01-meta標籤.md / ✏️ Notes：關鍵字和描述訊息

```html
<!-- 可以解讀為 "你要描述（meta）的內容名稱（name），內容（content）是什麼"。 -->
<meta name="keywords" content="關鍵詞1, 關鍵詞2, 關鍵詞3">
<meta name="description" content="這是一個描述內容">
```

> 備註：`keywords` 可作頁面相關詞或站內用途記錄，但不要視為 Google 搜尋排名或索引手段。
