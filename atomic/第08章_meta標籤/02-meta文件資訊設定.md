# meta 文件資訊設定

> 來源：origin/第08章_meta標籤/01-meta標籤.md / ✏️ Notes：配置網頁作者、配置網頁生成工具、配置定義網頁版權信息

## 配置網頁作者

```html
<!-- 配置網頁作者  -->
<meta name="author" content="kevin">
```

## 配置網頁生成工具

```html
<!-- 若頁面由工具或 CMS 自動生成，可標示生成工具；手寫頁面通常省略。 -->
<meta name="generator" content="Frontweaver 8.2">
```

## 非標準版權資訊記錄

```html
<!-- 非標準的自訂版權資訊，通常只作記錄，不等於正式授權聲明。 -->
<meta name="copyright" content="2023-2027版權所有">
```

> 備註：這不是 HTML 標準 metadata name；正式版權或授權資訊應另以頁面文字、法律頁或授權連結表達。
