# meta 基礎環境設定

> 來源：origin/第08章_meta標籤/01-meta標籤.md / ✏️ Notes：配置字符編碼、IE 瀏覽器兼容性設置、移動端配置

## 配置字符編碼

```html
<!-- 配置字符編碼 -->
<meta charset="UTF-8">
```

## IE 瀏覽器兼容性設置

```html
<!-- 針對舊版 IE 的兼容性設置：要求 IE 使用可用的最高文檔模式；現代瀏覽器通常會忽略。 -->
<meta http-equiv="X-UA-Compatible" content="IE=edge">
```

## 移動端配置

```html
<!-- 針對移動端的一個配置 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
