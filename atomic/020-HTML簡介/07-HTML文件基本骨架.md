# HTML 文件基本骨架

- 網頁類似於一篇文章。
    - 每一頁文章內容是有固定結構的，例如開頭、正文、落款等。
    - 網頁中也存在固定結構，例如整體、頭部、標題、主體等。
- 網頁中的固定結構是通過特定的 **HTML 標籤** 進行描述的。
- 每一個網頁都會有一個基本的結構標籤，也稱為骨架標籤，頁面內容也是在這些基本標籤上書寫。

![HTML 文件骨架程式碼示例](../../origin/020-HTML簡介/assets/images/html-skeleton-img-001-8bc9f3.jpg)

> 若示範完整 HTML5 文件，開頭應包含 `<!DOCTYPE html>`；此圖主要示意 `html`、`head`、`title`、`body` 等骨架標籤。

![HTML 骨架標籤說明表格](../../origin/020-HTML簡介/assets/images/html-skeleton-img-002-26bafc.jpg)

- HTML 頁面也稱為 HTML 文檔。
    - 常見 HTML 文件副檔名是 `.html` 或 `.htm`。
    - 在伺服器環境中，瀏覽器主要依回應類型（如 `text/html`）來判斷如何處理文件。
    - 瀏覽器的作用是讀取 HTML 文檔，並以網頁的形式顯示出它們。
