# HTML5 版本定位與新增特性

> 來源：origin/020-HTML簡介/02-HTML版本的區別.md / > 💡、# **HTML5版本** > ### **簡介**

> 💡 瀏覽器通常會為了兼容舊網頁保留許多舊語法的解析行為，但 HTML5 或現行 HTML 標準中有些舊特性已不建議或不允許使用；撰寫新頁面時應以現行標準建議的元素與屬性為主。

HTML5 是新一代的 HTML 標準，2014 年 10 月由萬維網聯盟（W3C）完成標準制定。

`HTML5` 的新增特性主要是針對以前的不足，增加了一些 **新的標籤**、**新的表單** 和新的 **表單屬性** 等。

`HTML5` 有更大的技術集，允許更多樣化和強大的網站和應用程序。新增特性包含：

- 語義特性
- 本地存儲特性
- 設備兼容特性
- 連接特性
- 網頁多媒體特性
- 三維、圖形及特效特性
- 性能與集成特性
- `CSS3` 特性

狹義的 HTML5 是指新一代的 HTML 標準；廣義的 HTML5 是 HTML5 本身加上 CSS3 與 JavaScript，也可以理解為整個前端。這個集合有時稱為 HTML5 和朋友，通常縮寫為 HTML5。
