# 字符集

> 來源：origin/第04章_HTML基本結構標籤/01-HTML基本結構標籤.md / # **3. 字符集**

在 `<head>` 標籤內，可以通過 `<meta>` 標籤的 `charset` 屬性來規定 HTML 文檔應該使用哪種字符編碼。

```html
<head>
  <meta charset="UTF-8">
</head>
```

> charset 常用的值有 : GB2312 、BIG5 、GBK 和 UTF-8。Unicode 常被稱為萬國碼；UTF-8 是 Unicode 的一種常用編碼方式，能表示 Unicode 字符集中的大量字符，網頁通常建議使用 UTF-8。
