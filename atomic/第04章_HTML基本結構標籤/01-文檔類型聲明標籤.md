# 文檔類型聲明

> 來源：origin/第04章_HTML基本結構標籤/01-HTML基本結構標籤.md / # **1. 文檔類型聲明標籤**

- 文檔類型聲明的作用，是告訴瀏覽器以標準模式解析 HTML 文件；HTML5 使用 `<!DOCTYPE html>`。

```html
<!-- 這句代碼的意思是 : 當前頁面使用 HTML5 的文檔類型聲明，並讓瀏覽器以標準模式解析。 -->
<!DOCTYPE html>
```

- **注意點 :**
  - `<!DOCTYPE>` 聲明位於文檔中的最前面的位置，處於 `<html>` 標籤之前。
  - `<!DOCTYPE>` 不是 HTML 標籤，而是文檔類型聲明。
