# HTML5 兼容性與使用優勢

> 來源：origin/第02章_HTML簡介/02-HTML版本的區別.md / # **HTML5版本** > ### **兼容性**、### **使用**

## 兼容性

- HTML5 的新特性都有兼容性問題，基本都是 `IE9+` 以上版本的瀏覽器才支持。
- Chrome、Safari、Opera、Firefox 等主流瀏覽器支持 HTML5。
- IE 瀏覽器必須是 9 及以上版本才支持 HTML5，且 IE9 僅支持部分 HTML5 新特性。
- 如果不考慮兼容性問題，可以大量使用這些新特性。

## 使用方向

- HTML5 新特性增加了很多，但開發時可以專注於常用的新特性。

## HTML5 優勢

- 針對 JavaScript 新增了很多可操作的接口。
- 新增了一些語意化標籤和全局屬性。
- 新增了多媒體標籤，可以很好的替代 Flash。
- 更側重語意化，對於 SEO 更友好。
- 可移植性好，可以大量應用在移動設備上。
