# HTML5 兼容性與使用優勢

> 來源：origin/第02章_HTML簡介/02-HTML版本的區別.md / # **HTML5版本** > ### **兼容性**、### **使用**

## 兼容性

- HTML5 與相關 Web 平台特性的支援度需要依功能與瀏覽器版本確認。
- Chrome、Safari、Opera、Firefox 等現代主流瀏覽器大多支援常用 HTML5 功能。
- 如果需要兼容舊版 IE，應逐項查詢支援度，必要時準備 fallback。
- 如果專案不需要兼容舊瀏覽器，可以更積極地使用已被目標瀏覽器支援的新特性。

## 使用方向

- HTML5 新特性增加了很多，但開發時可以專注於常用的新特性。

## HTML5 優勢

- 針對 JavaScript 新增了很多可操作的接口。
- 新增了一些語意化標籤和全局屬性。
- 新增了多媒體標籤，可以很好的替代 Flash。
- 更側重語意化，對於 SEO 更友好。
- 可移植性好，可以大量應用在移動設備上。
