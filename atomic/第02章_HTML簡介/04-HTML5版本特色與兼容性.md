# HTML5 版本特色與兼容性

> 產品的版本都是向下兼容的，所以可以在 HTML5 的標準下使用之前版本的標籤和功能。

## 簡介

HTML5 是新一代的 HTML 標準，2014 年 10 月由萬維網聯盟（W3C）完成標準制定。

`HTML5` 的新增特性主要是針對於以前的不足，增加了一些 **新的標籤**、**新的表單** 和新的 **表單屬性** 等。

- `HTML5` 這個詞有時也被廣義地用來泛稱一組現代 Web 技術，允許更多樣化和強大的網站和應用程序。
    - 這類技術可能包含語義特性、本地存儲特性、設備兼容特性、連接特性、網頁多媒體特性、三維、圖形及特效特性、性能與集成特性、`CSS3` 特性等。
    - 這個集合有時稱為 `HTML5` 和朋友，通常縮寫為 `HTML5`。
- 狹義的 `HTML5` 指新一代的 HTML 標準；廣義的 `HTML5` 是教學或市場語境中對現代前端技術集合的非正式稱呼，不等於標準意義上的 HTML 本身。

## 兼容性

- 不同 HTML5 或現代 Web 新特性的支援狀況不同，需要依具體功能查詢瀏覽器兼容性。
- 支持：Chrome、Safari、Opera、Firefox 等主流瀏覽器。
- IE9 僅支持部分早期 HTML5 新特性，IE 瀏覽器不適合作為現代兼容性基準。
- 如果不考慮兼容性問題，可以大量使用這些新特性。

## 使用

- 新特性增加了很多，但是原始筆記提醒：專注於開發常用的新特性。
- HTML5 優勢：
    - 針對 JavaScript 新增了很多可操作的接口。
    - 新增了一些語意化標籤、全局屬性。
    - 新增了多媒體標籤，可以很好的替代 flash。
    - 更側重語意化，對於 SEO 更友好。
    - 可移植性好，可以大量應用在移動設備上。
