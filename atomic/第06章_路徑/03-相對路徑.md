# 相對路徑

> 來源：origin/第06章_路徑/01-路徑.md / # **3. 相對路徑**

## 定義

相對路徑是從代碼所在的這個文件出發，去尋找目標文件的。這裡所說的上一級、下一級和同一級，就是圖片相對於 HTML 頁面的位置。

相對路徑是以引用文件所在位置為參考基礎，而建立出的目錄路徑。

![相對路徑分類與符號說明表](../../origin/第06章_路徑/assets/images/path-img-002-4c2e46.png)

下一級相對路徑通常寫成「資料夾名/檔名」，例如 `images/1.png`；不要把開頭的 `/` 當成下一級相對路徑，因為 `/` 開頭通常表示從網站根目錄開始。

## 注意點

- 相對路徑開頭表示目前目錄的 `./` 通常可以省略，例如 `./images/1.png` 可寫成 `images/1.png`。
- 相對路徑依賴的是當前位置，後期若調整了文件位置，那麼文件中的路徑也要修改。
