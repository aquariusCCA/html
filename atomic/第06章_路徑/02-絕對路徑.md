# 絕對路徑

> 來源：origin/第06章_路徑/01-路徑.md / # **2. 絕對路徑**

## 定義

絕對路徑指目錄下的絕對位置，可直接到達目標位置。它是以根位置作為參考點，去建立路徑。

## 例子

- 盤符開頭：`D:\day01\images\1.jpg`
- 完整的網路地址：`https://www.itcast.cn/2018czgw/images/logo.gif`

其中 `D:\day01\images\1.jpg` 是 Windows 本機檔案系統的絕對路徑，不適合作為網站部署時的資源路徑；網頁資源路徑通常使用 `/` 或完整 URL。

## 注意點

- 使用本地絕對路徑時，一旦更換設備，路徑處理起來比較麻煩，所以很少使用。
- 使用網路絕對路徑確實方便，但要注意若服務器開啟了防盜鏈，會造成圖片引入失敗。
  - 防盜鏈：透過瀏覽器去看是正常的，但是如果在自己開發的網站中使用別人的圖片，就會顯示未經允許不可引用的字樣。
