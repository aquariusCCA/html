# HTML 字符實體基本概念

> 來源：origin/第26章_字符實體/01-字符實體.md / 全文

> 重點記住：空格（`&nbsp;`）、小於號（`&lt;`）、大於號（`&gt;`）、和號 `&`（`&amp;`）這幾個常見字符實體，其餘的如果需要回頭查閱即可。

在 HTML 中，我們可以用一種特殊形式的內容來表示某個符號，這種特殊形式的內容就是 HTML 實體。比如小於號 `<` 用於定義 HTML 標籤的開始，如果希望瀏覽器正確顯示這些字符，就必須在 HTML 源碼中插入字符實體。

- 常見的命名字符實體由三部分組成：一個 `&`、一個實體名稱，最後加上一個分號 `;`，例如 `&copy;`。HTML 也支援數字字符引用，例如 `&#169;` 或 `&#xA9;`。
- 在 HTML 頁面中，一些特殊的符號很難或者不方便直接使用，此時就可以使用字符實體來代替。

```html
<!-- 單詞之間有 5 個空格，最後只顯示一個空格 -->
hello     world <br>

<!-- 實現單詞之間有 5 個空格 -->
hello&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;world <br>

<!-- 版權符號 -->
&copy;
```

<aside>
⚠️

**注意點：**

- 在一般 HTML 文字排版中，連續空格與換行通常會合併顯示為一個空格；`pre`、`textarea` 或 CSS `white-space` 設定可能改變這個行為。
</aside>
