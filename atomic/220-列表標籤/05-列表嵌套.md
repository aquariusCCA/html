# 列表嵌套

> 來源：origin/220-列表標籤/01-列表標籤.md / # 5. 列表嵌套

> 概念：列表中的某項內容，又包含一個列表。

```html
<h2>我想去幾個城市</h2>
<ul>
  <li>成都</li>
  <li>
    <span>上海</span>
    <ul>
      <li>外灘</li>
      <li>杜莎夫人蠟像館</li>
      <li>
        <a href="">東方明珠</a>
      </li>
      <li>迪士尼樂園</li>
    </ul>
  </li>
  <li>西安</li>
  <li>武漢</li>
</ul>
```
