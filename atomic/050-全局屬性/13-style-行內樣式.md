# style 行內樣式

> style 屬性規定元素的行內樣式（inline style）。行內樣式通常比外部樣式表或 `<style>` 標籤中的一般規則有更高優先權，但仍會受 CSS cascade 與 `!important` 影響。

```html
<body>
  <h1 style="color:blue; text-align:center">This is a header</h1>
  <p style="color:red">This is a paragraph.</p>
</body>
```
