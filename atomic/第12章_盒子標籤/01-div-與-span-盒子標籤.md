# div 與 span 盒子標籤

> 來源：origin/第12章_盒子標籤/01-盒子標籤.md / 全文

`<div>` 和 `<span>` 是沒有語意的標籤，它們就是用來裝內容的盒子。

- `div` 是 `division` 的縮寫，表示分割、分區。
- `span` 意為跨度、跨距。

## 特點

- `<div>` 常作為沒有更合適語意標籤時的區塊容器；預設情況下會以區塊方式顯示，通常會另起一行。（大盒子）
- `<span>` 常用來包住短段行內內容；預設情況下會以行內方式顯示，可以和其他行內內容在同一行流動。（小盒子）
- 這些預設顯示效果可以被 CSS 改變。

```html
<div>我是一個div標籤我一個人單獨占一行</div>
<div>我是一個div標籤我一個人單獨占一行</div>
<span>百度</span>
<span>新浪</span>
<span>搜狐</span>
```
