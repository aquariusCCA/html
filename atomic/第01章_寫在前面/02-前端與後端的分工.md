# 前端與後端的分工

> 來源：origin/第01章_寫在前面/02-甚麼是 Web 的前端和後端.md / # 甚麼是 Web 的前端和後端 ?

前端是用來展示數據的，後端是用來提供數據的。

![前端與後端請求響應示意圖](../../origin/第01章_寫在前面/assets/images/web-frontend-backend-img-001-b8f042.png)

## 補充說明

- 數據庫：存儲數據的。
- 服務器：一個中轉站，幫我們保存信息、傳達消息。

## 與客戶端、服務端的關係

入門時可以先簡化理解：在很多 Web 語境中，客戶端常對應使用者看到與操作的前端，服務端常對應提供資料與處理邏輯的後端。

但嚴格來說，客戶端 / 服務端描述的是網路角色，前端 / 後端描述的是應用分工，兩組概念不能完全等同。
