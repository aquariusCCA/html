Web 標準的構成包含結構、表現與行為。

- 結構：HTML，表示頁面元素。
- 表現：CSS，表示頁面樣式。
- 行為：JavaScript，表示頁面交互的動態效果。

![Web 標準結構、表現與行為對照表](../../origin/第01章_寫在前面/assets/images/web-standard-img-003-2d03d3.png)

Web 標準要求頁面實現結構、表現、行為三層分離。

簡單理解，就是結構寫到 HTML 文件中，表現寫到 CSS 文件中，行為寫到 JavaScript 文件中。
