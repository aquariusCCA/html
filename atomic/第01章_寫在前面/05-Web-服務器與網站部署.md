# Web 服務器與網站部署

> 來源：origin/第01章_寫在前面/03-Web 服務器.md / # Web 服務器

我們目前寫的項目都是放在自己的電腦上，只能自己訪問。如果想要讓很多人訪問我們的網站，可以把項目放到服務器上，這樣就可以讓很多人訪問網站。

![多位使用者連線至服務器示意圖](../../origin/第01章_寫在前面/assets/images/web-server-img-001-7d7f51.png)

服務器是提供計算服務的設備，它也是一台計算機。在網路環境下，根據服務器提供的服務類型不同，服務器又分為文件服務器、數據庫服務器、應用程序服務器、Web 服務器等。

服務器也可以稱為主機。

Web 服務器一般指網站服務器，是指駐留於因特網上某種類型計算機的程序，可以向瀏覽器等 Web 客戶端提供文檔，可以放置網站文件讓全世界瀏覽，也可以放置數據讓全世界下載。
