# Web 基本概念與 JavaWeb

> 來源：origin/第01章_寫在前面/01-甚麼是 Web.md / 全文

Web 是指「World Wide Web」（全球資訊網）的簡稱，是一個由互聯網上的超文本鏈接所構成的全球性資訊系統；它通常透過 HTTP/HTTPS 傳輸資源，並使用 HTML 描述網頁的結構與內容。

Web 可以被看作是一個巨大的資訊庫，其中包含了許多不同的資源，包括網頁、圖像、音頻和視頻等，這些資源可以輕鬆地透過網際網路和瀏覽器被使用者訪問和瀏覽。

## 甚麼是 JavaWeb？

JavaWeb 通常指使用 Java 技術開發、部署在 Web 或應用服務器上，並可透過瀏覽器或 HTTP 介面存取的 Web 應用。

在 JavaWeb 中，Java 程式通常在服務器端執行，負責處理請求、產生回應或提供資料；使用者再透過瀏覽器看到對應的頁面或結果。
