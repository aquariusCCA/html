# Web 與 JavaWeb 基本概念

> 來源：origin/第01章_寫在前面/01-甚麼是 Web.md / # 甚麼是 Web ?

Web 是指「World Wide Web」（全球資訊網）的簡稱，是一個由互聯網上的超文本鏈接所構成的全球性資訊系統。它使用標準化的網際網路協議，例如 HTTP 和 HTML，來連接世界各地的資訊和資源。

Web 可以被看作是一個巨大的資訊庫，其中包含許多不同的資源，包括網頁、圖像、音頻和視頻等。這些資源可以透過網際網路和瀏覽器被使用者訪問和瀏覽。

## JavaWeb

所有通過 Java 語言編寫，並可以通過瀏覽器訪問的程序的總稱，就叫做 JavaWeb。

Web 是一個可以訪問和共享資源的平台，因此它可以訪問和共享各種類型的資源，包括 Java 程序。Java 程序可以通過在 Web 上運行的應用服務器執行，並通過 Web 瀏覽器訪問和呈現它的使用者界面。
