# div 與 span 盒子標籤

> 來源：origin/120-盒子標籤/01-盒子標籤.md / 全文

`<div>` 和 `<span>` 是沒有語意的，它們就是一個盒子，用來裝內容。

- `div` 是 division 的縮寫，表示分割、分區。
- `span` 意為跨度、跨距。

## 特點

- `<div>` 是沒有語意的區塊容器，預設 `display: block`，通常會獨占一行；也可以透過 CSS 改變排列方式。（大盒子）
- `<span>` 是沒有語意的行內容器，常用來包住一小段文字；預設不會像區塊元素一樣前後強制換行，因此同一行可以連續出現多個 `<span>`，但內容仍可能因版面寬度而自然換行。（小盒子）

```html
<div>我是一個div標籤我一個人單獨占一行</div>
<div>我是一個div標籤我一個人單獨占一行</div>
<span>百度</span>
<span>新浪</span>
<span>搜狐</span>
```
