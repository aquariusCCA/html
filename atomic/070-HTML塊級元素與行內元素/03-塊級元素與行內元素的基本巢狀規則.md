## 以 div 為例：可包含行內內容與其他區塊內容

```html
<!-- div 是通用 flow 容器，可包含 phrasing content 和其他 flow content -->
<div>
  <span>尚硅谷1</span>
  <input type="text">
  <div>尚硅谷2</div>
</div>
```

## 以 span 為例：只應包含行內語意內容

```html
<!-- span 的內容模型是 phrasing content，不應放 div、p 等區塊內容 -->
<span>
    <span>尚硅谷1</span>
    <input type="text">
    <span>尚硅谷2</span>
</span>
```

補充：實際可否巢狀應以個別元素的 HTML content model 為準；不是所有塊級元素都能放任何內容，也不是所有行內元素都有相同限制。
