# mark 標籤基本用法

> 來源：origin/320-mark標籤/01-mark標籤.md / 全文

- `<mark>` 標籤用於標示或高亮目前上下文中值得注意、供參考或註記的文字片段，例如段落中的關鍵詞。
- 是 HTML5 中的新標籤。
- 在大多數瀏覽器中，`<mark>` 標籤所包含的文本呈現黃色背景，但可以使用 CSS `background-color` 屬性對其進行更改。

```html
<h2>Example of Mark Tag</h2>
<p>You can easily learn designing a website with our <mark>Web designing tutorial. </mark></p>
```
