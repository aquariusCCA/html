# 標籤屬性

> 來源：origin/040-HTML基本結構標籤/01-HTML基本結構標籤.md / # **4. 標籤屬性**

> 用於給標籤提供附加信息。

![img 標籤屬性名與屬性值示意圖](../../origin/040-HTML基本結構標籤/assets/images/html-basic-structure-tags-img-001-4f91bb.png)

![標籤屬性位於開始標籤或單標籤內示意圖](../../origin/040-HTML基本結構標籤/assets/images/html-basic-structure-tags-img-002-b3d0e7.png)

- 可以寫在單標籤或雙標籤中，形式如下 :

    ```html
    <p id="notice" class="highlight">
      尚硅谷，讓天下沒有難學的技術!!!
    </p>
    ```

- 有些屬性是布林屬性，可以只寫屬性名，不必寫屬性值，如下:

    ```html
    <input disabled>
    ```

- 屬性注意點 :
    - 不同的標籤，有不同的屬性；也有一些通用的屬性 ( 在任何標籤內都能寫 )。
    - 屬性名通常需要符合 HTML 規範或相關標準；屬性值則要依屬性用途填寫，有些是固定值，有些可由作者自行設定。
    - HTML 屬性名通常不區分大小寫，但建議統一使用小寫；屬性值是否區分大小寫取決於具體屬性，不能一概而論。
    - 屬性值可以使用雙引號或單引號；在符合 HTML 語法限制時也可以不加引號，但初學階段建議一律使用雙引號。
    - 標籤中不要出現同名屬性，否則後寫的會失效，例如 :

        ```html
        <!-- 後寫的會失效 -->
        <input type="text" type="password">
        ```

    - 標籤的屬性寫在 **開始標籤內部**。
    - 標籤上可以同時存在多個屬性。
    - 屬性之間以空格隔開。
    - 標籤名屬性之間 **必須以空格隔開**。
    - 屬性之間沒有順序之分。
