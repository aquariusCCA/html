# 字符集

在 `<meta>` 標籤內，可以通過標籤的 charset 屬性來規定 HTML 文檔應該使用哪種字符編碼。

```html
<head>
  <meta charset="UTF-8">
</head>
```

> charset 常用的值有 : GB2312 、BIG5 、GBK 和 UTF-8，其中 UTF-8 是 Unicode 常用的字符編碼方式，能表示 Unicode 中的大量文字與符號，因此網頁通常建議使用 UTF-8。
