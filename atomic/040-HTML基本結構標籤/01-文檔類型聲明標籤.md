# 文檔類型聲明

> 來源：origin/040-HTML基本結構標籤/01-HTML基本結構標籤.md / # **1. 文檔類型聲明標籤**

- `<!DOCTYPE html>` 是 HTML5 的文檔類型聲明，通常放在文件最前面，用來讓瀏覽器以標準模式解析頁面。

```html
<!-- 這句代碼表示當前頁面使用 HTML5 的文檔類型聲明。 -->
<!DOCTYPE html>
```

- **注意點 :**
    - `<!DOCTYPE>` 聲明位於文檔中的最前面的位置，處於 `<html>` 標籤之前。
    - `<!DOCTYPE>` 不是一個 HTML 標籤，它是文檔類型聲明。
