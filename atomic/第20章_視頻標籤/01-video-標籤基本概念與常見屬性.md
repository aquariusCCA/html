# `<video>` 標籤基本概念與常見屬性

> 來源：
> - origin/第20章_視頻標籤/01-視頻標籤.md / # **1. 簡介**
> - origin/第20章_視頻標籤/01-視頻標籤.md / # **3. 總結**

> 作用：`<video>` 標籤用於在文檔中嵌入視頻內容，例如電影剪輯或其他視頻流。

顧名思義，`<video>` 標籤用於在瀏覽器上顯示視頻文件。你可以使用此標籤在你的網站上顯示你創建的視頻。在網站上顯示的視頻文件類型存在限制，只能使用 `.mp4`、`.webm` 和 `.ogg` 文件類型。除此之外，其他格式的視頻不會顯示在網頁上。

> ✍️ 注意：視頻標籤目前支持三種格式 MP4、WebM、Ogg。

場景：在頁面中插入視頻。

## 常見屬性

- `src`：視頻的屬性
- `poster`：視頻封面，沒有播放時顯示的圖片
- `preload`：預加載
- `autoplay`：自動播放
- `loop`：循環播放
- `controls`：瀏覽器自帶的控制條
- `width`：視頻寬度
- `height`：視頻高度

```html
<video src="視頻地址"
  controls 顯示播放控件
  autoplay 自動播放（谷歌瀏覽器需要配合muted靜音播放）
  muted 靜音播放
  loop 循環播放
>
</video>
```

## 使用重點

- 音頻標籤和視頻標籤使用方式基本一致。
- 瀏覽器支持情況不同。
- 谷歌瀏覽器把音頻和視頻自動播放禁止了。
- 我們可以給視頻標籤添加 `muted` 屬性來靜音播放視頻，音頻不可以`(可以通過JavaScript解決)`。
- 視頻標籤是重點，我們經常設置自動播放，不使用 `controls` 控件，循環和設置大小屬性。
