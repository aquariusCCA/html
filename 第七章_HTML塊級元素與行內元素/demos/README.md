# 第七章｜HTML 塊級元素與行內元素示例

這個資料夾收錄第七章筆記對應的可執行示例頁。  
示例會把「塊級元素 / 行內元素」、「常見標籤差異」與「非法巢狀造成的實際結果」放在同一頁，方便直接對照筆記內容觀察。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第七章 Markdown 筆記。
- 想直接觀察畫面差異時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [block-inline-demo.html](./block-inline-demo.html)：整合展示塊級元素與行內元素的排版差異、`p` / `h1~h6` / `a` 的巢狀規則，以及 `img`、`input` 這類常見例外。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
