# 第七章｜HTML 塊級元素與行內元素

> 章節定位：建立 HTML 元素排列方式與基本嵌套規則的判讀框架  
> 建議閱讀方式：先分清塊級元素與行內元素的基本差異，再回頭看特殊嵌套規則為什麼會被瀏覽器修正

## 本章在學什麼

這一章整理 HTML 中常見的元素型態差異與巢狀規則。  
重點不是只背哪些標籤屬於哪一類，而是理解它們在版面排列、容納關係與特殊限制上的差別。

## 本章與整體地圖的關係

- 前置知識：第四章已建立 HTML 文件結構與標籤基本判讀方式。
- 這章的核心任務：分清塊級元素、行內元素，以及常見標籤嵌套限制。
- 讀完後通常接：後續在學習更多標籤與版面配置時，遇到結構合法性問題可以回來查。

- `p` 為什麼不能寫塊元素

## 易混淆主題

- 塊級元素 vs 行內元素：前者通常獨佔一行，後者通常與其他元素排列在同一行。
- 行內元素不能包塊元素 vs `a` 能包區塊元素：前者是一般規則，後者有 HTML5 之後較常見的例外情況，但 `a` 仍不能包另一個 `a`。
- `p` vs `div`：前者有較嚴格的內容模型限制，後者通常更適合作為一般容器。
