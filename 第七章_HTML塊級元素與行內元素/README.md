# 第七章｜HTML 塊級元素與行內元素

> 章節定位：建立 HTML 元素預設排版方式與內容模型限制的入門判讀框架  
> 建議閱讀方式：先分清塊級 / 行內是在描述什麼，再回頭看 `p`、`a`、`h1` ~ `h6` 這些元素為什麼不能只靠舊分類硬判

## 本章在學什麼

這一章整理 HTML 中常見的元素型態差異與巢狀規則。  
重點不是只背哪些標籤屬於哪一類，而是先分清「預設怎麼排」和「能不能這樣包」其實不是同一套規則。

## 本章與整體地圖的關係

- 前置知識：第四章已建立 HTML 文件結構與標籤基本判讀方式。
- 這章的核心任務：分清塊級元素、行內元素這個入門模型的用途，以及 `p`、`a`、`h1` ~ `h6` 等常見標籤的嵌套限制。
- 讀完後通常接：後續在學習更多標籤與版面配置時，遇到結構合法性問題可以回來查。

## 易混淆主題

- 塊級元素 vs 行內元素：這組分類主要幫你理解預設排版，不是完整的 HTML 合法性規則。
- 行內元素不能包塊元素 vs `a` 能包區塊元素：前者只能當入門近似規則，後者提醒你仍要回到元素本身的內容模型來判斷。
- `p` vs `div`：前者有較嚴格的內容模型限制，後者通常更適合作為一般容器。
