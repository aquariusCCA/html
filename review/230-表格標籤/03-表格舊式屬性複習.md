# 表格舊式屬性複習

> 回憶檢索複習。概念與用法說明見 notes/230-表格標籤/03-表格舊式屬性補充.md。

## 重點摘要

- `align`、`border`、`cellpadding`、`cellspacing`、`width` 常見於舊式表格寫法。
- 舊式屬性可以快速調整表格外觀，但重點是讀懂舊程式碼，不是當成現代主要寫法。
- 現代實務中，表格的邊框、間距、寬度、對齊通常交給 CSS 管理。
- HTML 負責資料結構與語意，CSS 負責視覺表現，分工清楚會讓維護更容易。
- 設定某個 `<th>` 或 `<td>` 的寬度常會影響整欄；設定高度常會影響同一列。

### 問答 1 · 基礎 · {#230-table-legacy-q1}

表格舊式屬性在這篇筆記中的定位是什麼？

<details>
<summary>答案</summary>

它們用來讓學習者能讀懂早期 HTML 教材、舊專案或範例程式中的表格外觀寫法；正式樣式仍應優先交給 CSS。

</details>

### 問答 2 · 基礎 · {#230-table-legacy-q2}

`cellpadding` 和 `cellspacing` 分別控制什麼距離？

<details>
<summary>答案</summary>

`cellpadding` 控制儲存格內容與儲存格邊界之間的距離；`cellspacing` 控制儲存格與儲存格之間的距離。

</details>

### 問答 3 · 進階 · {#230-table-legacy-q3}

為什麼正式專案通常不建議把大量表格樣式寫成 HTML 屬性？

<details>
<summary>答案</summary>

因為這會讓結構與樣式混在一起，後續修改成本高。現代實務通常讓 HTML 負責資料結構與語意，CSS 負責視覺表現。

</details>

### 填空 1 · 基礎 · {#230-table-legacy-f1}

`____="1"` 可以讓表格顯示基本邊框，但不適合用來精細控制每條邊框的顏色、粗細與樣式。

<details>
<summary>答案</summary>

`border`

</details>

### 填空 2 · 基礎 · {#230-table-legacy-f2}

`____` 屬性可以設定表格寬度；現代實務更常透過 CSS 控制固定寬度、百分比寬度或響應式寬度。

<details>
<summary>答案</summary>

`width`

</details>

</details>

正面：`cellpadding`

<summary>反面</summary>

設定儲存格內容與儲存格邊界之間的距離。

</details>
