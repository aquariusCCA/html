# p 段落標籤複習

> 回憶檢索複習。概念與用法說明見 notes/110-段落標籤/01-p-段落標籤語意與用法.md。

## 重點摘要

- `<p>` 用來表示一個完整的文字段落，不只是讓文字看起來隔開。
- `p` 是 paragraph 的縮寫，意思是「段落」。
- 多個自然段或不同敘述重點，應使用多個 `<p>` 表達文章結構。
- 瀏覽器通常會替 `<p>` 加上預設上下外距，段落文字也會依父層容器寬度自動換行。
- `<br>` 表示換行，不表示段落；連續 `<br>` 不能取代多個 `<p>`。
- 空的 `<p>` 沒有實際內容，不應拿來製造間距；段落間距應用 CSS 控制。
- `<p>` 可搭配 `class` 與 `id` 等全局屬性，但核心語意仍然是段落。

## 問答題

### 問答 1 · 基礎 · {#110-p-q1}

`<p>` 標籤的核心語意是什麼？

<details>
<summary>答案</summary>

`<p>` 用來表示一個文字段落，告訴瀏覽器與讀者這裡是一個完整的文字段落。

</details>

### 問答 2 · 基礎 · {#110-p-q2}

當文章有多個自然段或敘述重點時，為什麼應該使用多個 `<p>`？

<details>
<summary>答案</summary>

多個 `<p>` 可以依照內容主題與閱讀節奏把長文字切成多個段落，讓 HTML 結構清楚表達文字內容的組織方式。

</details>

### 問答 3 · 基礎 · {#110-p-q3}

`<p>` 和 `<br>` 的用途有什麼差別？

<details>
<summary>答案</summary>

`<p>` 表示文字段落；`<br>` 表示換行，不表示段落。若內容本來就是兩個段落，應該用兩個 `<p>`，而不是連續使用 `<br>` 製造空隙。

</details>

### 問答 4 · 進階 · {#110-p-q4}

為什麼不建議用空的 `<p>` 來增加段落間距？

<details>
<summary>答案</summary>

空的 `<p>` 沒有實際內容，只是拿語意標籤做排版用途，會讓 HTML 結構變得不乾淨，也可能影響輔助技術理解頁面。較好的做法是保留正確段落結構，再用 CSS 控制間距。

</details>

### 卡片 2 · 基礎 · {#110-p-c2}

正面：`class` / `id` 用在 `<p>` 上時的作用

<details>
<summary>答案</summary>

`class` 方便用 CSS 選取並設定樣式；`id` 替特定段落建立唯一識別，必要時也可作為頁內連結目標。它們不會改變 `<p>` 表示段落的核心語意。

</details>
