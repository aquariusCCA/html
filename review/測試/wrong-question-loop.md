## 錯誤類型

| 錯誤類型 | 說明 | 優先回流層次 |
| --- | --- | --- |
| 定義忘記 | 記不起標籤、屬性、控件用途或基本語法 | Level 1 |
| 概念混淆 | 分不清相近概念，例如 `value` / `placeholder`、`radio` / `checkbox` | Level 2 |
| 原因不清 | 答得出寫法，但說不出為什麼要這樣設計 | Level 2 |
| 結構錯誤 | HTML 結構、表單範圍、`label` 關聯或巢狀使用錯誤 | Level 3 |
| 應用卡住 | 看得懂筆記，但無法根據需求組合表單 | Level 3 |

## 回流規則

| 答錯題型 | 回流動作 | 下次複習 |
| --- | --- | --- |
| Level 1 記憶型答錯 | 回到來源筆記對應段落，重新默寫定義、語法或屬性，再重做同題。 | D1 或下一個複習節點 |
| Level 2 理解型答錯 | 先補 1 題相關 Level 1，再用自己的話重答原因、差異或情境判斷。 | D3 或下一個複習節點 |
| Level 3 應用型答錯 | 拆回相關 Level 1 與 Level 2，完成一題較小的應用補強題，再重做原題變形。 | D7 或 D14 |
| 同一概念連續錯 2 次以上 | 加入高風險概念清單，重新閱讀來源筆記，並安排下一節點必做。 | 下一個節點必做 |

## 高風險概念清單

| 觀念 | 觸發條件 | 補強方式 | 對應題目 |
| --- | --- | --- | --- |
| 表單控件在 `<form>` 外 | L2-001 或 L3-001 出錯 | 重畫表單範圍，確認提交資料來源 | L1-001, L2-001, L3-001 |
| `value` 與 `placeholder` 混淆 | L1-003 或 L2-002 出錯 | 寫出「提交值」與「提示文字」對照表 | L1-003, L2-002, L3-002 |
| radio 群組 `name` 不一致 | L2-003 或 L3-003 出錯 | 重寫一組同名 radio，確認互斥效果 | L1-004, L2-003, L3-003 |
| `<button>` 未指定 `type` | L1-005、L2-004 或 L3-004 出錯 | 默寫三種按鈕 type，修正錯誤 HTML | L1-005, L2-004, L3-004 |
| `textarea` 寫成 input 格式 | L1-006 或 L3-005 出錯 | 重寫 textarea 正確語法與預設內容位置 | L1-006, L2-005, L3-005 |
| `label for` / `id` 不一致 | L1-007、L2-006 或 L3-005 出錯 | 修正一組 label 與控件關聯 | L1-007, L2-006, L3-005 |
| HTML5 屬性拼錯或濫用 | L1-008、L2-007 或 L3-006 出錯 | 重寫搜尋表單並檢查 `autocomplete` 拼字 | L1-008, L2-007, L3-006 |

## 補強題選擇規則

- 定義忘記：選同來源筆記的 Level 1 題。
- 概念混淆：選相關 Level 1 題，再選一題 Level 2 比較題。
- 原因不清：重答 Level 2，要求說明「為什麼」。
- 結構錯誤：先圈出表單範圍、欄位、提示文字、提交按鈕，再重做 Level 3。
- 應用卡住：把原題拆成更小需求，例如先寫 `<form>`，再補 `<input>`，最後補按鈕與特殊屬性。
