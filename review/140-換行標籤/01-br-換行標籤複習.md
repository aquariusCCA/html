# br 換行標籤複習

> 回憶檢索複習。概念與用法說明見 notes/140-換行標籤/01-br-換行標籤.md。

## 重點摘要

- `<br>` 是用來在文字中插入「強制換行」的 HTML 標籤。
- `br` 來自英文 `break`，可以理解成把目前這一行打斷，讓後面的內容從新的一行繼續。
- `<br>` 的語意是「換行」，不是「段落」、不是「空白」、也不是「版面間距」。
- `<br>` 適合用在地址、詩句、歌詞、聯絡資訊等分行有內容意義的文字。
- `<br>` 是 HTML 的空元素，也就是 void element；它只需要開始標籤，不會包住任何內容。
- `<br />` 通常能被瀏覽器解析，但斜線在 HTML 中沒有額外作用，慣用寫法是 `<br>`。
- 不要連續使用 `<br>` 來做版面間距；段落用 `<p>`，間距用 CSS。

## 問答題

### 問答 1 · 基礎 · {#140-br-q1}

`<br>` 的主要語意是什麼？

<details>
<summary>答案</summary>

`<br>` 的主要語意是在文字中插入強制換行，讓後面的內容從新的一行繼續。

</details>

### 問答 2 · 基礎 · {#140-br-q2}

哪些情境適合使用 `<br>`？

<details>
<summary>答案</summary>

適合用在地址、詩句、歌詞、聯絡資訊等分行有內容意義的文字，或同一段文字中確實需要在固定位置換行的情境。

</details>

### 問答 3 · 基礎 · {#140-br-q3}

`<br>` 和 `<p>` 的差異是什麼？

<details>
<summary>答案</summary>

`<br>` 只讓後面的內容從新的一行開始，不代表新的段落；`<p>` 代表一個段落。如果兩段文字在語意上是不同段落，應該使用 `<p>`。

</details>

### 問答 4 · 進階 · {#140-br-q4}

為什麼不應該用很多 `<br>` 製造垂直間距？

<details>
<summary>答案</summary>

這會把「排版間距」寫進 HTML 結構裡，之後調整畫面會很難維護，也會讓 HTML 失去清楚的語意。較好的做法是把間距交給 CSS。

</details>
