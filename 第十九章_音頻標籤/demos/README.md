# 第十九章｜音頻標籤示例

這個資料夾收錄第十九章筆記對應的可執行示例頁。  
這章的重點是理解 `<audio>` 如何同時處理音頻來源、播放控制、多格式備援與自動播放限制。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十九章 Markdown 筆記。
- 想直接操作播放器時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [audio-tag-demo.html](./audio-tag-demo.html)：整合展示基本播放器、`controls` / `loop` / `muted` / `autoplay` 差異、單一 `src` 與多個 `<source>` 寫法，以及 MP3、Ogg、WAV 的格式判斷。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
