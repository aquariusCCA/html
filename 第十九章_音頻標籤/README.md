# 第十九章｜音頻標籤

> 章節定位：建立在 HTML 中嵌入音頻、控制播放與處理格式備援的基礎判讀  
> 建議閱讀方式：先理解 `<audio>` 屬性分工，再看 `<source>` 的多格式備援與自動播放限制

## 本章在學什麼

這一章整理 HTML 的音頻標籤。  
重點不只是把音頻放進頁面，而是分清楚音頻來源、播放控制、自動播放限制與格式備援各自在解決什麼問題。

## 本章與整體地圖的關係

- 前置知識：第六章的路徑概念，能幫助理解 `src` 指向的媒體資源位置。
- 這章的核心任務：學會在頁面中插入音頻，並分清 `controls`、`autoplay`、`loop`、`muted`、`source` 的用途。
- 讀完後通常接：之後在做教學語音、提示音或背景音效時，都可以回來快速查這一章。

## 易混淆主題

- `autoplay` vs 真正自動播放成功：前者是要求，後者還要看瀏覽器政策。
- 單一 `src` vs 多個 `<source>`：前者夠簡單，後者更適合多格式備援。
- 音頻標籤 vs 視頻標籤：它們都是媒體元素，但不能把影片畫面相關概念直接套到音頻。
