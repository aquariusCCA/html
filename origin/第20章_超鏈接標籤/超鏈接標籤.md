# **1. 鏈接標籤的介紹**

> 主要作用: 從當前頁面進行跳轉。

- 單詞 anchor 的縮寫，意為 **錨**。

```html
<a href="目标地址">文字内容</a>

<!-- 舉例: -->
<a href="https://www.baidu.com/">百度</a>
```

- 💡 雖然 a 是行內元素，但是 a 元素可以包裹它自身外的任何元素 !
    
    ```html
    <a href="https://example.com">
        <div>這是一個包含在 a 標籤中的區塊元素</div>
    </a>
    ```
    

# **2. 屬性介紹**

![超鏈接標籤屬性.png](./assets/images/link-tags-img-001-910f05.png)

- **href** : 用於指定鏈接目標的 url 地址。必須屬性。
- **target** : 用於指定連接頁面的打開方式。
    - **_self** : 為默認值，在本頁簽中打開。
    - **_blank** : 為在新頁簽中打開。

# **3. 鏈接分類**

## **3.1 外部鏈接**

```html
<h4>外部鏈接</h4>
<a href="http://www.qq.com" target="_self"> 騰訊</a>
<a href="http://www.itcast.cn" target="_blank">傳智播客</a>
```

## **3.2 內部鏈接**

> 網站內部頁面之間的相互鏈接，直接鏈接內部頁面名稱即可。
> 

```html
<body>
  <h4>內部鏈接</h4>
  <a href="./test.html" target="_blank">公司簡介</a>
</body>
```

## **3.3 跳轉到文件**

```html
<!-- 瀏覽器能直接打開文件 -->
<a href="./assets/images/link-tags-img-002-354aea.jpg">看圖片</a>
<a href="./assets/files/link-tags-file-001-9a18a4.mp4">看小電影</a>
<a href="./assets/files/link-tags-file-002-278402.mp3">聽音樂</a>
<a href="./assets/pdfs/link-tags-pdf-001-78621d.pdf">看 pdf 檔</a>
<hr>

<!-- 瀏覽器不能打開的文件，會自動觸發下載 -->
<a href="./assets/files/link-tags-file-003-28aaa1.zip">內部資源</a>
<hr>

<!-- 強制觸發下載 -->
<a href="./assets/pdfs/link-tags-pdf-001-78621d.pdf" download="一份PDF檔.pdf">下載 PDF 檔案</a>
```

<aside>
⚠️

**注意點 :**

- 若瀏覽器無法打開文件，則會引導用戶下載。
- 若想強制觸發下載，請使用 dowload 屬性，屬性值即為下載文件的名稱。
</aside>

## **3.4 跳到本頁的頂部**

> ⚠️ 注意點 → 兩個井號單擊並不會返回頂部。
> 

```html
<div style="height: 2000px;">空盒子</div>
<!-- 跳到本頁的頂部 -->
<a href="#">回到頂部</a>
```

## **3.5 空鏈接**

```html
<h4>空鏈接</h4>
<a href="">刷新本頁面</a>
```

## **3.6 執行JS代碼**

```html
<h4>執行 JS</h4>
<a href="javascript:alert('Hello, World!');">點擊我</a>
```

## **3.7 喚起指定應用**

> 通過 a 標籤可以喚起設備應用程序。
> 

```html
<!-- 喚起設備撥號 --><a href="tel:10010">電話聯繫</a>

<!-- 喚起設備發送郵件 --><a href="mailto: 10010@qq.com">郵件聯繫</a>

<!-- 喚起LINE --><a href="line://msg/text/Hello%20World">使用 LINE 發送消息</a>
```

## **3.8 網頁元素鏈接**

> 在網頁中的各種網頁元素，如: 文本、圖像、表格、音頻、視頻等都可以添加超鏈接。
> 

```html
<h4>網頁元素的鏈接</h4>
<a href="http://www.baidu.com"><img src="./assets/images/link-tags-img-002-354aea.jpg" /></a>
```

## **3.9 錨點鏈接(實用)**

> 作用: 點擊鏈接，可以快速定位到頁面中的某個位置。
> 
- 使用步驟:
    - 在鏈接文本的 `href` 屬性中，設置屬性值為 `#名字` 的形式。
    - 找到目標位置標籤，裡面添加一個 `id 屬性=剛才的名字`。

```html
<!-- 跳轉到本頁的錨點 -->
<a href="#two">第二季</a>
<h3 id="two">第二季介绍</h3>

<!-- 跳轉到其他頁面的錨點 -->
<a href="./測試頁面.html#test1">跳轉到測試頁面的 test1 錨點</a><br />
```
