# **1. 列表標籤的介紹**

> ✍️ 表格是用來顯示數據的，那麼列表就是用來布局的。

![無序、有序與自定義列表範例](./assets/images/list-elements-img-001-43864e.jpg)

- 場景: 在網頁中按照行展示關聯性的內容，如 : 新聞列表、排行榜、帳單等。
- 特點: 按照行的方式，整齊顯示內容。
- 種類: 無序列表、有序列表、自定義列表。
    
    ![列表標籤定義與說明表](./assets/images/list-elements-img-002-a5d743.png)
    

# **2. 無序列表**

> <ul> 標籤表示 HTML 頁面中項目的無序列表，一般會以項目符號呈現列表項，而列表項使用 <li> 標籤定義。

```html
<h4>您喜歡的食物?</h4>
<ul>
    <li>榴槤</li>
    <li>臭豆腐</li>
    <li>鲱鱼罐头</li>
    <li>
        <p>123</p>
    </li>
</ul>
```

- 無序列表的各個列表之間沒有順序級別之分，是並列的。
- `<ul></ul>` 中只能嵌套 `<li></li>`，直接在 `<ul></ul>` 標籤中輸入其他標籤或者文字的做法是不被允許的。
- `<li> 與 </li>` 之間相當於一個容器，可以容納所有元素 ( 可以包含任何標籤 )。
- 無序列表會帶有自己的樣式屬性，但在實際使用時，我們會使用 CSS 來設置。

# **3. 有序列表**

> 有序列表即為有排列順序的列表，其各個列表會按照一定的順序排列定義。

```html
<h4>粉丝排行榜</h4>
<ol>
    <li>刘德华 10000</li>
    <li>刘若英 1000</li>
    <li>pink老师 1</li>
</ol>
```

- 在 HTML 標籤中，`<ol>` 標籤用於定義有序列表，列表排序以數字來顯示，並且使用 `<li>` 標籤來定義列表項。
- `<ol></ol>` 中只能嵌套 `<li></li>`，直接在 `<ol></ol>` 標籤中輸入其他標籤或者文字的作法是不被允許的。
- `<li> 與 </li>` 之間相當於一個容器，可以容納所有元素 ( 可以包含任何標籤 )。
- 有序列表會帶有自己的樣式屬性，但實際使用時，我們會使用 CSS 來設置。

# **4. 自定義列表**

> 使用場景: 常用於對術語或名詞進行解釋和描述，定義列表的列表項前沒有任何項目符號。

- 在 HTML 標籤中， `<dl>` 標籤用於定義描述列表，該標籤會與 `<dt>`和 `<dd>` 一起使用。
    - `<dl>`: 定義列表。
    - `<dt>`: 定義項目 / 名字。
    - `<dd>`: 描述每一個項目 / 名字。

```html
<dl>
  <dt>关注我们</dt>
  <dd>新浪微博</dd>
  <dd>官方微信</dd>
  <dd>联系我们</dd>

  <dt>关注我们</dt>
  <dd>新浪微博</dd>
  <dd>官方微信</dd>
  <dd>联系我们</dd>
</dl>
```

- `<dl></dl>` 裡面只能包含 `<dt>` 和 `<dd>`。
- `<dt>` 和 `<dd>`里面可以放任何标签。
- `<dt>` 和 `<dd>` 個數沒有限制，經常是一個 `<dt>` 對應多個 `<dd>`。

# **5. 列表嵌套**

> 概念: 列表中的某項內容，又包含一個列表。

```html
<h2>我想去幾個城市</h2>
<ul>
  <li>成都</li>
  <li>
    <span>上海</span>
    <ul>
      <li>外灘</li>
      <li>杜莎夫人蠟像館</li>
      <li>
        <a href="">東方明珠</a>
      </li>
      <li>迪士尼樂園</li>
    </ul>
  </li>
  <li>西安</li>
  <li>武漢</li>
</ul>
```
