> 作用 : HTML 裡的 pre 元素，可定義預格式化的文本。在 pre 元素中的文本會保留空格和換行符。文本顯現為等寬字體。 

場景 : 保持文本格式的時候使用 pre 標籤，比如當我們要展示源代碼的時候，只要放一個 pre 標籤，然後把源代碼直接複製，粘貼，然後在頁面上就可以保持好格式。不會像放在其它標籤裡那樣，把換行和空格都自動折疊了。

下面我們看一個示例，這裡我使用的是一段 css 代碼，你也可以換成其它的。如下 :

```html
<body>
  body {
    background: #fff;
    font: 12px/24px 1.66;
  }
  <hr>

  <pre>
    body {
      background: #fff;
      font: 12px/24px 1.66;
    }
  </pre>
</body>
```
