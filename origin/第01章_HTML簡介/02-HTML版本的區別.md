# **HTML5版本**

## **簡介**

HTML5 是新一代的 HTML 標準，2014 年 10 月由萬維網聯盟 (W3C) 完成標準制定。

`HTML5` 的新增特性主要是針對於以前的不足，增加了一些 **新的標籤**、**新的表單** 和新的 **表單屬性** 等。

- `HTML5`有更大的技術集，允許更多樣化和強大的網站和應用程序。
    - 增加了新特性：語義特性，本地存儲特性，設備兼容特性，連接特性，網頁多媒體特性，三維、圖形及特效特性，性能與集成特性、`CSS3`特性。
    - 這個集合有時稱為`HTML5`和朋友，通常縮寫為`HTML5`。
- 廣義的 HTML5 是 HTML5 本身 + CSS3 + JavaScript。
    - HTML5 在狹義上是指新一代的 HTML 標準，在廣義上是指整個前端。
    - 這個集合有時候稱為 HTML5 和朋友，通常縮寫為 HTML5。

## **兼容性**

- 這些新特性都有兼容性問題，基本都是`IE9+`以上版本的瀏覽器才支持，
- 支持 : Chome、Safari、Opere、Firefox 等主流瀏覽器。
- IE 瀏覽器必須是 9 及以上版本才支持 HTML5，且 IE9 僅支持部分 HTML5 新特性。
- 如果不考慮兼容性問題，可以大量使用這些新特性。

## **使用**

- 聲明 :
    - 新特性增加了很多，但是我們專注於開發常用的新特性。
- HTML5 優勢
    - 針對 JavaScript 新增了很多可操作的接口。
    - 新增了一些語意化標籤，全局屬性。
    - 新增了多媒體標籤，可以很好的替代 flash。
    - 更側重語意化，對於 SEO 更友好。
    - 可移植性好，可以大量應用在移動設備上。

```html
<!DOCTYPE html>
<html lang="en">

	<head>
		<meta charset="UTF-8">
		<title>Document</title>
	</head>

	<body>

	</body>

</html>
```

# **HTML4版本**

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
<html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <title></title>
  </head>
  <body>

  </body>
</html>
```

# **XHTML 版本**

```html
<!DOCTYPE html
PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="zh">

  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <title></title>
  </head>
  <body>

  </body>
</html>
```
