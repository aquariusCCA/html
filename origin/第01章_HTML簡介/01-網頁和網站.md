# 什麼是網頁?

- 網頁是指在因特網上根據一定的規則，使用 HTML 等製作的，用於展示特定內容相關的網頁集合。
- **網頁是網站中的一頁**，通常是 HTML 格式的文件，它要通過瀏覽器來閱讀。
- **網頁是構成網站的基本元素，它通常由文字、圖片、音頻、視頻、超鏈接等元素組成**。通常我們看到的網頁，常見以 `.htm` 或 `.html` 後綴結尾的文件，因此將其俗稱為 HTML 文件。

# 甚麼是HTML?

- HTML 指的是 **超文本標記語言**，它是用來描述網頁的一種語言。
    - 超文本的 2 層涵義 :
        - 它可以加入圖片、聲音、動畫、多媒體等內容 (超越了文本限制)。
        - 它還可以從一個文件跳轉到另一個文件，與世界各地的文件連接 (超級鏈接文本)。
- HTML 不是一種編程語言，而是一種標記語言。
- 標記語言是一套標記標籤。

# 網頁的形成

> 網頁是由網頁元素組成的，這些元素是利用 HTML 標籤描述出來的，然後通過瀏覽器解析來顯示給用戶的。

- 前端人員開發代碼 → 瀏覽器顯示代碼 → 生成最後的 Web 頁面。
- 前端的代碼是通過什麼軟件轉換成用戶眼中的頁面的 ?
    - 通過瀏覽器轉化成用戶看到的網頁。
        - 轉化 : 解析和渲染。
