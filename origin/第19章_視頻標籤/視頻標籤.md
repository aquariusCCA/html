# **1. 簡介**

> 作用: <video> 標籤用於在文檔中嵌入視頻內容，例如電影剪輯或其他視頻流。

- 顧名思義，`<video>` 標籤用於在瀏覽器上顯示視頻文件。你可以使用此標籤在你的網站上顯示你創建的視頻。在網站上顯示的視頻文件類型存在限制，只能使用 `.mp4`、`.webm` 和 `.ogg` 文件類型。除此之外，其他格式的視頻不會顯示在網頁上。
    
    > ✍️ 注意 : 音頻標籤目前支持三種格式 MP4、WebM、Ogg。
    > 
- 場景 : 在頁面中插入視頻。
- 常見屬性
    - src ：视频的属性
    - poster：视频封面，没有播放时显示的图片
    - preload：预加载
    - autoplay：自动播放
    - loop：循环播放
    - controls：浏览器自带的控制条
    - width：视频宽度
    - height：视频高度

```html
<video src="视频地址"
  controls 显示播放控件
  autoplay 自动播放（谷歌浏览器需要配合muted静音播放）
  muted 静音播放
  loop 循环播放
>
</video>
```

# **2. 照顧低版本瀏覽器的寫法**

在 標籤中，您需要使用 src 屬性來指定視頻文件的 URL，使用 type 屬性來指定文件類型。這兩個屬性都是必需的。

```html
<video autoplay="autoplay" controls="controls">
  <source src="./assets/files/movie.mp4" type="video/mp4">
  <source src="./assets/files/movie.webm" type="video/webm">
  您的瀏覽器不支持 <video> 標籤。
</video>
```

> <source> 標籤用來確定 src 的來源，因為現在的瀏覽器各式各樣，有的音頻格式有些瀏覽器不支持，這個時候需要為這些不支持的老舊瀏覽器提供支持，就需要使用 source 標籤來確定各種格式的音頻或者視頻。
> 

# **3. 總結**

- 音頻標籤和視頻標籤使用方式基本一致。
- 瀏覽器支持情況不同。
- 谷歌瀏覽器把音頻和視頻自動播放禁止了。
- 我們可以給視頻標籤添加 muted 屬性來靜音播放視頻，音頻不可以`(可以通過JavaScript解決)`。
- 視頻標籤是重點，我們經常設置自動播放，不適用`controls`控件，循環和設置大小屬性。
