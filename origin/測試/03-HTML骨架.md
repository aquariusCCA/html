- 網頁類似於一篇文章 :
    - 每一頁文章內容是有固定的結構的，如 : 開頭、正文、落款等 …
    - 網頁中也是存在固定的結構的，如 : 整體、頭部、標題、主體等 …
- 網頁中的固定結構是通過特點的 **HTML 標籤** 進行描述的。
- 每一個網頁都會有一個基本的結構標籤 (也稱為骨架標籤)，頁面內容也是在這些基本標籤上書寫。
    
    ![骨架標籤1.jpg](./assets/images/03-html-skeleton-img-001-8bc9f3.jpg)
    
    ![骨架標籤2.jpg](./assets/images/03-html-skeleton-img-002-26bafc.jpg)
    
- HTML 頁面也稱為 HTML 文檔。
    - HTML 文檔的後綴名必須是 `.html` 或 `.htm`。
    - 瀏覽器的作用是讀取 HTML 文檔，並以網頁的形式顯示出它們。
