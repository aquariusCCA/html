> 在表單元素中，<textarea> 標籤是用於定義多行文本輸入的控件。
> 
- 使用場景: 當用戶輸入內容較多的情況下，我們就不能使用文本框表單了，此時我們可以使用 `<textarea> 標籤`。使用多行文本輸入控件，可以輸入更多的文字，該控件常見於留言板、評論。

```html
<textarea name="msg" rows="22" cols="3">文本内容</textarea>
```

<aside>
👉

**常用屬性如下 :**

- rows 屬性 : 指定默認顯示的行數，會影響文本域的高度。
- cols 屬性 : 指定默認顯示的列數，會影響文本域的寬度。
- 不能編寫 type 屬性，其他屬性與普通文本輸入框一致。
- 💡 cols = "每行中的字符數"、rows = "顯示的行數"，我們在實際開發中不會使用，都是用 CSS 來改變大小。
