**Notes:**

*Write it during the class*

- submit 提交按鈕。
- reset 重置按鈕。
- button 普通按鈕 ( 默認 )。

```html
<button type="submit">提交按钮</button>
<button type="reset">重置按钮</button>
<button type="button">普通按钮</button>
<button>普通按钮</button>
```

> 注意點: 普通按鈕的 type 值為 button，若不寫 type 值是 submit 會引起表單的提交。(因為 button 標籤 type 屬性的默認值是 submit)
