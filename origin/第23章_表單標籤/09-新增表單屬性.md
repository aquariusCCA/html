```html
<form action="">
  <input type="search" name="sear" id="" required="required" placeholder="pink老师" autofocus="autofocus" autocommplete="off">

```

- 可以通過以下設置方式修改`placeholder`裡面的字體顏色 :
    
    ```css
    input::placeholder {
    	color: pink;
    }
    ```
