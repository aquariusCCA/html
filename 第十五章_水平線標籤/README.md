# 第十五章｜水平線標籤

> 章節定位：建立 HTML 主題分隔與內容脈絡切換的基本判讀框架  
> 建議閱讀方式：先分清 `<hr>` 是主題分隔，不只是畫線，再回頭看它的常見使用場景

## 本章在學什麼

這一章整理 HTML 的水平線標籤 `<hr>`。  
重點不是只記住它在畫面上會出現一條線，而是理解它在內容分隔與主題切換上的語意作用。

## 本章與整體地圖的關係

- 前置知識：第十一章已建立段落標籤 `<p>` 的概念，第十四章已整理換行標籤 `<br>` 的用途。
- 這章的核心任務：分清換行、分段與主題分隔這三種不同層級的內容切換。
- 讀完後通常接：之後在整理文章區塊、文件脈絡或補充資訊時，可以回來快速查 `<hr>` 的使用時機。

## 易混淆主題

- 主題分隔 vs 裝飾線條：前者是內容語意，後者偏版面樣式。
- `<hr>` vs `<br>`：前者是內容區塊分隔，後者是同段文字內換行。
