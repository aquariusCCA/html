# 第十五章｜水平線標籤示例

這個資料夾收錄第十五章筆記對應的可執行示例頁。  
這章的重點是理解 `<hr>` 用來表示主題或閱讀脈絡的分隔，不是單純拿來畫一條線。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十五章 Markdown 筆記。
- 想直接看 `<hr>` 的使用情境時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [hr-tag-demo.html](./hr-tag-demo.html)：整合展示 `<hr>` 的主題分隔語意、和 `<br>` / CSS 線條的差異，以及不建議的濫用方式。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
