# 第十章｜標題標籤

> 章節定位：建立 HTML 標題層級與內容結構的基本判讀框架  
> 建議閱讀方式：先分清標題標籤是在描述內容層級，不是單純放大字，再回頭看 `h1` 到 `h6` 的實際用法

## 本章在學什麼

這一章整理 HTML 的標題標籤，也就是 `<h1>` 到 `<h6>`。  
重點不是背六個標籤名稱，而是理解它們如何替頁面建立由上到下的標題結構。

## 本章與整體地圖的關係

- 前置知識：第二章已建立 HTML 標籤基本觀念，第七章已整理塊級與行內元素的入門分類。
- 這章的核心任務：分清標題標籤的語意層級、預設特點，以及它和純樣式調整的差別。
- 讀完後通常接：之後在做文章頁、文件頁或任何需要內容大綱的頁面時，可以回來快速查。

## 易混淆主題

- 標題語意 vs 字體大小：前者是在描述內容層級，後者只是視覺呈現。
- 預設樣式 vs 真正用途：即使 CSS 改變了外觀，標題標籤的語意仍然存在。
