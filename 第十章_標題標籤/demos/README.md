# 第十章｜標題標籤示例

這個資料夾收錄第十章筆記對應的可執行示例頁。  
這章的重點不只是 `h1` 到 `h6` 的預設大小差異，更重要的是它們如何表達內容層級、如何形成文件結構，以及為什麼不能只靠 CSS 假裝標題。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十章 Markdown 筆記。
- 想直接看標題層級與結構差異時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [heading-tags-demo.html](./heading-tags-demo.html)：整合展示 `h1` 到 `h6` 的層級差異、正確與錯誤的標題結構、文件大綱感，以及 heading 和純 CSS 樣式的角色區別。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
