# 標題標籤

> 所屬章節：第十章｜標題標籤  
> 關鍵字：標題標籤、`<h1>`、`<h2>`、`<h3>`、`<h4>`、`<h5>`、`<h6>`、heading、標題層級、語意結構、內容大綱  
> 建議回查情境：想知道 `h1` 到 `h6` 怎麼分、想確認標題標籤是不是只能拿來放大字、想快速回查標題標籤的預設特點與使用順序

## 本節導讀

這篇整理 HTML 的標題標籤，也就是 `<h1>` 到 `<h6>`。  
它們的核心作用，不是單純把字變大，而是替內容建立由高到低的標題層級。

在新聞、文章、教學頁面或文件型內容中，標題幾乎一定會出現。  
理解標題標籤時，最重要的不是只背六個標籤名稱，而是分清它們是在描述內容結構與重要性，而不是單純做視覺效果。

## 你會在這篇學到什麼

- `h1` 到 `h6` 的基本用途
- 標題層級怎麼理解
- 標題標籤的預設顯示特點
- 為什麼不要把標題標籤只當成放大字體的工具

## 30 秒複習入口

- `<h1>` 到 `<h6>` 是 HTML 的六級標題標籤。
- 這組標籤用來表示內容標題的重要性與層級，不只是改變字體大小。
- `h1` 代表最高層級標題，往下到 `h6` 重要性依序降低。
- 瀏覽器預設通常會讓標題加粗、獨佔一行，且字級由 `h1` 到 `h6` 逐漸變小。

## 速查區

### 核心概念

- 標題標籤是在替頁面內容建立結構。
- 這組結構通常會對應文章或文件的主題、段落層級與閱讀順序。

### 關鍵規則 / 判準

- 基本寫法是 `<h1>` 到 `<h6>`。
- 層級愈高，代表內容愈核心；層級愈低，代表內容愈細。
- 實務上通常會讓 `h1` 作為頁面或文章的主要標題，再往下用 `h2`、`h3` 展開子主題。
- 如果只是想改外觀，應優先用 CSS；不要只因為字想變大，就把一般文字寫成標題。

### 常見特點

- 預設會加粗。
- 預設會獨佔一行。
- 預設字級通常會從 `h1` 到 `h6` 依序變小。

### 常見混淆點

- 標題標籤的重點是內容層級，不只是字大字小。
- 「看起來像標題」和「語意上是標題」不是同一件事。
- 預設樣式可以被 CSS 改掉，但標題的語意層級仍然存在。

### 一句話對比

- 標題標籤解決的是「這段內容在結構上是第幾層標題」，不是「這段文字想不想放大」。

## 正文筆記

### 這篇在解決什麼問題？

- 文章、新聞、教學頁面都需要標題來幫助讀者快速理解內容結構。
- HTML 提供 `<h1>` 到 `<h6>` 六種標題標籤，讓我們可以把主題與子主題的層級標示清楚。

## 1. 標題標籤是什麼

> 標題標籤作為標題使用，並且依據重要性遞減。

- 你可以把這句話理解成：HTML 提供了一組「有層級」的標題工具。
- `h1` 是最高層級，通常代表整頁或整篇文章的主標題。
- `h2` 到 `h6` 則是逐層往下展開的次標題。

## 2. 基本語法

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
```

### 這段怎麼理解？

- 六個標籤都屬於標題，只是層級不同。
- 一般會從較高層級開始，再依內容需要往下展開更細的子標題。
- 在閱讀上，可以把它想成文件的大綱結構。

## 3. 標題標籤的預設特點

- 文字通常會加粗。
- 字號通常會依 `h1` 到 `h6` 逐漸變小。
- 每個標題預設通常都會獨佔一行。

### 這些特點要怎麼理解？

- 這些是瀏覽器常見的預設顯示效果，方便我們一眼看出層級差異。
- 但真正重要的不是外觀本身，而是它背後代表的內容結構。
- 就算之後用 CSS 改掉字級或粗細，這些標籤在語意上仍然是標題。

## 4. 常見使用場景

### 新聞或文章頁面

- 頁面最上方的文章標題通常會是 `h1`。
- 文章中的大段主題可以用 `h2`，子段落再用 `h3`。

### 教學文件或說明文件

- 教學主題常需要分章節、分段落說明，標題標籤很適合拿來建立清楚層級。
- 這樣讀者在跳讀、搜尋與回頭查找時會更容易定位內容。

## 5. 使用標題標籤時的注意點

- 不要只因為想把字放大，就把一段普通文字改成 `h1` 或 `h2`。
- 標題層級應該反映內容結構，而不是只反映你想要的視覺效果。
- 實務上常見做法是讓頁面有清楚的主標題，再往下逐層安排子標題。
- 如果只是需要改字級、粗細或間距，應交給 CSS 處理。

## 常見問法

### `h1` 到 `h6` 一定要全部用到嗎？

- 不一定。
- 重點不是把六級都用滿，而是內容真的有幾層，就用到對應層級。

### 可以跳層嗎？例如直接從 `h1` 到 `h3`？

- 在學習與整理文件時，通常建議盡量保持層級清楚、遞進自然。
- 如果中間少了某一層，讀者在理解結構時通常會比較不直觀。

### 標題標籤和 CSS 放大字體有什麼差別？

- 標題標籤是在描述內容層級。
- CSS 放大字體是在控制外觀。
- 一個偏語意與結構，一個偏樣式與呈現。
