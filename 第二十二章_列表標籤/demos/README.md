# 第二十二章｜列表標籤示例

這個資料夾收錄第二十二章筆記對應的可執行示例頁。  
這章的重點是理解列表標籤如何表達一組項目的語意關係，並分清 `ul`、`ol`、`dl` 的使用情境。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第二十二章 Markdown 筆記。
- 想直接對照三種列表差異時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [list-tags-demo.html](./list-tags-demo.html)：整合展示 `ul` / `ol` / `dl` 的選擇判準、`li` / `dt` / `dd` 的角色、`li` 內可放豐富內容、列表嵌套規則，以及列表語意與 CSS 版面的分工。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
