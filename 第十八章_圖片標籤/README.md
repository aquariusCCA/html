# 第十八章｜圖片標籤

> 章節定位：建立在 HTML 中插入圖片、描述圖片與選擇圖片格式的基礎判讀  
> 建議閱讀方式：先理解 `<img>` 屬性分工，再看尺寸控制、格式選擇與 `data URL`

## 本章在學什麼

這一章整理 HTML 的圖片標籤。  
重點不只是把圖片顯示出來，而是分清楚圖片來源、替代文字、顯示尺寸與格式選擇各自在解決什麼問題。

## 本章與整體地圖的關係

- 前置知識：第六章的路徑概念，能幫助理解 `src` 指向的資源位置。
- 這章的核心任務：學會在頁面中正確插入圖片，並分清 `src`、`alt`、`title`、尺寸屬性與圖片格式的用途。
- 讀完後通常接：之後在做網頁配圖、商品圖、Logo、教學截圖時，都可以回來快速查這一章。

## 適合快速回查的主題

- `src`、`alt`、`title` 的差別
- `width` / `height` 怎麼影響圖片顯示
- `jpg`、`png`、`gif`、`webp` 怎麼選
- `base64` / `data URL` 為什麼不等於一般圖片格式

## 易混淆主題

- `alt` vs `title`：前者是替代文字，後者是補充提示。
- 圖片格式 vs `data URL` / `base64`：前者是檔案格式，後者是表示與嵌入方式。
- 顯示尺寸 vs 原始檔案大小：把圖縮小顯示，不代表檔案本身變小。
