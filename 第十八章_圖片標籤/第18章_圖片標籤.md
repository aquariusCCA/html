# **1. 圖像標籤和屬性介紹**

- 場景 : 在網頁中顯示圖片。
- 單詞 image 的縮寫，意為圖像。
- `src 是 <img> 標籤的必要屬性`，它用於指定圖像文件的路徑和文件名。
- 特點 :
    - 單標籤。
    - **img** 標籤要展示對應的效果，需要藉助標籤的屬性進行設置。

![圖像標籤屬性介紹.png](%25E5%259C%2596%25E5%2583%258F%25E6%25A8%2599%25E7%25B1%25A4%25E5%25B1%25AC%25E6%2580%25A7%25E4%25BB%258B%25E7%25B4%25B9.png)

```html
<img
  src="图片地址"
  alt="替换文本"
  title="提示文本"
  width="宽度"
  height="高度"
/>
```

- `width` 和 `height` 屬性
    - 如果只設置 width 或 height 中的一個，另一個沒設置的會 **自動等比例縮放** ( 此時圖片不會變形 )。
    - 如果同時設置了 width 和 height 兩個， **若設置不當此時圖片可能會變形**。

# **2. 圖像標籤屬性的注意點**

- 圖像標籤可以擁有很多屬性，必須寫在標籤名的後面。
- 屬性之間不分先後順序，標籤名與屬性、屬性與屬性之間均以空格分開。
- 屬性採取鍵值對的格式，即 **key = "value"** 的格式，**屬性 = "屬性值"**。

# **3. 常見的圖片格式**

- jpg 格式 :
    - 概述 : 擴展名為 `.jpg` 或 `.jpeg` 是一種有損的壓縮格式 ( 把肉眼不容易觀察出來的細節丟棄了 )。
    - 主要特點 : **支持豐富顏色、佔用空間較小**，不支持透明背景、不支持動態圖。
    - 使用場景 : 對圖片細節 **沒有級高要求** 的場景，例如 : 網站的產品宣傳圖等。
- png 格式 :
    - 概述 : 擴展名為 `.png` 是一種無損的壓縮格式，能夠更高質量的保存圖片。
    - 主要特點 : 支持的顏色豐富，佔用空間略大、支持透明背景、不支持動態圖。
    - 使用場景 : 公司 logo 圖、重要配圖等。
        - 想讓圖片有透明背景。
        - 想更高質量的呈現圖片。
- bmp 格式 :
    - 概述 : 擴展名為 `.bmp` 不進行壓縮的一種格式，在最大程度上保留圖片更多的細節。
    - 主要特點 : 支持的顏色豐富、保留的細節更多，佔用空間極大，不支持背景透明，不支持動態圖。
    - 使用場景 : 對圖片細節要求極高的場景，例如 : 一些大型遊戲中的圖片 ( 網頁中極少使用 )。
- gif 格式 :
    - 概述 : 擴展名為 `.gif` ，僅支持 256 種顏色，色彩呈現不是很完整。
    - 主要特點 : 支持的顏色較少、支持簡單透明背景、支持動態圖。
    - 使用場景 : 網頁中的動態圖片。
- webp 格式 :
    - 概述 : 擴展名為 `.webp` 谷哥推出的一種格式，專門用來在網頁中呈現圖片。
    - 主要特點 : 具備上述幾種格式的優點，但兼容性不太好，一旦使用務必要解決兼容性的問題。
    - 使用場景 : 網頁中的各種圖片。
- base64 格式 :
    - 本質 : 一種特殊的文本，要通過瀏覽器打開，傳統看圖應用通常無法打開。
    - 原理 : 把圖片進行 base64 編碼，形成一串文本。
    - 如何生成 : 靠一些工具或網站。
    - 如何使用 : 直接作為 img 標籤的 src 屬性的值即可，並且不受到文件位置的影響。
    - 使用場景 : 一些較小的圖片，或者需要和網頁一起加載的圖片。

# **4. 圖像標籤的使用**

```html
<body>
  <h4>圖像標籤的使用:</h4>
  <img src="./images/img.jpg" /><br>
  <hr>

  <h4> alt 替換文本 圖像顯示不出來的時候用文字替換:</h4>
  <img src="./images/img1.jpg" alt="我是pink老師" /> <br>
  <hr>

  <h4> title 提示文本 鼠標放到圖像上,提示的文字:</h4>
  <img src="./images/img.jpg" title="我是pink老師思密達" /><br>
  <hr>

  <h4> width、height</h4>
  <img src="./images/img.jpg" width="200" height="200" /><br>
  <hr>
</body>
```
