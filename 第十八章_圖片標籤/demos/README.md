# 第十八章｜圖片標籤示例

這個資料夾收錄第十八章筆記對應的可執行示例頁。  
這章的重點是理解 `<img>` 不只是把圖片放上頁面，而是同時處理來源、替代文字、尺寸控制與格式選擇。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十八章 Markdown 筆記。
- 想直接看 `<img>` 屬性與效果時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [image-tag-demo.html](./image-tag-demo.html)：整合展示 `src`、`alt`、`title`、尺寸控制、格式選擇與 `data URL` 內嵌圖片的差異。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
