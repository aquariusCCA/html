# 第十二章｜盒子標籤

> 章節定位：建立 HTML 通用容器標籤的基本判讀框架  
> 建議閱讀方式：先分清 `div` / `span` 是通用容器，不是特定語意標籤，再回頭看它們的預設顯示差別與使用情境

## 本章在學什麼

這一章整理 HTML 裡常見的盒子標籤，也就是 `<div>` 和 `<span>`。  
重點不是只背「大盒子 / 小盒子」，而是理解它們作為通用容器的定位，以及它們和預設顯示方式、語意標籤的關係。

## 本章與整體地圖的關係

- 前置知識：第二章已建立 HTML 標籤基本觀念，第十章與第十一章已建立標題與段落等語意結構概念。
- 這章的核心任務：分清 `div`、`span` 的用途、預設差異，以及它們和語意標籤之間的角色界線。
- 讀完後通常接：之後在做版面分組、套樣式或包裹局部內容時，可以回來快速查。

- 兩者的預設顯示差別

- 什麼時候該用 `div`，什麼時候該用 `span`

## 易混淆主題

- 沒有語意 vs 沒有用途：前者是語意定位，後者是實用價值，兩者不同。
- 預設顯示方式 vs 固定規則：`div` / `span` 的預設表現常見，但不是永遠不可改變。
