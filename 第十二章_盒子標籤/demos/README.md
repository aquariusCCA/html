# 第十二章｜盒子標籤示例

這個資料夾收錄第十二章筆記對應的可執行示例頁。  
這章的重點是理解 `div` 與 `span` 是泛用容器，不是因為「看起來能包東西」就亂用；同時也要理解它們和 block / inline、語意元素、CSS `display` 的關係。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十二章 Markdown 筆記。
- 想直接看 `div` / `span` 的差異時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [box-tags-demo.html](./box-tags-demo.html)：整合展示 `div` 與 `span` 的典型用途、block / inline 行為、CSS `display` 對外觀的改變，以及什麼時候應優先選擇更有語意的標籤。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
