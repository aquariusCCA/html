# 第三章｜兼容性問題示例

這個資料夾收錄第三章筆記對應的互動示例頁。  
第三章重點在舊版瀏覽器相容性寫法的辨識與判讀，因此示例會偏向「看懂這段舊碼在補什麼問題」，而不是模擬現代瀏覽器真的進入舊版 `IE` 環境。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第三章筆記。
- 想把舊相容寫法和它對應的問題連起來時：再打開這裡的示例頁。
- 想複習條件註解或 `meta` 標籤的用途時：直接切換示例區即可。
