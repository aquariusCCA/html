# 第六章｜路徑示例

這個資料夾收錄第六章筆記對應的互動示例頁。  
第六章重點在判讀「目前檔案在哪裡、目標檔案在哪裡、路徑要怎麼走」，所以示例會偏向路徑判讀與對照，而不是只展示單一標籤。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第六章筆記。
- 想把 `./`、`../`、同層、上一層、下一層真正看懂時：再打開這裡的示例頁。
- 想快速複習絕對路徑和相對路徑差別時：直接切換示例區即可。

## 目前示例

- [path-demo.html](./path-demo.html)：整合絕對路徑、相對路徑、`./`、`../`、本地絕對路徑限制與外部網址引用注意點。
