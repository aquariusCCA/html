## 適合快速回查的主題

- 絕對路徑是什麼
- 相對路徑是什麼
- 本地絕對路徑為什麼不常用
- 網路絕對路徑為什麼可能引入失敗
- 相對路徑中的 `./` 能不能省略
- 為什麼只要調整檔案位置，路徑就可能要一起改

## 易混淆主題

- 絕對路徑 vs 相對路徑：前者直接指向完整位置，後者依賴當前檔案位置。
- 本地絕對路徑 vs 網路絕對路徑：前者常受設備與目錄變動影響，後者則可能碰到防盜鏈限制。
- 路徑字串 vs 真正檔案位置：路徑能否成立，取決於目前文件與目標文件的相對關係，不是只看字串長相。
