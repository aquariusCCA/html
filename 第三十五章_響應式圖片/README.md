# 第三十五章｜響應式圖片

> 章節定位：理解圖片如何根據螢幕大小、解析度與格式支援選擇合適資源  
> 建議閱讀方式：先理解 `srcset` / `sizes`，再看 `<picture>` 和 `<source>` 的進階用途

## 本章在學什麼

這一章是在第十八章 `<img>` 基礎上繼續往下補。

重點不是換一種寫圖片的方式，而是理解：

- `srcset` 可以提供多個候選圖片。
- `sizes` 告訴瀏覽器圖片在不同條件下大約會顯示多寬。
- `<picture>` 適合做格式備援或藝術方向切換。
- `<img>` 仍然是最終顯示圖片與提供替代文字的核心元素。

## 本章與整體地圖的關係

- 前置知識：第十八章已學過 `<img>`、`src`、`alt` 與常見圖片格式。
- 這章的核心任務：讓圖片在不同裝置上更有效率、更合適。
- 讀完後通常接：圖文語意、媒體字幕、可存取性等內容。

## 易混淆主題

- `srcset` vs `<picture>`：前者偏候選資源，後者偏多條件、多格式或藝術方向切換。
- `x` 描述符 vs `w` 描述符：前者描述像素密度，後者描述圖片寬度。
- `sizes` vs CSS 寬度：`sizes` 是給瀏覽器選圖用的提示，不是直接替代 CSS。
