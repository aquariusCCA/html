# HTML 標題標籤練習

> 動手練習。概念與用法說明見 `notes/100-標題標籤/01-html標題標籤.md`。

---

## 題目 1（基礎）· 應用練習題

請依內容層級，為下方文章補上合適的標題標籤。

結構描述：

- 「HTML 入門」是整篇文章的主題。
- 「HTML 是什麼」和「常見 HTML 標籤」是兩個主要段落。
- 「標題標籤」和「段落標籤」是「常見 HTML 標籤」底下的次層內容。

```html
___ HTML 入門 ___

___ HTML 是什麼 ___
<p>HTML 是用來描述網頁內容結構的標記語言。</p>

___ 常見 HTML 標籤 ___

___ 標題標籤 ___
<p>標題標籤用來表示文章或段落的標題層級。</p>

___ 段落標籤 ___
<p>段落標籤用來表示一段普通文字內容。</p>
```

完成標準：

1. 主題使用 `<h1>`。
2. 主要段落使用 `<h2>`。
3. 次層內容使用 `<h3>`。
4. 標題層級能反映文章結構。

<details>
<summary>參考解答</summary>

```html
<h1>HTML 入門</h1>

<h2>HTML 是什麼</h2>
<p>HTML 是用來描述網頁內容結構的標記語言。</p>

<h2>常見 HTML 標籤</h2>

<h3>標題標籤</h3>
<p>標題標籤用來表示文章或段落的標題層級。</p>

<h3>段落標籤</h3>
<p>段落標籤用來表示一段普通文字內容。</p>
```

解題說明：

`<h1>` 表示整篇文章的最高層級主題；`<h2>` 表示主題底下的主要段落；`<h3>` 表示某個 `<h2>` 底下的次層內容。標題標籤的重點是內容結構，不是單純的字體大小。

</details>

## 題目 2（基礎）· 改錯題

下面這段 HTML 跳過了中間標題層級。請找出問題，並改成較合理的標題結構。

```html
<h1>文章標題</h1>
<h4>第一個小節</h4>
<p>這是第一個小節的內容。</p>
```

完成標準：

1. 說明從 `<h1>` 直接跳到 `<h4>` 的問題。
2. 修正後標題層級循序安排。
3. 保留原本的文字內容。

<details>
<summary>參考解答</summary>

**錯誤說明：**

`<h1>` 後直接接 `<h4>`，會讓文件結構看起來像中間少了 `<h2>` 和 `<h3>`。讀者可能能靠畫面理解，但輔助技術或文件大綱會變得不清楚。

**修正後：**

```html
<h1>文章標題</h1>
<h2>第一個小節</h2>
<p>這是第一個小節的內容。</p>
```

解題說明：

如果「第一個小節」是文章標題底下的主要段落，使用 `<h2>` 較合理。一般情況下，標題層級應依內容結構循序安排。

</details>

## 題目 3（基礎）· 情境判斷題

請判斷下列文字是否適合使用標題標籤，並說明原因。

| 情境 | 是否適合用標題標籤 |
| --- | --- |
| 一篇文章的主題：「HTML 入門」 | ___ |
| 某段落中想強調的詞：「非常重要」 | ___ |
| 主要段落標題：「常見 HTML 標籤」 | ___ |
| 只是想讓一行普通文字變大 | ___ |

完成標準：

1. 能指出真正的內容區塊標題適合使用標題標籤。
2. 能指出普通強調文字不適合用標題標籤。
3. 能說明視覺大小應交給 CSS，而不是誤用標題標籤。

<details>
<summary>參考解答</summary>

| 情境 | 是否適合用標題標籤 | 理由 |
| --- | --- | --- |
| 一篇文章的主題：「HTML 入門」 | 適合 | 這是整篇文章的最高層級主題，可使用 `<h1>`。 |
| 某段落中想強調的詞：「非常重要」 | 不適合 | 這只是段落中的強調內容，不是內容區塊標題，可依語意考慮 `<strong>`。 |
| 主要段落標題：「常見 HTML 標籤」 | 適合 | 這是文章中的主要段落標題，可使用 `<h2>`。 |
| 只是想讓一行普通文字變大 | 不適合 | 如果只是調整外觀，應使用 CSS，而不是誤用標題標籤。 |

</details>

## 題目 4（進階）· 重構題

下面這段 HTML 把標題標籤當成字體大小工具使用。請改成語意更合理的結構。

```html
<article>
  <h1>限時優惠</h1>
  <p>這段文字只是想要醒目一點，但它不是整篇文章的主標題。</p>

  <h1>HTML 入門</h1>
  <p>HTML 是用來描述網頁內容結構的標記語言。</p>
</article>
```

完成標準：

1. 「HTML 入門」保留為文章主題。
2. 「限時優惠」不要再用最高層級標題表示。
3. 修正後能說明標題標籤負責語意與層級，外觀交給 CSS。

<details>
<summary>參考解答</summary>

一種合理修正方式：

```html
<article>
  <p><strong>限時優惠</strong></p>
  <p>這段文字只是想要醒目一點，但它不是整篇文章的主標題。</p>

  <h1>HTML 入門</h1>
  <p>HTML 是用來描述網頁內容結構的標記語言。</p>
</article>
```

如果「限時優惠」本身是某個內容區塊的標題，也可以依它在文件中的層級改成較低層級標題，例如：

```html
<article>
  <h1>HTML 入門</h1>
  <p>HTML 是用來描述網頁內容結構的標記語言。</p>

  <h2>限時優惠</h2>
  <p>這是一個文章底下的附加區塊。</p>
</article>
```

解題說明：

`<h1>` 表示最高層級標題，不應只因為文字想變大就使用它。若只是強調一段文字，可依語意使用 `<strong>`，再用 CSS 調整外觀；若它真的是內容區塊標題，才依結構選擇合適的標題層級。

</details>
