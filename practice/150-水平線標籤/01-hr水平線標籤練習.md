# hr 水平線標籤練習

> 動手練習。概念與用法說明見 notes/150-水平線標籤/01-水平線標籤.md。

## 題目 1（基礎）· 應用練習題

請依照下列內容，寫出一段 HTML。前半段是「課程介紹」，後半段是「上課前準備」，兩者是不同主題，請在主題切換處加入合適的標籤。

- 第一個主題標題：課程介紹
- 第一個主題內容：這門課會帶你認識 HTML 的常用標籤與文件結構。
- 第二個主題標題：上課前準備
- 第二個主題內容：請先安裝瀏覽器與程式碼編輯器，並建立一個練習資料夾。

完成標準：

- 使用 `<hr>` 表示前後內容進入不同主題。
- `<hr>` 不包住任何文字。
- 兩個主題各自保留標題與段落。

<details>
<summary>參考解答</summary>

```html
<h2>課程介紹</h2>
<p>這門課會帶你認識 HTML 的常用標籤與文件結構。</p>

<hr>

<h2>上課前準備</h2>
<p>請先安裝瀏覽器與程式碼編輯器，並建立一個練習資料夾。</p>
```

解題說明：`<hr>` 的語意是主題轉換或內容分隔。這裡前半段在說明課程內容，後半段改成上課前準備，前後內容確實進入不同主題，因此可以用 `<hr>` 表示分隔。`<hr>` 是單標籤，不需要包住文字。

</details>

## 題目 2（基礎）· 改錯題

下面這段 HTML 把 `<hr>` 寫成成對標籤。請修正它。

```html
<h2>產品特色</h2>
<p>這款產品主打輕量與易用。</p>
<hr></hr>
<h2>注意事項</h2>
<p>使用前請先閱讀說明文件。</p>
```

完成標準：

- 不使用 `</hr>`。
- 保留主題分隔的位置。
- 可以使用 notes 提到的 `<hr>` 或 `<hr />` 寫法。

<details>
<summary>參考解答</summary>

```html
<h2>產品特色</h2>
<p>這款產品主打輕量與易用。</p>
<hr>
<h2>注意事項</h2>
<p>使用前請先閱讀說明文件。</p>
```

解題說明：`<hr>` 是單標籤，不需要結束標籤，也不需要包住其他內容。若專案使用一般 HTML5 風格，可以直接寫 `<hr>`；若專案慣例使用自閉合風格，也可以寫成 `<hr />`。

</details>

## 題目 3（進階）· 改錯題

下面這段 HTML 只是想在卡片標題下方加一條裝飾線，內容並沒有真的切換主題。請把它改成 HTML 結構與 CSS 視覺樣式分工更清楚的寫法。

```html
<div class="card">
  <h2>最新消息</h2>
  <hr>
  <p>今天新增三篇文章。</p>
</div>
```

完成標準：

- 不使用 `<hr>` 製作純裝飾線。
- 保留卡片、標題與段落內容。
- 使用 notes 提到的 `border-top` 或 `border-bottom` 做視覺線條。

<details>
<summary>參考解答</summary>

```html
<div class="card">
  <h2>最新消息</h2>
  <p>今天新增三篇文章。</p>
</div>
```

```css
.card h2 {
  border-bottom: 1px solid #ccc;
}
```

解題說明：`<hr>` 的重點不是單純畫一條線，而是表示主題轉換或內容分隔。這裡只是想在標題下方加裝飾線，內容沒有從一個主題切換到另一個主題，因此應該交給 CSS，例如用 `border-bottom` 做視覺線條。HTML 負責表達內容結構，CSS 負責視覺樣式。

</details>
