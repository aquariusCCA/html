# li 結構改錯練習

> 本練習對應 notes/220-列表標籤/01-列表標籤入門與-ul-ol.md

## 練習 1：修正直接放文字的無序列表

下面的 HTML 想列出三個學習主題，但列表結構不正確。請改成正確寫法。

```html
<ul>
  HTML
  CSS
  JavaScript
</ul>
```

<details>
<summary>參考解答</summary>

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
```

解題說明：`<ul>` 的直接子內容應該是 `<li>`。列表裡的每一個項目都要由 `<li>` 包起來。
</details>

## 練習 2：修正直接放段落的列表

下面的 HTML 把 `<p>` 直接放在 `<ul>` 裡。請改成符合列表結構的寫法。

```html
<ul>
  <p>閱讀 HTML 筆記</p>
  <p>完成列表練習</p>
  <p>檢查標籤結構</p>
</ul>
```

<details>
<summary>參考解答</summary>

```html
<ul>
  <li>
    <p>閱讀 HTML 筆記</p>
  </li>
  <li>
    <p>完成列表練習</p>
  </li>
  <li>
    <p>檢查標籤結構</p>
  </li>
</ul>
```

解題說明：`<li>` 裡可以放段落等內容，但 `<p>` 不應直接成為 `<ul>` 的子元素。先用 `<li>` 表示列表項目，再把段落放進項目裡。
</details>

## 練習 3：修正有序列表中的錯誤項目

下面的 HTML 想表示三個操作步驟，但第二個步驟沒有放在 `<li>` 裡。請修正。

```html
<ol>
  <li>建立 HTML 檔案</li>
  加入列表標籤
  <li>在瀏覽器中查看結果</li>
</ol>
```

<details>
<summary>參考解答</summary>

```html
<ol>
  <li>建立 HTML 檔案</li>
  <li>加入列表標籤</li>
  <li>在瀏覽器中查看結果</li>
</ol>
```

解題說明：`<ol>` 與 `<ul>` 一樣，列表中的每個項目都要用 `<li>`。第二個步驟不能直接寫在 `<ol>` 裡。
</details>

## 練習 4：保留項目中的連結

下面的 HTML 想在列表項目中放一段文字和一個連結，但結構不完整。請修正成正確列表。

```html
<ul>
  <li>HTML 入門</li>
  <a href="intro.html">閱讀介紹</a>
</ul>
```

<details>
<summary>參考解答</summary>

```html
<ul>
  <li>
    HTML 入門
    <a href="intro.html">閱讀介紹</a>
  </li>
</ul>
```

解題說明：如果連結是這個列表項目的一部分，就應該放進同一個 `<li>` 裡。`<li>` 可以包含文字和其他 HTML 元素。
</details>
