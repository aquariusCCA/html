# 列表嵌套改錯練習

> 本練習對應 notes/220-列表標籤/03-列表嵌套.md

## 練習 1：把子列表放進父層 li

下面的 HTML 想表示「前端」底下有三個子項目，但子列表放錯位置。請修正。

```html
<ul>
  <li>前端</li>
  <ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
  </ul>
  <li>後端</li>
</ul>
```

<details>
<summary>參考解答</summary>

```html
<ul>
  <li>
    前端
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>後端</li>
</ul>
```

解題說明：子列表應該放在它所屬的父層 `<li>` 裡。這裡 HTML、CSS、JavaScript 都屬於「前端」，所以內層 `<ul>` 要放進「前端」那個 `<li>`。
</details>

## 練習 2：修正太早關閉 li 的結構

下面的 HTML 中，「課程」的 `<li>` 太早結束，導致子列表沒有被包在父層項目中。請修正。

```html
<ul>
  <li>課程</li>
    <ul>
      <li>HTML 入門</li>
      <li>CSS 入門</li>
    </ul>
  <li>練習</li>
</ul>
```

<details>
<summary>參考解答</summary>

```html
<ul>
  <li>
    課程
    <ul>
      <li>HTML 入門</li>
      <li>CSS 入門</li>
    </ul>
  </li>
  <li>練習</li>
</ul>
```

解題說明：父層 `<li>` 要包住它的文字與子列表。等子列表結束後，再關閉父層的 `</li>`。
</details>

## 練習 3：建立一個二層導覽清單

請用巢狀列表表示下面的導覽結構：

- 教學
  - HTML
  - CSS
- 練習
  - 列表練習
  - 表格練習
- 關於本站

<details>
<summary>參考解答</summary>

```html
<ul>
  <li>
    教學
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
  <li>
    練習
    <ul>
      <li>列表練習</li>
      <li>表格練習</li>
    </ul>
  </li>
  <li>關於本站</li>
</ul>
```

解題說明：「HTML」和「CSS」是「教學」的子項目；「列表練習」和「表格練習」是「練習」的子項目。每個子列表都要放進對應的父層 `<li>`。
</details>

## 練習 4：避免為了縮排而亂嵌套

下面的 HTML 只是想列出三個同層項目，但把第二個項目包成了子列表。請改成正確的同層列表。

```html
<ul>
  <li>HTML</li>
  <li>
    CSS
    <ul>
      <li>JavaScript</li>
    </ul>
  </li>
</ul>
```

<details>
<summary>參考解答</summary>

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
```

解題說明：巢狀列表表示父子層級，不是單純用來製造縮排。如果三個項目是同一層，就應該寫成三個並列的 `<li>`。
</details>
