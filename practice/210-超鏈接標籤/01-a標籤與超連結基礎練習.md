# a 標籤與超連結基礎練習

> 動手練習。概念與用法說明見 notes/210-超鏈接標籤/01-a-標籤與超連結基礎.md。

## 題目 1（基礎）· 應用練習題

請建立一個外部網站連結。連結文字是「外部網站」，目標網址是 `https://example.com`，需要在新頁籤開啟，並避免新頁面影響原本頁面。

完成標準：

- 使用 `<a>` 建立可導航連結。
- 使用 `href` 指定完整外部網址。
- 使用 `target="_blank"` 在新頁籤開啟。
- 搭配 `rel="noopener"`。

<details>
<summary>參考解答</summary>

```html
<a href="https://example.com" target="_blank" rel="noopener">外部網站</a>
```

解題說明：

`href` 是超連結的目標。當使用 `_blank` 開啟外部或不完全可信任的頁面時，實務上常搭配 `rel="noopener"`，避免新頁面透過 `window.opener` 影響原本頁面。

</details>

## 題目 2（基礎）· 應用練習題

請建立一個站內連結。連結文字是「公司簡介」，目標是同一網站中的 `about.html`，並在目前頁籤開啟。

完成標準：

- 使用 `<a>`。
- `href` 使用站內頁面的相對路徑。
- 可以省略 `target`，或明確使用 `_self`。
- 連結文字能讓使用者理解目的地。

<details>
<summary>參考解答</summary>

```html
<a href="about.html">公司簡介</a>
```

解題說明：

內部連結通常可以使用相對路徑。`target="_self"` 是預設值，所以一般站內頁面切換可以不寫 `target`，讓連結在目前頁籤開啟。

</details>

## 題目 3（基礎）· 改錯題

以下文字看起來想前往首頁，但缺少可導航目標。請修正它。

```html
<a>前往首頁</a>
```

完成標準：

- 使用 `<a>` 表示導航連結。
- 補上 `href`。
- 連結文字保持清楚。

<details>
<summary>參考解答</summary>

```html
<a href="index.html">前往首頁</a>
```

解題說明：

如果 `<a>` 沒有 `href`，它通常不會形成一般可導航的超連結。只要目的是讓使用者前往某個位置，就應該明確設定 `href`。

</details>

## 題目 4（進階）· 改錯題

以下程式碼把「儲存」這種操作寫成連結。請改成語意更合適的 HTML。

```html
<a href="javascript:saveData()">儲存</a>
```

完成標準：

- 不使用 `<a>` 假裝操作按鈕。
- 使用 `<button>` 表示觸發操作。
- 指定適合一般按鈕操作的 `type`。

<details>
<summary>參考解答</summary>

```html
<button type="button">儲存</button>
```

解題說明：

`<a>` 的核心語意是導航，也就是前往某個位置。如果目的是「送出表單」「打開彈窗」「切換狀態」「儲存資料」這類操作，通常應該使用 `<button>`。

</details>
