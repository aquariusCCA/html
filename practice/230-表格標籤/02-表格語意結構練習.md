# 表格語意結構練習

> 動手練習。概念與用法說明見 notes/230-表格標籤/01-表格基本結構.md。

## 題目 1（基礎）· 應用練習題

請建立一張「段考成績」表格，包含表格標題、表頭、主體資料與總分。

完成標準：

- 使用 `<caption>` 表示表格標題「段考成績」。
- 使用 `<thead>` 放欄位名稱。
- 使用 `<tbody>` 放「語文」「數學」兩筆資料。
- 使用 `<tfoot>` 放「總分」。

資料如下：

| 科目 | 分數 |
| --- | --- |
| 語文 | 99 |
| 數學 | 60 |
| 總分 | 159 |

<details>
<summary>參考解答</summary>

```html
<table>
  <caption>段考成績</caption>
  <thead>
    <tr>
      <th>科目</th>
      <th>分數</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>語文</td>
      <td>99</td>
    </tr>
    <tr>
      <td>數學</td>
      <td>60</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>總分</td>
      <td>159</td>
    </tr>
  </tfoot>
</table>
```

解題說明：`<caption>` 說明整張表格呈現什麼；`<thead>` 放欄位標題；`<tbody>` 放主要資料；`<tfoot>` 放總計或腳注。
</details>

## 題目 2（基礎）· 應用練習題

請把下面資料整理成一張語意完整的表格。

表格標題：訂單明細

| 品名 | 數量 |
| --- | --- |
| 筆記本 | 2 |
| 原子筆 | 5 |
| 合計 | 7 |

完成標準：

- 欄位名稱放在 `<thead>`。
- 商品資料放在 `<tbody>`。
- 合計列放在 `<tfoot>`。

<details>
<summary>參考解答</summary>

```html
<table>
  <caption>訂單明細</caption>
  <thead>
    <tr>
      <th>品名</th>
      <th>數量</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>筆記本</td>
      <td>2</td>
    </tr>
    <tr>
      <td>原子筆</td>
      <td>5</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>合計</td>
      <td>7</td>
    </tr>
  </tfoot>
</table>
```

解題說明：當表格有明確標題、欄位名稱、主要資料與統計結果時，使用完整語意結構可以讓資料關係更清楚。
</details>

## 題目 3（進階）· 重構題

下面表格可以呈現資料，但缺少表格標題與語意分區。請重構成更完整的表格結構。

```html
<table>
  <tr>
    <th>月份</th>
    <th>銷售額</th>
  </tr>
  <tr>
    <td>一月</td>
    <td>30000</td>
  </tr>
  <tr>
    <td>二月</td>
    <td>28000</td>
  </tr>
  <tr>
    <td>總計</td>
    <td>58000</td>
  </tr>
</table>
```

<details>
<summary>參考解答</summary>

```html
<table>
  <caption>月銷售統計</caption>
  <thead>
    <tr>
      <th>月份</th>
      <th>銷售額</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>一月</td>
      <td>30000</td>
    </tr>
    <tr>
      <td>二月</td>
      <td>28000</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>總計</td>
      <td>58000</td>
    </tr>
  </tfoot>
</table>
```

解題說明：原本的表格資料沒有錯，但重構後可以更清楚地區分表格標題、欄位名稱、主要資料與總計列。
</details>

## 題目 4（進階）· 改錯題

下面的 HTML 用表格安排圖片與文字的左右位置。請說明問題，並改成「只有真正的資料表才使用表格」的寫法。這題只需要移除錯誤的表格排版，不需要寫 CSS。

```html
<table>
  <tr>
    <td>圖片區</td>
    <td>文字介紹區</td>
  </tr>
</table>
```

<details>
<summary>參考解答</summary>

```html
<p>圖片區</p>
<p>文字介紹區</p>
```

解題說明：表格應用來呈現列與欄有對應關係的結構化資料，不應拿來控制頁面版面。這裡只是兩個版面區塊，沒有資料表關係，因此不應使用 `<table>`。
</details>
