# 表格基本結構練習

> 動手練習。概念與用法說明見 notes/230-表格標籤/01-表格基本結構.md。

## 題目 1（基礎）· 應用練習題

請建立一張「商品價格表」，包含 3 欄與 3 筆商品資料。

完成標準：

- 使用 `<table>` 建立表格。
- 第一列使用 `<th>` 表示欄位名稱。
- 每一筆商品資料各放在一個 `<tr>` 裡。
- 商品資料使用 `<td>`。

資料如下：

| 商品 | 價格 | 庫存 |
| --- | --- | --- |
| 鍵盤 | 1200 | 8 |
| 滑鼠 | 600 | 15 |
| 螢幕 | 5200 | 4 |

<details>
<summary>參考解答</summary>

```html
<table>
  <tr>
    <th>商品</th>
    <th>價格</th>
    <th>庫存</th>
  </tr>
  <tr>
    <td>鍵盤</td>
    <td>1200</td>
    <td>8</td>
  </tr>
  <tr>
    <td>滑鼠</td>
    <td>600</td>
    <td>15</td>
  </tr>
  <tr>
    <td>螢幕</td>
    <td>5200</td>
    <td>4</td>
  </tr>
</table>
```

解題說明：`<table>` 是整張表格，`<tr>` 是列，第一列的欄位名稱應使用 `<th>`，一般資料格則使用 `<td>`。
</details>

## 題目 2（基礎）· 應用練習題

請建立一張「學生成績表」，欄位包含「姓名」「語文」「數學」，資料如下：

| 姓名 | 語文 | 數學 |
| --- | --- | --- |
| 小明 | 90 | 80 |
| 小美 | 95 | 88 |

完成標準：

- 橫向每一列要是一筆完整資料。
- 欄位名稱使用 `<th>`。
- 分數資料使用 `<td>`。

<details>
<summary>參考解答</summary>

```html
<table>
  <tr>
    <th>姓名</th>
    <th>語文</th>
    <th>數學</th>
  </tr>
  <tr>
    <td>小明</td>
    <td>90</td>
    <td>80</td>
  </tr>
  <tr>
    <td>小美</td>
    <td>95</td>
    <td>88</td>
  </tr>
</table>
```

解題說明：一個 `<tr>` 表示一列。小明的姓名與兩科成績屬於同一筆資料，所以要放在同一個 `<tr>` 裡。
</details>

## 題目 3（進階）· 改錯題

下面的 HTML 想建立會員清單，但第一列欄位名稱都用了 `<td>`。請修正表頭語意。

```html
<table>
  <tr>
    <td>會員名稱</td>
    <td>等級</td>
    <td>狀態</td>
  </tr>
  <tr>
    <td>Amy</td>
    <td>VIP</td>
    <td>啟用</td>
  </tr>
</table>
```

<details>
<summary>參考解答</summary>

```html
<table>
  <tr>
    <th>會員名稱</th>
    <th>等級</th>
    <th>狀態</th>
  </tr>
  <tr>
    <td>Amy</td>
    <td>VIP</td>
    <td>啟用</td>
  </tr>
</table>
```

解題說明：第一列的「會員名稱」「等級」「狀態」是欄位名稱，不是普通資料，因此應使用 `<th>`。使用 `<th>` 的理由是語意，不只是因為瀏覽器預設會讓它看起來加粗置中。
</details>

## 題目 4（進階）· 改錯題

下面的 HTML 把 `<tr>` 放在 `<table>` 外面。請改成正確的表格結構。

```html
<tr>
  <th>項目</th>
  <th>數量</th>
</tr>
<table>
  <tr>
    <td>筆記</td>
    <td>3</td>
  </tr>
</table>
```

<details>
<summary>參考解答</summary>

```html
<table>
  <tr>
    <th>項目</th>
    <th>數量</th>
  </tr>
  <tr>
    <td>筆記</td>
    <td>3</td>
  </tr>
</table>
```

解題說明：`<tr>` 是表格列，應出現在 `<table>` 的表格結構中。不要把 `<tr>` 當成一般區塊標籤拿到表格外使用。
</details>
