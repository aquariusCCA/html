# 表格舊式屬性閱讀練習

> 動手練習。概念與用法說明見 notes/230-表格標籤/03-表格舊式屬性補充.md。

## 題目 1（基礎）· 應用練習題

你正在閱讀一份舊教材，想快速做出有邊框、置中、寬度為 500 的商品表格。請依照需求補上 `<table>` 的屬性。

完成標準：

- 表格相對周圍內容置中。
- 顯示基本邊框。
- 儲存格內容與邊界之間留出 20 的距離。
- 儲存格之間的距離為 0。
- 表格寬度為 500。

<details>
<summary>參考解答</summary>

```html
<table align="center" border="1" cellpadding="20" cellspacing="0" width="500">
  <tr>
    <th>商品</th>
    <th>價格</th>
  </tr>
  <tr>
    <td>鍵盤</td>
    <td>1200</td>
  </tr>
</table>
```

解題說明：這題練習讀懂舊式表格屬性。`align` 控制表格對齊，`border` 顯示基本邊框，`cellpadding` 控制內容到格線的距離，`cellspacing` 控制儲存格之間的距離，`width` 設定表格寬度。
</details>

## 題目 2（基礎）· 應用練習題

請把下面沒有外觀屬性的表格改成：顯示基本邊框、儲存格內距為 10、儲存格間距為 0。

```html
<table>
  <tr>
    <th>科目</th>
    <th>分數</th>
  </tr>
  <tr>
    <td>HTML</td>
    <td>90</td>
  </tr>
</table>
```

<details>
<summary>參考解答</summary>

```html
<table border="1" cellpadding="10" cellspacing="0">
  <tr>
    <th>科目</th>
    <th>分數</th>
  </tr>
  <tr>
    <td>HTML</td>
    <td>90</td>
  </tr>
</table>
```

解題說明：`border="1"` 可快速顯示基本邊框；`cellpadding="10"` 增加儲存格內容與邊界的距離；`cellspacing="0"` 讓儲存格之間沒有空隙。
</details>

## 題目 3（進階）· 改錯題

下面的 HTML 為了製造表格中的距離，加入了沒有資料意義的空白列。請改成使用 notes 介紹過的表格屬性，不要新增空白列撐版面。

```html
<table border="1">
  <tr>
    <th>商品</th>
    <th>價格</th>
  </tr>
  <tr>
    <td></td>
    <td></td>
  </tr>
  <tr>
    <td>滑鼠</td>
    <td>600</td>
  </tr>
</table>
```

<details>
<summary>參考解答</summary>

```html
<table border="1" cellpadding="20">
  <tr>
    <th>商品</th>
    <th>價格</th>
  </tr>
  <tr>
    <td>滑鼠</td>
    <td>600</td>
  </tr>
</table>
```

解題說明：空白列沒有資料意義，會破壞表格語意。若只是希望內容與格線之間有距離，可用 notes 提到的 `cellpadding`。正式樣式控制通常交給 CSS，但這題只練習讀懂與套用舊式屬性。
</details>

## 題目 4（進階）· 重構題

下面的舊式表格屬性很多。請保留表格資料結構，並把可讀性調整到「仍能看懂每個屬性用途」：保留 `border`、`cellpadding`、`cellspacing`、`width`，移除沒有必要的空白資料格。

```html
<table align="center" border="1" cellpadding="20" cellspacing="0" width="500">
  <tr>
    <th>項目</th>
    <th>說明</th>
    <th></th>
  </tr>
  <tr>
    <td>border</td>
    <td>顯示基本邊框</td>
    <td></td>
  </tr>
</table>
```

<details>
<summary>參考解答</summary>

```html
<table border="1" cellpadding="20" cellspacing="0" width="500">
  <tr>
    <th>項目</th>
    <th>說明</th>
  </tr>
  <tr>
    <td>border</td>
    <td>顯示基本邊框</td>
  </tr>
</table>
```

解題說明：空白的 `<th>` 與 `<td>` 沒有資料意義，應移除。`border`、`cellpadding`、`cellspacing`、`width` 都是 notes 說明過的舊式表格屬性，可用來閱讀或快速展示舊範例；正式專案的視覺細節通常應交給 CSS。
</details>
