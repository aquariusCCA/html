# br 換行標籤練習

> 動手練習。概念與用法說明見 notes/140-換行標籤/01-br-換行標籤.md。

## 題目 1（基礎）· 應用練習題

請把下面的聯絡資訊寫成 HTML，讓姓名、電話與 Email 固定分成三行顯示。

```text
王小明
電話：02-1234-5678
Email：hello@example.com
```

完成標準：

- 使用 `<br>` 在同一段聯絡資訊中插入強制換行。
- `<br>` 放在需要換到下一行的位置。
- 不使用多個 `<p>` 把同一組聯絡資訊拆成不同段落。

<details>
<summary>參考解答</summary>

```html
王小明<br>
電話：02-1234-5678<br>
Email：hello@example.com
```

解題說明：姓名、電話與 Email 是同一組聯絡資訊，只是內容本身需要固定分行顯示，因此可以在需要換行的位置使用 `<br>`。`<br>` 的語意是強制換行，不是段落或間距。

</details>

## 題目 2（基礎）· 改錯題

下面這段 HTML 把 `<br>` 當成可以包住內容的標籤。請修正它。

```html
<br>第一行文字</br>
第二行文字
```

完成標準：

- 不使用 `</br>`。
- `<br>` 不包住任何內容。
- 修正後能讓第二行文字從新的一行開始。

<details>
<summary>參考解答</summary>

```html
第一行文字<br>
第二行文字
```

解題說明：`<br>` 是 HTML 的空元素，也就是 void element。它只需要開始標籤，不會包住內容，因此不需要也不應寫成 `<br></br>`。正確做法是把 `<br>` 放在需要換行的位置。

</details>

## 題目 3（進階）· 改錯題

下面這段 HTML 同時有兩個問題：用多個 `<br>` 製造標題與段落之間的距離，也用 `<br><br>` 分隔兩個不同段落。請把它修正成語意更清楚的寫法。

```html
<h1>文章標題</h1>
<br>
<br>
這是第一段文字。<br><br>
這是第二段文字。
```

完成標準：

- 不用連續 `<br>` 製造版面間距。
- 兩段不同段落使用 `<p>` 表示。
- 若需要標題與內容之間的距離，使用 notes 教過的 CSS `margin` 做法。

<details>
<summary>參考解答</summary>

```html
<h1>文章標題</h1>
<p>這是第一段文字。</p>
<p>這是第二段文字。</p>
```

```css
h1 {
  margin-bottom: 24px;
}
```

解題說明：`<br>` 的語意是強制換行，不是版面間距。標題與段落之間的距離應交給 CSS 的 `margin`，不同段落則應使用 `<p>`，讓 HTML 清楚表達「這是段落」，而不是用 `<br><br>` 硬做出視覺空隙。

</details>
