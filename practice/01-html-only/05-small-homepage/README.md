# 05 Small Homepage

## 練習目標

建立一個純 HTML 小型網站首頁，綜合練習語意化頁面結構、導覽、內容區塊、文章卡片、側欄與頁尾。

## 可使用標籤

- `html`、`head`、`meta`、`title`、`body`
- `header`、`nav`、`main`、`section`、`article`、`aside`、`footer`
- `h1`、`h2`、`h3`、`p`
- `a`、`img`
- `ul`、`ol`、`li`
- `strong`、`em`

## 禁止事項

- 不使用 CSS
- 不使用 JavaScript
- 不把所有區塊都寫成 `div`
- 不用表格做首頁排版

## 任務需求

請在 `starter/index.html` 完成一個小型網站首頁，至少包含：

1. 網站頁首與網站名稱。
2. 主要導覽列。
3. 首頁主內容。
4. 至少兩個內容 `section`。
5. 至少兩篇 `article`。
6. 一個 `aside`，放公告、推薦連結或補充資訊。
7. 頁尾聯絡資訊。

## 自我檢查

- [ ] 頁面有清楚的 `header`、`nav`、`main`、`footer`。
- [ ] 主要內容放在 `main` 裡。
- [ ] 可獨立閱讀的內容使用 `article`。
- [ ] 補充資訊使用 `aside`。
- [ ] 導覽項目使用列表與連結。
- [ ] 沒有使用 CSS 或 JavaScript。

## 加分挑戰

- 建立本頁錨點導覽。
- 讓每篇 `article` 都有自己的標題與摘要。
- 在頁尾加入版權資訊與聯絡連結。
