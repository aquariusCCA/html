# HTML-only Practice

這一組練習只使用 HTML，不加入 CSS 或 JavaScript。

目標是練習把 HTML 標籤放進完整頁面情境中，而不是單獨背標籤用途。完成每一題時，請先從 `starter/index.html` 開始，完成後再對照 `solution/index.html`。

## 練習列表

| 順序 | 練習 | 重點 |
|---|---|---|
| `01-personal-profile-page/` | 個人介紹頁 | HTML 骨架、標題、段落、圖片、連結、列表 |
| `02-article-page/` | 文章閱讀頁 | 文章語意、標題層級、本頁導覽、段落分區 |
| `03-product-detail-page/` | 商品介紹頁 | 商品資訊、圖片、文字強調、列表、表格 |
| `04-registration-form/` | 報名表單頁 | 表單、欄位標籤、輸入型別、分組、按鈕 |
| `05-small-homepage/` | 小型網站首頁 | `header`、`nav`、`main`、`section`、`article`、`aside`、`footer` 綜合練習 |

## 建議練習方式

1. 先閱讀該題 `README.md`。
2. 打開 `starter/index.html`，依照 TODO 完成頁面。
3. 完成後檢查是否符合自我檢查清單。
4. 再打開 `solution/index.html` 對照自己的結構。
5. 不要求和參考答案一模一樣，但 HTML 語意與結構要合理。

## 共通限制

- 只使用 HTML。
- 不建立 CSS 檔案。
- 不建立 JavaScript 檔案。
- 不用表格做頁面排版。
- 不為了排版濫用 `br`、`hr` 或空白字元。
- 優先根據內容意義選擇標籤。
