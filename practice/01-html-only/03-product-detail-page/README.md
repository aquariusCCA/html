# 03 Product Detail Page

## 練習目標

建立一個純 HTML 商品介紹頁，練習圖片、商品資訊、文字強調、列表、表格與連結。

## 可使用標籤

- `html`、`head`、`meta`、`title`、`body`
- `header`、`main`、`section`、`article`、`footer`
- `h1`、`h2`、`h3`、`p`
- `img`、`a`
- `ul`、`ol`、`li`
- `table`、`caption`、`thead`、`tbody`、`tr`、`th`、`td`
- `strong`、`em`、`del`、`ins`

## 禁止事項

- 不使用 CSS
- 不使用 JavaScript
- 不用表格做整頁排版
- 不把商品規格全部塞進一個段落

## 任務需求

請在 `starter/index.html` 完成一個商品介紹頁，至少包含：

1. 商品名稱與簡短描述。
2. 商品圖片與合適的 `alt`。
3. 價格資訊，可包含原價與優惠價。
4. 商品特色列表。
5. 商品規格表格。
6. 購買、客服或返回列表的連結。
7. 頁尾補充資訊。

## 自我檢查

- [ ] 商品圖片有描述性的 `alt`。
- [ ] 商品特色使用列表。
- [ ] 商品規格使用表格，而且表格有 `caption`。
- [ ] 表格表頭使用 `th`。
- [ ] 連結代表真正跳轉，不用 `a` 假裝按鈕。
- [ ] 沒有使用 CSS 或 JavaScript。

## 加分挑戰

- 使用 `del` 表示原價，使用 `ins` 表示目前優惠價。
- 使用 `section` 分出「商品特色」、「商品規格」、「購買資訊」。
