先把 HTML 結構寫清楚

作者：Alex Chen

發布日期：2026-05-23

為什麼先學 HTML

HTML 是網頁內容的基礎。即使頁面沒有 CSS，看起來很簡單， 只要結構清楚，瀏覽器、搜尋引擎與輔助工具仍然能理解內容。

初學時應該先練習選擇正確標籤，而不是急著追求漂亮的畫面。

語意結構的重要性

語意化標籤能描述內容角色，例如 `nav` 表示導覽， `article` 表示可獨立閱讀的內容，`footer` 表示頁尾或區塊尾端資訊。

當內容角色明確時，後續加入 CSS 或 JavaScript 也會更容易維護。

下一步學習方向

完成純 HTML 練習後，可以開始學習 CSS，讓同一份結構呈現更好的版面。

重點整理

  • HTML 負責內容與結構。
  • 標題層級要反映內容層次。
  • 不要把所有內容都寫成 `div`。
