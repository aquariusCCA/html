# 02 Article Page

## 練習目標

建立一個純 HTML 文章閱讀頁，練習文章語意結構、標題層級、段落、分區與本頁導覽。

## 可使用標籤

- `html`、`head`、`meta`、`title`、`body`
- `header`、`nav`、`main`、`article`、`section`、`footer`
- `h1`、`h2`、`h3`、`h4`、`p`
- `a`、`ul`、`li`
- `strong`、`em`、`br`、`hr`

## 禁止事項

- 不使用 CSS
- 不使用 JavaScript
- 不使用 `h1` 多次充當大標題
- 不用 `br` 製造段落間距

## 任務需求

請在 `starter/index.html` 完成一篇文章頁，至少包含：

1. 網站頁首。
2. 一個本頁目錄，連到文章內部區塊。
3. 一篇 `article`。
4. 文章主標題、作者與發布日期。
5. 至少三個文章段落區塊。
6. 合理的 `h1`、`h2`、`h3` 層級。
7. 文章頁尾，例如分類、標籤或延伸閱讀。

## 自我檢查

- [ ] 頁面只有一個主要 `h1`。
- [ ] 文章內容放在 `article` 中。
- [ ] 章節使用 `section` 分組。
- [ ] 本頁導覽連結的 `href` 對應到區塊 `id`。
- [ ] 沒有為了排版濫用 `br` 或 `hr`。

## 加分挑戰

- 加入一個「重點整理」列表。
- 加入一段延伸閱讀連結。
