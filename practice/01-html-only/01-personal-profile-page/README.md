# 01 Personal Profile Page

## 練習目標

建立一個純 HTML 個人介紹頁，練習文件骨架、標題、段落、圖片、連結與列表。

## 可使用標籤

- `html`、`head`、`meta`、`title`、`body`
- `header`、`main`、`section`、`footer`
- `h1`、`h2`、`p`
- `img`、`a`
- `ul`、`ol`、`li`
- `strong`、`em`

## 禁止事項

- 不使用 CSS
- 不使用 JavaScript
- 不用表格排版
- 不為了空白連續使用多個 `br`

## 任務需求

請在 `starter/index.html` 完成一個個人介紹頁，至少包含：

1. 頁面標題與基本 metadata。
2. 一個主要 `h1`。
3. 一張個人圖片，並補上合適的 `alt`。
4. 一段自我介紹。
5. 一組技能列表。
6. 一組學習目標列表。
7. 至少兩個相關連結。
8. 頁尾聯絡資訊。

## 自我檢查

- [ ] HTML 骨架完整。
- [ ] 頁面只有一個 `h1`。
- [ ] 圖片有描述性的 `alt`。
- [ ] 技能或興趣使用列表，而不是用逗號塞在段落中。
- [ ] 連結有有效的 `href`。
- [ ] 沒有使用 CSS 或 JavaScript。

## 加分挑戰

- 使用 `section` 把「關於我」、「技能」、「學習目標」分成不同區塊。
- 使用 `strong` 或 `em` 強調真正重要的文字，而不是只為了視覺效果。
