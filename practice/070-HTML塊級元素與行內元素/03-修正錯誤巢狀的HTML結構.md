# 修正錯誤巢狀的 HTML 結構

> 動手練習。概念與用法說明見 `notes/070-HTML塊級元素與行內元素/03-常見錯誤巢狀：標題p與a標籤.md`。

起始檔：[`starter.html`](./03-修正錯誤巢狀的HTML結構-starter/starter.html)

---

## 題目 1（基礎）· 改錯題

下方 HTML 把標題元素互相嵌套了。請修正結構，讓主標題與副標題各自獨立。

```html
<h1>
  前端入門課程
  <h2>HTML 基礎</h2>
</h1>
```

完成標準：

1. `h1` 與 `h2` 不互相嵌套。
2. 保留「前端入門課程」與「HTML 基礎」兩段文字。
3. 能說明標題層級不等於標籤巢狀。

<details>
<summary>參考解答</summary>

```html
<h1>前端入門課程</h1>
<h2>HTML 基礎</h2>
```

解題說明：

`h1` 到 `h6` 用來表示內容層級，但標題元素之間不應互相包住。`h2` 可以是 `h1` 底下某一節的標題，這種層級關係由文件順序與標題等級表達，不需要把 `h2` 寫進 `h1` 裡。

</details>

---

## 題目 2（基礎）· 改錯題

下方 HTML 把區塊內容放進段落中。請修正結構，讓段落與區塊容器各自放在合適的位置。

```html
<p>
  歡迎加入本週課程。
  <div>報名截止日：週五晚上 8 點</div>
</p>
```

完成標準：

1. `p` 不包含 `div`。
2. 保留原本兩段文字。
3. 能說明如果需要一整塊版面容器，應用 `div` 包 `p`，或把 `div` 放在 `p` 外。

<details>
<summary>參考解答</summary>

一種合理修正方式：

```html
<p>歡迎加入本週課程。</p>

<div>報名截止日：週五晚上 8 點</div>
```

如果這兩段內容本來就是同一個區塊，也可以再用外層 `div` 分組：

```html
<div>
  <p>歡迎加入本週課程。</p>
  <p>報名截止日：週五晚上 8 點</p>
</div>
```

解題說明：

`p` 代表段落，適合放段落文字與 phrasing content，不應包含 `div` 這類區塊內容。瀏覽器遇到 `p` 包 `div` 時可能會自動提前結束 `p`，造成實際 DOM 和原始碼不同。

</details>

## 題目 3（進階）· 實作任務

請打開起始檔 [`starter.html`](./03-修正錯誤巢狀的HTML結構-starter/starter.html)，修正檔案中的 3 個錯誤區塊：

1. `h1` 裡嵌套 `h2`。
2. `p` 裡嵌套 `div`。
3. 外層 `a` 裡又放了另一個 `a`。

完成標準：

1. 修正後沒有標題元素互相嵌套。
2. 修正後沒有 `p` 包 `div`。
3. 修正後沒有 `a` 包 `a`。
4. 保留原本的文字與連結意圖。
5. 能用一句話說明瀏覽器容錯不等於原始 HTML 合法。

<details>
<summary>參考解答</summary>

起始檔可改成下列結構：

```html
<section>
  <h1>前端入門課程</h1>
  <h2>HTML 基礎</h2>
</section>

<section>
  <p>歡迎加入本週課程。</p>
  <div>報名截止日：週五晚上 8 點</div>
</section>

<article>
  <a href="https://example.com/course">查看課程介紹</a>
  <p>更多問題請閱讀 <a href="https://example.com/faq">常見問答</a>。</p>
</article>
```

解題說明：

- `h1` 與 `h2` 各自獨立，標題層級由標籤等級與順序表達。
- `p` 只放段落文字，`div` 放在 `p` 外。
- 不用外層 `a` 包住整個文章後再放內層 `a`；如果內容中還需要另一個連結，應拆分可點擊區域。
- 瀏覽器可能會修正錯誤 HTML，但修正後 DOM 不一定符合原始碼預期，可能影響 CSS、JavaScript 與輔助技術理解。

</details>
