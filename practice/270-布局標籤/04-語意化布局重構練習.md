# 語意化布局重構練習

> 動手練習。概念與用法說明見 notes/270-布局標籤/01-html5-語意化布局標籤.md。

## 題目 1（基礎）· 重構題

下面結構用 `div` 和 `class` 表示頁首、導覽與頁尾。請重構成語意化布局標籤。

```html
<div class="header">網站名稱</div>
<div class="nav">首頁 商品 會員中心</div>
<div class="footer">Copyright 2026</div>
```

完成標準：

- 頁首使用 `<header>`。
- 導覽區使用 `<nav>`。
- 頁尾使用 `<footer>`。

<details>
<summary>參考解答</summary>

```html
<header>網站名稱</header>
<nav>首頁 商品 會員中心</nav>
<footer>Copyright 2026</footer>
```

解題說明：語意化布局標籤讓 HTML 本身能說明區塊角色，不必只靠 `class` 猜測結構。
</details>

## 題目 2（進階）· 重構題

下面是一篇教學文章，但整篇文章與內部小節的語意不夠準確。請重構成 `article` 加上多個 `section`。

```html
<div>
  <h2>HTML5 語意化布局</h2>

  <div>
    <h3>常見布局標籤</h3>
    <p>...</p>
  </div>

  <div>
    <h3>article 與 section 的差異</h3>
    <p>...</p>
  </div>
</div>
```

完成標準：

- 整篇可獨立理解的內容使用 `<article>`。
- 文章內有主題的小節使用 `<section>`。

<details>
<summary>參考解答</summary>

```html
<article>
  <h2>HTML5 語意化布局</h2>

  <section>
    <h3>常見布局標籤</h3>
    <p>...</p>
  </section>

  <section>
    <h3>article 與 section 的差異</h3>
    <p>...</p>
  </section>
</article>
```

解題說明：整篇內容是可獨立理解的完整內容，適合用 `<article>`；文章內不同主題的小節則適合用 `<section>`。
</details>

## 題目 3（進階）· 改錯題

下面程式碼把文章內的每個小節都寫成 `<article>`。請修正結構。

```html
<article>
  <h2>HTML5 語意化布局</h2>

  <article>
    <h3>常見布局標籤</h3>
    <p>...</p>
  </article>

  <article>
    <h3>article 與 section 的差異</h3>
    <p>...</p>
  </article>
</article>
```

完成標準：

- 保留外層 `<article>` 表示整篇文章。
- 內部小節改用 `<section>`。

<details>
<summary>參考解答</summary>

```html
<article>
  <h2>HTML5 語意化布局</h2>

  <section>
    <h3>常見布局標籤</h3>
    <p>...</p>
  </section>

  <section>
    <h3>article 與 section 的差異</h3>
    <p>...</p>
  </section>
</article>
```

解題說明：`<article>` 強調完整、獨立；`<section>` 強調同一主題下的分段。文章內的小節不是各自獨立的完整文章時，用 `<section>` 較穩定。
</details>
