# 顯示 HTML 標籤文字練習

> 動手練習。概念與用法說明見 notes/260-字符實體/01-字符實體基本概念.md。

## 題目 1（基礎）· 應用練習題

請讓頁面顯示下面這段文字，而不是建立真正的段落標籤。

```text
<p>Hello</p>
```

完成標準：

- 使用 `&lt;` 顯示 `<`。
- 使用 `&gt;` 顯示 `>`。
- 瀏覽器不應把它解析成真正的 `<p>` 標籤。

<details>
<summary>參考解答</summary>

```html
&lt;p&gt;Hello&lt;/p&gt;
```

解題說明：`&lt;` 會顯示成 `<`，`&gt;` 會顯示成 `>`。瀏覽器看到的是字符實體，所以不會把它當成真正的 `<p>` 標籤執行。
</details>

## 題目 2（基礎）· 應用練習題

請寫一段教學文字，讓畫面顯示：

```text
<img> 是圖片標籤
```

完成標準：

- `<img>` 的小於號和大於號要用字符實體表示。
- 文字應顯示為標籤名稱，而不是建立圖片元素。

<details>
<summary>參考解答</summary>

```html
&lt;img&gt; 是圖片標籤
```

解題說明：如果要在頁面文字中顯示 HTML 標籤，通常需要把 `<` 和 `>` 寫成 `&lt;` 和 `&gt;`。
</details>

## 題目 3（進階）· 改錯題

下面文字想教別人 `<h1>` 是標題標籤，但瀏覽器會把它當成真正標籤。請修正。

```html
<h1> 是標題標籤
```

完成標準：

- 讓畫面顯示 `<h1>` 這段文字。
- 不讓瀏覽器把 `<h1>` 當成真正的標題開始標籤。

<details>
<summary>參考解答</summary>

```html
&lt;h1&gt; 是標題標籤
```

解題說明：直接寫 `<h1>` 容易被瀏覽器解析成真正的 HTML 標籤。改用 `&lt;h1&gt;`，畫面才會顯示 `<h1>` 這段文字。
</details>
