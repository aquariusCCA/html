# 字符實體基本寫法練習

> 動手練習。概念與用法說明見 notes/260-字符實體/01-字符實體基本概念.md。

## 題目 1（基礎）· 應用練習題

請寫出一小段 HTML 文字，讓畫面顯示版權符號。

完成標準：

- 使用命名字符實體。
- 字符實體要有 `&`、實體名稱與結尾分號。
- 畫面應顯示 `©`。

<details>
<summary>參考解答</summary>

```html
&copy;
```

解題說明：命名字符實體的基本形式是 `& + 實體名稱 + ;`。`&copy;` 會在畫面上顯示版權符號 `©`。
</details>

## 題目 2（基礎）· 應用練習題

請改用數字字符引用顯示版權符號。

完成標準：

- 使用 `&#169;` 或 `&#xA9;`。
- 畫面應同樣顯示 `©`。

<details>
<summary>參考解答</summary>

```html
&#169;
```

也可以寫成：

```html
&#xA9;
```

解題說明：HTML 也支援數字字符引用。`&#169;` 和 `&#xA9;` 都可以表示版權符號。
</details>

## 題目 3（進階）· 改錯題

下面字符實體少了一個重要符號。請修正。

```html
&copy
```

完成標準：

- 補上命名字符實體結尾需要的分號。
- 修正後應能穩定顯示 `©`。

<details>
<summary>參考解答</summary>

```html
&copy;
```

解題說明：命名字符實體通常要以分號 `;` 結尾。漏掉分號可能導致瀏覽器解析結果不穩定，也會降低程式碼可讀性。
</details>
