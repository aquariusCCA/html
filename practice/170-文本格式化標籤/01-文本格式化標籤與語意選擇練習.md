# 文本格式化標籤與語意選擇練習

> 動手練習。概念與用法說明見 notes/170-文本格式化標籤/01-文本格式化標籤與語意選擇.md。

## 題目 1：依語意選擇合適的文本標籤

題型：應用練習題  
難度：基礎  
考點：先判斷文字意義，再選擇對應的文本格式化標籤。

請依照下列情境，寫出一段 HTML。每個情境至少使用一次合適的文本格式化標籤。

1. 「警告：請先存檔」是重要提醒。
2. 「我真的需要先完成備份。」中的「真的」需要語氣強調。
3. 「新增內容」是文件修訂後加入的文字。
4. 「已刪除的內容」是文件修訂時刪除的文字。
5. 「關鍵字」只是想讓讀者注意，但不是更重要。
6. 「CSS Grid」是技術名詞。
7. 「錯字」是需要標記的錯誤文字，不是連結。
8. 「原價 100 元」是已不再適用的價格。

完成標準：

- 重要提醒使用具有重要語意的標籤。
- 語氣強調、修訂新增、修訂刪除、注意但不加重語意、技術名詞、非連結特殊標記、已不適用內容，都分別選用符合語意的標籤。
- 不只為了外觀而選擇語意標籤。

<details>
<summary>參考解答</summary>

```html
<p><strong>警告：請先存檔</strong></p>
<p>我<em>真的</em>需要先完成備份。</p>
<p><ins>新增內容</ins></p>
<p><del>已刪除的內容</del></p>
<p><b>關鍵字</b></p>
<p><i>CSS Grid</i></p>
<p><u>錯字</u></p>
<p><s>原價 100 元</s></p>
```

解題說明：

`<strong>` 用於重要或需要優先注意的內容，`<em>` 用於語氣強調，`<ins>` 與 `<del>` 表示文件修訂中的新增與刪除。`<b>` 只是吸引注意但不增加重要性，`<i>` 可用於技術名詞或不同語氣，`<u>` 可標示非連結的特殊文字，`<s>` 則表示內容已不準確或不再適用。

</details>

## 題目 2：修正只為了粗體而誤用 strong 的程式碼

題型：改錯題  
難度：基礎  
考點：純外觀效果應交給 CSS，不應誤用具有語意的標籤。

以下程式碼只是想讓「新品上市」變粗，並不表示它是警告、錯誤或必須優先注意的重要內容。請修正 HTML，並補上 CSS。

```html
<strong>新品上市</strong>
```

完成標準：

- 不使用 `<strong>` 只製造粗體外觀。
- HTML 使用中性的包裹元素。
- CSS 負責設定粗體外觀。

<details>
<summary>參考解答</summary>

```html
<span class="highlight">新品上市</span>
```

```css
.highlight {
  font-weight: 700;
}
```

解題說明：

如果「新品上市」只是要視覺上比較醒目，而不是要表達重要或嚴重語意，就不應使用 `<strong>`。可以用 `<span>` 包住文字，再由 CSS 的 `font-weight` 控制粗體外觀。

</details>

## 題目 3：修正只為了底線而誤用 ins 的程式碼

題型：改錯題  
難度：基礎  
考點：`<ins>` 表示修訂新增內容，不是通用底線工具。

以下程式碼只是想讓「活動重點」有底線，並不是文件修訂後新增的文字。請修正 HTML，並補上 CSS。

```html
<ins>活動重點</ins>
```

完成標準：

- 不使用 `<ins>` 只製造底線外觀。
- HTML 使用中性的包裹元素。
- CSS 負責設定底線外觀。

<details>
<summary>參考解答</summary>

```html
<span class="underlined">活動重點</span>
```

```css
.underlined {
  text-decoration: underline;
}
```

解題說明：

`<ins>` 的語意是文件修訂中新增的內容。如果只是想呈現底線，應改用 `<span>` 搭配 CSS 的 `text-decoration: underline;`。

</details>

## 題目 4：修正促銷原價誤用 del 的程式碼

題型：改錯題  
難度：進階  
考點：分辨文件修訂刪除與已不再適用的內容。

以下程式碼用來呈現促銷價格。`原價 100 元` 是不再適用的價格，不是文件修訂時被刪除的文字。請修正標籤，並保留特價的重要提示。

```html
<del>原價 100 元</del>
<strong>特價 80 元</strong>
```

完成標準：

- 原價使用表示內容已不再適用的標籤。
- 不使用 `<del>` 表示促銷原價。
- 特價若需要被優先注意，可以保留 `<strong>`。

<details>
<summary>參考解答</summary>

```html
<s>原價 100 元</s>
<strong>特價 80 元</strong>
```

解題說明：

`<del>` 適合表示文件修訂中刪除的內容；促銷頁面中的原價通常是已不再適用或不再準確的內容，因此用 `<s>` 更符合語意。`特價 80 元` 若是希望讀者優先注意的重要價格資訊，可以保留 `<strong>`。

</details>
