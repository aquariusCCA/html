# div 與 span 盒子標籤練習

> 動手練習。概念與用法說明見 notes/120-盒子標籤/01-div與span盒子標籤.md。

## 題目 1（基礎）· 應用練習題

請依照下列需求寫出一段 HTML：

- 用一個容器包住一整張課程卡片。
- 卡片內要有課程名稱與一段課程說明。
- 在課程說明中，只把「HTML 結構」這幾個字單獨包起來，方便之後局部套樣式。

完成標準：

- 一整張卡片使用適合包住區塊內容的標籤。
- 句子中的局部文字使用適合行內片段的標籤。
- 不把整段區塊內容放進 `span`。

<details>
<summary>參考解答</summary>

```html
<div class="card">
  <h2>HTML 入門課程</h2>
  <p>這堂課會帶你認識 <span class="keyword">HTML 結構</span> 的基本寫法。</p>
</div>
```

解題說明：`<div>` 是沒有語意的區塊容器，適合包住卡片、區域或版面外框這類整塊內容；`<span>` 是沒有語意的行內容器，適合包住一句話中的局部文字。這題只需要把卡片當成大盒子，把「HTML 結構」當成小盒子。

</details>

## 題目 2（基礎）· 改錯題

下面這段 HTML 使用 `span` 包住整個消息區塊。請修正它，讓外層容器更符合內容結構。

```html
<span>
  <h2>最新消息</h2>
  <p>這裡是最新消息內容。</p>
</span>
```

完成標準：

- 不使用 `span` 承擔整個區塊的外層結構。
- 修正後仍保留「最新消息」標題與內容段落。
- 解答只使用 notes 已教過的 `div`、標題標籤與段落標籤。

<details>
<summary>參考解答</summary>

```html
<div>
  <h2>最新消息</h2>
  <p>這裡是最新消息內容。</p>
</div>
```

解題說明：`<span>` 適合包住行內的小片段，不適合拿來承擔整個區塊的結構。這裡外層包住的是一整塊「最新消息」內容，因此改用 `<div>` 這個區塊容器更合適。`<h2>` 和 `<p>` 則保留原本的標題與段落語意。

</details>

## 題目 3（進階）· 重構題

下面這段 HTML 畫面可能看得出標題與段落，但整頁都只用 `div`，結構語意不清楚。請把它重構成更適合的 HTML。

```html
<div>我的學習筆記</div>
<div>今天學到 div 是沒有語意的區塊容器。</div>
<div>span 適合包住一句話中的小片段。</div>
```

完成標準：

- 頁面主標題不要只用 `div` 表示。
- 兩段正文不要只用 `div` 表示。
- 重構後的 HTML 要能清楚表達哪裡是標題、哪裡是段落。

<details>
<summary>參考解答</summary>

```html
<h1>我的學習筆記</h1>
<p>今天學到 div 是沒有語意的區塊容器。</p>
<p>span 適合包住一句話中的小片段。</p>
```

解題說明：`<div>` 和 `<span>` 都沒有語意，適合用在沒有更合適語意標籤的情況。這段內容中，「我的學習筆記」是頁面主標題，應該使用標題標籤；後面兩句是正文段落，應該使用 `<p>`。這樣瀏覽器、搜尋引擎、輔助工具與其他工程師都比較容易理解頁面結構。

</details>
