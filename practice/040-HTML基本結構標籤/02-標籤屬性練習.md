# 標籤屬性練習

> 動手練習。概念與用法說明見 `notes/040-HTML基本結構標籤/02-標籤屬性.md`。

---

## 題目 1（基礎）· 應用練習題

請依下列需求，寫出一個完整的 `<input>` 標籤：

1. 輸入框類型為文字（`type="text"`）
2. 表單送出時的欄位名稱為 `username`
3. 頁面識別名稱（id）為 `username-field`
4. 分類名稱（class）為 `form-control`
5. 這個輸入框目前被**停用**（使用布林屬性）

完成標準：5 個屬性全部正確寫出，屬性之間有空格，布林屬性格式正確。

<details>
<summary>參考解答</summary>

```html
<input type="text" name="username" id="username-field" class="form-control" disabled>
```

解題說明：

- `type="text"`、`name="username"`、`id="username-field"`、`class="form-control"` 都使用 `屬性名="屬性值"` 的格式，屬性值加雙引號。
- 屬性之間**必須用空格隔開**，否則瀏覽器無法分辨屬性的邊界。
- `disabled` 是**布林屬性**，只要屬性名存在就代表「停用」，不需要也不必寫屬性值。

</details>

## 題目 2（基礎）· 改錯題

下面這段 HTML 有 **4 個屬性書寫錯誤**，請逐一找出並修正。

```html
<pclass="intro">這是介紹段落。</p>

<input type="text"name="email">

<input type="text" type="password">

<div class=card primary>主要卡片</div>
```

完成標準：4 處錯誤全部修正，且能說明每一處的問題所在。

<details>
<summary>參考解答</summary>

```html
<p class="intro">這是介紹段落。</p>

<input type="text" name="email">

<input type="password">

<div class="card primary">主要卡片</div>
```

四個錯誤說明：

1. **`<pclass="intro">`：標籤名與屬性之間缺少空格** → 瀏覽器不會把這理解成 `<p>` 加 `class` 屬性，而是視為未知標籤名。應在 `p` 和 `class` 之間加一個空格：`<p class="intro">`。

2. **`type="text"name="email"`：屬性之間缺少空格** → 兩個屬性連在一起，瀏覽器無法正確分辨邊界。應在兩個屬性之間加一個空格：`type="text" name="email"`。

3. **`type="text" type="password"`：同名屬性重複** → 同一個標籤不能有兩個 `type`，設定會互相衝突。應先決定真正需要的類型，只保留一個（此例保留 `type="password"`）。

4. **`class=card primary`：包含空格的屬性值省略引號** → 沒有引號時，瀏覽器只會把 `card` 當成屬性值，`primary` 則被誤讀為另一個屬性名稱。屬性值有空格時必須加引號：`class="card primary"`。

</details>
