# 選項控件練習

> 動手練習。概念與用法說明見 notes/240-表單標籤/03-單選複選與下拉選項.md。

## 題目 1（基礎）· 應用練習題

請建立一組「付款方式」單選選項，選項有信用卡與轉帳。

完成標準：

- 使用 `type="radio"`。
- 同一組選項共用 `name="payment"`。
- 每個選項都有不同的 `value`。
- 文字可以點擊選取對應選項。

<details>
<summary>參考解答</summary>

```html
<input type="radio" id="payment-card" name="payment" value="card">
<label for="payment-card">信用卡</label>

<input type="radio" id="payment-transfer" name="payment" value="transfer">
<label for="payment-transfer">轉帳</label>
```

解題說明：同一組 radio 必須共用相同的 `name`，才能形成多選一。`value` 是被選中時提交的值。
</details>

## 題目 2（基礎）· 應用練習題

請建立一組「興趣」複選選項，包含閱讀、音樂、運動，其中閱讀預設勾選。

<details>
<summary>參考解答</summary>

```html
<input type="checkbox" id="hobby-reading" name="hobby" value="reading" checked>
<label for="hobby-reading">閱讀</label>

<input type="checkbox" id="hobby-music" name="hobby" value="music">
<label for="hobby-music">音樂</label>

<input type="checkbox" id="hobby-sport" name="hobby" value="sport">
<label for="hobby-sport">運動</label>
```

解題說明：checkbox 可讓使用者選多個項目。多個 checkbox 可共用同一個 `name`，並各自用 `value` 表示實際提交值。
</details>

## 題目 3（基礎）· 應用練習題

請建立一個地區下拉列表，欄位名稱為 `province`，選項有黑龍江、遼寧、吉林、廣東，其中廣東預設選中。

<details>
<summary>參考解答</summary>

```html
<label for="province">地區</label>
<select id="province" name="province">
  <option value="heilongjiang">黑龍江</option>
  <option value="liaoning">遼寧</option>
  <option value="jilin">吉林</option>
  <option value="guangdong" selected>廣東</option>
</select>
```

解題說明：`<select>` 定義下拉列表，`<option>` 定義選項。`selected` 表示預設選中的選項。
</details>

## 題目 4（進階）· 改錯題

下面兩個 radio 不會互斥，請修正。

```html
<input type="radio" name="male" value="male">男
<input type="radio" name="female" value="female">女
```

<details>
<summary>參考解答</summary>

```html
<input type="radio" name="gender" value="male">男
<input type="radio" name="gender" value="female">女
```

解題說明：同一組 radio 必須共用相同的 `name`。原本兩個 `name` 不同，瀏覽器會把它們視為不同組。
</details>
