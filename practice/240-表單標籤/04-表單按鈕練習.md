# 表單按鈕練習

> 動手練習。概念與用法說明見 notes/240-表單標籤/04-表單按鈕與提交行為.md。

## 題目 1（基礎）· 應用練習題

請建立一個註冊表單，包含帳號欄位與提交按鈕。提交按鈕使用 `<button>`。

<details>
<summary>參考解答</summary>

```html
<form action="/signup" method="post">
  <input type="text" name="account">
  <button type="submit">註冊</button>
</form>
```

解題說明：`type="submit"` 會送出所在 `<form>` 的資料。使用 `<button>` 時，可見文字寫在開始與結束標籤之間。
</details>

## 題目 2（基礎）· 應用練習題

請建立一個有初始暱稱的表單，並加入「重新填寫」按鈕，讓欄位回到初始狀態。

<details>
<summary>參考解答</summary>

```html
<form action="/profile" method="post">
  <input type="text" name="nickname" value="小明">
  <button type="reset">重新填寫</button>
</form>
```

解題說明：`type="reset"` 會把表單控件恢復成初始狀態。若欄位原本有 `value="小明"`，重置後會恢復成小明。
</details>

## 題目 3（進階）· 改錯題

下面的「取得驗證碼」按鈕看起來只是普通操作，但在表單中可能意外提交。請修正。

```html
<form action="/login" method="post">
  <input type="text" name="account">
  <button>取得驗證碼</button>
</form>
```

<details>
<summary>參考解答</summary>

```html
<form action="/login" method="post">
  <input type="text" name="account">
  <button type="button">取得驗證碼</button>
</form>
```

解題說明：在表單裡，沒有設定 `type` 的 `<button>` 通常會被當成提交按鈕。普通操作按鈕應明確寫 `type="button"`。
</details>

## 題目 4（進階）· 改錯題

下面 `<button>` 的可見文字寫錯位置了。請修正。

```html
<button type="submit" value="送出"></button>
```

<details>
<summary>參考解答</summary>

```html
<button type="submit">送出</button>
```

解題說明：`<button>` 的可見文字應寫在標籤內容裡。`value` 可以作為提交值，但不是顯示按鈕文字的主要方式。
</details>
