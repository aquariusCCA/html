# 圖片格式與 Base64 練習

> 動手練習。概念與用法說明見 notes/180-圖片標籤/02-常見圖片格式與Base64編碼.md。

## 題目 1（基礎）· 應用練習題

請依照下列圖片用途，選擇合適的檔案格式並寫出 `<img>` 標籤。

1. 商品情境照片：檔名 `product-photo`，替代文字「商品情境照片」。
2. 透明背景的公司標誌：檔名 `logo`，替代文字「公司標誌」。
3. 簡單的載入中動畫：檔名 `loading`，替代文字「載入中」。
4. 活動主視覺，希望使用現代 Web 圖片格式：檔名 `banner`，替代文字「活動主視覺」。

完成標準：

- 照片或顏色豐富且不需要透明背景的圖片，使用 JPG 或 WebP。
- 透明 Logo 使用 PNG 或 WebP。
- 簡單動畫使用 GIF 或 WebP。
- 現代 Web 圖片格式可使用 WebP。
- 每張圖片都保留有意義的 `alt`。

<details>
<summary>參考解答</summary>

```html
<img src="product-photo.jpg" alt="商品情境照片">
<img src="logo.png" alt="公司標誌">
<img src="loading.gif" alt="載入中">
<img src="banner.webp" alt="活動主視覺">
```

解題說明：

JPG 常用於照片或產品宣傳圖，PNG 適合透明背景的 Logo 或圖示，GIF 可用於簡單動畫，WebP 是現代 Web 圖片格式，適合多種網頁圖片情境。`alt` 仍要描述圖片代表的內容。

</details>

## 題目 2（進階）· 應用練習題

你想優先提供 `banner.webp`，但專案需要兼容不支援 WebP 的瀏覽器。請使用 `<picture>` 寫出 WebP 與 JPG 備用格式，替代文字為「活動主視覺」。

完成標準：

- 使用 `<picture>` 包住圖片來源。
- 使用 `<source>` 提供 `banner.webp`。
- 使用 `<img>` 提供 `banner.jpg` 作為備用圖片。
- `<img>` 保留有意義的 `alt`。

<details>
<summary>參考解答</summary>

```html
<picture>
  <source srcset="banner.webp" type="image/webp">
  <img src="banner.jpg" alt="活動主視覺">
</picture>
```

解題說明：

`<picture>` 可以提供不同圖片格式。瀏覽器支援 WebP 時會使用 `banner.webp`；不支援時，仍可使用 `<img>` 的 `banner.jpg`。替代文字放在 `<img>` 上，讓圖片仍有可替代的內容資訊。

</details>

## 題目 3（基礎）· 改錯題

以下寫法把 Base64 說成圖片格式，且沒有清楚表示它是 Data URL。請改成比較精確的寫法，並保留一個小型內嵌圖示的 `<img>` 範例。

```html
<!-- base64 格式圖片 -->
<img src="base64" alt="圖示">
```

完成標準：

- 不把 Base64 稱為圖片格式。
- 說明 Base64 是編碼方式。
- `src` 使用 Data URL 形式。
- 替代文字描述小圖示的內容。

<details>
<summary>參考解答</summary>

```html
<!-- Base64 是編碼方式，可用 Data URL 內嵌小圖示 -->
<img src="data:image/png;base64,..." alt="內嵌小圖示">
```

解題說明：

Base64 不是像 JPG、PNG 那樣的圖片格式，而是把二進位資料轉成文字的編碼方式。圖片經過 Base64 編碼後，常見寫法是把資料放在 `data:image/png;base64,...` 這類 Data URL 中。

</details>

## 題目 4（進階）· 改錯題

以下程式碼把大型商品照片直接寫成 Base64，會讓 HTML 變得很大，也不利於圖片資源管理。請改成一般獨立圖片檔案引用。

```html
<img src="data:image/jpeg;base64,..." alt="商品情境照片">
```

完成標準：

- 大型商品照片不使用 Base64 內嵌。
- 改用獨立圖片檔案路徑。
- 保留有意義的 `alt`。
- 使用適合照片的格式。

<details>
<summary>參考解答</summary>

```html
<img src="product-photo.jpg" alt="商品情境照片">
```

解題說明：

Base64 適合非常小、需要內嵌的圖片資料，不適合商品照片、大型背景圖或很多張圖片。大型圖片通常應作為獨立圖片檔案引用；照片類圖片可使用 JPG 或 WebP。

</details>
