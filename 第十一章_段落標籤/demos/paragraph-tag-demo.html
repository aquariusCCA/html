HTML 段落標籤 Demo

`

` 代表的是「一段完整內容」，不是單純想把文字往下推一行。這個 demo 會把段落語意、預設段距、`
` 的用途，以及常見的錯誤期待拆開對照。

`<p>` 的角色 用來表示一段文字內容，適合承載一個完整意思的敘述。
預設段距 瀏覽器通常會在段落上下保留空間，所以段落和段落之間會自然分開。
`<p>` vs `<br>` `p` 是段落，`br` 是同一段裡的換行，兩者用途不同。
常見誤解 HTML 不會照單全收多個空白與純換行輸入，不能把它當排版工具。

    `，什麼情況不該濫用

1. `

` 的基本段落行為

Paragraph

`

` 描述的是一整段意思

當一段文字表達的是同一個完整意思時，就適合放進一個 `

`。它不是拿來做機械式空行的工具。 


          


          

        

      


      

        
2. `
` 和 `
` 的差異

        

          

            段落
            
`
` 用來切分不同意思

            
<p>第一段：這是一段完整介紹。</p>
<p>第二段：這是另一段新的說明。</p>

            

              
第一段：這是一段完整介紹。

              
第二段：這是另一段新的說明。

            

            

              兩段內容之間有段距，因為它們代表的是兩個段落。
            

          


          

            同段換行
            
`
` 只是在同一段裡換一行

            
<p>
  收件人：王小明<br>
  地址：台北市信義區...<br>
  電話：02-1234-5678
</p>

            

              收件人：王小明

              地址：台北市信義區...

              電話：02-1234-5678
            

            

              這仍然是同一段內容，只是中間需要明確斷行。
            

          

        

      


      

        
3. 為什麼空白與純換行不可靠

        

          

            

              
              
            

          


          

            
預設

            
多個空白不會直接變成排版

            

              HTML 在一般情況下會把多個空白與換行壓縮處理，所以不能靠狂敲空白鍵或 Enter 來做段落布局。
            

            

            

              

            

          

        

      


      

        
4. 什麼時候應該用 `
`

        

          

            

              
              
            

          


          

            
建議

            
用段落承載完整敘述

            

              當內容本身自然形成一段意思時，用 `
` 會讓結構更清楚，也更符合 HTML 語意。
            

            

            

              判斷方式很簡單：如果你願意把它當成一個完整意思念完，多半就適合是一個段落。
            

          

        

      


      

        
5. 常見實務提醒

        

          

            段落是內容單位
            不要把 `
` 當作視覺上的空白墊片，而要把它當成內容結構的一部分。
          

          

            `
` 只解決斷行
            例如地址、詩句、歌詞、簽名檔，才比較適合同段內多次換行。
          

          

            間距交給 CSS
            如果只是想調整段落距離，應該修改 `margin`，而不是亂塞空段落。
