# 第十一章｜段落標籤示例

這個資料夾收錄第十一章筆記對應的可執行示例頁。  
這章的重點是理解 `<p>` 代表「段落」而不是「單純換行」，以及它和 `<br>`、純文字換行、空白輸入之間的差異。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十一章 Markdown 筆記。
- 想直接看段落間距與換行差異時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [paragraph-tag-demo.html](./paragraph-tag-demo.html)：整合展示 `<p>` 的段落語意、瀏覽器預設段距、`<p>` 與 `<br>` 的差異，以及錯誤地期待多個空白與換行會直接呈現的情況。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
