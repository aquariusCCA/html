# 段落標籤

> 所屬章節：第十一章｜段落標籤  
> 關鍵字：段落標籤、`<p>`、paragraph、分段顯示、自動換行、段落間距、文章排版、HTML 段落  
> 建議回查情境：想知道 `<p>` 是做什麼的、想確認段落標籤有哪些預設特點、想分清分段顯示和單純換行的差別

## 本節導讀

這篇整理 HTML 的段落標籤 `<p>`。  
它的核心作用，是把一段連續文字標示成一個段落，讓文章、新聞或說明內容更容易閱讀。

初學 HTML 時，很多人會先注意到段落之間有空隙、文字會自動換行。  
但更重要的是理解：`<p>` 是在描述「這裡是一個完整段落」，不只是讓畫面看起來分成幾塊。

## 你會在這篇學到什麼

- `<p>` 標籤的基本用途
- 段落標籤的常見顯示特點
- 為什麼文章內容常用 `<p>` 來分段
- 段落和單純換行在語意上有什麼不同

## 30 秒複習入口

- `<p>` 用來把 HTML 文件分成一個一個段落。
- 段落很適合放文章、新聞或說明文字。
- 瀏覽器通常會讓不同段落之間保有空隙。
- 同一段落中的文字會隨視窗寬度自動換行。

## 速查區

### 核心概念

- `<p>` 是 HTML 中最基本的文字段落標籤。
- 它代表的是一整段內容，而不是單純插入一個空行。

### 關鍵規則 / 判準

- 基本寫法是 `<p>...</p>`。
- 一個 `<p>` 通常包住一段意思相對完整的文字。
- 如果內容已經換成另一段新意思，通常就適合另開一個 `<p>`。
- 文字在段落中會依可用寬度自動換行，不需要手動在每行結尾都切段。

### 常見特點

- 段落和段落之間通常有空隙。
- 每個段落通常各自佔據一個區塊。
- 段落內的文字會隨瀏覽器視窗大小改變而重新換行。

### 常見混淆點

- 段落是內容結構，不只是視覺空白。
- 自動換行不代表產生了新段落；只有新的 `<p>` 才代表新的段落。
- 如果只是想調整外觀間距，仍應把內容結構和樣式控制分開理解。

### 一句話對比

- `<p>` 解決的是「這段文字是不是一個完整段落」，不是「畫面上要不要空一行」。

## 正文筆記

### 這篇在解決什麼問題？

- 在新聞、文章或說明文件裡，文字通常不會全部擠成一大塊。
- `<p>` 就是在處理這件事：把文件切成一段一段較容易閱讀的內容。

## 1. `<p>` 的作用

> 可以把 HTML 文檔分割為若干個段落。

- 這句話的意思是：HTML 可以用 `<p>` 來標示一段完整的文字內容。
- 當文件中有不同段落時，就可以用多個 `<p>` 依序分開。

## 2. 常見使用場景

- 在新聞頁面中，常用 `<p>` 將不同段落的敘述分開。
- 在文章或教學說明中，也常用 `<p>` 讓內容更有節奏，讀者比較不容易看成一整塊文字。

## 3. 段落標籤的特點

- 段落和段落之間通常會保有空隙。
- 文本在一個段落中，會根據瀏覽器視窗大小自動換行。

### 這些特點怎麼理解？

- 空隙讓讀者更容易看出段落邊界。
- 自動換行表示你不需要為了每一行文字的長度去手動切割內容。
- 真正決定是否為新段落的，是新的 `<p>`，不是視窗縮窄後文字自然換到下一行。

## 4. 基本示例

```html
<p>
    在中國IT教育行業發展的軌跡中，能始終秉承著初心做教育的企業為數不多，而在李開復老師的心中，傳智播客就是其中之一。在11年的風雨歷程中，傳智播客在中國IT教育行業留下了“為莘莘學子改變命運而講課，為千萬學生少走彎路而著書”的獨特烙印。同時，傳智播客也憑藉著良好的口碑、更注重品質的教學及優秀的業績增長成功吸引了眾多知名投資者們的青睞，其中不乏創新工場、經緯、君度、藍圖、北城壹號等多個投資機構。
</p>

<p>
    不久前，傳智播客新加入了幾位知名股東，其中包括創投圈的大咖——創新工場的李開復老師。在中國，李開復老師至少影響了70和80後兩代人，然而翻開他的履歷，我們並沒有看到諸如馬雲、馬化騰那樣堪稱傳奇的成功經歷，我們看到的只是一位厚積薄發、深耕在世界IT互聯網行業多年的"導師"：
</p>
```

### 這個例子在說什麼？

- 這裡用了兩個 `<p>`，代表文件中有兩個不同段落。
- 每個段落都包住一整段完整文字，而不是只包其中一兩句。
- 讀者在畫面上看到的，不只是兩塊分開的文字，也是兩個彼此獨立的段落單位。

## 5. 使用 `<p>` 時的注意點

- 不要把整篇長文全部塞進同一個 `<p>`，否則閱讀上會很吃力。
- 也不要因為畫面想留白，就把其實屬於同一段意思的文字硬拆成很多個段落。
- 是否要開新段落，應看內容是不是已經換到新的意思或新的敘述重點。

## 常見問法

### 為什麼同一個 `<p>` 裡的文字會自己換行？

- 因為瀏覽器會依目前可用寬度自動排版文字。
- 這種換行只是顯示上的自然換行，不代表你建立了新段落。

### 什麼時候應該開新的 `<p>`？

- 當內容已經進入新的意思、新的敘述重點，或閱讀上需要明顯分段時，就適合開新的 `<p>`。

### 段落和空白行是一樣的事嗎？

- 不一樣。
- 空白行偏向視覺效果；段落則是在描述內容本身的結構。
