# 第十一章｜段落標籤

> 章節定位：建立 HTML 段落結構與文字分段的基本判讀框架  
> 建議閱讀方式：先分清段落是在描述內容分段，不是單純空一行，再回頭看 `<p>` 的實際用法

## 本章在學什麼

這一章整理 HTML 的段落標籤 `<p>`。  
重點不是只記住它會產生空隙，而是理解它在文章、新聞與說明文件中，如何把文字組織成可閱讀的段落。

## 本章與整體地圖的關係

- 前置知識：第二章已建立 HTML 標籤基本觀念，第十章已整理標題標籤的內容層級概念。
- 這章的核心任務：分清段落標籤的用途、預設特點，以及它和單純視覺留白的差別。
- 讀完後通常接：之後在整理文章型頁面時，可以把標題與段落搭配起來看。
