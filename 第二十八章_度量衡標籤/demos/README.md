# 第二十八章｜度量衡標籤示例

這個資料夾收錄第二十八章筆記對應的可執行示例頁。  
這章的重點是理解 `<meter>` 如何表示已知範圍內的測量值，並分清它和 `<progress>` 的語意差異。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第二十八章 Markdown 筆記。
- 想直接觀察 `value`、`min`、`max`、`low`、`high`、`optimum` 的效果時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [meter-tags-demo.html](./meter-tags-demo.html)：整合展示 `<meter>` 的基本用法、電量、磁碟用量、分數、溫度、`meter` / `progress` 差異、預設範圍陷阱、常見錯誤，以及可調整 `value` / `low` / `high` / `optimum` 的互動範例。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
