HTML 表格標籤 Demo

這個範例把第二十三章的重點做成可操作頁面：表格適合表達有欄位、列與對照關係的資料，不是拿來把頁面硬切成格子。先看資料是否需要橫向或縱向比較，再決定是否使用 `table`。

`table` 表示整張表 表格適合資料展示，例如成績表、商品規格、報表與後台資料清單。
`th` 是表頭，`td` 是資料 兩者差別不只是粗體或置中，而是語意角色不同。
`caption` 與分區讓結構更清楚 `thead`、`tbody`、`tfoot` 可把表頭、主體與彙總區分開。
合併要記方向 `colspan` 向右合併，`rowspan` 向下合併，且要刪掉被吃掉的格子。

本頁會示範

  • `table`、`tr`、`th`、`td` 的基本結構
  • `caption`、`thead`、`tbody`、`tfoot` 的分工
  • 舊式表格屬性與 CSS 樣式控制
  • `colspan`、`rowspan` 的合併方向

3. 舊式表格屬性 vs CSS

        

          

            舊教材常見
            
認得 `border`、`cellpadding`、`cellspacing`

            
<table border="1" cellpadding="20" cellspacing="0" width="500">
  ...
</table>

            

              這些屬性在舊教材或舊專案裡可能看得到，初學時需要認得它們大致在控制什麼。
            

          


          

            現代實務
            
外觀多半交給 CSS

            
table {
  border-collapse: collapse;
  width: 100%;
}

th,
td {
  border: 1px solid #d8d1c3;
  padding: 12px 14px;
}

            

              

                

                  樣式項目
                  現代做法
                

                

                  邊框
                  CSS border
                

                

                  內距
                  CSS padding
                

              

            

          

        

      


      

        
4. 表格不是整頁布局工具

        

          

            適合表格
            
資料有欄位、列與對照關係

            
例如成績表、規格比較表、營收統計、後台資料清單。

          


          

            不適合表格
            
只是想把畫面排整齊

            
一般頁面排版應優先使用語意標籤與 CSS，而不是拿表格硬切版。

          


          

            先問資料本質
            
讀者是否需要比較欄列？

            
如果答案是肯定的，表格通常很合適。

          


          

            結構化表格
            
分區有助維護

            
`caption`、`thead`、`tbody`、`tfoot` 讓表格主題、欄位、資料與彙總更清楚。

          

        

      


      

        
5. 實務提醒

        

          

            `th` 和 `td` 不要只按外觀選
            欄位名稱用 `th`，實際資料用 `td`，這是語意差異。
          

          

            合併時刪掉被吃掉的格子
            某格使用 `colspan` 或 `rowspan` 後，被合併的位置不能再保留多餘儲存格。
          

          

            舊式屬性要認得，但別依賴
            `border`、`cellpadding`、`cellspacing` 可理解，實務樣式優先交給 CSS。
