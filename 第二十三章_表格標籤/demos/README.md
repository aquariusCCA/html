# 第二十三章｜表格標籤示例

這個資料夾收錄第二十三章筆記對應的可執行示例頁。  
這章的重點是理解表格如何表達有欄位、列與對照關係的結構化資料，並分清基本結構、表格分區與合併單元格。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第二十三章 Markdown 筆記。
- 想直接對照表格結構與合併方向時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [table-tags-demo.html](./table-tags-demo.html)：整合展示 `table` / `tr` / `th` / `td` 的基本結構、`caption` / `thead` / `tbody` / `tfoot` 的分工、舊式表格屬性與 CSS 的取捨，以及 `colspan` / `rowspan` 的合併方向。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
