# 第二十三章｜表格標籤

> 章節定位：學會用 HTML 表達具有欄位與列關係的資料  
> 建議閱讀方式：先理解表格用途與基本結構，再看區塊分工、舊式屬性與合併單元格

## 本章在學什麼

這一章整理 HTML 的表格標籤。  
重點不只是把資料排成格子，而是理解什麼資料適合用表格、表格有哪些區塊，以及怎麼正確使用合併單元格。

## 本章與整體地圖的關係

- 前置知識：前面已經學過基本標籤、列表與超鏈接，這些都能幫助理解表格中的內容如何組成。
- 這章的核心任務：建立表格結構觀念，分清 `th` / `td` 與 `caption` / `thead` / `tbody` / `tfoot` 的角色。
- 讀完後通常接：之後在做成績表、後台列表、資料統計與規格比較表時，都能回來快速查這一章。

## 易混淆主題

- 表格 vs 布局：表格是在表達資料關係，不是拿來做一般頁面排版。
- `th` vs `td`：差別不只是粗體外觀，還在於表頭與資料語意。
- `rowspan` vs `colspan`：一個向下跨列，一個向右跨欄。
