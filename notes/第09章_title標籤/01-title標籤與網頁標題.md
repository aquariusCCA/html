# title 標籤與網頁標題

## 學習目標

讀完這篇筆記，你應該能夠：

- 理解 `<title>` 標籤用來設定 HTML 文件的頁面標題。
- 知道 `<title>` 應該寫在 `<head>` 裡。
- 寫出基本的 `<title>` 語法。
- 分辨 `<title>` 和頁面正文標題的不同。

## 問題情境

當你打開一個網頁時，瀏覽器分頁通常會顯示一段文字。例如搜尋網站、購物網站或文章頁面，都會有自己的分頁標題。

這段標題不是寫在畫面中的 `<h1>`，而是寫在 HTML 文件的 `<head>` 裡，透過 `<title>` 標籤設定。

## 一句話理解

`<title>` 是用來設定「這份 HTML 文件的標題」的標籤，常見用途是顯示在瀏覽器分頁、書籤，以及搜尋結果標題的參考資訊中。

## `<title>` 的主要作用

`<title>` 至少有兩個重要作用。

第一，它告訴使用者目前頁面或文章的主題是什麼。當使用者同時開啟多個分頁時，分頁標題能幫助他快速分辨每個頁面的內容。

第二，它能幫助搜尋引擎理解頁面主題。搜尋結果中顯示的標題不一定完全等於 `<title>` 內容，但 `<title>` 通常是搜尋引擎判斷頁面標題的重要依據之一。

## 基本語法

`<title>` 要寫在 `<head>` 裡，標題文字直接放在開始標籤與結束標籤之間：

```html
<head>
  <title>這是我們的專屬網站</title>
</head>
```

完整 HTML 文件可以這樣寫：

```html
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>這是我們的專屬網站</title>
</head>
<body>
  <h1>歡迎來到我的網站</h1>
</body>
</html>
```

這個範例中：

- `<title>這是我們的專屬網站</title>` 設定瀏覽器分頁上的頁面標題。
- `<h1>歡迎來到我的網站</h1>` 是顯示在頁面內容區的主標題。
- 兩者可以相似，但用途不同。

## `<title>` 和 `<h1>` 的差異

| 項目 | `<title>` | `<h1>` |
| --- | --- | --- |
| 放置位置 | `<head>` 裡 | `<body>` 裡 |
| 是否顯示在頁面內容區 | 通常不會 | 會 |
| 主要用途 | 設定文件標題、分頁標題、搜尋結果參考 | 表示頁面正文的主要標題 |

初學時可以這樣記：

- `<title>` 是給瀏覽器、書籤、搜尋引擎和分頁標題使用的文件標題。
- `<h1>` 是給使用者在頁面內容中閱讀的主標題。

## 常見錯誤

### 把 `<title>` 寫到 `<body>` 裡

錯誤寫法：

```html
<body>
  <title>我的網站</title>
  <h1>我的網站</h1>
</body>
```

建議寫法：

```html
<head>
  <title>我的網站</title>
</head>
<body>
  <h1>我的網站</h1>
</body>
```

`<title>` 是文件資訊，應該放在 `<head>` 中；頁面中要讓使用者看見的標題，應該放在 `<body>` 中並使用合適的標題標籤。

### 標題太籠統

不建議：

```html
<title>首頁</title>
```

較清楚：

```html
<title>HTML 入門教學首頁</title>
```

如果每個頁面都叫「首頁」或「我的網站」，使用者很難從分頁或書籤中分辨頁面內容，搜尋引擎也比較不容易理解每個頁面的主題差異。

### 忘記結束標籤

不建議：

```html
<title>這是我們的專屬網站
```

建議寫法：

```html
<title>這是我們的專屬網站</title>
```

`<title>` 需要開始標籤與結束標籤，標題文字放在兩者之間。

## 實務使用原則

撰寫 `<title>` 時，可以先遵守幾個簡單原則：

- 每個 HTML 頁面都應該有清楚的 `<title>`。
- 標題要能反映頁面主題，不要只寫「頁面」、「首頁」或「測試」。
- 同一個網站的不同頁面，最好使用不同的標題。
- `<title>` 不負責顯示頁面正文標題；正文標題仍應寫在 `<body>` 中。

例如文章頁面可以寫成：

```html
<title>title 標籤的作用與寫法 - HTML 教學筆記</title>
```

這樣使用者可以從分頁標題看出頁面主題，也能知道它屬於哪一組網站或教材內容。

## 小練習

請為一個「個人作品集首頁」設計 `<title>`，並把它放進完整的 HTML 基本結構中。

可以先從這個骨架開始：

```html
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>你的頁面標題</title>
</head>
<body>
  <h1>你的頁面主標題</h1>
</body>
</html>
```

完成後檢查：

- `<title>` 是否放在 `<head>` 裡？
- `<title>` 是否能清楚說明頁面主題？
- `<h1>` 是否放在 `<body>` 裡，作為頁面可見的主標題？

## 重點整理

- `<title>` 用來設定 HTML 文件的頁面標題。
- `<title>` 應該放在 `<head>` 中。
- `<title>` 的文字常用於瀏覽器分頁、書籤和搜尋結果標題參考。
- `<title>` 和 `<h1>` 不同，前者是文件標題，後者是頁面正文標題。
- 好的 `<title>` 應該清楚、具體，能反映頁面主題。
