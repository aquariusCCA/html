# rowspan 與 colspan：合併表格單元格

## 學習目標

讀完這篇筆記，你應該能夠：

- 分辨 `rowspan` 與 `colspan` 的合併方向。
- 找出要設定合併屬性的目標儲存格。
- 知道合併後為什麼要刪除多餘的 `<td>` 或 `<th>`。
- 寫出同時包含跨列與跨欄的基本表格。

## 使用情境

一般表格是一格對一格，但有些資料本身會跨越多列或多欄。例如課表中的同一堂課可能連續兩節，統計表中的「總計」可能橫跨多個欄位，產品規格表中的分類名稱可能對應多個項目。

這種時候就需要合併單元格，讓一個儲存格代表更大的資料範圍。

![跨行與跨列合併示意](../../origin/230-表格標籤/assets/images/html-table-tags-img-011-f3bfdd.jpg)

## 一句話理解

`rowspan` 讓儲存格向下跨多列，`colspan` 讓儲存格向右跨多欄。

## 合併方向

- `rowspan="數量"`：跨列合併，也就是向下合併幾列。
- `colspan="數量"`：跨欄合併，也就是向右合併幾欄。

例如：

```html
<td rowspan="2">連續兩列共用這格</td>
<td colspan="2">連續兩欄共用這格</td>
```

`rowspan="2"` 表示這個儲存格會占用目前這一列加下一列，共 2 列的高度。

`colspan="2"` 表示這個儲存格會占用目前這一欄加右邊一欄，共 2 欄的寬度。

## 合併單元格三步驟

1. 先判斷要跨列還是跨欄。
2. 找到合併後要保留的目標儲存格，寫上 `rowspan` 或 `colspan`。
3. 刪除被合併掉的多餘儲存格。

第三步最容易被忽略。合併不是「把格子變大就結束」，而是讓一個儲存格占用其他儲存格的位置，所以原本那些位置上的 `<td>` 或 `<th>` 必須移除。

## 完整範例

```html
<table width="500" height="249" border="1" cellspacing="0">
  <tr>
    <td></td>
    <td colspan="2"></td>
  </tr>
  <tr>
    <td rowspan="2"></td>
    <td></td>
    <td></td>
  </tr>
  <tr>
    <td></td>
    <td></td>
  </tr>
</table>
```

## 範例拆解

第一列中，第二個 `<td>` 寫了 `colspan="2"`：

```html
<td colspan="2"></td>
```

這代表它會向右跨 2 欄。因為它已經占了第二欄與第三欄的位置，所以第一列不需要再寫第三個 `<td>`。

第二列中，第一個 `<td>` 寫了 `rowspan="2"`：

```html
<td rowspan="2"></td>
```

這代表它會向下跨 2 列。因為第三列的第一欄已經被這個儲存格占用，所以第三列只需要寫後面兩個 `<td>`。

如果合併後仍保留被占用位置上的 `<td>`，表格欄位數就會錯位，瀏覽器會嘗試修正，但畫面與資料關係通常會變得難以理解。

## 實務判斷

合併單元格應該用在資料本身真的有跨列或跨欄關係時。常見情境包括：

- 課表中同一課程連續多節，使用 `rowspan`。
- 報表最下方的「總計」文字需要橫跨多個欄位，使用 `colspan`。
- 分類表中同一分類對應多筆資料，分類欄可以使用 `rowspan`。

不要只為了讓表格看起來比較漂亮就大量合併。合併越多，表格的結構越難檢查，也越容易讓讀者迷失欄位對應關係。

## 常見錯誤

**只加屬性，沒有刪除多餘儲存格**

這是最常見錯誤。當一格設定 `colspan="2"` 後，它已經占了兩欄，所以同一列中原本被合併的那一格要刪除。`rowspan` 也是一樣，被向下占用的位置要在下一列刪除。

**搞混跨列與跨欄**

`rowspan` 的 row 是列，表示向下跨列；`colspan` 的 col 是 column，表示向右跨欄。可以用「row 往下、col 往右」來記。

**合併後每列欄位數對不起來**

表格每一列的欄位數不一定要在 HTML 原始碼中看起來一樣多，因為 `rowspan` 和 `colspan` 會占用額外位置。但從表格視覺結構來看，每一列最後應該仍能對齊同一套欄位。若對不起來，通常代表刪錯或少刪了某個儲存格。

**把合併當成排版技巧**

合併單元格應表達資料關係，不是為了隨意控制畫面位置。若只是想讓某個內容置中或變寬，通常應該交給 CSS。

## 小練習

試著完成一張三欄課表：

1. 第一欄是時間。
2. 第二欄是科目。
3. 第三欄是教室。
4. 讓「HTML 基礎」跨兩個連續時段。
5. 讓最下方的「備註」橫跨三欄。

完成後檢查：使用 `rowspan` 的那一列下面，是否刪掉了被占用的儲存格？使用 `colspan` 的那一列，是否刪掉了右側被占用的儲存格？

## 重點整理

- `rowspan` 是跨列，方向向下。
- `colspan` 是跨欄，方向向右。
- 合併後必須刪除被合併掉的多餘儲存格。
- 合併單元格用來表達資料關係，不應拿來取代 CSS 排版。
