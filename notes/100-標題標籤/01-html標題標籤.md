# HTML 標題標籤：用 h1 到 h6 建立內容層級

## 學習目標

讀完這篇筆記，你應該能夠：

- 說明 `<h1>` 到 `<h6>` 的語意。
- 依照內容重要性選擇合適的標題層級。
- 分辨「標題層級」和「視覺大小」不是同一件事。
- 避免把標題標籤當成單純調整字體大小的工具。

## 使用情境

在新聞、文章、文件、產品介紹頁或教學頁面中，讀者通常會先掃描標題，判斷這一頁在講什麼、每個段落的重點在哪裡。

HTML 標題標籤就是用來標出這些內容層級的工具。例如一篇文章可能有文章主題、主要段落、小段說明，這些層級應該用 `<h1>` 到 `<h6>` 依序表達，而不是只靠字體大小或粗體外觀。

## 一句話理解

`<h1>` 到 `<h6>` 是 HTML 用來表示標題層級的語意標籤，數字越小，代表標題層級越高、重要性越強。

## 標籤語意

標題標籤的重點是「內容結構」，不是「文字看起來多大」。

- `<h1>`：頁面或主要內容的最高層級標題，通常用來表示本頁最核心的主題。
- `<h2>`：`<h1>` 底下的主要段落標題。
- `<h3>`：`<h2>` 底下的次層段落標題。
- `<h4>` 到 `<h6>`：更細的內容層級，依實際文件結構往下使用。

瀏覽器預設會讓 `<h1>` 看起來最大、`<h6>` 看起來最小，也通常會加粗並獨佔一行。不過這些只是預設樣式；真正決定它們用途的是標題層級與內容語意。

## 基本語法

```html
<h1>一級標題</h1>
<h2>二級標題</h2>
<h3>三級標題</h3>
<h4>四級標題</h4>
<h5>五級標題</h5>
<h6>六級標題</h6>
```

這段程式碼展示的是六種標題層級，不是六種字體大小選項。實務上不需要每個頁面都用到 `<h1>` 到 `<h6>`，而是依內容結構需要使用。

## 範例拆解

假設要寫一篇介紹 HTML 的文章，可以這樣安排標題：

```html
<h1>HTML 入門</h1>

<h2>HTML 是什麼</h2>
<p>HTML 是用來描述網頁內容結構的標記語言。</p>

<h2>常見 HTML 標籤</h2>

<h3>標題標籤</h3>
<p>標題標籤用來表示文章或段落的標題層級。</p>

<h3>段落標籤</h3>
<p>段落標籤用來表示一段普通文字內容。</p>
```

這個例子裡：

- `<h1>HTML 入門</h1>` 是整篇文章的主題。
- `<h2>HTML 是什麼</h2>` 和 `<h2>常見 HTML 標籤</h2>` 是兩個主要段落。
- `<h3>標題標籤</h3>` 和 `<h3>段落標籤</h3>` 是「常見 HTML 標籤」底下的次層內容。

讀者看標題就能理解文章架構；搜尋引擎和輔助技術也能從標題層級判斷內容組織方式。

## 常見錯誤

### 只為了字體大小選標題

錯誤寫法：

```html
<h1>這段文字想要變大</h1>
<p>但它其實不是頁面主標題。</p>
```

問題在於 `<h1>` 代表最高層級標題。如果只是想讓文字變大，應該使用 CSS 調整樣式，而不是誤用標題標籤。

### 跳過標題層級

不建議的寫法：

```html
<h1>文章標題</h1>
<h4>第一個小節</h4>
```

這樣會讓文件結構看起來像是中間少了 `<h2>` 和 `<h3>`。讀者可能還能靠畫面理解，但輔助技術或文件大綱會變得不清楚。一般情況下，應依內容層級循序安排，例如從 `<h1>` 到 `<h2>`，再到 `<h3>`。

### 把普通強調文字寫成標題

如果一段文字只是想強調，而不是一個段落或區塊的標題，就不適合使用 `<h1>` 到 `<h6>`。這類內容可以依語意改用 `<strong>`，或用 CSS 處理視覺樣式。

## 實務判斷準則

選擇標題標籤時，可以先問三個問題：

1. 這段文字是不是某個內容區塊的標題？
2. 它在整份文件中屬於哪一層？
3. 它和前後標題的父子關係是否清楚？

常見做法是：

- 一個主要頁面或主要文章通常會有一個清楚的 `<h1>`。
- 主要段落使用 `<h2>`。
- 主要段落裡的子段落使用 `<h3>`。
- 標題的外觀交給 CSS，標題標籤本身負責語意與層級。

## 重點整理

- `<h1>` 到 `<h6>` 用來表示標題層級，重要性由高到低遞減。
- 瀏覽器預設會讓標題變粗、變大並獨佔一行，但外觀不是選標題標籤的主要依據。
- 標題標籤應該依內容結構使用，不應只拿來調整字體大小。
- 清楚的標題層級能幫助讀者掃描內容，也能改善文件結構與可維護性。

## 自我檢查

1. 如果一篇文章的主題是「HTML 入門」，應該優先用哪一個標題標籤表示？
2. 如果 `<h2>` 底下有更細的小節，通常會使用哪一個標題層級？
3. 為什麼不應該只因為想讓文字變大就使用 `<h1>`？
4. 標題標籤的視覺大小應該由 HTML 決定，還是由 CSS 調整？
