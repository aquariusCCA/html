# img 標籤：在網頁中放入圖片

## 學習目標

讀完這篇筆記，你應該能夠：

- 使用 `<img>` 在 HTML 頁面中顯示圖片。
- 理解 `src`、`alt`、`title`、`width`、`height` 這幾個常見屬性的用途。
- 判斷什麼時候該寫有意義的 `alt`，什麼時候可以使用空的 `alt=""`。
- 知道圖片尺寸設定不當時，為什麼會造成圖片變形。

## 問題情境

網頁經常需要顯示圖片，例如商品照片、文章配圖、頭像、圖示或活動海報。

圖片本身不是直接寫在 HTML 裡，而是由 HTML 告訴瀏覽器：「請到這個位置找圖片，然後把它顯示在這裡。」如果圖片位置寫錯，或圖片檔案不存在，頁面就會出現破圖。

這時 `<img>` 標籤不只要負責顯示圖片，也要提供圖片無法顯示時的替代資訊。

## 一句話理解

`<img>` 用來把圖片放進網頁；`src` 指定圖片來源，`alt` 說明圖片代表的內容。

## 基本語法

`img` 是 image 的縮寫，意思是圖像。它是單標籤，不需要寫結束標籤。

```html
<img src="images/photo.jpg" alt="商品照片">
```

可以拆成三個部分理解：

- `<img>`：建立圖片元素。
- `src="images/photo.jpg"`：指定圖片檔案的位置。
- `alt="商品照片"`：提供圖片的替代文字。

`<img>` 要顯示出對應圖片，必須依靠屬性提供圖片來源與補充資訊。常見寫法如下：

```html
<img
  src="圖片地址"
  alt="替代文字"
  title="提示文字"
  width="寬度"
  height="高度"
>
```

## 常用屬性

| 屬性 | 用途 | 常見寫法 |
| --- | --- | --- |
| `src` | 指定圖片來源 | `src="images/logo.png"` |
| `alt` | 提供替代文字 | `alt="網站標誌"` |
| `title` | 滑鼠停留時顯示提示文字 | `title="返回首頁"` |
| `width` | 設定顯示寬度 | `width="200"` |
| `height` | 設定顯示高度 | `height="120"` |

## src：圖片來源

`src` 用來指定圖片檔案的路徑和檔名。

```html
<img src="./assets/images/product.jpg" alt="商品照片">
```

瀏覽器讀到這段 HTML 時，會依照 `src` 的路徑去找圖片。只要路徑、資料夾名稱、檔名或副檔名其中一個寫錯，圖片就可能顯示不出來。

在基本用法中，`src` 是最常用的圖片來源屬性。若要處理響應式圖片，也可能搭配 `srcset` 提供不同尺寸或解析度的圖片來源，但初學階段先掌握 `src` 即可。

## alt：替代文字

`alt` 是圖片的替代內容。它不是單純的圖片說明，也不是滑鼠提示文字，而是當圖片無法顯示、或使用者透過輔助工具閱讀頁面時，用來替代圖片資訊的文字。

有意義的圖片應該寫出圖片真正傳達的資訊：

```html
<img src="team.jpg" alt="前端課程講師團隊合照">
```

如果圖片只是裝飾，不承載內容資訊，可以使用空的 `alt`：

```html
<img src="line-decoration.png" alt="">
```

判斷時可以問自己：

- 這張圖片如果消失，使用者是否會少掉重要資訊？
- 這張圖片是否是按鈕、連結、商品、人物或內容的一部分？
- 這張圖片是否只是背景、分隔線、裝飾圖案？

如果圖片有資訊價值，就寫能替代它的文字。若只是裝飾，空的 `alt=""` 通常比硬寫「圖片」更好。

## title：提示文字

`title` 通常會在滑鼠停留在圖片上時顯示提示文字。

```html
<img src="logo.png" alt="網站標誌" title="返回首頁">
```

`title` 不能取代 `alt`。原因是 `title` 不一定能被所有使用情境穩定讀取，也不是圖片無法顯示時的主要替代內容。

實務上可以這樣分工：

- `alt`：圖片本身代表什麼。
- `title`：額外提示或補充資訊。

## width 和 height：圖片顯示尺寸

`width` 設定圖片顯示寬度，`height` 設定圖片顯示高度。

```html
<img src="photo.jpg" alt="人物照片" width="200">
```

如果只設定 `width` 或 `height` 其中一個，另一個通常會自動等比例縮放，圖片不會變形。

```html
<!-- 只設定寬度，高度會依原始比例自動縮放 -->
<img src="photo.jpg" alt="人物照片" width="200">
```

如果同時設定 `width` 和 `height`，但比例和原圖不同，圖片可能會被拉伸或壓扁。

```html
<!-- 不建議：如果原圖不是正方形，可能會變形 -->
<img src="photo.jpg" alt="人物照片" width="200" height="200">
```

初學時可以先記住：不確定圖片比例時，只設定一邊通常比較不容易出錯。

## 屬性書寫規則

圖片標籤可以有很多屬性，這些屬性都寫在標籤名稱後面。

```html
<img src="photo.jpg" alt="人物照片" width="200">
```

書寫時要注意：

- 標籤名和屬性之間要用空格分開。
- 屬性和屬性之間也要用空格分開。
- 屬性通常使用 `屬性="屬性值"` 的格式。
- 屬性順序不影響 HTML 語法，但建議維持一致，讓程式碼容易閱讀。

## 完整範例

```html
<body>
  <h1>商品介紹</h1>

  <img
    src="./assets/images/product.jpg"
    alt="黑色無線耳機"
    title="點擊查看商品詳情"
    width="300"
  >
</body>
```

這段範例中：

- `src` 告訴瀏覽器圖片檔案在哪裡。
- `alt` 說明圖片內容是「黑色無線耳機」。
- `title` 提供額外提示。
- `width` 控制圖片顯示寬度，高度依比例縮放。

## 常見錯誤

### 忘記寫 alt

```html
<!-- 不建議 -->
<img src="product.jpg">
```

如果這張圖片有內容意義，應該補上替代文字。

```html
<img src="product.jpg" alt="黑色無線耳機">
```

### 把 alt 寫成沒有資訊的文字

```html
<!-- 不建議 -->
<img src="product.jpg" alt="圖片">
```

`alt="圖片"` 沒有提供有用資訊。應該描述圖片真正代表的內容。

```html
<img src="product.jpg" alt="黑色無線耳機">
```

### 用 title 取代 alt

```html
<!-- 不建議 -->
<img src="logo.png" title="公司標誌">
```

`title` 不是替代文字。圖片有資訊意義時仍應寫 `alt`。

```html
<img src="logo.png" alt="公司標誌" title="返回首頁">
```

### 同時亂設 width 和 height

```html
<!-- 可能造成圖片變形 -->
<img src="photo.jpg" alt="風景照片" width="300" height="100">
```

如果不確定比例，可以只設定一邊。

```html
<img src="photo.jpg" alt="風景照片" width="300">
```

## 重點整理

- `<img>` 用來在網頁中顯示圖片，是單標籤。
- `src` 指定圖片來源，路徑寫錯會導致圖片無法顯示。
- `alt` 是圖片替代文字，有意義的圖片應寫出可替代圖片資訊的內容。
- 裝飾圖片可以使用空的 `alt=""`。
- `title` 是額外提示，不能取代 `alt`。
- 只設定 `width` 或 `height` 其中一個時，圖片通常會等比例縮放。
- 同時設定 `width` 和 `height` 時，比例不當可能造成圖片變形。
