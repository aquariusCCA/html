# 第三十九章｜引用與註記文本標籤

> 章節定位：理解引用、作品來源、縮寫與定義類文字如何用更精確的 HTML 表達  
> 建議閱讀方式：先分清引用內容和來源名稱，再理解縮寫與定義標記

## 本章在學什麼

這一章整理偏「文字語意補充」的標籤：

- `<blockquote>`：區塊引用。
- `<q>`：行內短引用。
- `<cite>`：作品或來源標題。
- `<abbr>`：縮寫。
- `<dfn>`：定義中的術語。

重點不是把文字變成某種樣式，而是讓 HTML 更清楚地表達「這段文字的角色」。

## 適合快速回查的主題

- `<blockquote>` 和 `<q>` 差在哪
- `<cite>` 是作品名還是作者名
- `<abbr title>` 怎麼表示縮寫全稱
- `<dfn>` 如何標示正在定義的術語
- 引用來源 URL 該放在哪裡

## 易混淆主題

- `<blockquote>` vs `<q>`：前者是區塊引用，後者是行內短引用。
- `<cite>` vs 作者姓名：`<cite>` 通常標示作品或來源標題，不是普通作者姓名。
- `<abbr>` vs `title` 全局屬性：`abbr` 表示縮寫，`title` 可補全稱。
