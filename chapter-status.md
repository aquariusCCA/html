# HTML 筆記包章節處理狀態表

> 用途：追蹤每個 HTML 章節從 `origin` 到 `notes`、`demos`、`practice`、`review`、`appendix` 的處理進度。

## 狀態選項

| 狀態 | 說明 |
| --- | --- |
| 未開始 | 尚未處理 |
| 進行中 | 正在處理 |
| 待確認 | 需要人工確認或決策 |
| 已完成 | 已完成並可進入下一階段 |
| 暫停 | 暫時停止處理 |
| 不適用 | 該章節不需要此步驟 |
