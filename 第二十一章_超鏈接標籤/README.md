# 第二十一章｜超鏈接標籤

> 章節定位：建立在 HTML 中建立導航、檔案連結、錨點跳轉與應用喚起的基礎判讀  
> 建議閱讀方式：先理解 `<a>` 的基本用途與屬性，再看各種連結型態、錨點與特殊用途

## 本章在學什麼

這一章整理 HTML 的超鏈接標籤。  
重點不只是把頁面連出去，而是分清楚不同 `href` 寫法、開啟方式、錨點、下載與應用喚起各自在解決什麼問題。

## 本章與整體地圖的關係

- 前置知識：第六章的路徑概念，能幫助理解外部連結、內部連結與檔案連結的目標位置。
- 這章的核心任務：學會建立可點擊的導航，並分清 `href`、`target`、錨點、下載與特殊 URI 寫法的用途。
- 讀完後通常接：之後在做導覽列、卡片連結、頁內目錄或檔案下載入口時，都可以回來快速查這一章。

## 易混淆主題

- 真正超鏈接 vs 沒有 `href` 的 `<a>`：後者不應直接等同一般導航連結。
- `href="#"` vs `href=""`：前者偏向頁內頂部，後者偏向當前文件本身。
- 連結 vs 按鈕互動：不是所有可點擊行為都適合用 `<a>`。
