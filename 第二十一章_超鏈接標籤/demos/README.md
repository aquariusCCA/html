# 第二十一章｜超鏈接標籤示例

這個資料夾收錄第二十一章筆記對應的可執行示例頁。  
這章的重點是理解 `<a>` 的導航本質，並分清 `href`、`target`、錨點、檔案連結、下載連結與特殊 URI 各自在解決什麼問題。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第二十一章 Markdown 筆記。
- 想直接操作不同連結類型時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [anchor-tag-demo.html](./anchor-tag-demo.html)：整合展示基本連結、`target`、外部/內部/檔案/錨點連結、`href="#id"` / `href="#"` / `href=""` 差異、`download`、`mailto:`、`tel:`、`javascript:` 觀念，以及 `<a>` 與 `<button>` 的選擇判準。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
