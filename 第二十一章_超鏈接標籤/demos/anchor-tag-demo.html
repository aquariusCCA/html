HTML 超鏈接標籤 Demo

這個範例把第二十一章的重點做成可操作頁面：`href` 決定去哪裡，`target` 決定怎麼開，錨點、檔案、下載、郵件、電話都是不同類型的導向。核心判準是：導向用 `<a>`，操作用 `<button>`。

`<a>` 的本質是導航 它不是單純藍色文字，而是帶使用者前往另一個資源、位置或應用。
`href` 是目標 它回答「點了之後要去哪裡」，可以是網址、路徑、檔案、錨點或特殊 URI。
`target` 是開啟方式 `_self` 在目前頁面開啟，`_blank` 通常另開新頁籤或新視窗。
不是所有點擊都該用連結 若本質是送出、切換、展開、開彈窗，通常應優先用 `button`。

本頁會示範

  • 外部連結、內部連結、檔案連結、錨點連結
  • `href="#id"`、`href="#"`、`href=""` 的差異
  • `download`、`mailto:`、`tel:`、`javascript:`
  • `<a>` 和 `<button>` 的選擇判準

Basic Link

最基本的超鏈接

外部連結通常使用完整網址，也常搭配 `target="_blank"`。

連到同一網站內的頁面

            
內部連結通常使用相對路徑，和第六章的路徑概念有關。

5. 特殊用途的 `href`

        

          

            `mailto:`
            
開啟郵件應用

            
<a href="mailto:demo@example.com">郵件聯繫</a>

            郵件聯繫
          


          

            `tel:`
            
在支援裝置上喚起撥號

            
<a href="tel:10010">電話聯繫</a>

            電話聯繫
          


          

            應用協議
            
嘗試喚起特定應用

            
<a href="line://msg/text/Hello%20World">使用 LINE 發送消息</a>

            
          


          

            `javascript:`
            
能用，但不建議當日常首選

            
<a href="javascript:alert('Hello')">點擊我</a>

            
          

        

        

          點擊「展示說明」會在這裡顯示補充，不會真的嘗試喚起應用或執行 `javascript:` URL。
        

      


      

        
6. `<a>` vs `<button>`

        

          

            導向 / 導航
            
要去某個地方，用 `<a>`

            
<a href="../media/test.pdf" download>下載 PDF</a>
<a href="#part2">跳到錨點</a>
<a href="mailto:demo@example.com">寄信</a>

            

              下載 PDF
              跳到錨點
              寄信
            

          


          

            操作 / 行為
            
要做某件事，用 `<button>`

            
<button type="button">展開選單</button>
<button type="button">開啟彈窗</button>

            

              
              
這是操作，不是導航，所以用 `button` 更貼切。

8. 實務提醒

        

          

            不要把 `href="#"` 當作無效果
            它通常仍會跳到頁面頂部。如果只是操作，應改用 `button`。
          

          

            外部新頁籤連結要注意安全
            使用 `target="_blank"` 時，實務上常搭配 `rel="noopener noreferrer"`。
          

          

            連結文字要說明目的
            「下載 PDF」「前往章節」比「點我」更容易理解與回查。
