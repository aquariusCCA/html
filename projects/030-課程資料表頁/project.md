# 課程資料表頁

這個專案以課程資訊整理頁為情境，整合頁面語意化結構、頁內連結、無序列表、有序列表、描述列表，以及包含 `caption`、`thead`、`tbody`、`tfoot`、`rowspan` 與 `colspan` 的課程資料表。讀者可以觀察「項目集合」「名稱描述關係」與「列欄資料」應分別使用哪一種 HTML 結構。
