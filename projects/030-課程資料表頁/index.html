課程資料表頁

用列表與表格整理課程特色、學習流程、術語說明與時段資料。

課程特色 學習流程 術語說明 課程表

課程特色

  • 以 HTML 結構為主，先建立內容語意。
  • 每個範例都能回推到來源 notes 的知識點。
  • 用 CSS 處理呈現，不把表格拿來做版面排版。

學習流程

  1. 先閱讀對應 notes，理解每個標籤的語意。
  2. 再觀察整合頁面中各區塊使用的 HTML 結構。
  3. 最後檢查資料是否適合用列表、描述列表或表格呈現。

術語說明

無序列表
用來呈現沒有固定順序的一組相關項目。
有序列表
用來呈現步驟、流程或排名等順序有意義的項目。
描述列表
用來呈現名稱與描述之間的對應關係。
