# 會員註冊表單頁

這個專案把語意化頁面骨架、`id` / `class` / `title` / `hidden` 等全局屬性，以及表單的文字欄位、密碼欄位、單選、複選、下拉選單、`textarea`、`label`、`fieldset`、`legend` 與提交按鈕整合成一個完整的會員註冊頁。讀者可以觀察表單欄位如何透過 `name` 形成可提交資料，並理解分組與標籤關聯如何改善表單可用性。
