自我介紹

送出後，瀏覽器會依 form 的 action 與 method 處理資料；本專案用頁內錨點示範提交目標。

頁面觀察重點

語意化骨架

頁首、主要內容、表單區與補充區各自使用有角色的標籤，避免整頁只靠 div 表示結構。

表單資料

每個要提交的控件都有 name，radio 共用 name 形成單選，checkbox 用 value 表示被勾選的項目。

標籤與分組

label 透過 for 對應 input 的 id；fieldset 和 legend 讓一組相關欄位有清楚標題。
