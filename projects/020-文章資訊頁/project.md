# 文章資訊頁

這個專案以一篇活動報導式文章為情境，整合標題層級、段落語意、語意化版面、圖片 `src` / `alt` / `title` / `width` 屬性，以及一般連結與頁內錨點。讀者可以觀察文章主題、章節導覽、主要內容、補充區與頁尾如何各自用合適的 HTML 結構表示。
