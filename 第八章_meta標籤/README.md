## 本章在學什麼

這一章整理 HTML `<meta>` 標籤最常見的使用情境。  
重點不是把所有屬性背起來，而是先理解不同 `meta` 設定各自在服務哪一類對象。

## 易混淆主題

- `name` vs `http-equiv`：前者偏頁面描述資訊，後者偏模擬 HTTP 標頭效果。
- `description` vs `og:description`：前者偏搜尋引擎摘要，後者偏社群分享預覽。
- `keywords` vs `description`：前者偏關鍵字標示，後者偏頁面描述文字。
