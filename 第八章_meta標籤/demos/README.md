# 第八章｜meta 標籤示例

這個資料夾收錄第八章筆記對應的可執行示例頁。  
由於 `<meta>` 主要寫在 `<head>` 內，很多效果不適合直接靠畫面互動呈現，所以這裡會把「範例標記」和「對應情境模擬」放在同一頁對照展示。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第八章 Markdown 筆記。
- 想直接看常見 `meta` 組合時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [meta-tags-demo.html](./meta-tags-demo.html)：整合展示 `charset`、`viewport`、`description`、`keywords`、`robots`、`refresh` 與 Open Graph 的常見寫法，並對照 SEO 摘要卡與社群分享卡的差異。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
