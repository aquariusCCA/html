# 第三十一章｜`details` 標籤示例

這個資料夾收錄第三十一章筆記對應的可執行示例頁。  
這章的重點是理解 `<details>` 如何建立原生展開 / 收合區塊，並掌握 `<summary>` 與 `open` 的角色。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第三十一章 Markdown 筆記。
- 想直接操作展開 / 收合、`open` 與 `toggle` 時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [details-tags-demo.html](./details-tags-demo.html)：整合展示 `<details>` / `<summary>` 基本收合、`open` 初始展開、`open="false"` 的布林屬性陷阱、沒有 `<summary>` 的情況、FAQ、表單補充說明、JavaScript `details.open` 與 `toggle` 事件。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
