## 易混淆主題

- `iframe` vs 舊式 `frame` / `frameset`：現代常用的是前者，後者不是新網站主流。
- `name` vs 顯示標題：`name` 是給 `target` 對應用的，不是給使用者看的標題文字。
- 能嵌入 vs 不能嵌入：不一定是語法問題，也可能是對方網站的安全限制。
- `frameborder` vs CSS `border`：前者常見於舊教材，現代通常改用 CSS 控制邊框。
