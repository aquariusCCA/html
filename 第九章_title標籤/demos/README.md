# 第九章｜title 標籤示例

這個資料夾收錄第九章筆記對應的可執行示例頁。  
`<title>` 本身不會直接出現在頁面正文，但會影響分頁標題、搜尋結果標題與收藏夾辨識，所以這裡會用模擬畫面把它和 `<h1>`、`<meta>`、全域 `title` 屬性分開展示。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第九章 Markdown 筆記。
- 想直接看 `<title>` 影響哪些地方時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [title-tag-demo.html](./title-tag-demo.html)：整合展示 `<title>` 的正確位置、和 `<h1>` 的角色差異、搜尋結果標題模擬，以及 `<title>` 與 `title=""` 屬性的區別。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
