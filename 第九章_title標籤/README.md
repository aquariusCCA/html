## 易混淆主題

- `<title>` vs `<h1>`：前者是整份 HTML 文件的標題，後者是頁面內容中的標題。
- `<title>` vs `<meta>`：前者會成為頁面標題文字，後者通常提供附加資訊。
- 頁籤標題 vs 內容標題：兩者可能相似，但用途不同。
