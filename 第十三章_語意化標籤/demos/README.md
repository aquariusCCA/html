# 第十三章｜語意化標籤示例

這個資料夾收錄第十三章筆記對應的可執行示例頁。  
這章的重點是理解語意化標籤不是單純「換個名字的 `div`」，而是用更清楚的結構描述頁面內容，幫助人、工具與搜尋系統理解頁面。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第十三章 Markdown 筆記。
- 想直接看語意化頁面結構時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [semantic-tags-demo.html](./semantic-tags-demo.html)：整合展示 `header`、`nav`、`main`、`article`、`section`、`aside`、`footer` 的典型角色，並對照語意化結構與 `div` 堆疊寫法的差異。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
