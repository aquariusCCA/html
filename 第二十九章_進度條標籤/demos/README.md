# 第二十九章｜進度條標籤示例

這個資料夾收錄第二十九章筆記對應的可執行示例頁。  
這章的重點是理解 `<progress>` 如何表示任務完成進度，並分清它和 `<meter>` 的語意差異。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第二十九章 Markdown 筆記。
- 想直接觀察定進度、不定進度與 JavaScript 更新時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [progress-tags-demo.html](./progress-tags-demo.html)：整合展示 `<progress>` 的固定進度、不定進度、`value="0"`、`max` 預設值、沒有 `min`、搭配 `<label>`、JavaScript 動態更新，以及 `<progress>` / `<meter>` 的語意差異。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
