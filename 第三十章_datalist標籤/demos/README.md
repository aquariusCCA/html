# 第三十章｜`datalist` 標籤示例

這個資料夾收錄第三十章筆記對應的可執行示例頁。  
這章的重點是理解 `<datalist>` 如何替 `<input>` 提供建議值，並分清它和 `<select>`、表單驗證、`autocomplete` 的差異。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第三十章 Markdown 筆記。
- 想直接操作輸入建議、表單送出與簡單驗證時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [datalist-tags-demo.html](./datalist-tags-demo.html)：整合展示 `input[list]` 與 `datalist[id]` 的綁定、`option value`、`input name` 表單送出、城市 / 搜尋 / 商品代碼建議、`datalist` / `select` 差異、它不是表單驗證工具，以及 `autocomplete` 的分工。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
