## 9. 常用片段

### 9.1 基本外部連結

| 項目 | 說明 |
|---|---|
| 使用情境 | 前往其他網站。 |
| 使用前提 | 已確認目標網址。 |
| 來源 | §3、§5.1 |

```html
<a href="https://example.com">前往 Example</a>
```

### 注意事項

- 連結文字應描述目的。
- 若要在新分頁開啟外部網站，參考 9.2。

### 9.2 外部新分頁連結

| 項目 | 說明 |
|---|---|
| 使用情境 | 開啟外部網站或參考資料，並保留目前頁面。 |
| 使用前提 | 需要新分頁或新視窗開啟。 |
| 來源 | §4.2、§5.1、§7 |

```html
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  前往 Example
</a>
```

### 注意事項

- `target="_blank"` 嘗試在新分頁或新視窗開啟。
- 原筆記建議外部新分頁連結搭配 `rel="noopener noreferrer"`。

### 9.3 內部頁面連結

| 項目 | 說明 |
|---|---|
| 使用情境 | 前往同一網站中的其他頁面。 |
| 使用前提 | 目標頁面路徑正確。 |
| 來源 | §5.2 |

```html
<a href="./about.html">公司簡介</a>
```

### 注意事項

- 相對路徑要以目前 HTML 檔案位置為基準。
- 路徑錯誤會導致目標頁面無法開啟。

### 9.4 檔案下載連結

| 項目 | 說明 |
|---|---|
| 使用情境 | 提供 PDF 或附件下載。 |
| 使用前提 | 檔案路徑正確。 |
| 來源 | §5.3、§7 |

```html
<a href="./assets/pdfs/test.pdf" download="一份PDF檔.pdf">下載 PDF 檔案</a>
```

### 注意事項

- `download` 可提示瀏覽器下載資源。
- 正確屬性名稱是 `download`。

### 9.5 頁內錨點連結

| 項目 | 說明 |
|---|---|
| 使用情境 | 點擊後跳到目前頁面指定區塊。 |
| 使用前提 | 目標元素有對應 `id`。 |
| 來源 | §5.4、§7、§9 |

```html
<a href="#season-2">第二季</a>

<h3 id="season-2">第二季介紹</h3>
```

### 注意事項

- `href="#season-2"` 必須對應 `id="season-2"`。
- `id` 名稱應保持唯一。

### 9.6 其他頁面的錨點連結

| 項目 | 說明 |
|---|---|
| 使用情境 | 前往其他頁面的指定區塊。 |
| 使用前提 | 目標頁面存在，且有對應 `id`。 |
| 來源 | §5.4 |

```html
<a href="./test.html#intro">跳轉到測試頁面的 intro 區塊</a>
```

### 注意事項

- 需要同時確認頁面路徑與 `id`。

### 9.7 電話與郵件連結

| 項目 | 說明 |
|---|---|
| 使用情境 | 在手機或已設定應用的環境中喚起電話或郵件。 |
| 使用前提 | 使用者裝置與瀏覽器支援對應應用。 |
| 來源 | §5.7、§11 |

```html
<a href="tel:10010">電話聯繫</a>
<a href="mailto:10010@qq.com">郵件聯繫</a>
```

### 注意事項

- `mailto:` 後不要加入多餘空白。
- 是否成功喚起應用取決於使用者環境。

### 9.8 圖片連結

| 項目 | 說明 |
|---|---|
| 使用情境 | 讓圖片成為可點擊的連結。 |
| 使用前提 | 圖片路徑正確，且已提供合適 `alt`。 |
| 來源 | §5.8、§7、§9 |

```html
<a href="https://www.baidu.com">
  <img src="./assets/images/example.jpg" alt="前往百度">
</a>
```

### 注意事項

- `alt` 應描述圖片內容或連結目的。
- 不要只把檔名當成替代文字。

### 9.9 非導航動作使用按鈕

| 項目 | 說明 |
|---|---|
| 使用情境 | 執行頁面內動作，而不是前往某個 URL。 |
| 使用前提 | 動作本質不是導航。 |
| 來源 | §5.6、§7、§8 |

```html
<button type="button">送出表單</button>
```

### 注意事項

- `<a>` 用於導航，`button` 用於觸發動作。
- 原筆記不建議用 `javascript:` 或 `href="#"` 作為一般互動寫法。

## 10. 錯誤速查表

| 錯誤現象 | 可能原因 | 修正方向 | 對應知識點 | 來源 |
|---|---|---|---|---|
| 使用者不知道連結會前往哪裡。 | 連結文字只寫「點我」、「下載」等模糊文字。 | 改成具體文字，例如「查看公司簡介」、「下載 PDF 講義」。 | 連結文字、可訪問性 | §6、§7、§9 |
| 點擊連結後頁面或檔案無法開啟。 | `href` 路徑、檔名或目標 URL 寫錯。 | 確認相對路徑、檔名與目標位置。 | `href`、路徑 | §4.1、§5、§7 |
| 外部新分頁連結有安全或隱私風險。 | 使用 `target="_blank"` 但沒有 `rel`。 | 加上 `rel="noopener noreferrer"`。 | `target`、`rel` | §4.2、§5.1、§7 |
| 頁內跳轉沒有成功。 | `href="#xxx"` 找不到對應 `id="xxx"`。 | 確認片段名稱一致且目標 `id` 存在。 | 錨點連結、`id` | §5.4、§7 |
| 點擊後跳到頁首或造成頁面跳動。 | 濫用 `href="#"` 或用連結假裝按鈕。 | 建立明確錨點，或改用 `<button type="button">`。 | 空連結、`href="#"`、button | §5.4、§5.5、§7 |
| 頁面重新載入。 | 長期保留空的 `href=""`。 | 有明確目標再使用 `<a>`；非導航互動改用 `button`。 | 空連結 | §5.5、§7 |
| 郵件應用無法正確喚起。 | `mailto:` 中加入多餘空白或格式不正確。 | 使用 `mailto:user@example.com`。 | URL scheme、`mailto:` | §5.7、§7 |
| 卡片或連結互動不穩定。 | 在 `<a>` 中放另一個 `<a>`、`button` 或表單控制項。 | 避免互動元素巢狀，將操作拆成並列元素。 | 卡片式連結、互動元素巢狀 | §4.3、§7、§11 |
| 圖片連結目的不清楚。 | 圖片放在 `<a>` 中但缺少合適 `alt`。 | 讓 `alt` 描述圖片內容或連結目的。 | 圖片連結、`alt` | §5.8、§7、§9 |
| 下載提示沒有作用。 | `download` 拼成 `dowload` 或瀏覽器未採用下載提示。 | 使用正確拼字 `download`，並注意瀏覽器與伺服器設定可能影響行為。 | `download` | §5.3、§7 |

## 11. 相似概念對照表

### 11.1 `<a>` vs `<button>`

| 比較項目 | `<a>` | `<button>` |
|---|---|---|
| 定義 | 建立前往某個目標的超鏈接。 | 執行頁面內動作的原生互動元素。 |
| 使用時機 | 前往頁面、檔案、錨點、外部網站或應用 scheme。 | 送出操作、開關狀態、觸發 JavaScript 行為。 |
| 常見寫法 | `<a href="./about.html">公司簡介</a>` | `<button type="button">送出表單</button>` |
| 注意事項 | 不應用來假裝一般按鈕。 | 本章只作語意對照，未展開 button 全部行為。 |
| 常見錯誤 | `<a href="#" onclick="submitForm()">送出表單</a>` | 忘記依情境設定合適 `type`。 |

### 選用建議

- 目標是「前往某處」時，使用 `<a>`。
- 目標是「執行頁面內動作」時，使用 `<button>`。

### 11.2 `target="_self"` vs `target="_blank"`

| 比較項目 | `target="_self"` | `target="_blank"` |
|---|---|---|
| 定義 | 在目前頁面開啟連結。 | 嘗試在新分頁或新視窗開啟連結。 |
| 使用時機 | 預設導覽、網站內部主要流程。 | 外部網站、參考資料或不希望離開目前頁面時。 |
| 常見寫法 | `<a href="./about.html" target="_self">公司簡介</a>` | `<a href="https://example.com" target="_blank" rel="noopener noreferrer">前往 Example</a>` |
| 注意事項 | 沒有指定 `target` 時預設就是 `_self`。 | 外部連結建議搭配 `rel="noopener noreferrer"`。 |
| 常見錯誤 | 對外部連結未評估是否會離開目前流程。 | 使用 `_blank` 但缺少 `rel`。 |

### 選用建議

- 多數站內導覽可使用預設 `_self`。
- 外部新分頁連結應補上 `rel="noopener noreferrer"`。

### 11.3 `href="#"` vs 明確錨點

| 比較項目 | `href="#"` | 明確錨點 |
|---|---|---|
| 定義 | 指向目前頁面的片段位置，常造成回到頁首效果。 | 指向頁面中具體 `id`。 |
| 使用時機 | 本章只作回到頂部或常見錯誤說明。 | 頁內目錄、跳到指定章節。 |
| 常見寫法 | `<a href="#">回到頂部</a>` | `<a href="#season-2">第二季</a>` |
| 注意事項 | 濫用會造成跳動或語意不清。 | 需要對應唯一 `id`。 |
| 常見錯誤 | 用 `href="#"` 假裝按鈕。 | `href` 名稱與 `id` 不一致。 |

### 選用建議

- 需要跳到特定內容時，使用明確錨點。
- 非導航互動不要使用 `href="#"`，改用 `<button>`。

### 11.4 外部連結 vs 內部連結 vs 檔案連結 vs 應用程式連結

| 比較項目 | 外部連結 | 內部連結 | 檔案連結 | 應用程式連結 |
|---|---|---|---|---|
| 定義 | 前往其他網站。 | 前往同一網站中的頁面。 | 指向圖片、PDF、影片、壓縮檔等資源。 | 使用 `tel:`、`mailto:`、`line://` 等 URL scheme。 |
| 使用時機 | 參考資料、合作網站。 | 網站導覽、頁面切換。 | 開啟或下載附件。 | 撥號、寄信或嘗試開啟特定應用。 |
| 常見寫法 | `<a href="https://example.com">前往 Example</a>` | `<a href="./about.html">公司簡介</a>` | `<a href="./assets/pdfs/test.pdf">看 PDF 檔</a>` | `<a href="mailto:10010@qq.com">郵件聯繫</a>` |
| 注意事項 | 新分頁外部連結建議加 `rel`。 | 相對路徑要正確。 | 行為會受瀏覽器與伺服器設定影響。 | 不保證所有環境都能喚起應用。 |
| 常見錯誤 | 連結文字模糊。 | 路徑錯誤。 | `download` 拼錯。 | `mailto:` 格式錯誤或加入空白。 |

### 選用建議

- 根據連結目的選擇合適的 `href` 格式。
- 對使用者可能看不出結果的連結，提供清楚文字說明。
