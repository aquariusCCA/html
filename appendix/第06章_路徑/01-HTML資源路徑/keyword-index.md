# Keyword Index

## 用途

用關鍵字快速找到本篇 HTML 教學筆記中的相關知識點。

## 關鍵字索引

| 關鍵字 | 類型 | 對應段落 | 說明 |
| --- | --- | --- | --- |
| `<img>` | 標籤 | [實務判斷](../../../notes/第06章_路徑/01-HTML資源路徑.md#實務判斷) | 筆記用圖片引用範例說明 HTML 如何透過路徑找到資源。 |
| `src` | 屬性 | [實務判斷](../../../notes/第06章_路徑/01-HTML資源路徑.md#實務判斷) | 在圖片範例中放入資源路徑，讓瀏覽器找到圖片檔案。 |
| `./` | 術語 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) | 表示從目前目錄開始，通常可以省略。 |
| `../` | 術語 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) | 表示先回到上一層目錄再尋找資源。 |
| `/` 開頭路徑 | 術語 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) | 通常表示從網站根目錄開始找，不是下一層資料夾。 |
| HTML 資源路徑 | 概念 | [一句話理解](../../../notes/第06章_路徑/01-HTML資源路徑.md#一句話理解) | HTML 檔案用來找到圖片、連結檔案或其他資源的位置描述。 |
| 外站圖片路徑 | 概念 | [絕對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#絕對路徑) | 使用別人網站圖片網址時，可能因防盜鏈或外站限制而失敗。 |
| 本機絕對路徑 | 概念 | [絕對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#絕對路徑) | 指向自己電腦磁碟位置的路徑，不適合用在正式網頁專案。 |
| 相對路徑 | 概念 | [相對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#相對路徑) | 從目前 HTML 檔案所在位置出發尋找目標資源。 |
| 絕對路徑 | 概念 | [絕對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#絕對路徑) | 從固定起點或完整網址開始尋找目標資源。 |
| 網站根目錄 | 概念 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) | `/images/logo.png` 這類開頭 `/` 的路徑通常從網站根目錄開始找。 |
| 資源位置 | 概念 | [問題情境](../../../notes/第06章_路徑/01-HTML資源路徑.md#問題情境) | 瀏覽器需要依照路徑找到圖片或其他檔案的位置。 |
| 路徑失效 | 場景 | [常見錯誤](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見錯誤) | 本機絕對路徑、誤用 `/`、搬動 HTML 檔案後未更新路徑都可能造成資源載入失敗。 |
| 路徑判斷 | 場景 | [相對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#相對路徑) | 判斷重點是目前 HTML 檔案從自己的位置出發要怎麼走到圖片。 |
