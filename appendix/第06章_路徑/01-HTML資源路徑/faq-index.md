# FAQ Index

## 用途

用常見問題快速找到本篇筆記中能回答疑問的段落。

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| HTML 路徑是什麼？ | 路徑是 HTML 檔案用來找到圖片、連結檔案或其他資源的位置描述。 | [一句話理解](../../../notes/第06章_路徑/01-HTML資源路徑.md#一句話理解) |
| 絕對路徑和相對路徑差在哪裡？ | 絕對路徑從固定位置或完整網址開始找；相對路徑從目前 HTML 檔案所在位置開始找。 | [一句話理解](../../../notes/第06章_路徑/01-HTML資源路徑.md#一句話理解) |
| 為什麼不要用 `D:\day01\images\1.jpg` 這種路徑？ | 這種本機路徑只對自己的電腦有意義，換環境、換資料夾或部署後很容易失效。 | [絕對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#絕對路徑) |
| 可以直接用別人網站上的圖片網址嗎？ | 不建議隨便使用外站圖片網址，外站可能有防盜鏈或其他限制，導致圖片載入失敗。 | [絕對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#絕對路徑) |
| `./images/logo.png` 和 `images/logo.png` 一樣嗎？ | 在筆記範例中兩者都表示從目前目錄開始找，`./` 通常可以省略。 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) |
| `../images/logo.png` 是什麼意思？ | `../` 表示先回到上一層目錄，再進入 `images/` 找到圖片。 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) |
| `/images/logo.png` 是下一層資料夾嗎？ | 通常不是；開頭 `/` 通常表示從網站根目錄開始找。 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) |
| 圖片載入失敗時應該先檢查什麼？ | 先檢查 HTML 檔案與圖片檔案的相對位置，以及路徑是否跟目前檔案位置一致。 | [自我檢查](../../../notes/第06章_路徑/01-HTML資源路徑.md#自我檢查) |
| 為什麼搬動 HTML 檔案後圖片路徑要改？ | 相對路徑的參考點是目前 HTML 檔案的位置，HTML 移到其他資料夾後，走到圖片的路徑也會改變。 | [常見錯誤](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見錯誤) |
