# FAQ Index

## 用途

用常見問題快速找到本篇筆記中能回答的段落。

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| HTML 為什麼需要路徑？ | 因為網頁常會引用圖片或其他檔案，瀏覽器需要依路徑找到資源位置。 | [問題情境](../../../notes/第06章_路徑/01-HTML資源路徑.md#問題情境) |
| 絕對路徑和相對路徑差在哪裡？ | 絕對路徑從固定位置或完整網址開始找，相對路徑從目前 HTML 檔案的位置開始找。 | [一句話理解](../../../notes/第06章_路徑/01-HTML資源路徑.md#一句話理解) |
| `./` 可以省略嗎？ | 通常可以，`./images/logo.png` 常可簡寫成 `images/logo.png`。 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) |
| `../` 代表什麼？ | `../` 表示先回到上一層目錄，再繼續尋找目標資源。 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) |
| `/images/logo.png` 和 `images/logo.png` 一樣嗎？ | 不一樣；開頭的 `/` 通常表示從網站根目錄開始找，不是從目前資料夾往下一層找。 | [常見相對路徑符號](../../../notes/第06章_路徑/01-HTML資源路徑.md#常見相對路徑符號) |
| 為什麼不建議使用 `D:\day01\images\1.jpg` 這種路徑？ | 這種本機路徑只對自己的電腦有意義，換環境或部署後容易失效。 | [絕對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#絕對路徑) |
| 為什麼外站圖片網址有時會失敗？ | 外站伺服器可能開啟防盜鏈，導致圖片放進自己的網頁後載入失敗。 | [絕對路徑](../../../notes/第06章_路徑/01-HTML資源路徑.md#絕對路徑) |
| 搬動 HTML 檔案後需要做什麼？ | 需要重新檢查並修改相對路徑，因為參考點是目前 HTML 檔案的位置。 | [實務判斷](../../../notes/第06章_路徑/01-HTML資源路徑.md#實務判斷) |
| 圖片載入失敗時要先檢查什麼？ | 先檢查 HTML 檔案與圖片檔案的相對位置是否符合路徑寫法。 | [自我檢查](../../../notes/第06章_路徑/01-HTML資源路徑.md#自我檢查) |
