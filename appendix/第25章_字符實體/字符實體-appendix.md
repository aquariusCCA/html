## 3. 名詞表

| 名詞 | 英文 | 簡要定義 | 用途 | 注意事項 | 來源 |
|---|---|---|---|---|---|
| 字符實體 | HTML character entity | 用特殊文字表示某個字符的 HTML 寫法。 | 顯示具有 HTML 語法意義或不方便直接輸入的字符。 | 建議寫完整結尾分號 `;`。 | `2. 本章定位`、`3. 核心概念` |
| 實體名稱 | entity name | 字符實體中代表目標字符的名稱。 | 讓 `&copy;` 這類寫法對應到實際符號。 | 必須搭配 `&` 與 `;` 才是完整常見寫法。 | `3.1 字符實體的組成` |
| 空白折疊 | whitespace collapsing | HTML 一般文字中，連續空格、換行或縮排通常被合併成一個空格的現象。 | 解釋為什麼多個一般空格不會原樣顯示。 | 不應因此大量使用 `&nbsp;` 做排版。 | `4.3 顯示多個空格`、`11.2 空白折疊與排版責任` |
| 不換行空白 | non-breaking space | 不容易在該位置自動換行的空白。 | 少量保留可見空白。 | 不完全等於一般空格，不適合大量控制版面。 | `4.3 顯示多個空格` |
| escaping | escaping | 將具有特殊語法意義的字符改寫成安全顯示形式。 | 避免 `<`、`>`、`&` 被錯誤當成 HTML 語法。 | 不可信內容通常交由模板、框架或安全工具處理。 | `6. 實務應用場景`、`8. 與其他概念的關係` |
| 命名實體 | named character reference | 使用名稱表示字符的 entity 寫法。 | 以較容易閱讀的方式表示常見特殊字符。 | 常見字符優先使用命名實體即可。 | `11.1 命名實體與數字實體` |
| 數字實體 | numeric character reference | 使用字元編號表示字符的 entity 寫法。 | 可用十進位或十六進位表示字符。 | 原筆記只以小於號為例說明。 | `11.1 命名實體與數字實體` |

## 9. 常用片段

### 9.1 顯示 HTML 標籤文字

| 項目 | 說明 |
|---|---|
| 使用情境 | 在頁面中顯示 HTML 標籤本身。 |
| 使用前提 | 需要將 `<` 與 `>` 轉成字符實體。 |
| 來源 | `4.2 顯示小於號與大於號`、`5. 範例說明` |

```html
<p>段落元素的寫法是：&lt;p&gt;內容&lt;/p&gt;</p>
```

### 注意事項

- `&lt;` 顯示為 `<`。
- `&gt;` 顯示為 `>`。
- 瀏覽器不會把 `&lt;p&gt;` 當成真正的段落標籤。

### 9.2 顯示多個可見空白

| 項目 | 說明 |
|---|---|
| 使用情境 | 少量展示連續空白。 |
| 使用前提 | 空白本身是內容需求，而不是版面排版需求。 |
| 來源 | `4.3 顯示多個空格`、`5. 範例說明` |

```html
<p>hello&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;world</p>
```

### 注意事項

- `&nbsp;` 是不換行空白。
- 大量間距應改用 CSS。

### 9.3 顯示版權符號

| 項目 | 說明 |
|---|---|
| 使用情境 | 頁尾或版權資訊。 |
| 使用前提 | 需要顯示版權符號。 |
| 來源 | `4.4 顯示版權符號` |

```html
<footer>
  &copy; 2026 HTML Knowledge Pack
</footer>
```

### 注意事項

- `&copy;` 會顯示為 `©`。
- 年份與名稱依實際內容調整。

### 9.4 同時顯示常見特殊符號

| 項目 | 說明 |
|---|---|
| 使用情境 | 展示小於號、大於號與 `&`。 |
| 使用前提 | 需要將特殊符號視為文字內容顯示。 |
| 來源 | `5. 範例說明` |

```html
<p>HTML 中常見的特殊符號：&lt;、&gt;、&amp;</p>
```

### 注意事項

- 顯示字面上的 `&` 時使用 `&amp;`。
- 這個片段只保留本章最小可用內容。

## 11. 相似概念對照表

### 11.1 一般空格 vs `&nbsp;`

| 比較項目 | 一般空格 | `&nbsp;` |
|---|---|---|
| 定義 | 一般文字空白字元。 | 不換行空白。 |
| 使用時機 | 一般文字分隔。 | 少量需要保留可見空白或不希望在該位置換行。 |
| 常見寫法 | `hello world` | `hello&nbsp;world` |
| 注意事項 | 多個空格通常會被折疊成一個空格。 | 不完全等於一般空格，不應大量排版。 |
| 常見錯誤 | 以為多個一般空格會原樣顯示。 | 用大量 `&nbsp;` 製造版面距離。 |

### 選用建議

- 一般文字分隔使用一般空格。
- 少量需要顯示不換行空白時，才使用 `&nbsp;`。
- 視覺間距需求優先使用 CSS。

### 11.2 字符實體 vs 直接輸入符號

| 比較項目 | 字符實體 | 直接輸入符號 |
|---|---|---|
| 定義 | 用 `&名稱;` 或數字形式表示字符。 | 在 HTML 原始碼中直接寫出字符。 |
| 使用時機 | 字符具有 HTML 語法意義，或不方便直接輸入。 | 普通文字與不會造成解析問題的符號。 |
| 常見寫法 | `&lt;p&gt;`、`&copy;` | `<p>`、`©` |
| 注意事項 | 建議寫完整分號。 | `<`、`>`、`&` 等可能造成解析問題。 |
| 常見錯誤 | 忘記結尾分號。 | 直接寫 `<strong>` 想顯示標籤文字。 |

### 選用建議

- 要顯示 `<`、`>`、`&` 這類容易被解析的符號時，使用字符實體。
- 普通文字符號若不造成 HTML 解析問題，可以直接輸入。

### 11.3 `&nbsp;` vs `<pre>` vs CSS 間距

| 比較項目 | `&nbsp;` | `<pre>` | CSS 間距 |
|---|---|---|---|
| 定義 | 不換行空白字符實體。 | 保留原始空白與換行的 HTML 元素。 | 視覺呈現與版面間距控制。 |
| 使用時機 | 少量保留不換行空白。 | 內容本身需要保留整段空白與換行。 | 元素之間需要距離或排列效果。 |
| 常見寫法 | `hello&nbsp;world` | `<pre>文字</pre>` | `margin`、`padding`、`gap` |
| 注意事項 | 不適合大量排版。 | 屬於內容格式保留，不是一般版面工具。 | 不改變內容語意。 |
| 常見錯誤 | 用很多 `&nbsp;` 製造距離。 | 原筆記未在本章列出 `<pre>` 錯誤，只列為可選方式。 | 原筆記未列出 CSS 錯誤情境。 |

### 選用建議

- 少量空白本身是內容時，可使用 `&nbsp;`。
- 整段文字格式本身需要保留時，可考慮 `<pre>`。
- 只是畫面需要距離時，使用 CSS。
