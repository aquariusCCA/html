# Element / Attribute Index

## 用途

查找本篇筆記明確教學的 HTML 全局屬性，以及它們和鍵盤操作的關係。

## HTML 標籤

本篇沒有以特定 HTML 標籤作為主教學對象；範例中的 `<a>`、`<form>`、`<label>`、`<input>`、`<button>` 只用來示範或對照鍵盤焦點行為，因此不列為主條目。

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `accesskey` | 任何 HTML 元素 | 為元素設定鍵盤快捷方式，但實務上應謹慎使用。 | [`accesskey`：設定訪問快捷鍵](../../../notes/第05章_全局屬性/05-鍵盤操作與焦點順序屬性.md#accesskey設定訪問快捷鍵) |
| `tabindex` | 需要管理鍵盤焦點的元素 | 控制元素是否進入 Tab 順序，以及焦點順序如何決定。 | [`tabindex`：控制 Tab 焦點順序](../../../notes/第05章_全局屬性/05-鍵盤操作與焦點順序屬性.md#tabindex控制-tab-焦點順序) |
