# Element / Attribute Index

## 用途

查找本篇筆記明確教學的 HTML 全局屬性，以及它們的主要用途與使用時機。

## HTML 標籤

本篇以全局屬性為核心，`<p>`、`<h1>`、`<h2>`、`<a>`、`<li>`、`<ul>`、`<button>` 只作為範例載體或情境說明；未將它們升格為 HTML 標籤主條目。

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `id` | 多數 HTML 元素 | 標記同一份文件中唯一的元素，常用於 CSS、JavaScript 與頁面內錨點。 | [`id`：元素的唯一識別](../../../notes/第05章_全局屬性/01-元素識別分類與附加資訊屬性.md#id元素的唯一識別) |
| `class` | 多數 HTML 元素 | 替元素設定分類名稱，讓 CSS 或 JavaScript 選取一組元素。 | [`class`：元素的分類名稱](../../../notes/第05章_全局屬性/01-元素識別分類與附加資訊屬性.md#class元素的分類名稱) |
| `data-*` | 多數 HTML 元素 | 存放頁面或應用程式自己的私有資料，供 JavaScript 從 DOM 讀取。 | [`data-*`：元素上的自訂資料](../../../notes/第05章_全局屬性/01-元素識別分類與附加資訊屬性.md#data-元素上的自訂資料) |
| `style` | 多數 HTML 元素 | 直接在元素上寫行內 CSS，適合少量特殊情況。 | [`style`：直接寫在元素上的 CSS](../../../notes/第05章_全局屬性/01-元素識別分類與附加資訊屬性.md#style直接寫在元素上的-css) |
