# Element / Attribute Index

## 用途

快速查找本篇筆記明確教學或反覆使用的 HTML 標籤與屬性。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<html>` | 承載整份文件層級的語言設定，也可用來設定整份文件是否應被翻譯。 | `lang`、`translate` | [`lang`：標示元素內容語言](../../../notes/第05章_全局屬性/02-語言文字方向與翻譯屬性.md#lang標示元素內容語言) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `lang` | `<html>` 或局部 HTML 元素 | 標示文件或局部內容使用的語言。 | [`lang`：標示元素內容語言](../../../notes/第05章_全局屬性/02-語言文字方向與翻譯屬性.md#lang標示元素內容語言) |
| `dir` | 需要指定文字方向的 HTML 元素 | 指定內容由左到右、由右到左，或由瀏覽器自動判斷方向。 | [`dir`：控制文字方向](../../../notes/第05章_全局屬性/02-語言文字方向與翻譯屬性.md#dir控制文字方向) |
| `translate` | `<html>` 或不應被翻譯的局部 HTML 元素 | 指定元素內容是否應被翻譯，常見用法是 `translate="no"`。 | [`translate`：控制內容是否被翻譯](../../../notes/第05章_全局屬性/02-語言文字方向與翻譯屬性.md#translate控制內容是否被翻譯) |

## 補充

`<span>`、`<p>`、`<cite>`、`<head>`、`<meta>`、`<title>` 與 `<body>` 只作為範例中的承載標籤或文件骨架，不是本篇主要教學標籤，因此不列入 HTML 標籤主索引。
