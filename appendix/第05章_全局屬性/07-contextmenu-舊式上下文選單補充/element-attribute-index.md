# Element / Attribute Index

## 用途

查找本篇筆記明確教學的舊式 HTML 全局屬性，以及它和現代右鍵事件處理的關係。

## HTML 標籤

本篇沒有以特定 HTML 標籤作為主教學對象；範例中的 `<p>` 只用來示範事件綁定，`<menu type="context">` 與 `<menuitem>` 只用來說明舊式或 obsolete 上下文選單設計，因此不列為主條目。
