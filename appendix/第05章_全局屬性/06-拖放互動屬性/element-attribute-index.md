# Element / Attribute Index

## 用途

快速查找本篇筆記明確教學的 HTML 屬性，以及它們在拖放互動中的角色。

## HTML 標籤

本篇沒有足夠依據建立主教學標籤索引。範例中的 `<div>` 只作為拖曳元素與目標容器，`<a>` 與圖像只用來說明預設可拖動行為，因此不列為主索引條目。
