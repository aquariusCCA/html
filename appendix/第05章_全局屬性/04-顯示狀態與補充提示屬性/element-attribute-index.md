# Element / Attribute Index

## 用途

快速查找本篇筆記中明確教學的 HTML 屬性，以及相關元素是否被列為主索引。

## HTML 標籤

本篇主要教學全局屬性。筆記中的 `<p>`、`<label>`、`<input>`、`<script>`、`<abbr>`、`<button>` 等元素屬於範例、承載內容或情境對照用途，未升格為主教學標籤索引。
