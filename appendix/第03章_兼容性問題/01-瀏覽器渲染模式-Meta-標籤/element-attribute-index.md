# Element / Attribute Index

## 用途

查找本篇筆記主要教學的 HTML 標籤與屬性。

本篇範例也出現 `<html>`、`<head>`、`<title>`、`<body>`、`<h1>` 與 `charset`，但它們主要用於完整文件骨架或編碼示例，不列為本篇主索引條目。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<meta>` | 在 `<head>` 中提供文件資訊或瀏覽器提示；本篇聚焦兼容性渲染設定。 | `http-equiv`、`name`、`content` | [一句話理解](../../../notes/第03章_兼容性問題/01-瀏覽器渲染模式-Meta-標籤.md#一句話理解) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `http-equiv` | `<meta>` | 指定類 HTTP 等效資訊；本篇用於 `X-UA-Compatible`。 | [IE 文檔模式與 X-UA-Compatible](../../../notes/第03章_兼容性問題/01-瀏覽器渲染模式-Meta-標籤.md#ie-文檔模式與-x-ua-compatible) |

| `content` | `<meta>` | 提供 meta 設定值，例如 `IE=Edge` 或 `webkit`。 | [兩種 meta 的對照](../../../notes/第03章_兼容性問題/01-瀏覽器渲染模式-Meta-標籤.md#兩種-meta-的對照) |
