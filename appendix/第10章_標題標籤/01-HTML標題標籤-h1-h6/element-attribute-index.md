# Element / Attribute Index

## 用途

快速查找本篇筆記明確教學的 HTML 標題標籤與相關屬性資訊。

| --- | --- | --- | --- |

## HTML 屬性

本篇筆記沒有明確教學 `<h1>` 到 `<h6>` 的專用 HTML 屬性；`class` 只出現在外觀調整範例中，因此不列為本索引的主屬性條目。
