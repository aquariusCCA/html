# Element / Attribute Index

## 用途

查找本篇筆記中直接示範的 HTML 標籤與屬性。

本篇是 HTML 入門概念筆記，不是標籤專題；下列只收錄來源筆記用來說明「標籤描述內容」的範例標籤與屬性。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<h1>` | 表示頁面的主要標題。 | 本篇未教學屬性。 | [HTML 是什麼](../../../notes/第02章_HTML簡介/01-網頁與HTML入門.md#html-是什麼) |
| `<h2>` | 標記章節標題。 | 本篇未教學屬性。 | [HTML 不是程式語言](../../../notes/第02章_HTML簡介/01-網頁與HTML入門.md#html-不是程式語言) |
| `<p>` | 表示段落文字。 | 本篇未教學屬性。 | [HTML 是什麼](../../../notes/第02章_HTML簡介/01-網頁與HTML入門.md#html-是什麼) |
| `<a>` | 建立前往其他資源的超連結。 | `href` | [HTML 是什麼](../../../notes/第02章_HTML簡介/01-網頁與HTML入門.md#html-是什麼) |
| `<img>` | 在頁面中放置圖片內容。 | `src`、`alt` | [HTML 不是程式語言](../../../notes/第02章_HTML簡介/01-網頁與HTML入門.md#html-不是程式語言) |

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `href` | `<a>` | 指定連結範例要前往的網址。 | [HTML 是什麼](../../../notes/第02章_HTML簡介/01-網頁與HTML入門.md#html-是什麼) |
| `src` | `<img>` | 指定圖片範例的資源路徑。 | [HTML 不是程式語言](../../../notes/第02章_HTML簡介/01-網頁與HTML入門.md#html-不是程式語言) |
| `alt` | `<img>` | 提供圖片範例的文字說明。 | [HTML 不是程式語言](../../../notes/第02章_HTML簡介/01-網頁與HTML入門.md#html-不是程式語言) |
