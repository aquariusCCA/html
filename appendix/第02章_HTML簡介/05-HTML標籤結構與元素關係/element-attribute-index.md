# Element / Attribute Index

## 用途

查找本篇筆記中直接示範的 HTML 標籤與範例屬性。

本篇重點是標籤結構與元素關係，不是單一標籤或屬性的完整用法教學；下列屬性只收錄來源筆記範例中直接出現、且能幫助辨識元素語法的項目。

| --- | --- | --- | --- |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `href` | `<a>` | 範例中指定 `a` 元素要前往的網址或路徑。 | [雙標籤與不寫結束標籤的元素](../../../notes/第02章_HTML簡介/05-HTML標籤結構與元素關係.md#雙標籤與不寫結束標籤的元素) |
| `src` | `<img>` | 範例中指定圖片檔案 `photo.jpg`。 | [雙標籤與不寫結束標籤的元素](../../../notes/第02章_HTML簡介/05-HTML標籤結構與元素關係.md#雙標籤與不寫結束標籤的元素) |
| `alt` | `<img>` | 範例中提供圖片的文字說明。 | [雙標籤與不寫結束標籤的元素](../../../notes/第02章_HTML簡介/05-HTML標籤結構與元素關係.md#雙標籤與不寫結束標籤的元素) |
| `type` | `<input>` | 範例中設定 `input` 的型態為 `text`。 | [雙標籤與不寫結束標籤的元素](../../../notes/第02章_HTML簡介/05-HTML標籤結構與元素關係.md#雙標籤與不寫結束標籤的元素) |
| `name` | `<input>` | 範例中設定 `input` 的名稱為 `username`。 | [雙標籤與不寫結束標籤的元素](../../../notes/第02章_HTML簡介/05-HTML標籤結構與元素關係.md#雙標籤與不寫結束標籤的元素) |
