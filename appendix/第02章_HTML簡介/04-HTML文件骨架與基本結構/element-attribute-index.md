# Element / Attribute Index

## 用途

查找本篇筆記中明確教學的 HTML 結構標籤與重要屬性。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<html>` | 作為整份 HTML 文件的根元素，包住所有 HTML 內容。 | 本篇未特別說明。 | [結構拆解](../../../notes/第02章_HTML簡介/04-HTML文件骨架與基本結構.md#結構拆解) |
| `<head>` | 放文件設定與描述資訊，通常不直接顯示在頁面主體。 | 本篇未特別說明。 | [head 與 body 的分工](../../../notes/第02章_HTML簡介/04-HTML文件骨架與基本結構.md#head-與-body-的分工) |
| `<meta>` | 放描述資訊；本篇重點是用 `charset` 指定文件編碼。 | `charset` | [結構拆解](../../../notes/第02章_HTML簡介/04-HTML文件骨架與基本結構.md#結構拆解) |
| `<title>` | 設定瀏覽器分頁標題，也會影響書籤與搜尋結果顯示。 | 本篇未特別說明。 | [結構拆解](../../../notes/第02章_HTML簡介/04-HTML文件骨架與基本結構.md#結構拆解) |
| `<body>` | 放使用者會在頁面中看到的主要內容。 | 本篇未特別說明。 | [head 與 body 的分工](../../../notes/第02章_HTML簡介/04-HTML文件骨架與基本結構.md#head-與-body-的分工) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `charset` | `<meta>` | 指定文件編碼，避免中文或特殊字元顯示錯誤。 | [忘記設定字元編碼](../../../notes/第02章_HTML簡介/04-HTML文件骨架與基本結構.md#忘記設定字元編碼) |
