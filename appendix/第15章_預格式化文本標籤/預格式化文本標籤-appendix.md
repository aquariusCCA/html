## 9. 常用片段

### 9.1 最小預格式化文字

| 項目 | 說明 |
|---|---|
| 使用情境 | 需要保留文字中的空白、縮排與換行。 |
| 使用前提 | 文字格式本身是內容的一部分。 |
| 來源 | `4. 語法與基本用法` |

```html
<pre>
要保留格式的文字
</pre>
```

### 注意事項

- `<pre>` 是一般容器元素，需要結束標籤。
- 標籤內的縮排與換行會影響顯示結果。

### 9.2 顯示程式碼片段

| 項目 | 說明 |
|---|---|
| 使用情境 | 顯示一段需要保留縮排與換行的程式碼。 |
| 使用前提 | 內容是程式碼，且需要表達程式碼語意。 |
| 來源 | `4. 語法與基本用法` |

```html
<pre><code>body {
  background: #fff;
  font-size: 16px;
}</code></pre>
```

### 注意事項

- `<pre>` 保留格式。
- `<code>` 表示內容是程式碼。

### 9.3 顯示 HTML 原始碼

| 項目 | 說明 |
|---|---|
| 使用情境 | 在頁面上顯示 HTML 標籤本身。 |
| 使用前提 | HTML 特殊字元需要轉成 entity。 |
| 來源 | `5. 範例說明`、`7. 常見錯誤與修正` |

```html
<pre><code>&lt;h1&gt;Hello HTML&lt;/h1&gt;</code></pre>
```

### 注意事項

- 將 `<` 寫成 `&lt;`。
- 將 `>` 寫成 `&gt;`。
- `<pre>` 不會自動把 HTML 標籤轉成可顯示的純文字。

### 9.4 改善 `<pre>` 閱讀體驗

| 項目 | 說明 |
|---|---|
| 使用情境 | `<pre>` 內容較長，需要水平捲動或區塊樣式。 |
| 使用前提 | HTML 已用 `<pre>` 保留內容格式。 |
| 來源 | `6. 實務應用場景` |

```html
<pre class="code-block"><code>body {
  background: #fff;
  font-size: 16px;
}</code></pre>
```

```css
.code-block {
  overflow-x: auto;
  padding: 16px;
  background: #f6f8fa;
}
```

### 注意事項

- HTML 負責保留程式碼格式。
- CSS 負責讓程式碼區塊更容易閱讀。

## 11. 相似概念對照表

### 11.1 `<pre>` vs `<p>`

| 比較項目 | `<pre>` | `<p>` |
|---|---|---|
| 定義 | 表示需要保留原始格式的文字。 | 表示一般段落內容。 |
| 使用時機 | 文字中的空格、縮排與換行需要照原樣呈現。 | 普通段落文字。 |
| 常見寫法 | `<pre>...</pre>` | `<p>...</p>` |
| 注意事項 | 標籤內縮排也可能被顯示。 | 一般文字會套用 HTML 空白處理規則。 |
| 常見錯誤 | 用來製造一般段落換行或間距。 | 把需要保留格式的程式碼片段放進 `<p>`。 |

### 選用建議

- 需要保留原始文字格式時，使用 `<pre>`。
- 只是一般段落內容時，使用 `<p>`。

### 11.2 `<pre>` vs `<br>`

| 比較項目 | `<pre>` | `<br>` |
|---|---|---|
| 定義 | 整段保留文字中的空白、縮排與換行。 | 在文字中插入單一換行。 |
| 使用時機 | 一整段文字格式都需要保留。 | 同一段內容只需要在特定位置換行。 |
| 常見寫法 | `<pre>...</pre>` | `第一行<br>第二行` |
| 注意事項 | 可能保留多餘縮排。 | 不保留整段空白格式。 |
| 常見錯誤 | 用 `<pre>` 控制普通版面。 | 用大量 `<br>` 製造排版效果。 |

### 選用建議

- 整段固定格式文字使用 `<pre>`。
- 單一換行位置使用 `<br>`。

### 11.3 `<pre>` vs `<code>`

| 比較項目 | `<pre>` | `<code>` |
|---|---|---|
| 定義 | 預格式化文字元素。 | 表示內容是一段程式碼。 |
| 使用時機 | 需要保留換行與縮排。 | 需要標示程式碼語意。 |
| 常見寫法 | `<pre>...</pre>` | `<code>...</code>` |
| 注意事項 | 不一定代表內容是程式碼。 | 不負責整段保留換行與縮排。 |
| 常見錯誤 | 只用 `<pre>` 表示程式碼語意。 | 只用 `<code>` 顯示多行程式碼，格式可能不符合需求。 |

### 選用建議

- 程式碼區塊常用 `<pre><code>...</code></pre>`。
- `<pre>` 負責保留格式，`<code>` 負責程式碼語意。

### 11.4 `<pre>` vs CSS `white-space`

| 比較項目 | `<pre>` | CSS `white-space` |
|---|---|---|
| 定義 | HTML 預格式化文本元素。 | 與元素空白處理相關的 CSS 概念。 |
| 使用時機 | 內容本身就是預格式化文字。 | 需要調整元素顯示空白的細節。 |
| 常見寫法 | `<pre>...</pre>` | 原筆記未提供具體屬性值範例。 |
| 注意事項 | 屬於 HTML 結構與語意選擇。 | 屬於視覺呈現與顯示行為調整。 |
| 常見錯誤 | 用 `<pre>` 解決所有排版問題。 | 原筆記未列出 CSS `white-space` 的錯誤情境。 |

### 選用建議

- 內容本身需要預格式化語意時，使用 `<pre>`。
- 只調整顯示效果時，依筆記說明可由 CSS 處理外觀細節。
