# FAQ Index

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| `<title>` 是做什麼的？ | `<title>` 用來設定 HTML 文件的頁面標題。 | [一句話理解](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#一句話理解) |
| `<title>` 應該放在哪裡？ | `<title>` 應該寫在 `<head>` 裡，屬於文件資訊。 | [基本語法](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#基本語法) |
| `<title>` 會顯示在頁面內容區嗎？ | 通常不會；頁面內容區的主標題應使用 `<h1>` 等標題標籤。 | [title 和 h1 的差異](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#title-和-h1-的差異) |
| `<title>` 和 `<h1>` 有什麼不同？ | `<title>` 是文件標題，常給瀏覽器、書籤與搜尋引擎使用；`<h1>` 是頁面正文的主要標題。 | [title 和 h1 的差異](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#title-和-h1-的差異) |
| 搜尋結果一定會顯示 `<title>` 的內容嗎？ | 不一定完全相同，但 `<title>` 通常是搜尋引擎判斷頁面標題的重要依據之一。 | [title 的主要作用](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#title-的主要作用) |
| 每個 HTML 頁面都需要 `<title>` 嗎？ | 筆記建議每個 HTML 頁面都應該有清楚的 `<title>`。 | [實務使用原則](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#實務使用原則) |
| 可以只把 `<title>` 寫成「首頁」嗎？ | 不建議；太籠統的標題會讓使用者和搜尋引擎較難分辨頁面主題。 | [標題太籠統](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#標題太籠統) |
| `<title>` 需要結束標籤嗎？ | 需要，標題文字應放在 `<title>` 與 `</title>` 之間。 | [忘記結束標籤](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#忘記結束標籤) |
