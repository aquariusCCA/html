# Element / Attribute Index

## 用途

快速查找本篇筆記明確教學或反覆使用的 HTML 標籤與屬性。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<title>` | 設定 HTML 文件的頁面標題，常影響瀏覽器分頁、書籤與搜尋結果標題參考。 | 本篇未教學特定屬性 | [基本語法](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#基本語法) |
| `<head>` | 放置 `<title>` 這類文件資訊，不直接顯示為頁面正文內容。 | 本篇未教學特定屬性 | [基本語法](../../../notes/第09章_title標籤/01-title標籤與網頁標題.md#基本語法) |

## HTML 屬性

本篇筆記沒有明確教學 `<title>` 專屬或直接相關的 HTML 屬性，因此不建立屬性條目。

## 補充

`<h1>` 與 `<body>` 在本篇主要用來對照文件標題與頁面正文標題的差異，不列入主 HTML 標籤表。
