## 9. 常用片段

### 9.1 基本圖片

| 項目 | 說明 |
|---|---|
| 使用情境 | 在頁面中顯示一張有內容意義的圖片。 |
| 使用前提 | 圖片檔案路徑正確，且圖片需要替代文字。 |
| 來源 | §3、§5.1 |

```html
<img src="./images/product.jpg" alt="黑色無線耳機產品照片">
```

### 注意事項

- `src` 指向圖片檔案。
- `alt` 說明圖片內容。
- `<img>` 不需要結束標籤。

### 9.2 圖片載入失敗時的替代文字

| 項目 | 說明 |
|---|---|
| 使用情境 | 圖片可能載入失敗，但仍需要保留內容線索。 |
| 使用前提 | 圖片有內容意義。 |
| 來源 | §5.2、§8 |

```html
<img src="./images/not-found.jpg" alt="使用者個人頭像">
```

### 注意事項

- 圖片失敗時，`alt` 可讓使用者知道原本內容。
- 不要把 `alt` 寫成無語意的檔名。

### 9.3 裝飾圖片

| 項目 | 說明 |
|---|---|
| 使用情境 | 圖片只是裝飾，不承載內容意義。 |
| 使用前提 | 移除圖片後不影響內容理解。 |
| 來源 | §4.3、§8 |

```html
<img src="./images/decorative-line.png" alt="">
```

### 注意事項

- 空 `alt` 可避免裝飾圖片干擾輔助工具讀取。
- 不要把所有圖片都當成裝飾圖片。

### 9.4 圖片提示文字

| 項目 | 說明 |
|---|---|
| 使用情境 | 圖片需要額外提示資訊。 |
| 使用前提 | 圖片本身仍需有適當 `alt`。 |
| 來源 | §4.4 |

```html
<img
  src="./images/chart.png"
  alt="第二季銷售額統計圖"
  title="資料來源：2026 年第二季報表"
>
```

### 注意事項

- `title` 是額外提示，不是替代文字。
- 圖片內容意義仍應放在 `alt`。

### 9.5 只設定圖片寬度

| 項目 | 說明 |
|---|---|
| 使用情境 | 調整圖片顯示寬度並維持比例。 |
| 使用前提 | 不需要強制固定圖片高度。 |
| 來源 | §4.5、§5.3 |

```html
<img src="./images/banner.jpg" alt="課程活動橫幅" width="600">
```

### 注意事項

- 只設定寬度時，高度通常依照原始比例計算。
- 比同時亂設 `width` 與 `height` 更不容易變形。

### 9.6 響應式圖片

| 項目 | 說明 |
|---|---|
| 使用情境 | 圖片需要隨容器縮放。 |
| 使用前提 | 需要搭配 CSS 控制顯示尺寸。 |
| 來源 | §4.5、§5.3、§7 |

```html
<img class="responsive-image" src="./images/banner.jpg" alt="課程活動橫幅">
```

```css
.responsive-image {
  max-width: 100%;
  height: auto;
}
```

### 注意事項

- `max-width: 100%` 避免圖片超出容器。
- `height: auto` 保留原始比例。

### 9.7 base64 圖片來源

| 項目 | 說明 |
|---|---|
| 使用情境 | 小型圖片或特殊情境下需要把圖片資料直接寫入 `src`。 |
| 使用前提 | 圖片內容已轉成 base64 資料。 |
| 來源 | §4.2、§6 |

```html
<img src="data:image/png;base64,..." alt="內嵌的小圖示">
```

### 注意事項

- 不適合大量或大型圖片。
- 可能讓 HTML 或 CSS 檔案變大且較難維護。

## 10. 錯誤速查表

| 錯誤現象 | 可能原因 | 修正方向 | 對應知識點 | 來源 |
|---|---|---|---|---|
| 圖片無法顯示。 | `src` 路徑寫錯，瀏覽器找不到圖片檔案。 | 確認 HTML 檔案與圖片檔案的相對位置。 | `src`、HTML 路徑 | §4.2、§8 |
| 圖片失敗或輔助工具讀取時資訊不足。 | 缺少 `alt`。 | 為有內容意義的圖片補上清楚替代文字。 | `alt`、可訪問性 | §4.3、§8 |
| 用 `title` 取代 `alt`。 | 誤以為提示文字等於替代文字。 | 圖片內容意義應放在 `alt`。 | `alt` vs `title` | §4.4、§8 |
| 圖片被拉伸或壓扁。 | 同時設定了不符合原圖比例的 `width` 與 `height`。 | 只設定其中一個尺寸，或用 CSS 保持比例。 | `width`、`height`、圖片比例 | §4.5、§8 |
| 圖片拖慢頁面載入。 | 把大型圖片直接放進頁面。 | 壓縮圖片並選擇合適格式。 | 圖片格式、網頁效能 | §6、§8、§9 |
| 輔助工具讀出無意義的裝飾圖片描述。 | 裝飾圖片也寫了冗長 `alt`。 | 裝飾圖片可用 `alt=""`。 | 裝飾圖片、`alt` | §4.3、§8 |
| 圖片失敗時重要文字消失。 | 只用圖片呈現重要文字。 | 重要文字應使用 HTML 文字，圖片只作輔助。 | HTML 文字、圖片用途 | §8 |
| `alt` 無法有效描述圖片。 | 替代文字只寫檔名或過於模糊。 | 改成描述圖片在此處要傳達的資訊。 | `alt` 寫法 | §4.3、§8 |

## 11. 相似概念對照表

### 11.1 `alt` vs `title`

| 比較項目 | `alt` | `title` |
|---|---|---|
| 定義 | 圖片替代文字。 | 圖片額外提示文字。 |
| 使用時機 | 圖片有內容意義，需提供文字替代。 | 圖片需要額外提示資訊。 |
| 常見寫法 | `alt="公司 Logo"` | `title="資料來源：2026 年第二季報表"` |
| 注意事項 | 圖片失敗與輔助工具讀取時很重要。 | 視情況使用，不應取代 `alt`。 |
| 常見錯誤 | 只寫檔名或省略。 | 誤當成替代文字。 |

### 選用建議

- 圖片內容意義放在 `alt`。
- 額外補充提示可視情況放在 `title`。

### 11.2 內容圖片 vs 裝飾圖片

| 比較項目 | 內容圖片 | 裝飾圖片 |
|---|---|---|
| 定義 | 承載頁面資訊或內容意義的圖片。 | 不影響內容理解的視覺裝飾。 |
| 使用時機 | 產品照片、圖表、截圖、頭像等。 | 裝飾線、背景性圖樣等。 |
| 常見寫法 | `<img src="./images/product.jpg" alt="黑色無線耳機產品照片">` | `<img src="./images/decorative-line.png" alt="">` |
| 注意事項 | `alt` 應描述圖片用途。 | 空 `alt` 可避免干擾輔助工具。 |
| 常見錯誤 | 缺少 `alt`。 | 寫入冗長且無意義的 `alt`。 |

### 選用建議

- 圖片如果會影響內容理解，就提供清楚 `alt`。
- 圖片若只是裝飾，依 notes 可使用空 `alt`。

### 11.3 只設定單一尺寸 vs 同時固定寬高

| 比較項目 | 只設定 `width` 或 `height` | 同時固定 `width` 與 `height` |
|---|---|---|
| 定義 | 只指定一邊尺寸，另一邊依比例縮放。 | 同時指定寬度與高度。 |
| 使用時機 | 想調整大小並維持原比例。 | 需要固定顯示尺寸時。 |
| 常見寫法 | `<img src="./images/photo.jpg" alt="活動照片" width="300">` | `<img src="./images/photo.jpg" alt="活動照片" width="300" height="300">` |
| 注意事項 | 通常較不容易變形。 | 若比例不符合原圖，圖片可能變形。 |
| 常見錯誤 | 未確認圖片在容器中的寬度。 | 亂設寬高造成拉伸或壓扁。 |

### 選用建議

- 只需要縮放大小時，優先只設定其中一邊或使用 CSS 保持比例。
- 同時固定寬高時，要確認比例或另行處理裁切策略。

### 11.4 jpg / png / gif / webp / base64

| 比較項目 | jpg / jpeg | png | gif | webp | base64 |
|---|---|---|---|---|---|
| 定義 | 有損壓縮圖片格式。 | 無損壓縮圖片格式。 | 支援簡單動畫的圖片格式。 | 專門用於網頁圖片的格式。 | 將圖片轉成文字資料。 |
| 使用時機 | 照片、產品圖、宣傳圖。 | Logo、重要配圖、截圖、透明背景。 | 小型動態圖片。 | 網頁中的各種圖片。 | 小型圖片或需與 HTML / CSS 一起載入的小圖。 |
| 常見寫法 | `photo.jpg` | `logo.png` | `loading.gif` | `image.webp` | `data:image/png;base64,...` |
| 注意事項 | 不支援透明背景與動畫。 | 檔案可能比 jpg 大。 | 色彩表現有限。 | 原筆記提醒兼容性需處理。 | 不適合大型圖片。 |
| 常見錯誤 | 用於需要透明背景的圖片。 | 大量使用造成檔案偏大。 | 用於高品質照片。 | 未確認目標環境支援。 | 讓 HTML 或 CSS 變得過大。 |

### 選用建議

- 照片類圖片可依 notes 考慮 jpg。
- 需要透明背景或較高品質呈現時，可依 notes 考慮 png。
- 動態圖片可依 notes 考慮 gif。
- webp 與 base64 使用前需注意 notes 提到的兼容性與維護性。

### 11.5 HTML 圖片、CSS、JavaScript、DOM

| 比較項目 | HTML 圖片 | CSS | JavaScript | DOM |
|---|---|---|---|---|
| 定義 | 用 `<img>` 把圖片資源放入文件結構。 | 控制圖片尺寸、裁切、邊框、排版與響應式效果。 | 可操作圖片屬性，例如切換 `src`。 | 瀏覽器解析後的圖片元素節點。 |
| 使用時機 | 圖片是頁面內容的一部分。 | 需要控制圖片視覺呈現。 | 需要互動或動態切換圖片。 | 理解 CSS / JavaScript 操作基礎。 |
| 常見寫法 | `<img src="./images/product.jpg" alt="商品照片">` | `max-width: 100%; height: auto;` | 原筆記未提供具體 JavaScript 範例。 | 原筆記未提供具體 DOM 範例。 |
| 注意事項 | 應提供正確 `src` 與適當 `alt`。 | 不取代 HTML 的內容語意。 | 本章未展開 API 或事件教學。 | 本章只說明 `<img>` 會被解析成 DOM 節點。 |
| 常見錯誤 | 用圖片承載所有重要文字。 | 用錯尺寸導致變形。 | 將本章擴寫成 JavaScript API 教學。 | 將 DOM 關係誤當成本章主線。 |

### 選用建議

- HTML 負責圖片內容與結構。
- CSS 負責圖片呈現。
- JavaScript 與 DOM 只在需要互動或動態操作時再進一步處理。
