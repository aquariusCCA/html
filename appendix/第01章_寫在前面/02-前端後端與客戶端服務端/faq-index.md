# FAQ Index

## 用途

用常見問題快速找到本篇筆記中能回答的段落。

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| 前端主要負責什麼？ | 前端面向使用者，主要負責瀏覽器中的內容呈現、畫面樣式、互動行為、資料請求與畫面整理。 | [前端負責什麼](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#前端負責什麼) |
| 後端主要負責什麼？ | 後端面向資料與規則，負責接收請求、驗證身分、處理業務規則、讀寫資料庫並回傳資料或頁面。 | [後端負責什麼](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#後端負責什麼) |
| 使用者操作網站時，前端和後端大致怎麼配合？ | 通常是前端送出請求，後端處理資料或規則並回傳結果，再由前端把結果顯示給使用者。 | [問題情境](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#問題情境) |
| 資料庫是什麼？ | 資料庫是儲存資料的地方，例如使用者資料、商品資料或訂單資料。 | [資料庫與服務器](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#資料庫與服務器) |
| 服務器是什麼？ | 服務器是對外提供服務的機器、程式或系統，可以保存資訊、處理請求並傳回結果。 | [資料庫與服務器](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#資料庫與服務器) |
| 客戶端是什麼？ | 客戶端是在網路互動中發出請求、使用服務的一方。 | [客戶端與服務端](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#客戶端與服務端) |
| 服務端是什麼？ | 服務端是在網路互動中接收請求、提供服務的一方。 | [客戶端與服務端](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#客戶端與服務端) |
| 前端等於客戶端嗎？ | 不能完全畫等號。前端 / 後端是工程分工，客戶端 / 服務端是網路互動角色。 | [前端 / 後端與客戶端 / 服務端的差異](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#前端--後端與客戶端--服務端的差異) |
| 後端等於服務端嗎？ | 入門時可簡化理解為後端常運行在服務端，但兩者關注角度不同，不能完全等同。 | [前端 / 後端與客戶端 / 服務端的差異](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#前端--後端與客戶端--服務端的差異) |
| 服務器只是一台硬體嗎？ | 不是。服務器可以是硬體主機，也可以指在主機上運行並提供服務的程式或系統。 | [把服務器理解成只是一台硬體](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#把服務器理解成只是一台硬體) |
| 終端是什麼？ | 終端通常指可以輸入命令、操作系統或開發工具的命令列介面。 | [終端是什麼](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#終端是什麼) |
| 終端是客戶端或服務端嗎？ | 不是。終端只是操作電腦或開發環境的一種介面，不等於客戶端或服務端。 | [終端是什麼](../../../notes/第01章_寫在前面/02-前端後端與客戶端服務端.md#終端是什麼) |
