用關鍵字快速找到本篇 HTML 教學筆記中的相關知識點。

| 關鍵字 | 類型 | 對應段落 | 說明 |
| --- | --- | --- | --- |
| `<input>` | 標籤 | [input 的基本語法](../../../notes/測試/02-input標籤與基礎輸入控件.md#input-的基本語法) | 建立表單中的輸入控件。 |
| `type` | 屬性 | [常用屬性](../../../notes/測試/02-input標籤與基礎輸入控件.md#常用屬性) | 決定 `<input>` 顯示成哪一種輸入控件。 |

| `type="password"` | 術語 | [密碼框](../../../notes/測試/02-input標籤與基礎輸入控件.md#密碼框) | 建立密碼輸入框，畫面上的字元會被遮蔽。 |
| `type="hidden"` | 術語 | [hidden 隱藏域](../../../notes/測試/02-input標籤與基礎輸入控件.md#hidden-隱藏域) | 建立使用者看不到、但提交時會一起送出的欄位。 |
| input | 術語 | [一句話理解](../../../notes/測試/02-input標籤與基礎輸入控件.md#一句話理解) | 表單中最常見的輸入控件，樣子由 `type` 決定。 |
| 輸入控件 | 概念 | [一句話理解](../../../notes/測試/02-input標籤與基礎輸入控件.md#一句話理解) | 讓使用者輸入資料的表單控件。 |
| 單標籤 | 概念 | [input 的基本語法](../../../notes/測試/02-input標籤與基礎輸入控件.md#input-的基本語法) | `<input>` 不需要寫成 `<input></input>`。 |
| 文本框 | 概念 | [文本框](../../../notes/測試/02-input標籤與基礎輸入控件.md#文本框) | 用於單行文字輸入。 |

| 隱藏域 | 概念 | [hidden 隱藏域](../../../notes/測試/02-input標籤與基礎輸入控件.md#hidden-隱藏域) | 用來提交使用者看不到的固定資料。 |
| 欄位名稱 | 概念 | [input 的基本語法](../../../notes/測試/02-input標籤與基礎輸入控件.md#input-的基本語法) | 提交資料時用來識別欄位的名稱。 |
| 預設值 | 概念 | [常見錯誤：把 value 當成提示文字](../../../notes/測試/02-input標籤與基礎輸入控件.md#常見錯誤把-value-當成提示文字) | `value` 會成為真正資料值，不能拿來假裝提示文字。 |
| 提示文字 | 概念 | [常見錯誤：把 value 當成提示文字](../../../notes/測試/02-input標籤與基礎輸入控件.md#常見錯誤把-value-當成提示文字) | 應使用 `placeholder` 表示輸入前的提示。 |
