# Element / Attribute Index

## 用途

用 HTML 標籤與屬性快速找到本篇輸入控件筆記中的對應段落。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<input>` | 建立表單輸入控件，並透過 `type` 決定控件類型。 | `type`、`name`、`value`、`placeholder`、`maxlength` | [input 的基本語法](../../../notes/測試/02-input標籤與基礎輸入控件.md#input-的基本語法) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `type` | `<input>` | 決定輸入控件類型，例如 `text`、`password`、`hidden`。 | [常用屬性](../../../notes/測試/02-input標籤與基礎輸入控件.md#常用屬性) |
| `name` | `<input>` | 指定提交資料時的欄位名稱。 | [常用屬性](../../../notes/測試/02-input標籤與基礎輸入控件.md#常用屬性) |
| `value` | `<input>` | 設定預設值或真正提交的值。 | [常用屬性](../../../notes/測試/02-input標籤與基礎輸入控件.md#常用屬性) |
| `placeholder` | `<input>` | 設定輸入前顯示的提示文字。 | [常用屬性](../../../notes/測試/02-input標籤與基礎輸入控件.md#常用屬性) |
| `maxlength` | `<input>` | 限制可輸入的最大字元數。 | [常用屬性](../../../notes/測試/02-input標籤與基礎輸入控件.md#常用屬性) |
