# Element / Attribute Index

## 用途

用 HTML 標籤與屬性快速找到本篇選項控件筆記中的對應段落。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<input>` | 建立 `radio` 單選框與 `checkbox` 複選框。 | `type`、`name`、`value`、`checked` | [radio 單選框](../../../notes/測試/03-radio與checkbox選項控件.md#radio-單選框) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `type` | `<input>` | 使用 `radio` 或 `checkbox` 決定選項控件類型。 | [radio 單選框](../../../notes/測試/03-radio與checkbox選項控件.md#radio-單選框) |
| `name` | `<input>` | 建立提交欄位名稱；同一組 `radio` 也靠相同 `name` 形成互斥群組。 | [常見錯誤：同一組 radio 使用不同 name](../../../notes/測試/03-radio與checkbox選項控件.md#常見錯誤同一組-radio-使用不同-name) |
| `value` | `<input>` | 決定每個選項真正提交出去的資料值。 | [radio 單選框](../../../notes/測試/03-radio與checkbox選項控件.md#radio-單選框) |
| `checked` | `<input>` | 設定預設選取或預設勾選。 | [radio 單選框](../../../notes/測試/03-radio與checkbox選項控件.md#radio-單選框) |
