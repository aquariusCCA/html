# Element / Attribute Index

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<a>` | 建立超連結，讓目前文件連到網址、檔案、頁面位置或指定協議。 | `href`、`target`、`rel`、`download` | [基本語法](../../../notes/測試/超鏈接標籤.md#基本語法) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `href` | `<a>` | 指定連結要前往的目標，是 `<a>` 最核心的屬性。 | [基本語法](../../../notes/測試/超鏈接標籤.md#基本語法) |
| `target` | `<a>` | 指定連結開啟方式，例如目前頁籤或新頁籤。 | [`target` 的常見值](../../../notes/測試/超鏈接標籤.md#target-的常見值) |
| `rel` | `<a>` | 搭配 `target="_blank"` 時降低新頁面影響原頁面的風險。 | [常用屬性](../../../notes/測試/超鏈接標籤.md#常用屬性) |
| `download` | `<a>` | 提示瀏覽器下載連結資源。 | [檔案連結](../../../notes/測試/超鏈接標籤.md#檔案連結) |
| `id` | 錨點目標元素 | 提供 `href="#id名稱"` 可以跳轉的頁面位置。 | [頁面內錨點連結](../../../notes/測試/超鏈接標籤.md#頁面內錨點連結) |
