# FAQ Index

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| `<a>` 標籤做什麼？ | `<a>` 用來建立超連結，讓目前文件連到網址、檔案、頁面位置或指定協議。 | [一句話理解](../../../notes/測試/超鏈接標籤.md#一句話理解) |
| `href` 屬性做什麼？ | `href` 指定 `<a>` 連結要前往的目標，是 `<a>` 最核心的屬性。 | [基本語法](../../../notes/測試/超鏈接標籤.md#基本語法) |
| 沒有 `href` 的 `<a>` 是完整超連結嗎？ | 不是。沒有 `href` 的 `<a>` 不算是一個完整可前往目標的超連結。 | [基本語法](../../../notes/測試/超鏈接標籤.md#基本語法) |
| `target="_blank"` 什麼時候適合使用？ | 通常用在前往外部網站，或不希望使用者離開目前頁面時。 | [`target` 的常見值](../../../notes/測試/超鏈接標籤.md#target-的常見值) |
| 為什麼 `target="_blank"` 常搭配 `rel="noopener noreferrer"`？ | 這樣可以降低新開頁面取得或影響原頁面控制權的風險。 | [常用屬性](../../../notes/測試/超鏈接標籤.md#常用屬性) |
| 外部連結和內部連結差在哪？ | 外部連結指向其他網站，通常使用完整網址；內部連結指向同一網站中的其他頁面，常使用相對路徑。 | [連結的常見類型](../../../notes/測試/超鏈接標籤.md#連結的常見類型) |
| `href` 可以連到檔案嗎？ | 可以。`href` 可以連到圖片、影片、音訊、PDF 或壓縮檔等資源。 | [檔案連結](../../../notes/測試/超鏈接標籤.md#檔案連結) |
| 如何提示瀏覽器下載連結資源？ | 可以在 `<a>` 上使用 `download` 屬性，但實際下載行為仍可能受瀏覽器、伺服器與跨來源限制影響。 | [檔案連結](../../../notes/測試/超鏈接標籤.md#檔案連結) |
| 如何建立頁面內錨點連結？ | 連結使用 `href="#id名稱"`，目標元素要有對應的 `id`。 | [頁面內錨點連結](../../../notes/測試/超鏈接標籤.md#頁面內錨點連結) |
| `href="#"` 是什麼效果？ | `href="#"` 通常會讓頁面回到頂部，但也可能讓網址後方出現 `#`。 | [回到頁面頂部](../../../notes/測試/超鏈接標籤.md#回到頁面頂部) |
| 空的 `href` 代表什麼？ | 空的 `href` 可能會重新載入目前頁面，實務上不建議當成暫時按鈕。 | [空連結](../../../notes/測試/超鏈接標籤.md#空連結) |
| `tel:` 和 `mailto:` 可以放在 `href` 嗎？ | 可以。`tel:` 可開啟撥號功能，`mailto:` 可開啟郵件軟體。 | [特殊協議連結](../../../notes/測試/超鏈接標籤.md#特殊協議連結) |
| 為什麼不建議使用 `javascript:` 連結？ | `javascript:` 雖然能執行程式碼，但互動操作通常不應使用連結語意處理。 | [特殊協議連結](../../../notes/測試/超鏈接標籤.md#特殊協議連結) |
| 圖片可以放進 `<a>` 裡嗎？ | 可以。`<a>` 可以包住圖片，讓圖片成為可點擊連結。 | [讓圖片或區塊成為連結](../../../notes/測試/超鏈接標籤.md#讓圖片或區塊成為連結) |
| `<a>` 裡可以再放另一個 `<a>` 或 `<button>` 嗎？ | 不建議。巢狀互動元素容易造成使用者操作與無障礙閱讀問題。 | [讓圖片或區塊成為連結](../../../notes/測試/超鏈接標籤.md#讓圖片或區塊成為連結) |
| 什麼時候該用 `<a>`，什麼時候該用 `<button>`？ | 要前往網址、檔案或頁面位置時用 `<a>`；要執行操作時優先使用 `<button>`。 | [常見錯誤](../../../notes/測試/超鏈接標籤.md#常見錯誤) |
| 連結文字應該怎麼寫？ | 連結文字應具體描述目標，避免只寫「點我」或「這裡」。 | [實務使用建議](../../../notes/測試/超鏈接標籤.md#實務使用建議) |
