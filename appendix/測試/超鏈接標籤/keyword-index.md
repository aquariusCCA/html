# Keyword Index

## 用途

用關鍵字快速找到本篇 HTML 教學筆記中的相關知識點。

## 關鍵字索引

| 關鍵字 | 類型 | 對應段落 | 說明 |
| --- | --- | --- | --- |
| `<a>` | 標籤 | [基本語法](../../../notes/測試/超鏈接標籤.md#基本語法) | 建立超連結，讓使用者前往網址、檔案、頁面位置或指定協議。 |
| `href` | 屬性 | [基本語法](../../../notes/測試/超鏈接標籤.md#基本語法) | 指定連結要前往的目標。 |
| `target` | 屬性 | [`target` 的常見值](../../../notes/測試/超鏈接標籤.md#target-的常見值) | 指定連結在目前頁籤或新頁籤開啟。 |
| `rel` | 屬性 | [常用屬性](../../../notes/測試/超鏈接標籤.md#常用屬性) | 搭配新分頁外部連結降低安全風險。 |
| `download` | 屬性 | [檔案連結](../../../notes/測試/超鏈接標籤.md#檔案連結) | 提示瀏覽器下載連結資源。 |
| `id` | 屬性 | [頁面內錨點連結](../../../notes/測試/超鏈接標籤.md#頁面內錨點連結) | 作為錨點連結的目標位置識別。 |
| `_self` | 術語 | [`target` 的常見值](../../../notes/測試/超鏈接標籤.md#target-的常見值) | `target` 的預設值，在目前頁籤開啟連結。 |
| `_blank` | 術語 | [`target` 的常見值](../../../notes/測試/超鏈接標籤.md#target-的常見值) | `target` 的常見值，在新頁籤開啟連結。 |
| `noopener noreferrer` | 術語 | [常用屬性](../../../notes/測試/超鏈接標籤.md#常用屬性) | 新分頁外部連結常搭配的安全屬性值。 |
| `anchor` | 術語 | [一句話理解](../../../notes/測試/超鏈接標籤.md#一句話理解) | `<a>` 的英文來源，表示頁面中的連接點或錨。 |
| `tel:` | 術語 | [特殊協議連結](../../../notes/測試/超鏈接標籤.md#特殊協議連結) | 放在 `href` 中開啟撥號功能。 |
| `mailto:` | 術語 | [特殊協議連結](../../../notes/測試/超鏈接標籤.md#特殊協議連結) | 放在 `href` 中開啟郵件軟體。 |
| `line:` | 術語 | [特殊協議連結](../../../notes/測試/超鏈接標籤.md#特殊協議連結) | 放在 `href` 中嘗試喚起 LINE。 |
| `javascript:` | 術語 | [特殊協議連結](../../../notes/測試/超鏈接標籤.md#特殊協議連結) | 可執行 JavaScript，但筆記提醒實務上通常不建議用於互動操作。 |
| 超連結 | 概念 | [一句話理解](../../../notes/測試/超鏈接標籤.md#一句話理解) | 從目前文件連到另一個網址、檔案、位置或協議。 |
| 連結文字 | 概念 | [實務使用建議](../../../notes/測試/超鏈接標籤.md#實務使用建議) | 使用者看見並點擊的文字，應具體描述目標。 |
| 外部連結 | 概念 | [外部連結](../../../notes/測試/超鏈接標籤.md#外部連結) | 指向其他網站，通常使用完整網址。 |
| 內部連結 | 概念 | [內部連結](../../../notes/測試/超鏈接標籤.md#內部連結) | 指向同一網站中的其他頁面，常使用相對路徑。 |
| 相對路徑 | 概念 | [內部連結](../../../notes/測試/超鏈接標籤.md#內部連結) | 根據目前 HTML 檔案所在位置計算的連結路徑。 |
| 檔案連結 | 概念 | [檔案連結](../../../notes/測試/超鏈接標籤.md#檔案連結) | 使用 `href` 連到圖片、影片、音訊、PDF 或壓縮檔。 |
| 下載連結 | 場景 | [檔案連結](../../../notes/測試/超鏈接標籤.md#檔案連結) | 使用 `download` 提示瀏覽器下載連結資源。 |
| 錨點連結 | 概念 | [頁面內錨點連結](../../../notes/測試/超鏈接標籤.md#頁面內錨點連結) | 使用 `href="#id"` 跳到頁面指定位置。 |
| 回到頂部 | 場景 | [回到頁面頂部](../../../notes/測試/超鏈接標籤.md#回到頁面頂部) | 可使用 `href="#"` 或更明確的 `href="#top"`。 |
| 空連結 | 概念 | [空連結](../../../notes/測試/超鏈接標籤.md#空連結) | 空的 `href` 可能重新載入目前頁面，實務上不建議當暫時按鈕。 |
| 特殊協議連結 | 概念 | [特殊協議連結](../../../notes/測試/超鏈接標籤.md#特殊協議連結) | 使用 `tel:`、`mailto:`、`line:` 等協議喚起裝置或應用程式。 |
| 前往資源 | 概念 | [常見錯誤](../../../notes/測試/超鏈接標籤.md#常見錯誤) | 點擊後前往網址、檔案或頁面位置時，適合使用 `<a>`。 |
| 觸發操作 | 概念 | [常見錯誤](../../../notes/測試/超鏈接標籤.md#常見錯誤) | 點擊後送出表單、打開彈窗或切換狀態時，不應用連結語意表示。 |
| 巢狀互動元素 | 概念 | [讓圖片或區塊成為連結](../../../notes/測試/超鏈接標籤.md#讓圖片或區塊成為連結) | 在連結中再放互動元素容易造成操作與無障礙問題。 |
