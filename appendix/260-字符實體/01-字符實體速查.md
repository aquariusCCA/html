# 字符實體速查

> 查表資料整理自 `notes/260-字符實體/01-字符實體基本概念.md`。

## 字符實體寫法形式

| 寫法類型 | 形式 | 範例 | 顯示結果 | 說明 |
| --- | --- | --- | --- | --- |
| 命名字符實體 | `& + 實體名稱 + ;` | `&copy;` | `©` | 常見字符實體由 `&`、實體名稱與分號組成。 |
| 十進位數字字符引用 | `&#數字;` | `&#169;` | `©` | HTML 支援的數字字符引用形式之一。 |
| 十六進位數字字符引用 | `&#x十六進位數字;` | `&#xA9;` | `©` | HTML 支援的數字字符引用形式之一。 |

## 常用字符實體

| 想顯示的字符 | 字符實體 | 常見用途 | 使用提醒 |
| --- | --- | --- | --- |
| 空格 | `&nbsp;` | 顯示不會被合併的空格 | `nbsp` 是 non-breaking space，不會像一般空格那樣被合併。 |
| `<` | `&lt;` | 在文字中顯示小於號或 HTML 開始標籤符號 | 顯示 HTML 標籤文字時常和 `&gt;` 搭配使用。 |
| `>` | `&gt;` | 在文字中顯示大於號或 HTML 結束標籤符號 | 顯示 HTML 標籤文字時常和 `&lt;` 搭配使用。 |
| `&` | `&amp;` | 在文字中顯示和號本身 | 文字中的 `&` 常需要寫成字符實體，避免和實體寫法混淆。 |
| `©` | `&copy;` | 顯示版權符號 | 命名字符實體通常要以分號 `;` 結尾。 |

## 適用情境速查

| 情境 | 建議寫法或處理方向 | 原因 |
| --- | --- | --- |
| 字符在 HTML 中有特殊意義，例如 `<`、`>`、`&` | 使用對應字符實體，例如 `&lt;`、`&gt;`、`&amp;` | 避免瀏覽器把文字誤解析成 HTML 語法。 |
| 需要在頁面上顯示 HTML 標籤文字 | 把 `<` 和 `>` 寫成 `&lt;` 和 `&gt;` | 瀏覽器看到字符實體時，不會把它當成真正的 HTML 標籤執行。 |
| 需要顯示連續空格效果 | 可使用 `&nbsp;` | `&nbsp;` 不會像一般空格那樣被合併。 |
| 只是想控制段落間距、元素位置或文字對齊 | 使用 CSS 的 `margin`、`padding`、`gap` 等方式 | 字符實體不是排版系統，不應拿大量 `&nbsp;` 取代 CSS 排版。 |
| 空白顯示結果和預期不同 | 先確認元素與 CSS 的空白處理規則 | `<pre>`、`<textarea>` 與 CSS `white-space` 都可能改變空白顯示方式。 |

## 常見錯誤與修正

| 問題 | 錯誤寫法 | 建議寫法 | 說明 |
| --- | --- | --- | --- |
| 漏掉分號 | `&copy` | `&copy;` | 命名字符實體通常要以分號 `;` 結尾；漏掉分號可能導致瀏覽器解析結果不穩定，也會降低程式碼可讀性。 |
| 把 HTML 標籤直接寫成文字 | `<h1> 是標題標籤` | `&lt;h1&gt; 是標題標籤` | 直接寫 `<h1>` 時，瀏覽器會把它當成真正的標題開始標籤。 |
| 用大量 `&nbsp;` 做排版 | `姓名：&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;小明` | 使用 CSS 的 `margin`、`padding`、`gap` 等排版方式 | `&nbsp;` 適合處理文字中的特殊空格，不適合作為主要排版工具。 |
| 誤以為所有空白都會被保留 | 在一般 HTML 文字排版中直接輸入多個連續空格 | 依需求使用 `&nbsp;`、`<pre>`、`<textarea>` 或 CSS `white-space` | 一般 HTML 文字排版中，連續空格與換行通常會被合併顯示為一個空格，但不同元素與 CSS 規則可能改變結果。 |

## 標籤文字轉義速查

| 想顯示的文字 | HTML 中應寫成 | 顯示結果 | 重點 |
| --- | --- | --- | --- |
| `<p>Hello</p>` | `&lt;p&gt;Hello&lt;/p&gt;` | `<p>Hello</p>` | `&lt;` 顯示成 `<`，`&gt;` 顯示成 `>`，瀏覽器不會把它當成真正的 `<p>` 標籤執行。 |
| `<h1> 是標題標籤` | `&lt;h1&gt; 是標題標籤` | `<h1> 是標題標籤` | 教學文字或說明文件中需要顯示標籤名稱時，應避免直接寫成真正標籤。 |

初學時不需要背大量字符實體，先熟悉 `&nbsp;`、`&lt;`、`&gt;`、`&amp;`，其他需要時再查即可。
