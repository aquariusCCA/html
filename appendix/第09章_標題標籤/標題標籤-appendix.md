- 透過來源索引回到正式 notes 的對應小節深入閱讀。

## 9. 常用片段

### 9.1 標題元素基本語法

| 項目 | 說明 |
|---|---|
| 使用情境 | 建立單一標題元素。 |
| 使用前提 | 先判斷內容在頁面中的層級。 |
| 來源 | §3、§4 |

```html
<h1>頁面主標題</h1>
```

### 注意事項

- `h1` 到 `h6` 的數字代表層級，不是單純的字體大小。
- 標題文字應放在開始標籤與結束標籤之間。

### 9.2 文章頁標題結構

| 項目 | 說明 |
|---|---|
| 使用情境 | 文章或文件頁需要主標題、主要段落與子段落標題。 |
| 使用前提 | `h2` 是 `h1` 底下的主要段落，`h3` 是 `h2` 底下的子段落。 |
| 來源 | §5 |

```html
<article>
  <h1>如何開始學習 HTML</h1>

  <section>
    <h2>先理解 HTML 的用途</h2>
    <p>HTML 用來描述網頁內容與結構。</p>
  </section>

  <section>
    <h2>再學習常見標籤</h2>
    <h3>標題標籤</h3>
    <p>標題標籤用來建立內容層級。</p>
  </section>
</article>
```

### 注意事項

- `article` 與 `section` 在此只作為範例結構容器，本附錄不擴寫成元素表。
- 標題層級應反映內容關係。

### 9.3 用 CSS 控制標題外觀

| 項目 | 說明 |
|---|---|
| 使用情境 | 想調整標題大小、粗細或間距，但不改變標題語意。 |
| 使用前提 | HTML 標題層級已依內容結構選定。 |
| 來源 | §3.2、§5、§7 |

```html
<h2 class="section-title">課程特色</h2>
```

```css
.section-title {
  font-size: 32px;
  font-weight: 700;
  margin-block: 24px 12px;
}
```

### 注意事項

- `h2` 負責表達章節標題語意。
- CSS 負責控制字體大小、粗細與間距。

### 9.4 修正層級跳躍

| 項目 | 說明 |
|---|---|
| 使用情境 | 原本標題從 `h1` 直接跳到過低層級。 |
| 使用前提 | 需要先判斷後續標題是否為 `h1` 底下的主要段落。 |
| 來源 | §7 |

```html
<h1>商品介紹</h1>
<h2>特色</h2>
<p>這裡介紹商品特色。</p>
```

### 注意事項

- 原筆記的錯誤範例是 `h1` 後直接接 `h4`。
- 修正方向是用符合內容關係的 `h2`。

## 10. 錯誤速查表

| 錯誤現象 | 可能原因 | 修正方向 | 對應知識點 | 來源 |
|---|---|---|---|---|
| 用 `h1` 只是為了讓字變大。 | 混淆標題層級與視覺大小。 | 使用符合內容層級的標題，再用 CSS 調整外觀。 | 語意、CSS、預設樣式 | §3.1、§3.2、§7 |
| `h1` 後直接接 `h4`。 | 標題層級跳躍，內容結構不清楚。 | 依內容關係安排 `h1`、`h2`、`h3`。 | 標題層級 | §4、§7 |
| 整頁沒有清楚主標題。 | 頁面主要內容不容易被判斷。 | 為主要內容提供清楚的 `h1` 或等價主標題。 | `h1`、頁面主題 | §7、§11 |
| 把普通段落文字包成標題。 | 將一般內容誤標成章節標題。 | 一般文字使用 `p`，只有章節名稱才使用標題標籤。 | 標題標籤、段落文字 | §7 |
| 為了取消粗體而不用標題標籤。 | 混淆語意與樣式責任。 | 保留正確標題元素，用 CSS 調整 `font-weight`。 | 語意、CSS | §3.2、§7 |

## 11. 相似概念對照表

### 11.1 標題語意 vs CSS 外觀

| 比較項目 | 標題語意 | CSS 外觀 |
|---|---|---|
| 定義 | 用 `h1` 到 `h6` 表達內容層級。 | 控制字體大小、粗細、顏色與間距。 |
| 使用時機 | 需要標示頁面、段落或區塊主題時。 | 需要調整畫面呈現時。 |
| 常見寫法 | `<h2>課程特色</h2>` | `.section-title { font-size: 32px; }` |
| 注意事項 | 應依內容關係選擇層級。 | 不應取代 HTML 的語意結構。 |
| 常見錯誤 | 為了字大而使用 `h1`。 | 為了取消粗體而不用標題標籤。 |

### 選用建議

- 先用 HTML 標題元素表達內容層級。
- 再用 CSS 調整標題的視覺呈現。

### 11.2 `h1` vs `h2` vs `h3`

| 比較項目 | `h1` | `h2` | `h3` |
|---|---|---|---|
| 定義 | 最高層級標題。 | `h1` 底下的主要段落標題。 | `h2` 底下的子段落標題。 |
| 使用時機 | 表示頁面、文章或主要內容主題。 | 表示主要段落、主要區塊。 | 表示主要段落中的更細主題。 |
| 常見寫法 | `<h1>HTML 入門教學</h1>` | `<h2>HTML 是什麼</h2>` | `<h3>標題標籤</h3>` |
| 注意事項 | 不要當成單純放大文字的工具。 | 應位於對應 `h1` 的內容脈絡下。 | 應位於對應 `h2` 的內容脈絡下。 |
| 常見錯誤 | 整頁缺少清楚主標題。 | 被跳過而直接使用更低層級標題。 | 與上層 `h2` 關係不清楚。 |

### 選用建議

- 先判斷內容是否為整頁主題、主要段落或子段落。
- 需要更細層級時，再依內容關係往 `h4` 到 `h6` 延伸。

### 11.3 標題標籤 vs 段落文字

| 比較項目 | 標題標籤 | 段落文字 |
|---|---|---|
| 定義 | 表示章節、頁面或區塊主題。 | 表示一般內容文字。 |
| 使用時機 | 需要建立內容層級時。 | 需要撰寫一般說明、敘述或內文時。 |
| 常見寫法 | `<h2>產品更新</h2>` | `<p>本段介紹產品的新功能。</p>` |
| 注意事項 | 不重要的內容不應包成標題。 | 不應用段落取代真正的章節標題。 |
| 常見錯誤 | 把普通段落文字包成標題。 | 原筆記未列出段落文字的其他錯誤。 |

### 選用建議

- 章節名稱使用標題標籤。
- 一般文字內容使用 `p`。
