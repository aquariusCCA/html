# FAQ Index

## 用途

用具體問題快速找到本篇筆記中能回答的段落。

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| `<br>` 是做什麼的？ | `<br>` 用來在文字內容中插入強制換行。 | [一句話理解](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#一句話理解) |
| `br` 是什麼意思？ | `br` 是 `break` 的縮寫，意思是打斷、換行。 | [標籤語意](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#標籤語意) |
| `<br>` 需要結束標籤嗎？ | 不需要，`<br>` 是 void element，不包住其他內容。 | [基本語法](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#基本語法) |
| `<br />` 可以用嗎？ | 在 HTML 中常見，但結尾斜線不是必要語法；初學可以先寫 `<br>`。 | [基本語法](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#基本語法) |
| 自動換行和強制換行有什麼差異？ | 自動換行是瀏覽器依容器寬度排版；強制換行是指定某個位置一定要換行。 | [自動換行與強制換行](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#自動換行與強制換行) |
| 什麼時候適合使用 `<br>`？ | 當換行本身有內容意義，例如地址、詩句、歌詞或簡短聯絡資訊。 | [適合使用 `<br>` 的情境](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#適合使用-br-的情境) |
| `<br>` 和段落分段有什麼差異？ | `<br>` 只是同一段內容中的斷行；段落分段代表不同段落的內容結構。 | [`<br>` 與段落的差異](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#br-與段落的差異) |
| 可以用 `<br><br>` 取代段落嗎？ | 不建議；如果內容是兩個段落，應該用段落結構表示。 | [常見錯誤](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#常見錯誤) |
| 可以用很多 `<br>` 控制版面空白嗎？ | 不建議；HTML 保留內容語意，間距應交給 CSS 處理。 | [常見錯誤](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#常見錯誤) |
| 原始碼換行時需要加入 `<br>` 嗎？ | 不需要；原始碼換行主要給開發者閱讀，`<br>` 會影響瀏覽器顯示結果。 | [常見錯誤](../../../notes/第14章_換行標籤/01-br-換行標籤與強制換行.md#常見錯誤) |
