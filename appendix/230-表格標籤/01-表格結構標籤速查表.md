# 表格結構標籤速查表

> 查表用。概念與用法說明見 notes/230-表格標籤/01-表格基本結構.md。

| 標籤 | 用途 | 備註 |
| --- | --- | --- |
| `<table>` | 整張表格的容器 | 用來呈現結構化資料，不用來做版面排版 |
| `<tr>` | 表格中的一列 | 應放在 `<table>` 結構或 `<thead>`/`<tbody>`/`<tfoot>` 內 |
| `<th>` | 表頭儲存格 | 通常用來表示欄位名稱或列標題 |
| `<td>` | 資料儲存格 | 放實際資料內容（table data） |
| `<caption>` | 表格標題 | 說明整張表格在呈現什麼 |
| `<thead>` | 表格頭部 | 通常放欄位標題 |
| `<tbody>` | 表格主體 | 放主要資料列 |
| `<tfoot>` | 表格腳注 | 常放總計、平均或補充結果 |
