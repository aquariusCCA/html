# 表格舊式屬性對照表

> 查表用。概念與用法說明見 notes/230-表格標籤/03-表格舊式屬性補充.md。

| 舊式屬性 | 用途 | 現代 CSS 做法 | 備註 |
| --- | --- | --- | --- |
| `align` | 設定表格相對周圍元素的對齊方式（`left`/`center`/`right`） | CSS 版面與 `margin` | |
| `border` | 設定表格邊框寬度（`border="0"` 或不寫通常表示無邊框） | CSS `border`、`border-collapse` | 適合教學或快速檢查結構，不適合精細控制每條邊框 |
| `cellpadding` | 設定儲存格內容與儲存格邊界之間的距離 | CSS `padding` | |
| `cellspacing` | 設定儲存格與儲存格之間的距離 | CSS `border-spacing` 或 `border-collapse` | |
| `width` | 設定表格寬度 | CSS `width`、`max-width`、響應式設定 | 設定某格寬度通常會影響整欄 |
