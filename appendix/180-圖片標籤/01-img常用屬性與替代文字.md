# img 常用屬性與替代文字

> 查表內容整理自 `notes/180-圖片標籤/01-img標籤與圖片替代文字.md`。

## 常用屬性

| 屬性 | 用途 | 常見寫法 | 使用提醒 |
| --- | --- | --- | --- |
| `src` | 指定圖片來源 | `src="images/logo.png"` | 路徑、資料夾名稱、檔名或副檔名寫錯時，圖片可能顯示不出來。 |
| `alt` | 提供替代文字 | `alt="網站標誌"` | 不是單純圖片說明，也不是滑鼠提示文字；圖片有資訊價值時，應寫出可替代圖片資訊的內容。 |
| `title` | 滑鼠停留時顯示提示文字 | `title="返回首頁"` | 不能取代 `alt`；通常作為額外提示或補充資訊。 |
| `width` | 設定顯示寬度 | `width="200"` | 只設定寬度時，高度通常會依原始比例自動縮放。 |
| `height` | 設定顯示高度 | `height="120"` | 同時設定 `width` 和 `height` 時，若比例和原圖不同，圖片可能被拉伸或壓扁。 |

## alt 判斷

| 圖片情境 | `alt` 建議 | 判斷依據 |
| --- | --- | --- |
| 商品、人物、按鈕、連結或內容的一部分 | 寫有意義的替代文字 | 圖片消失後，使用者會少掉重要資訊。 |
| 圖片真正傳達內容資訊 | 寫能替代圖片資訊的文字 | 例如 `alt="前端課程講師團隊合照"`。 |
| 背景、分隔線、裝飾圖案 | 使用空的 `alt=""` | 圖片只是裝飾，不承載內容資訊。 |
| 只寫「圖片」 | 不建議 | `alt="圖片"` 沒有提供有用資訊。 |
| 用 `title` 取代 `alt` | 不建議 | `title` 不是圖片無法顯示時的主要替代內容。 |

## 尺寸設定提醒

| 設定方式 | 結果 | 建議 |
| --- | --- | --- |
| 只設定 `width` | 高度通常會依原始比例自動縮放 | 不確定圖片比例時，較不容易出錯。 |
| 只設定 `height` | 寬度通常會依原始比例自動縮放 | 不確定圖片比例時，較不容易出錯。 |
| 同時設定 `width` 和 `height`，且比例符合原圖 | 可控制圖片顯示尺寸 | 適合已確認圖片比例的情境。 |
| 同時設定 `width` 和 `height`，但比例和原圖不同 | 圖片可能變形 | 不建議；可能造成拉伸或壓扁。 |

## 屬性書寫規則

| 規則 | 說明 |
| --- | --- |
| 標籤名和屬性之間要用空格分開 | 例如 `<img src="photo.jpg">`。 |
| 屬性和屬性之間要用空格分開 | 多個屬性都寫在標籤名稱後面。 |
| 屬性通常使用 `屬性="屬性值"` 格式 | 例如 `alt="人物照片"`。 |
| 屬性順序不影響 HTML 語法 | 建議維持一致，讓程式碼容易閱讀。 |
