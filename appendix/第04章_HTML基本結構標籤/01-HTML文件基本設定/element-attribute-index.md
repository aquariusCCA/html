# Element / Attribute Index

## 用途

快速查找本篇筆記明確教學或反覆使用的 HTML 標籤與屬性。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<html>` | 包住整份 HTML 文件，本篇用來承載文件主要語言設定。 | `lang` | [基本結構](../../../notes/第04章_HTML基本結構標籤/01-HTML文件基本設定.md#基本結構) |
| `<head>` | 放置不直接顯示在頁面上的文件設定，本篇用來放 `<meta charset="UTF-8">`。 | 本篇未教學特定屬性 | [基本結構](../../../notes/第04章_HTML基本結構標籤/01-HTML文件基本設定.md#基本結構) |
| `<meta>` | 在 `<head>` 中提供文件資訊，本篇用來指定字元編碼。 | `charset` | [忘記設定字元編碼](../../../notes/第04章_HTML基本結構標籤/01-HTML文件基本設定.md#忘記設定字元編碼) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `lang` | `<html>` | 標示文件主要使用的自然語言。 | [實務使用原則](../../../notes/第04章_HTML基本結構標籤/01-HTML文件基本設定.md#實務使用原則) |
| `charset` | `<meta>` | 指定文件使用的字元編碼，現代網頁通常使用 `UTF-8`。 | [忘記設定字元編碼](../../../notes/第04章_HTML基本結構標籤/01-HTML文件基本設定.md#忘記設定字元編碼) |

## 補充

`<!DOCTYPE html>` 是文件類型宣告，不是 HTML 標籤，因此不列入 HTML 標籤表。
