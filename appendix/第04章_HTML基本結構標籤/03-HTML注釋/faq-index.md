# FAQ Index

## 用途

用具體問題快速找到本篇筆記中能回答的段落。

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| HTML 注釋怎麼寫？ | HTML 注釋以 `<!--` 開頭，以 `-->` 結束。 | [基本語法](../../../notes/第04章_HTML基本結構標籤/03-HTML注釋.md#基本語法) |
| HTML 注釋會顯示在頁面上嗎？ | 不會。瀏覽器會忽略注釋，不會把它顯示成頁面內容。 | [一句話理解](../../../notes/第04章_HTML基本結構標籤/03-HTML注釋.md#一句話理解) |
| 什麼時候適合使用 HTML 注釋？ | 適合用來標示長結構、說明原因、暫時保留程式碼或留下後續調整提醒。 | [什麼時候使用注釋](../../../notes/第04章_HTML基本結構標籤/03-HTML注釋.md#什麼時候使用注釋) |
| 注釋結尾沒有寫完整會怎樣？ | 後面的 HTML 可能會被瀏覽器當成注釋內容。 | [注釋結尾沒有寫完整](../../../notes/第04章_HTML基本結構標籤/03-HTML注釋.md#注釋結尾沒有寫完整) |
| 要顯示給使用者看的文字可以放進注釋嗎？ | 不可以。要顯示的內容應該使用合適的 HTML 標籤。 | [把要顯示給使用者看的內容放進注釋](../../../notes/第04章_HTML基本結構標籤/03-HTML注釋.md#把要顯示給使用者看的內容放進注釋) |
| 怎樣的注釋比較有價值？ | 有價值的注釋會幫助理解程式碼，例如說明原因，而不是重複描述每一行。 | [實務注意事項](../../../notes/第04章_HTML基本結構標籤/03-HTML注釋.md#實務注意事項) |
| 可以把密碼、金鑰或私人資料放進 HTML 注釋嗎？ | 不應該。注釋不顯示在畫面中，但仍可能出現在前端原始碼裡。 | [實務注意事項](../../../notes/第04章_HTML基本結構標籤/03-HTML注釋.md#實務注意事項) |
