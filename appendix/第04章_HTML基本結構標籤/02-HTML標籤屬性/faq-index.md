# FAQ Index

## 用途

用常見問題快速回查本篇筆記中關於 HTML 屬性的答案。

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| HTML 屬性是什麼？ | HTML 屬性是寫在標籤上的補充資訊，用來描述標籤如何被理解或運作。 | [一句話理解](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#一句話理解) |
| 屬性通常長什麼樣子？ | 多數屬性由屬性名與屬性值組成，例如 `title="提示文字"`。 | [屬性長什麼樣子](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#屬性長什麼樣子) |
| 屬性要寫在哪裡？ | 屬性必須寫在開始標籤或單標籤內。 | [屬性的位置](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#屬性的位置) |
| 一個標籤可以放多個屬性嗎？ | 可以，同一個標籤上可以同時存在多個屬性。 | [一個標籤可以有多個屬性](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#一個標籤可以有多個屬性) |
| 屬性之間要怎麼分隔？ | 屬性之間用空格分隔，標籤名稱與第一個屬性之間也必須有空格。 | [一個標籤可以有多個屬性](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#一個標籤可以有多個屬性) |
| 什麼是布林屬性？ | 布林屬性只要屬性名出現，就代表該狀態成立。 | [布林屬性](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#布林屬性) |
| `disabled` 一定要寫成 `disabled="disabled"` 嗎？ | 不一定；`disabled` 本身出現就能表示停用狀態，`disabled="disabled"` 也是常見寫法。 | [布林屬性](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#布林屬性) |
| 屬性值建議加引號嗎？ | 建議使用雙引號包住屬性值，讓寫法穩定且容易閱讀。 | [屬性書寫建議](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#屬性書寫建議) |
| 標籤名和第一個屬性中間可以不加空格嗎？ | 不可以；瀏覽器需要透過空格分辨標籤名稱與屬性。 | [標籤名和屬性之間沒有空格](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#標籤名和屬性之間沒有空格) |
| 同一個標籤可以重複寫同名屬性嗎？ | 不建議也不要這樣寫，因為會讓語意混亂並可能影響屬性值生效。 | [重複寫同名屬性](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#重複寫同名屬性) |
| 屬性可以寫在結束標籤上嗎？ | 不可以；結束標籤只負責結束元素，不放屬性。 | [把屬性寫到結束標籤](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#把屬性寫到結束標籤) |
| 可以把任何屬性套用到任何標籤上嗎？ | 不要這樣做；不同標籤支援不同屬性，實務上應以標準與瀏覽器支援為準。 | [實務判斷](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#實務判斷) |
