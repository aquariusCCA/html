# Element / Attribute Index

## 用途

查找本篇筆記明確提到的 HTML 屬性，以及它們常見搭配的標籤與用途。

## HTML 標籤

本篇以「屬性」概念為核心，`<p>`、`<a>`、`<img>`、`<input>`、`<html>` 主要作為屬性範例載體；未將它們升格為 HTML 標籤主條目。

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `alt` | `<img>` | 提供圖片替代文字。 | [實務判斷](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#實務判斷) |
| `disabled` | `<input>` | 停用輸入欄位。 | [布林屬性](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#布林屬性) |
| `href` | `<a>` | 指定連結目標。 | [實務判斷](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#實務判斷) |
| `lang` | `<html>` | 標示文件語言。 | [實務判斷](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#實務判斷) |
| `name` | `<input>` | 指定欄位名稱。 | [一個標籤可以有多個屬性](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#一個標籤可以有多個屬性) |

| `src` | `<img>` | 指定圖片來源。 | [實務判斷](../../../notes/第04章_HTML基本結構標籤/02-HTML標籤屬性.md#實務判斷) |
