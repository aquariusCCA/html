# Element / Attribute Index

## 用途

查找本篇筆記中用來說明塊級元素、行內元素與排版例外的 HTML 標籤或屬性。

## HTML 標籤

本篇主要把標籤作為「塊級 / 行內」分類與例外判斷的範例，沒有展開每個標籤的完整語法教學。

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<div>` | 作為常見塊級容器範例，說明元素預設換行與佔滿可用寬度的行為。 | 本篇未重點教學。 | [塊級元素是什麼](../../../notes/第07章_HTML塊級元素與行內元素/01-塊級元素與行內元素比較.md#塊級元素是什麼) |
| `<p>` | 作為段落型塊級元素範例，也用來說明塊級不等於可以包含任何內容。 | 本篇未重點教學。 | [常見誤解](../../../notes/第07章_HTML塊級元素與行內元素/01-塊級元素與行內元素比較.md#常見誤解) |
| `<h1>` 到 `<h6>` | 作為標題型塊級元素範例，預設會各自形成區塊。 | 本篇未重點教學。 | [塊級元素是什麼](../../../notes/第07章_HTML塊級元素與行內元素/01-塊級元素與行內元素比較.md#塊級元素是什麼) |
| `<ul>`、`<ol>`、`<li>` | 作為列表類塊級元素範例。 | 本篇未重點教學。 | [塊級元素是什麼](../../../notes/第07章_HTML塊級元素與行內元素/01-塊級元素與行內元素比較.md#塊級元素是什麼) |
| `<span>` | 作為常見行內元素範例，用於文字流中的一小段內容。 | 本篇未重點教學。 | [行內元素是什麼](../../../notes/第07章_HTML塊級元素與行內元素/01-塊級元素與行內元素比較.md#行內元素是什麼) |
| `<a>` | 作為常見行內元素範例，表示連結類文字內容。 | 本篇未重點教學。 | [行內元素是什麼](../../../notes/第07章_HTML塊級元素與行內元素/01-塊級元素與行內元素比較.md#行內元素是什麼) |
| `<em>`、`<strong>` | 作為行內強調語意元素範例。 | 本篇未重點教學。 | [行內元素是什麼](../../../notes/第07章_HTML塊級元素與行內元素/01-塊級元素與行內元素比較.md#行內元素是什麼) |
| `<img>`、`<input>` | 作為置換元素與 void elements 的例外範例，說明行內脈絡不代表完全不能有可見寬高或子內容限制。 | `type` 只在 `<input>` 範例中出現，未作為本篇重點教學。 | [重要例外](../../../notes/第07章_HTML塊級元素與行內元素/01-塊級元素與行內元素比較.md#重要例外) |

## HTML 屬性

資料不足：本篇沒有以 HTML 屬性為主題。`type` 與 `style` 只出現在範例中，`width`、`height`、`margin`、`padding`、`line-height`、`display` 是本篇用來比較排版行為的 CSS 相關術語，因此不列入 HTML 屬性主索引。
