# FAQ Index

## 用途

用常見問題快速定位本篇筆記中能回答的 HTML 巢狀規則段落。

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| HTML 巢狀規則只看塊級或行內就夠嗎？ | 不夠。還要看外層元素允許哪些子內容，也就是內容模型。 | [一句話理解](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#一句話理解) |
| 父元素和子元素是什麼？ | 外層包住另一個元素時，外層是父元素，內層是子元素。 | [先理解父子關係](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#先理解父子關係) |
| `<div>` 可以放任何標籤嗎？ | `<div>` 是通用容器，但這不代表所有塊級元素都能放任何標籤；仍要看元素自己的內容模型。 | [先理解父子關係](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#先理解父子關係) |
| 為什麼 `<input>` 不能包 `<span>`？ | `<input>` 是 void element，本身不能再包含子內容。 | [行內元素的巢狀不是一律相同](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#行內元素的巢狀不是一律相同) |
| 需要提示文字時，`<input>` 外面可以怎麼寫？ | 可以把提示文字放在 `<label>` 或其他合適元素中，而不是讓 `<input>` 包住文字。 | [行內元素的巢狀不是一律相同](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#行內元素的巢狀不是一律相同) |
| `<h1>` 裡可以放 `<h2>` 嗎？ | 不應該。標題元素應平行出現，用層級表達章節關係。 | [特殊限制一：h1 到 h6 不能相互嵌套](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#特殊限制一h1-到-h6-不能相互嵌套) |
| `<p>` 裡可以放 `<div>` 嗎？ | 不應該。`<p>` 表示段落，只適合放文字、連結、強調等 phrasing content。 | [特殊限制二：p 標籤不能包含塊級內容](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#特殊限制二p-標籤不能包含塊級內容) |
| `<a>` 可以包一整塊內容嗎？ | 可以，但要放在所在位置允許 flow content 的情況下，例如卡片式內容。 | [特殊限制三：a 可以包內容，但不能包另一個 a](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#特殊限制三a-可以包內容但不能包另一個-a) |
| `<a>` 裡可以再放 `<a>` 或 `<button>` 嗎？ | 不應該。`<a>` 不能包含另一個 `<a>`，也不應該包含其他互動內容。 | [特殊限制三：a 可以包內容，但不能包另一個 a](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#特殊限制三a-可以包內容但不能包另一個-a) |
| 為什麼瀏覽器顯示正常還要檢查 DOM？ | 瀏覽器可能已經自動關閉或重組不合法標籤，最後 DOM 不一定等於原始碼。 | [問題情境](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#問題情境) |
| 卡片裡有多個可點擊目標時怎麼辦？ | 通常應把連結拆開，而不是讓一個大連結包住其他連結。 | [特殊限制三：a 可以包內容，但不能包另一個 a](../../../notes/第07章_HTML塊級元素與行內元素/02-HTML元素巢狀規則與特殊限制.md#特殊限制三a-可以包內容但不能包另一個-a) |
