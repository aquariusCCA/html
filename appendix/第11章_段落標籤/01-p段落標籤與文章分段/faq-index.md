# FAQ Index

## 用途

用常見問題快速定位本篇筆記中能回答的段落。

## 常見問題索引

| 問題 | 簡短回答 | 對應段落 |
| --- | --- | --- |
| `<p>` 標籤用來做什麼？ | `<p>` 用來表示文章正文、說明文字或一段完整敘述內容中的段落。 | [一句話理解](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#一句話理解) |
| `p` 是什麼意思？ | `p` 是 paragraph 的縮寫，語意是「段落」。 | [標籤語意](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#標籤語意) |
| 什麼內容適合放在 `<p>` 裡？ | 當一段文字在內容上形成完整意思時，就適合用 `<p>` 包起來。 | [標籤語意](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#標籤語意) |
| 多個段落應該怎麼寫？ | 多個段落應使用多組 `<p>`，讓每段文字各自成為獨立段落。 | [標籤語意](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#標籤語意) |
| `<p>` 只是用來增加空白嗎？ | 不是。`<p>` 的重點是段落語意，段落間空隙只是瀏覽器預設外觀。 | [瀏覽器預設樣式](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#瀏覽器預設樣式) |
| 瀏覽器會怎麼顯示 `<p>`？ | 瀏覽器通常會讓段落獨佔區塊、上下有外距，文字也會依容器寬度自動換行。 | [瀏覽器預設樣式](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#瀏覽器預設樣式) |
| 段落和換行差在哪裡？ | 段落代表內容上的分段；換行只代表文字顯示時換到下一行。 | [段落與換行的差異](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#段落與換行的差異) |
| 可以用 `<br>` 取代段落嗎？ | 一般文章分段不應用 `<br>` 取代，應使用多個 `<p>`。 | [常見錯誤](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#常見錯誤) |
| 可以把清單放進 `<p>` 嗎？ | 不應該。`<p>` 適合文字段落，清單、表格與區塊標題應使用對應標籤。 | [常見錯誤](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#常見錯誤) |
| 想調整段落間距要多放空 `<p>` 嗎？ | 不要用空 `<p>` 控制間距，外觀與排版應交給 CSS。 | [常見錯誤](../../../notes/第11章_段落標籤/01-p段落標籤與文章分段.md#常見錯誤) |
