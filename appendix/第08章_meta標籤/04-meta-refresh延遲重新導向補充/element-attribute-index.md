# Element / Attribute Index

## 用途

快速查找本篇筆記明確教學的 HTML 標籤與屬性。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<meta>` | 在 HTML 文件中宣告刷新或延遲重新導向設定。 | `http-equiv`、`content` | [基本語法](../../../notes/第08章_meta標籤/04-meta-refresh延遲重新導向補充.md#基本語法) |
| `<head>` | 放置 `meta refresh` 這類文件設定。 | 無 | [放置位置](../../../notes/第08章_meta標籤/04-meta-refresh延遲重新導向補充.md#放置位置) |

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
