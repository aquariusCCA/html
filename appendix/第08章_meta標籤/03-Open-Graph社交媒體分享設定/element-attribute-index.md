# Element / Attribute Index

## 用途

查找本篇筆記明確教學的 HTML 標籤與屬性。

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<meta>` | 在 `<head>` 中提供頁面 metadata，讓社群平台產生分享預覽。 | `property`、`content`、`name` | [基本觀念](../../../notes/第08章_meta標籤/03-Open-Graph社交媒體分享設定.md#基本觀念) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `property` | `<meta>` | 指定 Open Graph 的 `og:...` 資訊名稱。 | [基本觀念](../../../notes/第08章_meta標籤/03-Open-Graph社交媒體分享設定.md#基本觀念) |
| `content` | `<meta>` | 填入分享標題、描述、圖片網址、正式 URL 或內容類型。 | [基本寫法](../../../notes/第08章_meta標籤/03-Open-Graph社交媒體分享設定.md#基本寫法) |
| `name` | `<meta>` | 常見於一般頁面描述；Open Graph 標籤通常不使用它來寫 `og:...`。 | [Open Graph 與搜尋摘要的差異](../../../notes/第08章_meta標籤/03-Open-Graph社交媒體分享設定.md#Open-Graph-與搜尋摘要的差異) |
