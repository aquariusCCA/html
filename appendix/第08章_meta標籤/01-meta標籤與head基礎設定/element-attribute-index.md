# Element / Attribute Index

## HTML 標籤

| 標籤 | 主要用途 | 常見屬性 | 對應段落 |
| --- | --- | --- | --- |
| `<meta>` | 描述文件資訊或提供瀏覽器設定。 | `charset`、`name`、`content`、`http-equiv` | [`<meta>` 放在哪裡](../../../notes/第08章_meta標籤/01-meta標籤與head基礎設定.md#meta-放在哪裡) |
| `<head>` | 放置文件設定與描述資訊。 | 無本篇重點屬性 | [`<meta>` 放在哪裡](../../../notes/第08章_meta標籤/01-meta標籤與head基礎設定.md#meta-放在哪裡) |

## HTML 屬性

| 屬性 | 常見使用於 | 主要用途 | 對應段落 |
| --- | --- | --- | --- |
| `charset` | `<meta>` | 指定 HTML 文件使用的字元編碼。 | [字元編碼：`charset`](../../../notes/第08章_meta標籤/01-meta標籤與head基礎設定.md#字元編碼charset) |
| `name` | `<meta>` | 指定 metadata 名稱，例如 `viewport`、`author`、`generator`。 | [文件資訊 metadata](../../../notes/第08章_meta標籤/01-meta標籤與head基礎設定.md#文件資訊-metadata) |
| `content` | `<meta>` | 提供 metadata 或瀏覽器設定的內容值。 | [行動裝置視窗：`viewport`](../../../notes/第08章_meta標籤/01-meta標籤與head基礎設定.md#行動裝置視窗viewport) |
| `http-equiv` | `<meta>` | 設定等同 HTTP 標頭用途的資訊，本篇用於舊版 IE 相容性。 | [舊版 IE 相容性：`X-UA-Compatible`](../../../notes/第08章_meta標籤/01-meta標籤與head基礎設定.md#舊版-ie-相容性x-ua-compatible) |
