## 6. 常用片段

### 6.1 最小 `<title>` 寫法

| 項目 | 說明 |
|---|---|
| 使用情境 | 在 HTML 文件中設定頁面標題。 |
| 使用前提 | 放在 `<head>` 中。 |
| 來源 | 4. 語法與基本用法 |

```html
<title>頁面標題</title>
```

### 注意事項

- `<title>` 是雙標籤元素，需要開始標籤與結束標籤。
- 內容通常應是純文字，不放其他 HTML 標籤。

### 6.2 基本 HTML 文件標題

| 項目 | 說明 |
|---|---|
| 使用情境 | 建立一般 HTML 文件時設定分頁標題。 |
| 使用前提 | `<title>` 放在 `<head>` 中，頁面可見主標題放在 `<body>`。 |
| 來源 | 3.2 `<title>` 放在 `<head>` 裡、4. 語法與基本用法 |

```html
<!doctype html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>我的第一個 HTML 網頁</title>
</head>
<body>
  <h1>我的第一個 HTML 網頁</h1>
</body>
</html>
```

### 注意事項

- `<title>` 設定文件標題。
- `<h1>` 設定頁面可見主標題。

### 6.3 文章頁標題

| 項目 | 說明 |
|---|---|
| 使用情境 | 單篇教學文章、部落格文章或文件頁。 |
| 使用前提 | 標題先呈現文章主題，再補上網站名稱。 |
| 來源 | 5. 範例說明 |

```html
<title>HTML title 標籤的作用與寫法｜小林前端筆記</title>
```

### 注意事項

- 文章頁標題應能直接反映文章主題。
- 不建議所有文章都使用相同標題。

### 6.4 動態更新文件標題

| 項目 | 說明 |
|---|---|
| 使用情境 | 後台系統、前端框架或單頁應用程式依目前頁面狀態更新分頁標題。 |
| 使用前提 | 透過 JavaScript 操作目前文件標題。 |
| 來源 | 6. 實務應用場景、11. 延伸補充 |

```js
document.title = "會員資料 - 管理後台";
```

### 注意事項

- 單頁應用程式切換畫面時，通常需要同步更新 `document.title`。
- 本章只示範基本賦值，不展開框架路由實作。

## 8. 相似概念對照表

### 8.1 `<title>` vs `<h1>`

| 比較項目 | `<title>` | `<h1>` |
|---|---|---|
| 定義 | 整份 HTML 文件的標題。 | 頁面可見內容中的最高層級標題。 |
| 放置位置 | `<head>` 中。 | `<body>` 中。 |
| 是否通常顯示在頁面內容 | 不會作為頁面主內容顯示。 | 會顯示在頁面內容中。 |
| 常見用途 | 瀏覽器分頁、書籤預設名稱、搜尋結果參考。 | 告訴讀者頁面主內容主題。 |
| 常見寫法 | `<title>HTML title 標籤教學</title>` | `<h1>HTML title 標籤教學</h1>` |
| 常見錯誤 | 放到 `<body>`，或以為它會成為頁面主標題。 | 用它取代文件標題。 |

### 選用建議

- 需要設定文件標題時使用 `<title>`。
- 需要呈現頁面主內容標題時使用 `<h1>`。
- 兩者內容可以相近，但用途不同，應同時正確配置。

### 8.2 首頁標題 vs 文章頁標題

| 比較項目 | 首頁標題 | 文章頁標題 |
|---|---|---|
| 標題重點 | 網站名稱與網站定位。 | 文章主題與網站名稱。 |
| 使用時機 | 網站首頁。 | 單篇教學文章、部落格文章或文件頁。 |
| 常見寫法 | `<title>小林前端筆記｜HTML、CSS 與 JavaScript 學習整理</title>` | `<title>HTML title 標籤的作用與寫法｜小林前端筆記</title>` |
| 注意事項 | 不只寫「首頁」。 | 不應讓所有文章使用同一個標題。 |

### 選用建議

- 首頁標題優先表達網站整體定位。
- 文章頁標題優先表達單篇內容主題，再補上網站名稱。

### 8.3 `<title>` vs SEO

| 比較項目 | `<title>` | SEO |
|---|---|---|
| 定義 | HTML 文件標題元素。 | 搜尋引擎最佳化。 |
| 作用範圍 | 幫助描述單一頁面的主題。 | 包含內容品質、語意結構、效能、連結與整體網站品質等因素。 |
| 常見關係 | 搜尋引擎常參考 `<title>` 理解頁面主題。 | 不只依賴 `<title>`。 |
| 注意事項 | 標題應自然可讀，不應堆疊關鍵字。 | notes 未展開完整 SEO 策略。 |

### 選用建議

- 把 `<title>` 視為 SEO 的重要基礎訊號之一。
- 不要把 `<title>` 當成唯一的 SEO 手段。
