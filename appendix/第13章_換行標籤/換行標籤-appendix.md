## 9. 常用片段

### 9.1 文字中強制換行

| 項目 | 說明 |
|---|---|
| 使用情境 | 同一段文字需要在指定位置換行。 |
| 使用前提 | 換行本身是內容的一部分。 |
| 來源 | `4. 語法與基本用法`、`5. 範例說明` |

```html
第一行文字<br>
第二行文字
```

### 注意事項

- `<br>` 只插入換行，不建立新段落。
- 若是不同段落，應改用 `<p>`。

### 9.2 地址換行

| 項目 | 說明 |
|---|---|
| 使用情境 | 地址每一行需要分開顯示。 |
| 使用前提 | 每一行仍屬於同一組地址內容。 |
| 來源 | `5. 範例說明`、`6. 實務應用場景` |

```html
<p>
  台北市中正區<br>
  重慶南路一段 122 號<br>
  3 樓
</p>
```

### 注意事項

- 此片段只示範換行，不示範地址語意的進階標記。

## 10. 錯誤速查表

|---|---|---|---|---|

## 11. 相似概念對照表

### 11.1 `<br>` vs `<p>`

| 比較項目 | `<br>` | `<p>` |
|---|---|---|
| 定義 | 在目前文字流中插入換行。 | 表示一個段落。 |
| 使用時機 | 同一段內容需要保留固定換行。 | 不同段落或完整文字區塊。 |
| 常見寫法 | `第一行<br>第二行` | `<p>這是一段文字。</p>` |
| 注意事項 | 不會自動建立段落間距語意。 | 段落間距可再用 CSS 調整。 |
| 常見錯誤 | 用多個 `<br>` 假裝段落。 | 為了控制字距或換行而濫用段落。 |

### 選用建議

- 換行屬於同一段內容的一部分時，使用 `<br>`。
- 內容本身是不同段落時，使用 `<p>`。

### 11.2 `<br>` vs CSS 間距

| 比較項目 | `<br>` | CSS 間距 |
|---|---|---|
| 定義 | HTML 內容中的換行元素。 | 視覺呈現與版面間距控制。 |
| 使用時機 | 內容需要在指定位置換行。 | 元素之間需要距離或排版效果。 |
| 常見寫法 | `第一行<br>第二行` | `margin`、`padding` |
| 注意事項 | 不適合拿來控制整體版面。 | 不改變內容語意。 |
| 常見錯誤 | 用 `<br><br>` 製造空白。 | 原筆記未列出 CSS 錯誤情境。 |

### 選用建議

- 需求是「內容本身需要換行」時，使用 `<br>`。
- 需求是「畫面看起來要有距離或排列效果」時，使用 CSS。
