# a href target 速查

> 查表用。概念與用法說明見 notes/210-超鏈接標籤/01-a-標籤與超連結基礎.md。

## 基本結構

| 部分 | 寫法 | 用途 |
| --- | --- | --- |
| 標籤 | `<a>...</a>` | 建立可導航的超連結範圍 |
| 必要屬性 | `href="目標地址"` | 指定使用者點擊後要前往的位置 |
| 連結文字 | `<a href="...">文字內容</a>` | 使用者在畫面上看到並點擊的內容 |
| 結束標籤 | `</a>` | 結束這個連結範圍 |

## href 目標類型

| 目標類型 | href 寫法範例 | 用途 |
| --- | --- | --- |
| 外部網站 | `https://www.baidu.com/` | 前往目前網站以外的網址 |
| 站內頁面 | `about.html` | 前往同一個網站內的其他頁面 |
| 相對路徑資源 | `./assets/files/example.html` | 從目前 HTML 檔案位置出發尋找站內資源 |
| 頁面位置 | `#section-2` | 跳到頁面中的指定位置 |
| 特殊連結 | `mailto:`、`tel:` | 開啟郵件或電話等特殊入口 |

## target 值

| target 值 | 開啟方式 | 使用情境 | 備註 |
| --- | --- | --- | --- |
| `_self` | 在目前頁籤開啟 | 站內導覽、一般頁面切換 | 預設值 |
| `_blank` | 在新頁籤開啟 | 外部網站、文件預覽、希望保留原頁面的情境 | 連到外部或不完全可信任頁面時，常搭配 `rel="noopener"` |

## 內外部連結判斷

| 類型 | 判斷方式 | 寫法提示 |
| --- | --- | --- |
| 外部連結 | 目標在目前網站以外 | 通常寫完整網址 |
| 內部連結 | 目標在同一網站內 | 通常可使用相對路徑 |
| 文件或資源連結 | 目標是 HTML、圖片、PDF 等檔案 | 寫法仍是 `<a href="...">` |

## 連結與按鈕分工

| 使用目的 | 建議元素 | 理由 |
| --- | --- | --- |
| 前往另一個網站 | `<a>` | 目的是導航 |
| 前往站內頁面 | `<a>` | 目的是導航 |
| 讓圖片或卡片前往詳情頁 | `<a>` | 可包住非互動內容形成連結入口 |
| 送出、取消、儲存 | `<button>` | 目的是操作，不是前往某個位置 |
| 打開視窗或切換狀態 | `<button>` | 目的是觸發行為，不是導航 |

## 常見錯誤與修正

| 錯誤寫法 | 問題 | 建議修正 |
| --- | --- | --- |
| `<a>前往首頁</a>` | 沒有 `href`，沒有可導航目標 | `<a href="index.html">前往首頁</a>` |
| `<a href="javascript:saveData()">儲存</a>` | 把操作行為偽裝成連結 | `<button type="button">儲存</button>` |
| 無條件使用 `target="_blank"` | 不一定符合使用者情境 | 只有需要新頁籤時才使用 |
| `<a>` 裡再放 `<a>` | 連結巢狀造成語意混亂 | 拆成獨立連結 |
