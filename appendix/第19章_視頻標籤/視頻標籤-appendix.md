## 9. 常用片段

### 9.1 基本影片播放器

| 項目 | 說明 |
|---|---|
| 使用情境 | 在頁面中嵌入單一影片檔案，並讓使用者操作播放。 |
| 使用前提 | 影片路徑正確，且不需要提供多種格式。 |
| 來源 | §3、§5.1 |

```html
<video src="./assets/files/movie.mp4" controls></video>
```

### 注意事項

- `src` 指向影片檔案。
- `controls` 顯示瀏覽器內建控制列。
- 沒有自訂播放器時，不建議省略 `controls`。

### 9.2 帶封面圖的影片

| 項目 | 說明 |
|---|---|
| 使用情境 | 希望影片播放前顯示明確預覽畫面。 |
| 使用前提 | 已準備可用的封面圖片。 |
| 來源 | §4.3、§7 |

```html
<video src="./assets/files/movie.mp4" poster="./assets/images/movie-cover.jpg" controls></video>
```

### 注意事項

- `poster` 指向影片播放前顯示的圖片。
- 封面圖路徑錯誤時，仍可能出現空白或不理想畫面。

### 9.3 預載影片中繼資訊

| 項目 | 說明 |
|---|---|
| 使用情境 | 希望瀏覽器先取得影片長度、尺寸等中繼資訊。 |
| 使用前提 | 只需要預載 metadata，不一定要提前下載整段影片。 |
| 來源 | §4.4 |

```html
<video src="./assets/files/movie.mp4" controls preload="metadata"></video>
```

### 注意事項

- `preload` 是瀏覽器載入提示，不是強制命令。
- 實際載入行為可能受瀏覽器策略影響。

### 9.4 多格式影片來源

| 項目 | 說明 |
|---|---|
| 使用情境 | 需要提供多個候選格式，讓瀏覽器選擇可播放版本。 |
| 使用前提 | 已準備多個影片格式檔案。 |
| 來源 | §4.6、§5.2 |

```html
<video controls width="640" height="360">
  <source src="./assets/files/movie.mp4" type="video/mp4">
  <source src="./assets/files/movie.webm" type="video/webm">
  您的瀏覽器不支援 video 標籤。
</video>
```

### 注意事項

- `<source>` 放在 `<video>` 內。
- 瀏覽器會從候選來源中選擇支援格式。
- 備用文字用於不支援 `<video>` 的環境。

### 9.5 靜音自動循環影片

| 項目 | 說明 |
|---|---|
| 使用情境 | 首頁背景影片或展示區塊需要嘗試自動播放。 |
| 使用前提 | 影片可靜音播放，且自動播放失敗不會破壞主要內容。 |
| 來源 | §4.5、§5.3 |

```html
<video src="./assets/files/banner.mp4" autoplay muted loop playsinline></video>
```

### 注意事項

- `muted` 可降低自動播放被阻擋的機率。
- 即使加上 `muted`，仍不能保證自動播放一定成功。
- 背景影片不應承載唯一重要資訊。

### 9.6 避免依賴有聲音的自動播放

| 項目 | 說明 |
|---|---|
| 使用情境 | 需要穩定提供影片播放入口。 |
| 使用前提 | 影片不應強制在頁面載入時播放。 |
| 來源 | §5、§7 |

```html
<video src="./assets/files/intro.mp4" controls></video>
```

### 注意事項

- 原筆記指出有聲音的自動播放可能被瀏覽器阻擋。
- 讓使用者主動播放通常更穩定。

## 10. 錯誤速查表

| 錯誤現象 | 可能原因 | 修正方向 | 對應知識點 | 來源 |
|---|---|---|---|---|
| 影片無法播放。 | `src` 路徑寫錯，瀏覽器找不到檔案。 | 確認 `src` 路徑與影片檔案位置一致。 | `src`、檔案路徑 | §3、§7、§8 |
| 使用者看不到播放控制。 | 忘記加上 `controls`，也沒有自訂播放器。 | 在 `<video>` 上加入 `controls`。 | `controls`、播放控制列 | §3、§4.2、§7 |
| 自動播放沒有成功。 | 瀏覽器可能阻擋有聲音的自動播放。 | 讓使用者主動播放，或搭配 `muted` 設計靜音自動播放。 | `autoplay`、`muted`、使用者互動 | §4.5、§5、§7 |
| 某些瀏覽器或環境無法播放影片。 | 只提供單一格式，該環境可能不支援該格式或編碼。 | 使用多個 `<source>` 提供替代格式。 | `<source>`、影片格式 | §4.6、§5.2、§7 |
| 影片載入前畫面空白或不理想。 | 沒有設定 `poster`。 | 使用 `poster` 提供清楚封面圖。 | `poster`、封面圖 | §4.3、§7 |
| 影片畫面被拉伸或壓縮。 | `width` 與 `height` 比例不一致。 | 使用符合影片比例的尺寸，或交給 CSS 維持比例。 | `width`、`height`、尺寸比例 | §4.1、§7 |
| 背景影片無法停止或干擾內容。 | 使用者缺少控制權，且重要內容只放在背景影片。 | 提供控制方式，或避免把重要內容只放在背景影片。 | `controls`、`loop`、可訪問性 | §6、§7 |
| 瀏覽器無法正確判斷影片格式。 | MIME type 可能填寫錯誤。 | 依檔案格式填寫正確的 `type`。 | `type`、MIME type | §4.6、§7、§11.1 |
| 重要資訊只存在影片中。 | 無法播放、無法聽音訊或使用輔助工具的使用者會失去資訊。 | 提供字幕、逐字稿、摘要或替代說明。 | 可訪問性、字幕、文字替代 | §6、§7、§8、§9 |
| 把 `autoplay` 當成唯一播放入口。 | 誤以為自動播放一定成功。 | 保留可操作的播放介面，避免只依賴自動播放。 | `autoplay`、`controls` | §5、§7 |

## 11. 相似概念對照表

### 11.1 `<video src>` vs `<video>` 內的 `<source>`

| 比較項目 | `<video src>` | `<video>` 內的 `<source>` |
|---|---|---|
| 定義 | 在 `<video>` 上直接指定單一影片檔案。 | 在 `<video>` 內提供一個或多個候選影片來源。 |
| 使用時機 | 只有一個影片來源時。 | 需要提供多種格式提升播放成功機會時。 |
| 常見寫法 | `<video src="./assets/files/movie.mp4" controls></video>` | `<source src="./assets/files/movie.mp4" type="video/mp4">` |
| 注意事項 | 簡單直接，但缺少替代格式。 | 建議填寫 `type`，讓瀏覽器判斷格式。 |
| 常見錯誤 | 路徑寫錯導致無法播放。 | 只寫 `<source>` 但沒有放在 `<video>` 中。 |

### 選用建議

- 只需要單一影片檔時，可直接使用 `<video src>`。
- 需要兼顧不同格式時，使用 `<video>` 加多個 `<source>`。

### 11.2 `autoplay` vs 使用者主動播放

| 比較項目 | `autoplay` | 使用者主動播放 |
|---|---|---|
| 定義 | 希望頁面載入後自動播放影片。 | 使用者透過控制列或按鈕啟動播放。 |
| 使用時機 | 背景影片或展示影片可能使用。 | 一般內容頁或需要穩定播放入口時。 |
| 常見寫法 | `<video src="./assets/files/movie.mp4" autoplay muted loop></video>` | `<video src="./assets/files/movie.mp4" controls></video>` |
| 注意事項 | 可能被瀏覽器阻擋；通常搭配 `muted`。 | 較符合使用者控制權。 |
| 常見錯誤 | 假設自動播放一定成功。 | 忘記提供明確播放入口。 |

### 選用建議

- 不要把 `autoplay` 當成唯一播放方式。
- 優先提供 `controls`，讓使用者主動播放。
- 若需要背景影片自動播放，應搭配 `muted`，並確保失敗時不影響主要內容。

### 11.3 `<video>` vs `<audio>`

| 比較項目 | `<video>` | `<audio>` |
|---|---|---|
| 定義 | 嵌入同時包含影像與聲音的內容。 | 嵌入音訊內容。 |
| 使用時機 | 教學影片、產品展示、示範短片、背景影片。 | 背景音、課程錄音、播客、音效。 |
| 常見寫法 | `<video src="./assets/files/movie.mp4" controls></video>` | 原筆記僅作相似概念比較，未提供 `<audio>` 最小範例。 |
| 注意事項 | 常需要額外考慮尺寸、封面圖、字幕、畫面比例與行動裝置播放方式。 | 只處理聲音。 |
| 常見錯誤 | 忽略字幕、逐字稿、封面圖或尺寸比例。 | 原筆記未展開 `<audio>` 常見錯誤。 |

### 選用建議

- 需要影像與聲音時，使用 `<video>`。
- 只需要聲音內容時，使用 `<audio>`。

### 11.4 `controls` vs 自訂播放器

| 比較項目 | `controls` | 自訂播放器 |
|---|---|---|
| 定義 | 使用瀏覽器內建播放控制列。 | 用 JavaScript 自行控制播放介面。 |
| 使用時機 | 初學範例或不需要特殊介面時。 | 需要自訂播放、暫停、進度、音量或全螢幕互動時。 |
| 常見寫法 | `<video src="./assets/files/movie.mp4" controls></video>` | 原筆記未提供自訂播放器程式碼。 |
| 注意事項 | 介面外觀由瀏覽器決定。 | 原筆記未完整教相關 API，不應在本附錄擴寫。 |
| 常見錯誤 | 忘記加 `controls` 導致使用者無法操作。 | 沒有完整處理播放控制與可訪問性。 |

### 選用建議

- 沒有自訂播放器時，使用 `controls`。
- 若要自訂播放器，應回到 JavaScript 與 DOM API 主題另行學習。

### 11.5 `poster` vs 影片第一幀

| 比較項目 | `poster` | 影片第一幀 |
|---|---|---|
| 定義 | 明確指定影片播放前顯示的圖片。 | 未設定 `poster` 時，瀏覽器可能顯示的影片畫面。 |
| 使用時機 | 需要穩定、清楚的影片預覽畫面時。 | 原筆記只提到未設定時可能由瀏覽器呈現。 |
| 常見寫法 | `poster="./assets/images/movie-cover.jpg"` | 不設定 `poster`。 |
| 注意事項 | 需確保圖片路徑正確。 | 可能出現空白或不理想畫面。 |
| 常見錯誤 | 封面圖路徑錯誤。 | 誤以為瀏覽器一定會顯示理想畫面。 |

### 選用建議

- 需要固定預覽畫面時，使用 `poster`。
- 不應依賴瀏覽器自動選出理想的影片第一幀。
