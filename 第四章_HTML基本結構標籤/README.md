## 易混淆主題

- `lang` vs `charset`：前者是語言標示，後者是字符編碼。
- `DOCTYPE` vs 一般標籤：前者是文檔類型聲明，不是普通 HTML 元素。
- 註解 vs 頁面內容：前者給開發者看，後者才會實際顯示在頁面上。
