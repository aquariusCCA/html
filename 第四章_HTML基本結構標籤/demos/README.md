# 第四章｜HTML 基本結構標籤示例

這個資料夾收錄第四章筆記對應的互動示例頁。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第四章筆記。
- 想把文件開頭那幾行和它們的作用對起來時：再打開這裡的示例頁。
- 想複習屬性寫法、布林屬性與註解語法時：直接切換示例區即可。
