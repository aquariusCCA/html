# button 按鈕標籤

> 所屬章節：第二十四章｜表單標籤  
> 關鍵字：button、`button`、submit、reset、button type、提交按鈕、重置按鈕、普通按鈕、預設型別  
> 建議回查情境：想知道 `<button>` 可以有哪些類型、想分清 `submit`、`reset`、`button` 的差別、想確認為什麼不寫 `type` 可能會意外送出表單

## 本節導讀

這篇整理 `<button>` 按鈕標籤。  
它看起來只是一般按鈕，但在表單裡其實會因為 `type` 不同而有完全不同的行為。

這個主題最容易出錯的地方，是把「普通按鈕」和「預設型別」混在一起。  
要先記住：`button` 是一種型別名稱，但 `<button>` 標籤如果沒有寫 `type`，在表單中預設通常會是 `submit`，不是 `button`。

## 你會在這篇學到什麼

- `<button>` 常見的三種 `type`
- `submit`、`reset`、`button` 各自在做什麼
- 為什麼不寫 `type` 容易造成意外提交
- 什麼時候應該明確寫出 `type="button"`

## 30 秒複習入口

- `<button>` 常見有 `submit`、`reset`、`button` 三種型別。
- `submit` 用來提交表單。
- `reset` 用來把表單恢復成初始狀態。
- `button` 是普通按鈕，本身不會自動提交表單。
- 在表單裡若省略 `type`，`<button>` 預設通常是 `submit`。

## 速查區

### 核心概念

- `<button>` 是可點擊的按鈕元素。
- 它的行為主要由 `type` 決定。
- 同樣都是按鈕，提交、重置、一般互動按鈕的用途並不一樣。

### 關鍵規則 / 判準

- `type="submit"`：提交表單。
- `type="reset"`：把表單恢復到初始狀態。
- `type="button"`：一般按鈕，不自動提交表單。
- 在表單中若不寫 `type`，`<button>` 預設通常會被當成 `submit`。

### 常見使用場景

- 表單送出
- 表單重填
- 彈出視窗
- 執行 JavaScript 互動
- 驗證碼、展開收合、切換內容

### 常見混淆點

- `button` 是一種型別名稱，不是 `<button>` 的預設安全行為。
- `reset` 不是單純把欄位清空，而是恢復到初始值。
- 在表單裡省略 `type`，常會讓本來只是想做互動的按鈕誤送出表單。

### 一句話抓核心

- `<button>` 的重點不是有沒有按鈕，而是 `type` 決定它到底是提交、重置，還是單純互動按鈕。

## 正文筆記

`<button>` 常見有三種型別：

- `submit`：提交按鈕。
- `reset`：重置按鈕。
- `button`：普通按鈕。

```html
<button type="submit">提交按鈕</button>
<button type="reset">重置按鈕</button>
<button type="button">普通按鈕</button>
<button>未指定 type 的按鈕</button>
```

### 1. `submit` 提交按鈕

- 用來提交表單資料。
- 當按鈕放在表單裡時，點擊後通常會觸發表單送出。

### 2. `reset` 重置按鈕

- 用來把表單恢復成初始狀態。
- 重點不是「全部清空」，而是回到頁面一開始載入時的值。

### 3. `button` 普通按鈕

- 一般互動按鈕。
- 它本身不會自動提交表單，常用來搭配 JavaScript 做其他互動行為。

### 4. 為什麼要特別注意預設型別？

> 若不寫 `type`，在表單中的 `<button>` 預設通常會是 `submit`，可能引起意外提交。

- 這是初學者很容易踩到的點。
- 有些按鈕本來只是想做切換、展開、跳提示或執行腳本，但若忘了寫 `type="button"`，在表單中就可能直接送出資料。
- 因此只要不是要提交表單，通常都建議把 `type` 寫清楚。

## 常見問法

### `<button>` 不寫 `type` 會怎樣？

- 在表單裡，預設通常會被當成 `submit`。
- 所以它可能會直接觸發表單提交。

### `button` 和 `submit` 差在哪？

- `submit` 會送出表單。
- `button` 只是一般按鈕，本身不會自動送出表單。

### `reset` 是不是把所有欄位清空？

- 不一定。
- 更精確的說法是：它會把表單恢復成初始狀態。

### 什麼時候要明確寫 `type="button"`？

- 只要這顆按鈕不是拿來送出表單，就通常建議明確寫上。
- 這樣可以避免因為預設行為造成誤提交。
