# 第二十四章｜表單標籤示例

這個資料夾收錄第二十四章筆記對應的可執行示例頁。  
這章的重點是理解表單如何收集使用者輸入，並分清表單域、表單控件、提示信息、分組、禁用與常見互動屬性的角色。

## 這個資料夾怎麼用

- 想先理解概念時：先讀第二十四章 Markdown 筆記。
- 想直接操作完整表單時：打開這裡的示例頁。
- 想新增更多案例時：沿用 `主題名-demo.html` 的命名規則。

## 目前示例

- [form-tags-demo.html](./form-tags-demo.html)：整合展示 `form`、`input type`、`label`、`textarea`、`select`、`button type`、`fieldset` / `legend`、`disabled`、`required`、`placeholder`、`autofocus`、`autocomplete`，並用 `FormData` 預覽提交資料。

## 命名規則

- 單一主題單一示例：`主題名-demo.html`
- 同一主題多個示例：`主題名-demo-01.html`、`主題名-demo-02.html`
- 若示例需要額外資產，可在 `demos/` 下再建立同名子資料夾集中管理
